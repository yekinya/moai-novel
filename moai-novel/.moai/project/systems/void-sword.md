# 공허검 (空虛劍) — 원류·정통·습득 정통

> 본 문서는 *카엘이 Chapter 2 후반 ~ Chapter 3 초반에 본격 습득하는 검술 정통 공허검 (空虛劍)* 의 사양서.
>
> 본 정통은 *마법 시전이 봉인된 카엘 (두 팔 상실 + 마법 봉인)* 이 *마력→내공 운용 (앤트로퍼 특전) 을 의수 검술에 활용* 하면서도 *검기를 제대로 두르지 못하는 한계* 를 *우회* 하기 위한 *유일한 검술 정통*.

---

## 0. 한 줄 정의

> *검기 (劍氣) 없이 검사를 제압하는 검술* — *상대의 검기·마법을 *받지 않고 흘리거나 비키게 함 + 호흡·중심·간격의 *완벽한 운용* 으로 *상대의 의지가 꺾이도록 함*.

---

## 1. 원류 — 동부마녀연맹

### 지리적 출처
- **공허검 원류 = 동부마녀연맹 지역** — 마녀령 동부국 (모르간 가문 흑혈·백혈 분파)
- 전승 시기 = *대현자 시대 후기 — 칠흑성 전투 직전 (CC 870~890 추정)*. 약 130년 역사
- *마녀령의 *배타적 성격* 으로 *외부 사피엔에게 거의 전수되지 않음*

### 창시자 — *9혈 외부 사부 1인* (정확한 정체 보류)
- 창시자 = *9혈 원탁 *외부 손님 정통의 *1대 시조* 격*. 마녀령 출신 아니나 *9혈 원탁이 *비공식 사사 허용한 첫 외부인*
- *마녀령 검술 정통 (흑혈·백혈) 의 *호흡·간격* 원리를 *마법사 검사의 *검기 한계 극복용* 으로 *재구성한 창시자*
- *익명·잠적* — 정통 후예에게도 *이름은 전승되지 않음*. *공허(空虛) 의 첫 시전자* 라는 호칭만 남음

### 후계자 5인 (현존 *공허검 마스터 5인*)
- **세이른 스틸만** (강철제국 본가 방계, 60대) — 본작 정통의 *현존 5인 중 가장 젊음*. Ch1 시드의 인물 ([`../novel/characters.md`](../novel/characters.md) 참조)
- 동부국 *흑혈 외부 노 검사 1인* (정통 의 정점 유지)
- 마녀령 *서부 잠적 분파 1인* (대외 노출 거의 없음)
- 강철제국 *남부 변경 도장 1인* (세이른의 옛 동문)
- 헤븐즈 *국립상아탑 비공개 검술 사부 1인* (라카르 9세의 *비공식 호위 검사*)

### 정통의 본질 — *마법사·검사·마검사 모두를 위협하는 정통*
- 마법사 = *검기 없이 *마나 흐름을 비키게 함* → 마법사의 *마법 시전이 제압*
- 검사 = *검기 없이 *상대의 검 흐름을 받지 않음* → 검사의 *검술이 무력화*
- 마검사 = *둘 다 동시 무력화* → *공허검 정통의 최강 적용 대상*

## 2. 검술 원리

### 3대 원리

1. **불교(不交) — 받지 않음**
   - *상대의 검기·마법을 *직접 받아치지 않음*. *검을 *비키게 하고·흘리고·간격으로 피함*
   - 본작 정통 *마법사 검사들의 *검기 정면 충돌* 정통과 *근본적으로 다름*
   - *완벽한 호흡·중심·간격 운용* 으로 *상대의 모든 공격을 *받지 않은 채 흘려보냄*

2. **공출(空出) — 비움에서 출수**
   - *자기 안의 *살의·욕망·승부심을 *완전히 비운 상태* 에서 *출수*
   - *상대가 *공격의 의도를 *전혀 예측 불가*. *비어 있는 자의 검은 *어디서 나올지 모름*
   - 이 원리는 *카엘의 *비뚤린 시야와 *상극* — *카엘이 *공허검을 *온전히 익히기 위해서는 *비뚤린 시야가 *최소 50% 회복되어야 함*. Ch2 P1 의 *재정체화 후 50% 회복 정착* 이 *공허검 습득의 정통 진입 조건*

3. **허(虛) — 빈 곳을 침**
   - *상대의 *빈 곳 (틈·약점·심리적 허점) 을 *짧고 정밀한 검 한 점으로 침*
   - *검기를 두르지 않은 검* 이 *상대의 갑옷·결계·신성마법 보호를 *물리적 충격만으로 *뚫음*. 이는 *상대의 *심리적 허점을 매개로* 만 가능 — *공허검의 *심리 매개 정통*

### 호흡 정통 — *3박자 호흡* (마녀령 검술 정통 원형 재구성)
- *들숨 1박* — 상대 동선 관찰
- *멈춤 1박* — 상대 의도 *비움에서 직관*
- *날숨 1박* — 출수

3박자가 *완벽히 일치* 하면 *상대의 *모든 검기·마법이 비킴*. 일반 검사·마법사·마검사는 *이 호흡을 따라가지 못함*.

## 3. 카엘의 공허검 습득 (Ch2 P4~P5 정통)

### 진입 조건 (Ch2 P4 시점)
- 카엘 = *두 팔 상실 + 마법 시전 봉인 + 강화강철 의수 단계*. *검기 운용이 일반의 70% 수준에 머무름*
- *세이른 스틸만의 옛 인연* (Ch1 P3 검술 사사 알린 마스반 경의 후계 라인) 으로 *공허검 정통 존재 인지*
- *동부마녀연맹의 *배타적 성격* 으로 *직접 사사 거의 불가능*

### 사사 과정
- 카엘이 *동부마녀연맹 진입 — 모르간 가문 외곽 분파 1인의 *비공식 안내*
- *9혈 원탁 외부 손님 정통* 으로 *비공식 사사 허락 받음* — *모르간 솔테 (흑혈 종주, 모르간 린이엔의 조모) 의 *비공식 결단*
- 사사 기간 = 약 *3~6개월*. *기본 호흡 + 1·2 원리 (불교·공출) 부분 습득*. *3원리 허(虛) 는 *완전 습득 불가*
- *9혈 원탁 외부 손님 *8번째 등록*** 시드 (Chapter 3 본격 회수)

### 카엘의 공허검 운용 — *불완전 정통*
- *공허검의 *완벽한 정통은 50% 만 습득*. *나머지 50% 는 *자기 응용*
- *자기 응용 = *마검사 잡학 응용 + 공허검 원리 = 모자이크 정통*. *세이른 스틸만의 비판 — *너의 공허는 *온전하지 않다. 그러나 *너에게는 그게 더 맞을지도*
- *팔 전체에 검기를 두르는 방식 (Ch2 P6 카엘 개발) 의 *내적 동력원 = 공허검 호흡 + 마력→내공 + 의수 마나 감응의 *3중 융합*

### 룬·아티팩트 지식 부수 획득
- 공허검 사사 중 *마녀령 룬 마법사의 *비공식 룬 결각 + 아티팩트화 지식 부분 양도* — *Chapter 2 P6 강철제국 복귀 후 의수 룬 결각 작업의 핵심 매개*
- 정확한 룬 결각 기술은 *앨더스톰 (Ch2 P7) + 드워프 지하세계 (Ch2 P8) 에서 본격 보강*

## 4. 공허검 vs 다른 검술 정통

| 정통 | 핵심 | 카엘 시각 |
|------|------|---------|
| **마검사 (Ch1 정통)** | 마법 + 검술 모자이크 | *두 팔 상실 + 마법 봉인으로 *전면 봉인*. 의수에서 일부 재현 시도* |
| **정통 기사 검술** | 명예의 5합 + 검기 정면 충돌 | *공허검과 *대척점*. 카엘이 *떠돌이 시기부터 못마땅* |
| **공허검 (Ch2 정통)** | 받지 않고 흘림 + 비움에서 출수 + 빈 곳 침 | *카엘의 *마법 봉인 + 의수 한계의 *유일한 우회 정통* |
| **소드 엠퍼러 정통** (Chapter 3 후속) | 검기 절대 농도 + 광역 위협 | *카엘의 *공허검 + 의수 4단계 결합 후 *호각 도전 가능* |
| **흑혈·백혈 마검술** (모르간 가문) | 마녀령 정통 검술 + 검기 | *카엘과 *별개 라인. Chapter 3+ 카이엔(?) 와의 결투에서 *충돌* |
