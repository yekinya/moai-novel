## 1. 기본 정의

마법은 *마나 응용 기술* 의 총칭. 신비한 힘처럼 보이나 실제로는 *세계의 최소 구성 원리(세프)와 그 배열·진동·결합* 을 다루는 기술이다.

### 핵심 용어

- **세프 (Sef)**: 마나의 최소 단위. 학술 표기. 사피엔은 흔히 *마나* 라 부른다
- **셉터 (Septer)**: 세프의 근원을 탐구하는 마법사
- **세프렌 (Sefren)**: 마나 *감응성·응축성* 을 높이는 단계 (선천 의존)
- **셉터렌 (Septeren)**: 마나 *응용성* 을 높이는 단계 (후천 수련)
- **스펠 (Spell)**: 룬·시동어·양피지·스크롤 등 마법 매개체. 데미안 스펠른 발명
- **써클 (Circle)**: 마법 숙련도 1~9등급 (편의상 분류 — §6 참조)

> *"마법은 사실 거대한 세프 응축체인 드래곤이 한순간에 뿜어내는 브레스를 보고 지적생명체들이 흉내내며 정립한 것이라는 것이 정설이다."*
> — 라인하르츠 행커만 (변신 드래곤 본인)

## 2. 마법의 공통 3요소

모든 마법 시전은 다음 3단계로 분해된다.

1. **감응 (Sensing)** — 마나를 *느끼고 읽는* 능력
2. **배열 (Arrangement)** — 마나에 *형태와 목적* 을 부여하는 기술
3. **고정 (Fixation)** — 배열을 현실에 *유지하거나 몸·도구·공간에 묶는* 기술

3요소 중 어느 한 단계라도 실패하면 마법은 *형성되지 않거나 즉시 흩어진다*.

## 3. 5원소의 발현 원리 — 모든 마법은 세프 운영의 변형

### 핵심 명제

**원소는 별개의 힘이 아니다**. 모두 *세프(마나)의 운영 원리* 의 다른 표현이다. 화염·물·바람은 *결과적 현상* 일 뿐, 진짜는 *세프를 어떻게 움직였는가* 다.

### 5원소별 발현 메커니즘

| 원소 | 발현 원리 | 학습 난이도 |
|------|----------|------------|
| **불 (火)** | 세프의 *가열 + 마찰 + 응축 + 폭발*. 작은 세프 알갱이 두 개를 강하게 충돌시키면 열이 생긴다. 그 열을 연쇄로 만들면 화염이 된다 | 가장 쉬움 |
| **물 (水)** | 세프의 *팽창 + 냉각*. 가열의 *역방향* 운영. 공기 중 미세 수분 + 세프 응축으로 물 결정화 | 쉬움 |
| **바람 (風)** | 세프의 *이동 + 대류 + 수축·팽창*. 공기 자체가 아니라 *세프의 흐름* 을 만들어 공기를 끌고 다님 | 중간 |
| **바위 (土)** | 불 + 압력 응용. 흙·먼지 입자를 가열·압축해 결정화 | 중간 |
| **얼음** | 물 + 극냉각. 일레인 맥필렌의 *얼음창 중력 응축* 정수 | 중간 |
| **번개** | 세프의 *극도 가속 + 정전 응축*. 짧은 거리만 가능 | 어려움 |
| **비행** | 대류 + 가열 응용. 자기 신체 아래에 *상승 기류* 를 만든다. 장기 비행 = 마나 고갈 | 어려움 |
| **광명** | 세프 알갱이 *2개의 강한 충돌* — 가장 기초적이면서 *모든 원소의 원형* | 가장 쉬우나 응용은 가장 깊음 |
| **암흑 / 그림자** | 세프의 *분산 + 흡수*. 빛의 반대가 아닌 *광명의 역방향 운영* | 어려움 |
| **중력** | **가장 다루기 힘든 힘**. 직접 발현 거의 불가능. *질량 압축 + 바람의 회전 응용* 으로만 *간접* 실현. 일레인 맥필렌이 거의 유일한 9써클 중력 응용 마스터 | 극악 |

### 시전 메커니즘 상세 — *화염구 5단계 구현 원리* (정통)

본작 마법 묘사의 *표준 정통성*. 화염구 1개 시전 시 *반드시 5단계* 의식적 진행.

#### 5단계 구현

1. **마나 회전 + 압축** — 손 안의 마나를 *회전 또는 압축*. 공기가 함께 회전 시작
2. **마찰 + 흡입** — 마찰로 *뜨거워지는 순간* 주변 마나를 *중앙으로 흡입*
3. **연소 + 유지** — 공기 순환으로 *어느 순간 열기 생성*. 산소 연소로 *엄청난 속도로 주변 공기 순환* — 공기를 빨아들이며 *열구 유지*
4. **앤트로픽 억제** — *주변 앤트로픽이 빠르게 흩어지므로* 일반 구현은 매우 어려움. *자기 마나가 순간적으로 주변 마나보다 월등히 많아야* 앤트로픽 억누름 — 그래서 *큰 화염구는 마나 폭증* 필요
5. **성장 + 던지기**:
   - 큰 화염은 *산소 보충* 필요 (대기 산소가 빠르게 고갈됨)
   - **방법 A** — 바람 마법 응용: *한쪽에서 엄청난 기류의 공기량 흡입*
   - **방법 B** — *앤트로픽 원자 단위 재구성*: *탄소를 산소로 강제 변환* (변환계 마법)
   - 두 방법 *모두 훨씬 어려움*. 변환계 응용 시 *훨씬 농밀한 화염*
   - 던지는 방식 = *바람의 칼날에 실어서* 지속 공기 주입
   - **매질 활용** = *마력탄·열염석탄* 등으로 *훨씬 손쉽게 품질 향상*

#### 스펠과 고서클 시전의 차이

| 단계 | 시전 방식 |
|------|----------|
| **스펠 시전** (저~중급 셉터) | 5단계의 복잡한 감응·구현 과정을 *레시피화* 해 *룬어·시동어·양피지* 매개로 간단히 발동. *수박 겉핥기* 위험 |
| **고감응 고서클** (9써클 + 마스터급) | 스펠·레시피 의존 *벗어남*. *느끼는 대로 운용 → 결과를 스스로 창조*. 마법 종류별로 외울 필요 없이 *마나 흐름 원리* 를 자유자재로 다루면 *원하는 결과 생성* |

### 본작 카엘의 시전 방식 변천

| 시기 | 시전 방식 | 비고 |
|------|----------|------|
| 1.5부 떠돌이 | *현대마법총론과 상이* — 앤트로퍼 감응으로 *자기만의 방식* 발전 | 정통 스펠 모름. 직관 + 본능 |
| 2부 학습 | 정통 스펠 학습. 일레인식 응용 | 5년에 걸쳐 정통 마스터 |
| 3~4부 학살 | 스펠 + 자기 응용. 광역 마법 정점 | 앤트로퍼 이득 + 정통 결합 |
| 5~6부 의수 후 | *몸의 감응 + 검기 융합* — 스펠 점차 사용 안 함 | 의수 + 마법공학 결합 |
| Part 4 | **마나 흐름 원리 자유자재 운용** — *결과를 스스로 창조* | 진정한 의미의 *9써클 + 소드 엠퍼러급* |

### 원소 융합 — 응용의 정점

> *"광명과 화염폭풍 마법은 근본적으로 다르지 않다. 다른 것은 시전자가 *몇 가지를 동시에 제어할 수 있는가* 일 뿐이다."*
> — 행커만 임명식 시연 발언

- **백열의 구슬** = 화염구 + 광명 + 중력 압축의 3중 응용. 주먹만한 화염구를 *구슬 크기로* 압축하면 위력이 *수만 배*
- **얼음 거울 반사 백열** = 광명 + 얼음의 결합. 행커만 시연
- **얼음창 중력 응축** = 얼음 + 중력 응용. 일레인 맥필렌 정수
- **운석강하 분해** = 4방위 중력장 + 압축 얼음의 *동시 시전*. 9써클 마법사의 진짜 강함

> **응용의 본질 = 동시에 몇 가지를 제어할 수 있느냐**. 단일 마법의 위력이 아니라 *다중 시전과 응축* 이 9써클의 핵심.

## 4. 마나 응축 — 4가지 수련 타입

마법사의 강함은 *한 가지 척도* 가 아니다. 4가지 수련 타입이 있고, 어느 한 타입에 특화하거나 균형형이 된다.

### 타입 A — 마나통 (용량) 확장형

- **목표**: 최대 마나 용량(맥스 용량) 자체를 늘림
- **방식**: 장시간 마나 흡수 명상. 호흡과 함께 *큰 단위* 마나를 빨아들임
- **장점**: 1회 대규모 마법 시전 가능. 9써클 *운석강하* 시전에 필수
- **단점**: 회복 속도 느림. 한 번 쓰면 *수 시간 ~ 하루* 휴식
- **대표 인물**: 모티어스 매리앤 (최초 9써클)

### 타입 B — 빠른 충전형

- **목표**: 마나통은 작아도 *빠르게 차오르는* 회복 속도
- **방식**: 짧고 강한 마나 흡수 사이클 반복. *호흡·맥박과 동기화*
- **장점**: 장기전·연속 시전 강력. 전투 중 *끊임없는 화염구* 가능
- **단점**: 1회 한도 작아 *대규모 마법 불가*
- **대표 인물**: 데미안 스펠른 (스펠 매개로 작은 마나로도 강력 시전)

### 타입 C — 감응 효율형

- **목표**: 마나 감응을 극한으로 끌어올려 *아주 적은 마나로* 큰 효과
- **방식**: 정밀한 감응 훈련. 평생 *세프 한 알갱이 단위* 감응 연습
- **장점**: 마나 효율 압도적. 작은 회로로도 9써클급 마법 가능
- **단점**: 선천 재능 의존도 절대적. 후천 수련만으로는 한계
- **대표 인물**: 일레인 맥필렌 (행커만의 1/4~1/5 마나로 9써클 마법 시전)

### 타입 D — 지식 효율형

- **목표**: 마나 자체는 평범하되 *고효율 마법 레시피* 로 지구력 보완
- **방식**: 학문 연구·고대 문헌 해독. *적은 마나로 큰 효과* 의 레시피 수집
- **장점**: 평범한 셉터도 *지식만으로* 8써클급 위력 가능
- **단점**: 레시피가 없으면 무력. *창조성* 부족
- **대표 인물**: 슐로브스 마레 (마법 클래스 정립자), 본작 *카르엔 사이엔*

### 4타입 매트릭스 — 카엘의 위치

카엘 = **A + B + C + D 모두 비정상 (앤트로퍼 이득)**. 자기 노력 없이 *4타입 모두 평균 이상 도달* 했음. 5-2 카르엔 봉인 문서로 *이것이 자기 재능이 아니었음* 을 깨닫고 공허에 빠진다.

## 5. 마법사 vs 기사 — 거리별 우위 (절대우위 없음)

### 거리별 상성표

| 거리 | 마법사 | 기사 |
|------|--------|------|
| **원거리 (50m+)** | **유리** — 광역 마법 / 운석강하 / 화염구 | 불리 — 검 도달 불가, 원거리 무기로만 대응 |
| **중거리 (10~50m)** | 유리하나 *기사의 돌격에 신중* | 점진적 유리 — 돌격 거리 확보 |
| **근거리 (3~10m)** | 불리 — 스펠 시전 시간 부족 | 유리 — 검기 발현 + 돌격 가속 |
| **밀착 (3m 이하)** | **거의 절대 불리** — 마법사는 기사를 거의 이길 수 없다 | **거의 절대 유리** — 검기 한 합으로 결판 |

> *"절대우위는 없다. 다만 거리에 따른 *확률 우위* 가 있을 뿐이다."*
> — 슐로브스 마레, 강의록

### 마법사의 대응 전략

- **빠른 스펠** — 데미안 스펠른의 양피지 매개로 시전 시간 단축
- **회피 우선** — 점멸(블링크) + 순간이동 스펠 활용
- **보호·회피 아티팩트**:
  - 점멸 반지 (1일 사용 횟수 제한 → *비밀 카드* 로 활용)
  - 결계 부적
  - 자동 방어 결계 망토
- **시간 벌기** — 결계로 적의 접근을 *몇 초만* 지연시켜 그 사이 강한 스펠 시전

### 기사의 대응 전략

- **접근 또는 원거리 공격으로 마법사의 스펠·시전 방해**
- **방어 아티팩트 — 마법공격·정신계 공격 대비**:
  - 마법 저항 룬 갑주
  - 결계 무효화 검
  - 정신계 방어 투구 (환각·정신 조작 차단)
- **마법공학 결합** — *마나 지속 보관 물질* 을 검·갑주에 부착해 자기 마나 보조

### 본작 적용

- 카엘은 *원거리 광역 마법형 셉터* (1~3부). 기사 근접 전투 약함
- 5-5 이실의 *밀착 거리에서의 본태 일격* 에 카엘이 양팔을 잃은 이유 — 9써클 마법사여도 *밀착 거리 절대 불리*
- 6부 이후 카엘의 *공허검 전환* = 이 약점을 신체적으로 보완

## 6. 1~9등급 (써클) — 편의상 분류의 진실

### 공식 정의 (슐로브스 마레)

- 1~9써클로 마법 숙련도 구분
- **시니어 셉터**: 자연계 가장 작은 세프 감응 가능
- **9써클 마스터**: 모든 세프 *동시 감응* 가능 — 자격일 뿐 *마지막* 아님

### 진실 — *등급은 인간들의 편의를 위한 척도일 뿐*

> *"실제로 등급은 의미가 없다. 필요성에 따른 발현 원리를 연구·실현한 결과물일 뿐이다."*
> — 본작 명제

- 등급은 **사피엔들만의 기준점**. 다른 종족·드래곤·고대 마법은 *등급으로 분류되지 않는다*
- **같은 등급, 같은 마법이어도 구현 방법·위력·실현 결과가 판이하게 다르다**

### 예시 — *화염구* 의 진실

| 시전자 | 화염구 크기 | 위력 | 구현 난이도 |
|--------|-----------|------|-----------|
| 5써클 평범 셉터 | 주먹 크기 | 평민 1~2명 살상 | 보통 |
| 7써클 숙련 셉터 | 머리 크기 | 갑옷 기사 1명 절명 | 보통 |
| 9써클 일반 시전 | 집 크기 | 작은 마을 1회 전소 | 어려움 |
| 9써클 압축 시전 | **구슬 크기** (압축) | **수만 배 위력 — 도시 1구역 폭파** | **극악** |

> 압축 화염구 = 일반 화염구를 *광명 + 중력 응용* 으로 *구슬 크기로 축소*. 백열 구슬 형태. *집 크기 × 수만 배* 의 위력.

### 예시 — *구원 마법* 의 두 얼굴

> 구원 마법 = 빛으로 생체 회복력을 극적으로 높이는 마법
>
> **그 구현 원리에 물 + 침습 기술을 극한으로 섞으면** — *대규모 세포괴사를 일으키는 저주의 기운* 이 된다

- *같은 빛, 같은 원리, 정반대 결과*
- 본작 함의: **금기 응용은 정통 응용과 종이 한 장 차이**. 마녀령 회혈 가문의 인공 저주가 *르세프 치유 마법의 역설* 임을 시사

### 본작 적용

- 카엘은 본인이 *9써클* 임을 자랑하나, 진실은 *앤트로퍼 이득으로 도달한 잘못된 9써클*
- 알레이언 6세 = 신성마법 5등급이나 *균열의 빛 봉인 의식 즉발판* 시전 = 등급 표가 의미 없는 *천 년 비전*

## 7. 사피엔의 특이 응용 — 통념을 벗어난 마법 활용

대부분 피조물은 *기초 원리 + 수련* 의 정통 마법을 따르나, *전승·응용·학습에 매진* 하는 사피엔은 *통념을 벗어난 특이 응용성* 을 보인다.

### 7.1 마도공학 결합 무기 — *아다만티움 + 화약 + 부싯돌*

- **원리**: 자연계 최강 매질 *아다만티움* (드워프 합금)을 매개로 고열 응축
- **고열 발생 방식의 특이성**: *스펠 시전 시간조차 없도록* 고속 연사용 — *화약 + 부싯돌* 로 *초기 고열* 발생
- **응용 무기**:
  - *용접기형 해머* — 발생한 열을 *대기 컨트롤* 로 지속화. 망치 형태이나 *용접기처럼* 뜨거운 열이 지속 발생
  - *연사식 화염 발사기* — 화약 → 부싯돌 → 1회 점화. 마법사 스펠 없이 *기사가 화염 무기 사용 가능*
- **사회 의의**: 마법 재능 없는 평민도 *마법 무기 사용 가능* → 군 평준화
- **출처**: 타르모리스 → 제국 데미안 마도공학 정통. *카르덴 스펠른* (데미안 마도공학원 원장) 현 발전 중심

### 7.2 점멸 양피지 — 시전 시간 0의 단거리 순간이동

- **원리**: 점멸(블링크) 스펠을 *양피지 + 룬어* 에 미리 결박. *양피지를 찢는 순간* 즉발 시전
- **응용**:
  - 기본 1매 = 1회 사용. 5~10미터 점멸
  - 고급 양피지 = 20미터 점멸. 가격 폭등
  - 일레인 맥필렌이 사용한 *룬어 팔찌형 점멸* = 양피지 변형. 1전투당 *5회 시전*
- **사피엔 특이성**: 다른 어떤 종족도 이런 *마법 압축 양피지 기술* 을 개발하지 않았다 — *기록 본능* 의 부산물

### 7.3 비밀 카드 — *1일 한정 점멸 반지*

- **원리**: 점멸 스펠을 반지에 결박. *하루 사용 횟수가 정해져 있어* 시전이 *공개 표시 없이* 발생
- **전투 활용**: *상대가 모르는* 반지의 즉발 점멸로 *시야에서 사라졌다 다시 나타나기*. 결정적 1격을 노린 *비밀 무기*
- **가격**: 9써클 마법사 1년 연봉. 보통 *왕실 + 최상위 셉터* 만 보유
- **본작 함의**: 라이세린 공작의 *알 수 없는 행적 공백* — 점멸 반지 보유 가설

### 7.4 잔혹 전투 마법 — 윤리의식 없는 셉터의 응용

본작에서 *변경 용병 셉터* 또는 *암살자 셉터* 가 사용하는 *비공식·금기적 마법*. 정식 학원에서는 *공식 금지*.

#### 7.4.1 광명 + 눈 공격 — *시선 유도 + 눈동자 백열*

- **방식**:
  1. 강한 광명으로 *상대의 시선 유도* (반사적으로 광원을 봄)
  2. 그 *순간* — 가장 작고 빠른 광명을 *상대 눈동자에* 발사
  3. 눈동자 표면이 *백열로 가열* 되어 *영구 실명*
- **잔혹성**: 즉사가 아닌 *영구 장애*. 회복 불가
- **시전 난이도**: *5써클로도 가능*. 정밀 제어가 핵심

#### 7.4.2 바람 + 독 공격 — *공기 통로로 독 흘리기*

- **방식**:
  1. 미리 *독* 을 휴대 (액체·기체)
  2. *바람의 통로* 마법으로 자신 손바닥에서 *상대 얼굴* 까지 *눈에 보이지 않는 공기 흐름 통로* 형성
  3. 통로 안에 *독* 을 흘려보냄
  4. 상대는 *호흡으로 흡수* → 즉시 중독
- **잔혹성**: 적이 *공격을 인지하지 못함*. 평범한 대화 중 살해 가능
- **시전 난이도**: 6써클. *바람 + 정밀 통로* 의 결합

#### 7.4.3 바람 압축 + 귀 공격 — *고막 파괴*

- **방식**:
  1. *바람 압축* 으로 작은 음파 폭탄 형성
  2. 상대 *귀 직전* 에 폭발
  3. 고막 파괴 → *평형 감각 상실, 영구 청각 장애*
- **잔혹성**: 회복 불가. *집요한 반복* 시 정신 붕괴
- **시전 난이도**: 4써클. *바람 정밀 압축* 만 가능하면 시전 가능

### 본작 적용

- 라이세린 공작의 *비공식 측근* 중 *잔혹 전투 마법사* 1~2인 가설
- 카엘은 잔혹 마법을 *배우지 않는다* — 변질된 시야에서도 *너무 명백한 비윤리* 라 *본능적 거부*. 이 한 가지가 카엘에게 남아있던 *마지막 도덕적 잔재*
- 후반부 *백색 사냥단의 잔혹 마법 사용 사례* 직접 목격 → 카엘의 *분노가 처음으로 정당한 방향* 으로 사용되는 계기

## 8. 마법 8계통 (정통 분류)

### 8.1 마법계

- 주체: 학원 셉터, 마법사
- 방식: 공식·주문·룬·마법진으로 마나 배열
- 용도: 원소·결계·탐지·변환·기록·대규모 의식
- 비용: 복잡한 배열의 장기 고정 → 엔트로픽 농도 상승

### 8.2 검기계

- 주체: 기사
- 방식: 마나를 검·갑주에 흘려 절단·방어·기동 강화
- 핵심: 기술 + 마나 순환 결합
- 한계: 무기·육체 숙련 없이는 마나만으로 강해질 수 없음
- 상세: [`swordsmanship.md`](./swordsmanship.md)

### 8.3 기공계

- 주체: 격투가·수도사·용병 유파·몽크
- 방식: 마나를 호흡·혈류에 맞춰 *내부 순환*
- 용도: 신체 강화, 충격파, 방어, 회복 촉진
- 특징: 외부 주문보다 안정적, 수련 의존도 높음

### 8.4 마도공학계

- 주체: 공방·제국 기술자·데미안 마도공학원·그라스 룬 장인
- 방식: 마나를 동력으로 기계·무기·이동수단 작동
- 핵심: *마나 지속 보관 물질* 의 발견. 기사들이 이를 활용해 *전투 지속력* 보완
- 비용: 대량 운용 시 마나 정체·도시 농도 상승

### 8.5 악마계

- 주체: 마녀 (특히 금혈 가문), 금기 연구자
- 방식: 마계 해체 질서와 계약·오염·역류 마나 이용
- 용도: 저주·변이·해체·금기 소환·대가 교환
- 위치: 신좌침묵 의식에 사용되었으나 현재 *공식 금기*. 마녀령에서만 합법

### 8.6 성직계

- 주체: 성기사, 수녀, 무녀, 성직자
- 방식: 마나의 빛 계열을 *신앙심으로 정렬*
- 용도: 치유·정화·결계·축복·마계 오염 지연
- 한계: 생명을 늘리고 보존하므로 *엔트로픽 농도 문제 해결 못함*
- 상세: [`holy-magic.md`](./holy-magic.md)

### 8.7 소환계

- 주체: 소환사·계약술사·금혈 마녀
- 방식: 마나 매개로 다른 존재·정령·마계 창조물과 통로 형성
- 용도: 정령 계약, 사역마, 이계 물질 호출, 전장 보조
- 위험: 통로 역류 시 마계 균열의 씨앗

### 8.8 환각계

- 주체: 환술사·암살자·청혈 마녀
- 방식: 마나로 *감각·기억의 해석* 왜곡
- 용도: 은신·기만·공포·기억 조작·군중 통제
- 연결: 앤트로퍼의 감각 왜곡은 환각계보다 *더 근본적인 인식 변조* — 환각계는 *외부 감각* 만, 앤트로퍼는 *해석 회로* 자체

### 8.9 기타 분야

- **연금계**: 물질 배열 변환·약물·촉매
- **점성계**: 대륙 마나 흐름·전조 해석 (자혈 마녀 정점)
- **사령계**: 죽은 자의 잔향·기억. 농도 상승 위험 → 금기
- **혈계**: 혈통과 육체에 마나 배열 고정 (적혈 마녀 정점)
- **음률계**: 소리·리듬으로 마나 흐름 조율

## 9. 계통 간 긴장 — 마나 평형 차원

| 계통 | 농도 영향 | 본작 위치 |
|------|----------|----------|
| 마법계·마도공학계 | **농도 ↑** (문명 발전 동시) | 헤븐즈·데미안의 *자기 모순* |
| 성직계 | 생명 보존 → 농도 ↑ | 카르샤의 *작은 영토*는 농도 부담 작음 |
| 악마계·해체 기술 | **농도 ↓** (마계 위험 동반) | 마녀령의 *작은 해체* 윤리 |
| 검기계·기공계 | 개인 단위. 농도 무관 | 강철제국·세이른 |
| 환각계·소환계 | 농도 *중립* (장기적으로 약상승) | 마녀령 청혈·금혈 |

> 강력하고 유용한 기술일수록 *세계를 지키는 동시에 다음 리셋의 원인* 이 될 수 있다.

## 10. 본작 활용 — 마법 시전 묘사 5원칙

1. **5원소 발현은 반드시 *세프 운영 원리* 로 묘사**. *그냥 불이 나왔다* 는 금지. *세프가 가열되어 불이 되었다* 로 묘사
2. **써클은 *호명만*** — *9써클 운석강하* 라는 호명은 OK. 그러나 *9써클이라 강하다* 식의 단정 금지. 구현 방법·압축·응용으로 위력 차이 표현
3. **마법사 vs 기사 거리 상성을 의식적으로 활용** — 카엘은 4부까지 *원거리 학살*, 5-5에서 *밀착 거리 패배* 의 의미가 명확해야 함
4. **사피엔 특이 응용 (양피지·반지·마도공학)** 을 일상화. *마법은 곧 첨단 기술* 임을 묘사로 표현
5. **잔혹 마법은 *드물게, 단호하게* 사용** — 한 회차에 2회 이상 묘사 금지. 사용 시 *카엘 본능적 거부* 명시
