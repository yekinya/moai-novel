## 1. 마법사 등급 (1~9써클)

마법사 등급은 *써클 (Circle)* 단위로 측정. 1써클 = 견습 / 9써클 = 인간으로 도달 가능한 현실적 최고 등급.

| 써클 | 호칭 | 일반 정통 | 사회적 위치 |
|------|------|----------|------------|
| 1써클 | 견습 마법사 (Apprentice) | 기본 마법 1~2종 시전. 마나 사용 효율 낮음 | 도제·학생 |
| 2써클 | 초급 마법사 (Initiate) | 기본 공격·방어 마법 안정 시전 | 도제 졸업·하급 보조관 |
| 3써클 | 중급 마법사 (Adept) | 4원소 마법 + 단순 변형 마법 | 정규 군 마법사·궁정 보조 |
| 4써클 | 정식 마법사 (Mage) | 광역 1단계 + 복합 시전 가능. 학회 정식 인가 | 학회 정식 회원·왕실 인가 |
| 5써클 | 상급 마법사 (Senior Mage) | 광역 2단계 + 비상식 응용 가능. *국가 공인 마법사*의 일반적 하한 | 외교실 자문·고급 학자 |
| 6써클 | 전쟁 마법사 (Battle Mage) | 광역 학살·전장 차원 시전. *국가 전쟁 자산* | 전쟁 참모·왕실 직속 |
| 7써클 | 대마법사 (Archmage) | 진영 차원 영향 + 비전 정통 보유. *대륙 100여 명 수준* | 학회 원로·대륙 권위 |
| 8써클 | 마도 거장 (Grand Archmage) | *대규모 전장의 결정적 변수*. *전쟁의 흐름을 바꿈*. 대륙 10~20명 | 왕실 직속·대마법사 협회 정점 |
| 9써클 | 대마도사 (Magus / Grand Sage) | **인간으로 도달 가능한 현실적 최고 등급**. 대륙 1~3명만 존재. *생존 시 국가의 전략 자산* | 왕실 호위 최정점·전설 |

### 8써클과 9써클의 *결정적 차이*

- 8써클 = *대규모 전장에 결정적 영향*. 그러나 *다른 8써클·9써클의 견제 가능*. 단일 광역 마법 위력 = 보병 1,000명 ~ 5,000명 영향
- **9써클 = *전장 그 자체를 한 점으로 압축***. *광역 마법 1발이 보병 1만 명 단위 절멸 가능*. *다른 9써클이 없는 한 견제 불가*. 9써클 1명 = 8써클 5명을 압도. **국가에 9써클이 있느냐 없느냐가 전쟁의 본질적 양상을 결정**

### 시전 매개 정통 (3등급 구분)

| 단계 | 시전 매개 | 정통 |
|------|----------|------|
| 1~6써클 | *반드시 손 + 스펠 + 마법진* 매개 필요 | 시전 시간 + 매개 동작이 *시전의 *필수 조건*. 적의 직격에 *동작이 차단되면 시전 무효* |
| 7~8써클 | *일부 스펠 + 손으로 마법진* 매개 필요 | *대부분 시전은 단축 가능*하나 *광역·복합 마법은 여전히 매개 동작 필요*. 단축 시전 = 위력 일부 감소 |
| **9써클** | **온몸 마나 감응 — 손·스펠 없이 시전 가능** | *마나 회로가 *체내 전체에 확장 + 생각만으로 시전*. *약점 = 시전 의식의 명료성 필요 → 정신력 흔들림이 시전 차단 가능*. 본작 9써클 = *칼란 베르냐스* (상세 [`../world/06-국가별-군사력과-왕실정예.md`](../world/06-국가별-군사력과-왕실정예.md)) |

### 9써클 시전속도와 *기동 정통의 필요성*

- *9써클 광역 마법 = 8~12초 시전*. *시전 시간 동안 자기 보호가 필요*
- 본작 *칼란 베르냐스의 정통* = *공중부양 (5분 부유·30초 낙하) + 블링크 반지 2종 1분 반복 시전 + 바람 방어막 지속*의 *3중 기동 정통*으로 *시전 시간 약점을 완벽 보완*
- *대륙 9써클의 정통은 *시전 위력 정점만 보유 ≠ 기동 정통 보유*. *칼란은 *위력 + 기동 융합의 대륙 유일 정통*. 행커만도 *위력 정통 + 폴리모프 매개*는 보유했으나 *기동은 폴리모프 비행에 의존*

### 정통 부가 분류

- **마검사 (Sword Mage)** — *5써클 전후 마법 + 소드 마스터 또는 그 직전 단계 검술*의 *드문 융합 정통*. 일반적이지 않으며 *개인 단위 정통 — 일괄 척도 없음*. 검술과 마법이 *서로 다른 등급으로 측정될 수 있음*. 본작 카엘은 *5써클 + 소드 마스터 미만의 검술 + 비상식 응용*의 *변종 마검사*
- **신성 마법사 (Holy Mage)** — *써클 등급 대신 신성 위계로 측정* (해당 국가의 신앙 정통에 따름). 카르샤 신성제국 = *대주교·주교·신부·수도사 위계*
- **룬 마법사 (Rune Mage)** — *써클보다 룬 결각 등급 + 아티팩트화 정통으로 측정*. 앨더스톰 + 드워프 정통

## 2. 검사 등급 (5단계)

검사는 *써클이 아닌 단계 (Tier)* 단위로 측정. 일반검사가 1단계, 소드 엠퍼러가 최고 5단계.

| 단계 | 호칭 | 정통 | 사회적 위치 |
|------|------|------|------------|
| 1단계 | 일반 검사 (Soldier) | 기본 검술 + 진형 전투. 검기 없음 | 정규 보병·하급 호위 |
| 2단계 | 시니어 검사 (Senior) | 단독 전투 + 기본 살상 직관. 짧은 검기 1~2초 가능 | 정규 호위·중급 지휘관 |
| 3단계 | 익스퍼트 (Expert) | 검기 안정 운용 + 1합 다수 처리. *국가 정규 기사의 일반적 등급* | 기사단 지휘관·정규 기사 |
| 4단계 | **소드 마스터 (Sword Master)** | 검기 자유 운용 + 다수 동시 처리 + 일반 마법 1~3써클 비기 직접 반사·분쇄. *국가 전략 자산* | 왕실 직속 호위·전쟁 결정 변수 |
| 5단계 | **소드 엠퍼러 (Sword Emperor)** | **검의 인간적 정점 — 검 하나로 8써클 광역 마법 분쇄·반사 가능**. 대륙 5~10명 미만 존재. *왕실의 진정한 핵심 무력* | 국가 최정점·전설 |

### 소드 마스터와 소드 엠퍼러의 *결정적 차이*

- 소드 마스터 = *검기 + 살상 직관 + 마법 분쇄 일부*. *광역 6~7써클 마법을 짧게 회피·분쇄 가능*. 그러나 *8써클 광역 마법 직격은 위험*
- **소드 엠퍼러 = *검 하나로 광역 8써클 마법을 정면 분쇄·반사 가능***. *9써클을 직접 베지는 못하나, 9써클의 광역 마법조차 한 점에 압축해 비킬 수 있음*. 9써클 마법사와의 일대일 = *오래 끌면 마법사 우세, 짧으면 엠퍼러 우세*. **국가에 엠퍼러가 있느냐 없느냐가 전쟁의 무력 균형을 결정**

### 엠퍼러급 *전투 정통 6중* (본작 칼라난 발도르 정통 매개)

- **온몸 검기화** — *검기를 검 자루뿐 아니라 팔·다리·가슴 전체에 두름*. *근육 강화 + 신경 반응 강화 + 마나 감응 강화의 3중 동시 효과*. 물리 능력 = 일반 인간의 5~7배
- **한 검 적중 = 두동강** — *일반 강철 갑옷·일반 강철 검은 직격 시 동강. 아다만티움급만 부분 견딤*
- **마법 시전 1초 이내만 회피 가능** — *적의 1초 이상 시전 = 엠퍼러의 접근·검 한 합으로 목 날림*
- **검은 검 (Black Sword)** — *검기를 강하게 모은 응축 정점 일격*. *일반 강철 = 모두 동강. 카엘 같은 아다만티움 검 = 검과 함께 조각*
- **카보나이트 합금 갑옷** — *경량 + 매우 단단 + 마나 주입 가능 + 외부 마법 부분 반사*. *카보나이트 = 마나 전도 우수 + 가벼움의 결합으로 갑옷 정점 매개*
- **검기 단검 던지기** — *약 1.5초 검기 유지 + 자동 추적 보조*. *기동·원거리 보완 + 1초 시전 차단 정통과 결합 → 적의 거리 매개 자체를 차단*

본작 엠퍼러 = *카르난 발도르* (상세 [`../world/06-국가별-군사력과-왕실정예.md`](../world/06-국가별-군사력과-왕실정예.md)). *마법 자질 없는 자가 30년 검 수련 끝 도달한 정점 — 일반적으로 성취하기 힘든 정통의 직접 표현*

### 익스퍼트와 마스터 사이 *깊은 벽*

- 익스퍼트 → 마스터로의 진입은 *재능 + 깨달음 + 수련의 3중 일치*가 필요. *수련만으로 도달 불가*. 정규 기사 대부분은 *익스퍼트 정점에 머무름*
- 본작 카엘 = *익스퍼트 정점 (Ch1 P3 알린 마스반 경 사사 + 외곽 연구소 5년 자기 단련의 결과) + 비상식 마검사 응용*. *소드 마스터급은 아님*

### 특수 정통

- **마검사** — [`./swordsmanship.md`](./swordsmanship.md) + [`./magic-system.md`](./magic-system.md) 융합. *5써클 + 익스퍼트 ~ 마스터 미만의 검술 + 자기 변칙 5종이 일반적*
- **신성 기사 (Holy Knight)** — *카르샤 신성제국 정통*. *검술 + 신성 마법 + 축복 무장의 융합*. *소드 마스터 단계 아님 — 중간급 기사단 (익스퍼트 + 신성 가호) 수준*. 그러나 *대규모 진형에서는 마스터급 1인 분량의 전장 영향력*
- **공허검 마스터** — *마녀령 동부국 비전 5인 한정 (Ch2 P3 모르간 솔테·카이렌 스틸반·세이른 스틸만 포함)*. *소드 마스터 + 공허검 3원리의 융합 — 검기 없이 검사 제압*. 정통상 *소드 마스터의 변종 정점*

## 3. 전쟁의 핵심 존재 6종

대규모 전쟁에서 *전세를 결정짓는 무력 유형*. 한 국가의 전쟁 수행 능력은 *이 6종의 보유 + 조합 + 운용*으로 측정.

### (1) 8~9써클 마법사

- 8써클 = *전쟁의 결정적 변수*. 9써클 = *전쟁의 양상 자체를 결정*
- 9써클 1명의 광역 마법 1발 = *보병 1만 명 단위 절멸*. *다른 9써클·소드 엠퍼러가 없는 한 견제 불가*
- 대륙 9써클 = *총 1~3명 수준*. 본작 시점 (CC 1000~1001) 확인된 9써클 = *2명 (바이서스 왕실 직속 1인 + 일레인 맥필렌 — 단 일레인은 본작 시점 잠적 또는 사망 미확정)*

### (2) 소드 엠퍼러 / 소드 마스터

- 엠퍼러 = *왕실의 진정한 핵심 무력*. 대륙 5~10명 미만
- 마스터 = *국가 정규 군의 결정적 무력*. 대륙 200~300명 수준 (대부분 정규 군 또는 왕실 직속)

### (3) 마법공학 무기·공성 무기

- *바이서스 + 데미안의 대륙 최대 정통*. *룬 결각 + 아티팩트화 + 마도 회로의 융합*
- 대규모 공성포·마력 발사기·마법 차단 결계 발생기·연발 화살기·이동식 마법 함정 등
- *비용이 높아 다수 동원이 곧 국가 경제력의 척도*. 바이서스 = 대륙 1위, 데미안 = 대륙 2위 (정통은 더 깊으나 보유량은 적음)

### (4) 다양한 아티팩트

- *희귀 마나석 + 룬 결각 + 비전 정통의 융합*. 1회용·다회용·결계용·전략용 등 다양
- *카엘의 블링크 반지·마나응축 보석 장갑*은 *개인 단위 아티팩트의 정점*. 국가 단위로는 *대규모 결계 + 광역 차단·증폭 아티팩트*가 핵심
- 앨더스톰 = 대륙 룬 결각 정통의 최고 보고. 드워프 지하세계 = *아다만티움·미스릴 + 비전 룬*

### (5) 신성 사제단

- *대규모 힐·축복·결계 방어의 정통*. 정규 보병 1만 명을 *짧은 시간에 회복시킬 수 있음*. 군 전체의 *지속 가능성·사기·생존성*을 좌우
- **카르샤 신성제국 = 대륙 신성 사제단 정통의 정점**. *대주교 1인 + 주교 다수 + 신부·수도사 다수의 군 동원 가능*
- *바이서스는 신성 사제단이 거의 없음* (자세한 사유는 §4 종교·정치 정통 참조)

### (6) 마검사

- *5써클 + 익스퍼트 검술의 융합*. *국가 단위 다수 동원은 어려움 (개인 단위 정통)*
- 그러나 *전장에서 마법사·검사 양면 동시 견제 가능*. *마법사로는 검사를 견제 + 검사로는 마법사를 견제*. *전술적 가치 매우 높음*
- 본작 카엘 = *변종 마검사 + 5써클 + 5인 한정 비기 (광역 학살 합성 시 호스반이 카엘 명의로 사용)*

## 4. 종교·정치 정통과 군 무력의 관계

### 바이서스에 신성 사제단이 거의 없는 이유

- 바이서스는 *세속적 왕권 중심 + 마법 학회·마도공학 중심* 국가. *공식 국교는 라카르 9세 시기 정통 신앙 2종 (자세한 명칭은 [`../world/08-종교와-신앙.md`](../world/08-종교와-신앙.md) 참조) 의 *형식적 명목 신앙*만 유지*
- *대규모 신성 사제단 = 종교 권위의 정치적 자산*. 바이서스 왕가는 *역사적으로 신성 권위를 왕권 위협으로 간주 → 사제단을 약화·세속화 + 권위 제한*
- 결과 = *바이서스 신성 마법 위계는 4단계 신부급이 정점*. *5단계 이상 (주교·대주교 수준)은 존재하지 않음*. *대규모 전장 힐·축복은 사실상 불가능*
- 이는 바이서스 군의 *지속 가능성·사기·생존성 약점*. *물량·무기·마법공학으로 보완*

### 카르샤에 신성 사제단이 강한 이유

- 카르샤는 *신성제국 — 신성 신앙 정통의 대륙 정점*. *왕가 = 신성 권위와 융합*. *알레이언 6세 = 종교적·정치적 동시 정점*
- 신성 사제단 = *왕실의 직속 무력 일부 + 시민의 신앙 중심 + 대규모 전장 회복 자산*
- 신성 기사단 = *검술 + 신성 마법 + 축복 무장의 융합 군 직제*. *순수 검술 단계는 익스퍼트급이나, 신성 가호 매개로 마스터급에 근접한 전장 영향력*

### 그 외 국가의 특화

- **데미안** — 마도공학 정통의 정점. *공성 무기·아티팩트 보유량은 바이서스보다 적으나 정통 깊이는 우월*
- **강철제국** — *마법 거부 검술 정통*. *마법 봉인 검사 직제*. 일레인 양식 외 *마법공학 기피*. *순수 검술 + 강철 정통의 결합*
- **마녀령 동부국** — *9혈 마법 정통 + 룬 결각 + 마검사 변종 정통*. *왕가 없음 — 모르간 가문 흑혈 종주 중심*
- **앨더스톰** — 룬 마법 + 고대 마법 도서관. *군 직제는 작으나 비공식 동맹망의 정통*
- **드워프 지하세계** — 별도 종족 정통. *외부 거래 매개로만 인간계 영향*

## 5. 등급 측정의 한계와 변종

- *마검사·신성 기사·룬 마법사·공허검 마스터*는 *단순 등급으로 측정 불가*. *해당 정통의 깊이 + 융합 정도 + 전장 상황에 따라 평가 변동*
- 본작 카엘 = *Ch1 시기 5써클 + 익스퍼트 정점 + 비상식 응용 + 직관 1단계*의 *변종 마검사*. *광역 학살 1,400명은 불가능 — 호스반의 6써클 합작·강 독살 누적·시각 효과 조작이 합성된 결과*
- 본작 카엘 = *Ch2 시기 마법 봉인 완전 + 공허검 1·2원리 + 마력→내공 4중 회로 + 아다만티움 의수 + 룬 결각 전체 = 정통 등급으로는 측정 불가능한 변종 정점*. *전장에서는 소드 마스터급 직접 견제 + 8써클 광역 마법 일부 회피 가능 수준*

## 2. 8왕국 후신별 병법 정통

### 2.1 헤븐즈 킹덤 — *마법 군단 중심 + 결계 외교*

#### 핵심 교리

> *"마법사가 가장 강한 군대다. 검은 마법사를 *지키는 용도* 일 뿐."*

#### 군 구성

- **9써클 마스터급 1~2명** — 광역 마법 / 운석강하
- **시니어 셉터 50~100명** — 화염구·결계·치유 일제 시전
- **검사형 셉터 200~500명** — 마법사 호위
- **일반 보병 2,000~5,000명** — 후위 정리

#### 전술

- **결계 외교** — 적군 진군로에 *대규모 결계* 미리 깔고 진격 강요
- **광역 일제 사격** — 시니어 셉터 100명이 *동시 화염구 시전* → 적군 1열 일격 무력화
- **마지막 카드 — 9써클 운석강하** — 결정적 순간 1회. 적 후방 보급선 파괴

#### 약점

- *근접전 약함* — 검사형 셉터 약함. 적 기사 돌격에 무너짐
- *비용 극심* — 시니어 셉터 1명 양성 = 평민 100명 평생 비용

#### 본작 4-1 카엘의 응용

- 단신으로 *시니어 셉터 100명 분의 광역 마법* 시전 — 앤트로퍼 이득
- 적군 1열 1,400명 *형태가 무너진 채 사망* = 운석강하급 위력의 *압축 화염구* 응용

### 2.2 엘더스톰 — *마법 군국 + 영웅 가호*

#### 핵심 교리

> *"우리는 모두 일레인의 후예다. 사기와 결속이 마법 위력을 *배가* 한다."*

#### 군 구성

- **셉터 정예 100~200명** — 일레인 정통 *중력·공간 마법*
- **마검사 50~100명** — 마검 일제 돌격
- **일반 셉터 + 보병 5,000명**
- 군 *전체* 가 셉터 — 70%가 시니어 셉터 이상 (강도 격차)

#### 전술

- **중력 함정** — 적 진군로에 *중력장 폭탄* 매설. 보이지 않다가 폭발
- **얼음 거울 함정 전선** — 칠흑성 전투 응용. *반사 백열* 로 적 전선 일제 소각
- **마왕군 돌격** — 일레인 광장에서 *결속 의식* 후 출정. *사기 진작 신성마법* 으로 군 전체 *2배 강화*
- **오우거 환각 훈련 응용** — 적군에 *집단 환각* 시전. 적군이 *서로를 공격*

#### 약점

- *외부 결계 마법 약함* — 자기 진영 결계 우선이라 공성 시 약함
- *고비용 군대 — 작은 영지 침공에는 비효율*

### 2.3 강철제국 — *중기병 + 검기 일점 돌파*

#### 핵심 교리

> *"적 마법사 100명에 검 5자루로 응대한다 — *옆에서* 들어가는 5자루다."*

#### 군 구성

- **다섯 검 (그랜드 마스터급) 5인** — 황실 직속. 핵심 일격
- **마스터급 강철검사 200~300명** — 중기병 돌격
- **정식검사 1,000명** — 보병 정예
- **마검사 50명** — 마법사 진영 직접 침투
- **일반 보병 5,000~8,000명**

#### 전술

- **검의 돌격 (Charge of Steel)** — 중기병이 *적 마법사 진영의 측면* 으로 일거 돌격. 마법사들이 영창 마치기 전에 *밀착 거리* 도달
- **결계 분쇄 검 (Ward-Breaker)** — 다섯 검의 *검기 일격* 으로 적 결계 *물리적 분쇄*
- **소규모 정예 침투** — 일반전 대신 *5인 다섯 검 + 마검사 5인* 의 *10인 정예* 가 적 본진 후방 침투. 적 사령관 직접 처치
- **얼음의 왕좌 농성** — 카르나는 *천 년 함락 0회* — *영하 6개월 + 천연 절벽 + 결계* 의 3중 방어

#### 약점

- *광역 마법 결여* — 1차 광역 사격에 *보병 손실 큼*
- *후방 보급 의존* — 강철 갑주의 무게로 장거리 행군 어려움

### 2.4 카르샤 왕국 — *신성검 + 결계 농성*

#### 핵심 교리

> *"우리는 침공하지 않는다. 우리는 *지킨다*. 우리의 검은 신을 위해 쓰는 *마지막 도구* 다."*

#### 군 구성

- **신성검단 30명** — 신성검 마스터급 + 5등급 신성마법
- **신관 100명** — 결계·치유 전문 수녀단
- **정식 보병 1,500명**
- 총 *2,000명 미만* — **대륙 최소 규모 군대**

#### 전술

- **신성 결계 농성** — 세이렌 도성 주변 *7중 신성 결계*. 마계 창조물·언데드에는 절대 강함
- **마지막 카드 — *균열의 빛 즉발판*** — 알레이언 본인이 *자기 죽음으로* 시전. *앤트로퍼 사슬 균열* 만 가능
- **신성 치유 전선** — 수녀단이 부상병 즉시 복구 → 전쟁 지속력 극대화
- **사절 외교 우선** — *전쟁 자체를 회피*. 카일렌 베르하르트 경의 평화 사절단(3-1)이 정통

#### 약점

- *공격력 압도적 부족* — 일반 전쟁에서는 *3일 안에 함락*
- 본작 4부: 균열의 빛은 *앤트로퍼에게만 유효* → 일반 군사 침공에는 무력. 7일 만에 세이렌 함락

### 2.5 마녀령 동부국 — *9혈 분산 게릴라 + 저주 화공*

#### 핵심 교리

> *"우리는 정규 군대가 없다. 우리는 *9개의 작은 가시* 다. 가시는 *언제 어디서나* 찌른다."*

#### 군 구성

- **각 종파 추장 직속 무력** — 종파마다 50~200명. 총 1,000~1,500명
- *통합 군대 없음* — 9혈 종주별 *독립 운용*

#### 종파별 전술

| 종파 | 전술 |
|------|------|
| 흑혈 | 언데드 사역. 죽은 자가 다시 일어남 |
| 적혈 | 혈주술 일제 돌격. 마검사 정통 |
| 청혈 | 환각·기억 조작. 적 사령관 *자기 군대를 적으로 인식* |
| 백혈 | 후방 치유·정화. 외교 중재 |
| 회혈 | 인공 역병 화공. *적군 1주일 후 발병* |
| 녹혈 | 자연 독·식물 매복 |
| 자혈 | 점성 예언으로 *적 전술 사전 파악* |
| 금혈 | 마계 창조물 1체 임시 소환 |
| 무혈 | *마법 자체를 무력화* — 적 시전 즉시 *해체* |

#### 전체 전술

- **분산 동시 다발** — 9혈이 *9개 방향에서 동시 침투*. 적군은 어디를 막아야 할지 모름
- **장기 소모전** — 6개월 ~ 1년의 *느린 학살*. 적군은 *왜 죽어가는지* 모름
- ***정확한 표적, 작은 규모*** — 9혈 강령 준수. 일반 시민 학살 *금지*

#### 약점

- *대규모 정면전 부적합* — 한 번도 *대륙 정복* 시도한 적 없음
- *9혈 분열* — 종주 합의 깨지면 군사 작전 불가
- **앤트로퍼 1체 (카엘 같은) 에 의해 *전 마녀령 학살 가능*** — 변질된 운영의 결과

### 2.6 제국 데미안 — *마도공학 화기 + 용병단*

#### 핵심 교리

> *"마법사 1명을 양성하는 비용으로 *마도공학 화기 100자루* 를 만든다. 100자루가 1명을 이긴다."*

#### 군 구성

- **마도공학 화기병 5,000~10,000명** — 평민도 운용. *마법 재능 불필요*
- **마검사 100명** — 황실 직속
- **고용 용병단 + 백색 사냥단 2,000명** — 변경 무인
- **상인 사병 길드 군대 3,000명**
- **총 ~20,000명** — **대륙 최대 규모**

#### 전술

- **연발 화기 일제 사격** — *아다만티움 + 화약 + 부싯돌* 결합 화기. *시전 시간 0* 의 즉발 화염
  - *연사식 화염 발사기* — 적 1열 일제 무력화. 마법 결계로 막을 수 없음 (마법이 아니라 *순수 화학·기계 화염*)
  - *용접기형 해머* — 근접전. 적 갑주 *녹여 관통*
- **점멸 양피지 공격** — 보병이 *점멸 양피지* 로 *순간이동* — 적 후방 즉시 도달
- **상인 사병 — 비대칭 전쟁** — 적국 *경제 거점 직접 타격* (시장·항만 파괴)
- **백색 사냥단 — 정밀 암살** — 적 마법사·왕족 표적 사냥

#### 약점

- *대규모 마법 결여* — 9써클급 마법사 *0명* (영웅 시대 데미안 스펠른 사후 단절). 적의 운석강하에 *무력*
- *용병 의존 — 자금 끊기면 즉시 와해*

### 2.7 국립상아탑 — *비무장 / 학문 가호*

- *공식 군대 없음*. 헤븐즈 킹덤 군 일부가 *경비 임무*
- *학문 가치를 인정한 모든 왕국* 이 *상아탑 불가침 합의*
- 위협 시 *학장 라모르* 가 *모든 학파 동시 영창* 으로 *결계 농성* 가능 — 가설

### 2.8 칠흑숲 변경 엘프 — *부족 산발 게릴라*

- 통일 군대 없음. *부족별 사냥꾼·전사 100~300명*
- *침묵의 매복* — 일레인 시대 *오우거 환각* 의 원전. 적이 *몇 명인지조차 모름*
- 다크엘프 침공 시 *전면 봉기* 가능 — 본작 후반부 카드

## 3. 공성전 — 세부 사양

### 공성 측 (Attacker)

#### 도구

- **공성 망치 (Battering Ram)** — 마도공학 화기 결합 시 *3배 가속*
- **공성탑 (Siege Tower)** — 마법 결계 무효화 룬 부착
- **마법 포격** — 화염구 / 운석강하 / 압축 폭발
- **마계 균열 임시 소환** — 금혈 마녀 전용. *극단 카드*

#### 전술

- **결계 분석** — 결계 약점 (모서리·이음매) 정밀 공략
- **보급선 차단** — 농성 측이 결국 *식량 부족으로 항복*
- **사절 위장 침투** — 평화 사절단으로 들어가 *내부 파괴*

### 농성 측 (Defender)

#### 도구

- **다중 결계** — 7중까지 가능 (카르샤 세이렌)
- **성벽 마법 함정** — 접근자에게 *순간 광역 마법*
- **마도공학 자동 사격탑** — 화기로 외벽 방어. 데미안 미라돈 정통
- **신성 결계** — 마계·언데드 절대 방어. 카르샤 신성검단

#### 전술

- **지속력** — 식량 비축 + 신성 치유로 *수개월 농성*
- **밤 침투 격퇴** — 적 야간 공격에 대비. 환각계 마녀 협력 시 *적군이 자기 진영을 공격*
- **마지막 카드 — *균열의 빛 즉발판***: 카르샤만 보유

### 본작 4-2 세이렌 함락 분석

- 카르샤 농성 = 7중 결계 + 신성검단 + 신성 치유
- 카엘 측: 단신 9써클급 화력 + 라이세린 보급 + 정규군 5,000명
- 결과: **7일 함락** — 결계는 카엘 단신 광역 마법에 3일 만에 분쇄. 신성검단은 *근접전* 으로 4일 더 버팀. *균열의 빛* 카드는 알레이언 본인 보존을 위해 미사용 → 옥좌의 방까지 끌어들임

## 4. 국지전·매복 — 세부 전술

### 매복 (Ambush)

- **환각 매복** — 청혈 마녀 + 엘프 협력
- **독 통로 매복** — *바람 + 독* 잔혹 마법
- **점멸 매복** — 데미안 화기병이 *점멸 양피지* 로 적 진군로에 *순간 출현*
- **소드 엠퍼러 1인 매복** — 역사상 가장 효과적. *수백 명 적군이 1인의 검에 무너짐*

### 정찰

- **점성 정찰** — 자혈 마녀가 *미래 예지* 로 적 위치 파악
- **환각 정찰** — 청혈 마녀가 *적 사령관의 꿈* 침투
- **마도공학 정찰** — 데미안의 *마나 감지 장치* 활용

## 5. 본작 4-1~4-3 전쟁 묘사 응용

### 4-1 — 세번 강 회전

- 카엘 = 단신 *시니어 셉터 100명 분* 의 화력
- 카르샤 측 = 정규 보병 3,000명 + 신성검단 일부
- 핵심 묘사: **광역 *역엔트로피 폭압*** = 압축 화염구 + 중력 응용의 9써클급 응용
- 적군 1열 1,400명 *형태가 무너진 채* 사망 = *마나 회로가 즉시 해체* 된 결과 (앤트로퍼 효과)
- 적군이 *비명도 분노도 없이* 죽는 이유 = *형태 해체는 신경 신호 전달 자체를 차단* — 통증 인식 전에 사망

### 4-2 — 세이렌 함락

- 결계 분쇄: *카엘 단신 광역 마법 vs 7중 결계*. 3일 만에 결계 분쇄
- 함락: 시민 사절단의 *항복 화환* 을 *폭발 매개* 로 오인 → 어린아이 화환잡이 사망 = *프롤로그 단서 #1 회수*

### 4-3 — 옥좌의 방 진입

- 신성검단 호위 = 마지막 30명. *근접전 정통* 으로 카엘에 응전
- 카엘은 *마법사 vs 기사 근접 절대 불리* 상태였으나 — 앤트로퍼 이득의 *광역 즉발 마법* 으로 우위 유지
- 카일렌 베르하르트 경 = 신성검 부단장. *왕을 회피시키기 위한 방어 진형* — 카엘은 *왕을 미끼로 함정* 으로 오해
- 회랑의 *도망치소서* 비명을 카엘은 *주문 영창* 으로 들음 = *프롤로그 단서 #2 회수*

## 6. 본작 활용 — 전쟁 묘사 5원칙

1. **광역 학살 묘사 시 *마나 메커니즘 명시*** — *형태가 무너진다*, *마나 회로가 해체된다* 등 물리적 디테일
2. **마법사 vs 기사 거리 상성 의식적 사용** — 카엘의 *광역* 우위와 *근접* 약점의 일관성
3. **국가별 병법 특색을 *행동* 으로 표현** — 카르샤 = 결계+치유 / 헤븐즈 = 결계+화력 / 마녀령 = 환각+매복
4. **공성전 묘사 시 *기간 명시*** — 4-2 *7일* 의 무게
5. **잔혹 마법 사용 시 *카엘 본능적 거부*** — 1회 묘사 후 *그는 그 마법을 쓰지 않았다* 명시
