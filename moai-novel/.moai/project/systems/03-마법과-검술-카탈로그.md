본 문서의 목적 = *전투 시나리오의 다양성과 개연성 확보*. 등급 체계는 [`./military-system.md`](./military-system.md). 화폐·보석 정통은 [`../world/07-상업과-경제.md`](../world/07-상업과-경제.md).

## 1. 마법 시전속도 등급 정통

본작 정통상 *마법의 *써클 단계*가 *시전속도*를 결정한다. 같은 마법사도 *익숙한 마법은 단축·익숙하지 않은 마법은 느림*.

| 써클 단계 | 일반 시전 시간 | 본작 카엘 적용 (Ch1 P5 시점) |
|----------|---------------|------------------------------|
| 1써클 | 즉발~0.5초 | 즉발 (반복 시전 매개) |
| 2써클 | 0.5~1초 | 0.5초 이내 |
| 3써클 | 1초 이내 | **약 1초 이내** + *동시 시전·압축 응용 가능 (변칙 마검사 매개)* |
| 4써클 | 2~3초 | **약 2초** + *익숙·반복 가능* |
| 5써클 | 5~10초 | **수초 (대개 5~7초) — 쓰기 어렵지만 가능. 일부 익숙한 5써클만 안정 운용** |
| 6써클 | 10~20초 | (카엘 시전 불가 — Ch2 마력→내공으로 변환 후 시도) |
| 7써클 | 15~30초 | (대륙 정규 7써클 대마법사 영역) |
| 8써클 | 30~60초 | (호스반 8써클 시전 정점) |
| 9써클 | 60~720초 (8~12분 본격 광역) | (칼란 베르냐스 9써클 정점) |

### 마검사·앤트로퍼 변종의 *시전 단축 정통*

- **변칙 마검사** (카엘 Ch1 시기) = *3써클 이내 마법을 *압축 시전·동시 시전·검기 결합 매개*로 *1초 이내 다발 발사*
- **앤트로퍼 특전 마기** (카엘 Ch1 P5 직후) = *체외 마법 시전 외에 *전신 마기 회로*를 *체내에 흘려 신체 가속·내구·공격력 강화*. *마법 시전 자체는 단축 불가하나 *육체 보조의 즉발 효과*
- **공허검 호흡** (카엘 Ch2 P3 이후) = *마나 운용 안정성 향상 → 5써클 시전 시간 7~10초 → 5~7초로 단축 가능*

### 신성 마법의 *시전속도 특화*

- 신성 마법은 *써클 등급이 아닌 *신성 위계*로 측정 (대주교·주교·신부·수도사)
- *신성 마법 = 일반 마법 대비 시전시간 *1.5~2배* — 의식·기도 매개가 필요*
- 그러나 *위력 = 광역 회복·축복·결계 방어*로 *일반 공격 마법보다 *전장 지속성·생존성 우위*
- **본작 정통 — 카르샤 신성제국 정통조차 *시전속도 약점*이 본질**. *알레이언 6세 같은 신성검 운용자도 *신성 마법은 시전 시간 매개로 *전투 중 운용이 매우 제한적*

## 2. 마법 카탈로그 (35종)

### 화염 계열 (5종)

1. **화염구 (Fireball)** — 2써클·1초·5~10m 사정·중급 폭압. 카엘 익숙
2. **광역 화염 폭압 (Flame Burst)** — 4써클·2초·반경 10~15m·표면 일시 가열. 카엘 익숙. *Ch1 P5 옥좌의 방 1합 시전*
3. **화염 화살 (Fire Arrow)** — 2써클·즉발·다발 발사 (3~5발). 카엘 반복 가능
4. **화염 봉인 (Flame Bind)** — 3써클·1초·적 1체 1~2초 동작 차단
5. **별빛 강하 (Star Fall)** — 7써클·30초·광역 화염 운석 강하. 대마법사급 (행커만의 칠흑성 전투 시전)

### 얼음 계열 (5종)

6. **얼음 송곳 (Ice Spike)** — 2써클·1~2초·반복 발사. 칼란 베르냐스 주력
7. **대형 얼음창 (Giant Ice Lance)** — 5써클·5~8초·반경 30m 직격·보병 100~200명 절멸. 칼란 정점
8. **동결 폭발 (Frost Burst)** — 4써클·2초·반경 5~8m·움직임 일시 마비
9. **얼음 결계 (Ice Wall)** — 3써클·1초·단발 방어막 (1~2회 직격 견딤)
10. **빙판 보법 (Ice Step)** — 1써클·즉발·발바닥 빙판 형성·변칙 보법 매개. 카엘 마검사 비기

### 번개 계열 (4종)

11. **번개 연쇄 (Chain Lightning)** — 3써클·1초·3~5체 연쇄 전기 직격. 카엘 익숙·*Ch1 P5 옥좌의 방 1합 카본코어 매개 시전*
12. **번개 화살 (Lightning Arrow)** — 2써클·즉발·1체 직격
13. **전기 충격 (Static Shock)** — 1써클·즉발·접촉 전기 마비. 카엘 마검사 비기 (맞댄 검 전기 충격)
14. **천둥 폭압 (Thunder Burst)** — 5써클·5초·광역 음파 + 전기 + 마비

### 바람 계열 (5종)

15. **바람 칼날 (Wind Blade)** — 2써클·즉발·중거리 베기. 카엘 익숙
16. **바람 손날 (Wind Hand)** — 3써클·1초·근접 베기·검과 합성 가능. 카엘 마검사 비기
17. **바람 폭압 (Wind Burst)** — 4써클·2초·반경 10m 푸시·물체 날려버림
18. **바람 도약 (Wind Leap)** — 2써클·즉발·발바닥 폭발 + 3~5m 도약. 카엘 비기
19. **바람 통로 (Wind Channel)** — 5써클·7초·일직선 공기 흐름 매개 (독·연기·미세물 운반)

### 흙·대지 계열 (4종)

20. **흙 벽 (Earth Wall)** — 3써클·1초·일시 방벽 (1~2회 견딤)
21. **흙 결박 (Earth Bind)** — 3써클·1초·발 결착·이동 차단
22. **지면 진동 (Tremor)** — 5써클·5초·반경 15m 진동·기동 차단. 칼란 정점
23. **진흙 결착 (Mud Hold)** — 2써클·1초·발·다리 결착

### 물 계열 (3종)

24. **물 화살 (Water Arrow)** — 2써클·즉발·1체 직격
25. **물 결계 (Water Veil)** — 3써클·1초·일시 보호막 + 화염 약화
26. **얼음 결착 (Ice Lock)** — 4써클·2초·적 1체 1~2초 결착

### 비전·고급 (5종)

27. **마나창 (Mana Spear)** — 4써클·2~3초·관통 위력·검과 합성 가능. *Ch1 P5 옥좌의 방 5합 시전*
28. **마나검 (Mana Blade)** — 4써클·2초·임시 검 빚기·일격 후 소실. *Ch1 P5 옥좌의 방 2합*
29. **광역 마나 파동 (Mana Pulse)** — 5써클·5초·자기 회로 직접 격발·광역 파동·자해 위험. *Ch1 P5 옥좌의 방 3합*
30. **동시성 제어 (Concurrent Cast)** — 5써클 매개·아티팩트 보조·*카엘 비기*. *3~4개 마법을 순간 압축 시전 → 1발 광역 효과로 외관 위력 7~8써클*
31. **마기 응축 검 (Dark Mana Sword)** — 4써클·2초·빛이 아닌 어둠에 가까운 검신·결계 분쇄 매개. *Ch1 P5 옥좌의 방 2합*

### 결계·방어 (3종)

32. **마법진 결계 (Magic Circle Shield)** — 3~5써클·시전 시간 매개·정통 방어막
33. **광역 차단 결계 (Barrier Field)** — 5써클·5~10초·반경 20m 마법 차단
34. **결계 분쇄 폭열탄 (Barrier Breaker)** — 6써클 매개·아티팩트 보조·*호스반 비기*

### 신성·금기 (참조)

35. **신성 마법 (Holy Magic)**·**룬 결각**·**아티팩트화**·**엔트로피 역연산**·**마계 균열 시전** — 별도 분류. 시전속도·위계·금기 정통은 *각 도메인 문서 참조*

## 4. 카엘 Ch1 P5 시점 전투 정통 본격 정합

### 마법 운용 (Ch1 P5 시점)

| 단계 | 시전 시간 | 운용 상태 |
|------|----------|----------|
| 1~2써클 | 즉발~1초 | 익숙·반복 가능 |
| 3써클 | 1초 이내 | **익숙 + 동시 시전·압축 응용 가능 (변칙 마검사 매개)** |
| 4써클 | 약 2초 | **익숙·반복 가능** |
| 5써클 | 수초 (5~7초) | **일부 익숙한 5써클만 안정 운용. 쓰기 어려우나 가능** |
| 6써클 이상 | 불가 | 단독 시전 불가. *호스반 합작 매개로 외관 7~8써클 출력* |

### 마검사 비기 (Ch1 P5 시점)

- **앤트로퍼 특전 마기 전신 운용** — *전신에 마기 흘림으로 *신체 가속·내구력·공격력 강화*
- **마나검기** (검기 아님 — 마기로 검 두름)
- **마법 + 검 합성**: 바람 손날·검 막은 화염·맞댄 검 전기 충격·빙판 보법·바람 도약
- **동시성 제어 아티팩트** = *3~4개 마법 압축 시전 → 외관 위력 7~8써클*
- **자기 회로 직접 격발** = 5써클 광역 마나 파동. *자해 위험 (Ch1 P5 4합)*
- **심장 회로 직접 격발** = *본래 한 번 쓰면 다음 한 시간을 살 수 없는 비기*

### 정의롭게 싸우는 믿음 (Ch1 P5 전까지)

- 카엘은 *엔트로퍼 의식 침해로 심각한 사고 저하*. *라이세린의 조작 — 카르샤가 마계 차원의 균열을 획책한다*는 정보를 *완전히 믿고* *정의롭게 싸움*
- 의식 침해 = *비뚤린 시야*. *적의 평화 사절단을 *함정으로 해석. 항복 사절을 *예방 섬멸*. 모든 적대적 해석이 *자기 안에서 정의로움으로 합리화*
- **Ch1 P5 4-4 *왕의 한마디 + 자기 희생*이 *이 사고 저하에 *첫 본격 균열을 낸다***. *5합 결투의 1·2·3·4합 = 정의의 도구 / 5합 = 균열의 빛*

## 5. 알레이언 6세 능력 정통 (사용자 너프 정합)

### 본래 등급

- **5써클 신성마법사** (대륙 신성 위계상 *대주교 직전급 + 왕가 비전 매개*)
- **소드 마스터 *바로 전단계* = 익스퍼트 정점**
- 카르샤 신성제국 = *무력 약함·주변국 의지 정통*. *왕가 본질 = 신성 권위 + 정치 + 종교 동시 정점 — 군 무력의 정점은 아님*

### 본작 Ch1 P5 시점 — 추가 너프 (호스반 강 독살 매개)

- **카르샤 광범위 중독 + 알레이언 자기 해독 거부의 결과**:
  - *전투력 = 익스퍼트 정점의 *약 절반 (소드 마스터 레벨의 반)*
  - *검기 운용 제한적* — 일반 익스퍼트 검기보다 *유지 시간 짧음·밀도 낮음*
  - *신성 마법 시전 = 시전 시간 매개로 *전투 중 운용 매우 제한적*. *5합 결투 중 사용 가능한 신성 마법 = 1~2회 부분 시전 정도*
  - *반응 속도·신체 능력 = 정상의 60~70%*

### 5합 결투의 본질

- *알레이언이 정상 상태 + 신성 마법 자유 운용이었다면 카엘은 절대 이길 수 없는 상대*
- *그러나 독 중독 + 검기 운용 제한 + 신성 마법 시전 시간 매개로 *알레이언의 본격 정점 시전은 *5합 결투 중 거의 불가능*
- **+ 알레이언 본인의 *자기 죽음 결단*** — *마지막 5합에서 *검을 거꾸로 바닥에 박고 두 손을 가슴에 모음 = 자기 희생의 정통*. *알레이언이 *자기를 막지 않은 결과로 카엘의 마나창 직격*
- 카엘 = *4써클 광역 화염 폭압 + 3써클 번개 연쇄 + 4써클 마나검 + 5써클 자기 회로 격발 + 4써클 마나창의 5합 누적 시전*. *알레이언의 너프 + 자기 희생 매개로 *카엘이 가까스로 1합 직격*

### 알레이언의 신성 마법 단편 (5합 중 사용)

- **판결의 가호 (Verdict's Grace)** — *신성 5써클급 부분 시전*. *5합 중 3합 시점에 *짧은 시전 매개로 *카엘 광역 파동을 *분산·흡수*. 본격 위력은 *독 중독으로 50% 출력*. *시전 시간 부족으로 광역 반격 불가 — 흡수·분산만 가능*
- **신성 결계 접기** (1합) — *익스퍼트 검기 + 부분 신성 결계 매개*로 카엘의 폭압을 *부분 반사*. 본격 신성 결계는 시전 시간 매개로 *접기 정도만 가능*
