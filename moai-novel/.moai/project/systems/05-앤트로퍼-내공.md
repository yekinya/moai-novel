# 앤트로퍼 특전 — 마력 → 내공 운용 (Anti-Entroper Internal Energy)

> 본 문서는 *카엘이 두 팔 상실 + 마법 시전 완전 봉인 상태에서 *마력 자체를 내공으로 *변환·운용하는 특이 신체 정통* 의 사양서.
>
> 본 정통은 *앤트로퍼 (특히 강 발현) 만 *생리적으로 가능* — *일반 마법사·기사·마검사는 *원리상 불가*. 본작 *카엘의 Chapter 2~4 검술 진화의 *내적 동력원*.

## 0. 한 줄 정의

> *체외 마법 시전 봉인 상태에서 *체내에 잔존한 마나를 *근육·골격·신경·호흡 회로의 *동력으로 변환해 *내공 (內功) 으로 활용* 하는 정통. 마법 시전과 *근본적으로 다른 매커니즘*.

## 0a. 앤트로퍼 특전 — 점진적 신체·마법 강화 정통 ★

본 정통 = *앤트로퍼 (특히 강 발현)의 *생리적 자동 효과*. *Ch2 P1 마력→내공 발견과 별개로 *Ch1 시기부터 카엘 본인 무의식적으로 잠복·점진 강화*. *마법 시전 봉인 이후 매커니즘 본격 가시화*.

### 두 축의 점진 강화

| 축 | 강화 항목 | 정통 |
|----|-----------|------|
| **체 (體)** | *힘 · 근력 · 체력 · 민첩* 모두 *시간에 비례하여 점차 강해짐* | 일반 인간 대비 약 1.3배 → 5년 후 1.5배 → 10년 후 1.8~2배 |
| **마법 (魔)** | *시전속도 · 마나 감응 · 회복속도* 모두 *시간에 비례하여 점차 강해짐* | 같은 써클 마법도 *시전 시간 단축 + 응축 효율 + 회복 사이클 단축* |

*카엘 본인은 *Ch1 시기 이를 자기 *재능*이나 *부지런함의 결과*로 오해. *Ch2 P5 세이른 진단 + Ch2 P6 마르덴 정통 + Ch2 P8 라카르 9세 리셋 진실*을 거치며 *앤트로퍼 특전의 본질* 자각.

### 원리 — 금제가 흘리는 *마기 (魔氣) 일주천*

본 정통의 핵심 원리:

1. **앤트로퍼 금제가 흘리는 마기 (魔氣 = 마나 기운)** — *금제는 *외부 봉인이 아니라 *체내 마기의 지속적 미세 누출 회로*. 이 *마기가 *전신을 자연 일주천 (一週天 = 한 바퀴 순환)*
2. **자연 일주천 = 마법 시전 없이도 전신에 마나가 *지속적으로 흐름***
3. **신체 효율의 상당한 상승** — 마나가 *근육·뼈·신경·내장·감각의 *구석구석에 흐르면서 *물리적 활동의 *효율을 본질적으로 향상*. 일반인 대비 *회복·반응·인내·힘의 4중 우위*
4. **마나·체력 빠른 회복** — *체내 마기가 *상시 순환하므로 *부상·피로 회복이 일반인의 2~3배*. *Ch2 P1 카엘이 두 팔 절단·척추 손상·두부 충격에도 *3~5일 만에 도주 가능했던 결정적 매개*

### 9써클 *온몸 마나 감응*의 *상시 훈련 효과* 동등 정통 ★

본 정통의 *대륙적 의의*:

- *9써클 대마도사 칼란 베르냐스가 *60년 수련 끝 도달한 *온몸 마나 감응 정통* = *마나 회로의 체내 전체 확장*
- **앤트로퍼 강 발현자는 *그 정통을 *생리적 자동 작동으로 상시 진행*** — *수련 없이도 *9써클 온몸 마나 감응의 본질적 효과를 매일 누적*
- *결과 — 앤트로퍼는 *마법 봉인 상태에서도 *내공 운용 + 신체 강화 + 회복 가속의 *3중 효과로 *대마법사 수련 60년을 *생리로 단축*
- **다만 본질적 한계**: *체외 출수 통로 0의 결과로 *마법 시전 자체는 불가*. *대마법사 위력 = 0 / 신체·내공 효율 = 9써클급*의 *극단적 비대칭 정통*

### 금제 약화 → 특전 효과 동반 약화

- 본 정통은 *앤트로퍼 금제에 *연동된 정통*. *금제가 깊을수록 특전 강함 + 금제가 약해지면 특전도 약함*
- 본작 시점 카엘 = *Ch2 P1 잔향 50% 회복 정착 = 금제 50% 잔존*. *특전 효과도 *최대치의 약 절반 수준*
- *Ch3+ 카엘이 *자기 본성 본격 자각 + 금제 추가 약화 시 *특전도 일부 약화*. 그러나 *대신 본성의 결단·자기 정통의 명료성이 강화되는 *대가성 거래*
- **Ch4+ 자결 결단 정통**: *금제 완전 해체 시점 = 특전 완전 소멸 + 본성 완전 회복*. *카엘이 자기 죽음을 *선택할 수 있는 결정적 매개*

### 극의 (極意) — 발·등·엉덩이·전신 시전 정통 ★

본 정통의 *최정점 발현*. *Ch3+ 카엘의 본격 진화 곡선 시드*.

- **마법 시전의 극의** — *손에 마법진을 그리는 정통을 *완전히 초월*. *전신의 *어느 부위로도 마법 시전 가능*:
  - *발바닥에서 광역 폭압*
  - *등에서 화염 폭발 (자기 후방 보호)*
  - *엉덩이에서 바람 도약 폭발 (자기 추진)*
  - *어깨·가슴·이마 등에서 동시 다발 시전*
- **검기 시전의 극의** — *검 자루를 잡은 손에서만 검기를 빚는 정통을 *초월*. *전신의 *어느 부위에서도 검기 발현 가능*:

  - *등의 검기 — 자기 후방 동시 방어*

본작 적용 시점 = *Ch3 중후반 ~ Ch4 시점*. *Ch2 시기는 *기초 일주천 + 4중 회로 활성화*까지만.

## 1. 작동 조건 — *3중 필수*

### 1.1 *마법 시전 봉인* — 매커니즘 자체의 조건
- 카엘 = *두 팔 상실 → 마법 회로의 *외부 출수 통로 0*. 외부 마법 시전 불가
- 다만 *마나 자체는 체내에 *정상적으로 누적·회복*. *체내 마나량은 5써클 + 4중 효율 1.5배 수치 유지*
- **마법 시전 출수 통로 봉인 + 체내 마나 잉여 = 내공 변환 진입 조건**

### 1.2 *앤트로퍼 강 발현 본능* — 종족적 특화
- 앤트로퍼의 *역마나 흡수 본능* 이 *체내 마나의 *흐름 재배치* 를 *생리적으로 자동화*
- 일반 마법사·기사는 *체내 마나가 *근육·신경에 *섞이지 않음 — 마법 회로와 신체 회로가 *분리됨*
- *앤트로퍼는 *마법 회로 = 신체 회로 일부* — *원래 살의·역마나 흡수의 *신체 매개로 통합되어 있음*. 마법 시전 봉인 시 *통합된 회로가 *내공 운용으로 *자연 변환*

### 1.3 *극한 훈련 매개* — 의지적 매핑
- *자동으로 작동하지 않음*. *카엘의 의식적 훈련 + 의수 검술 시도의 *반복 실패 + 재시도* 매개로 *체득*
- 훈련 시기 = Ch2 P1 후반 ~ P2 (강철 의수 검술 훈련 + 마력→내공 운용 발견)
- *기존 마법사의 *체외 시전 정통과 *근본 다른 매핑 학습* — 카엘 본인도 *왜 이게 되는지 모름 — 단지 *몸이 이렇게 하라고 부른다*

## 2. 운용 메커니즘

### 2.1 내공의 4중 출수

체내 마나가 *내공으로 변환된 후 *4가지 회로로 출수*:

| 회로 | 효과 | 검술 응용 |
|------|------|---------|
| **근육 강화** | 일시적 *물리적 힘 1.3~1.5배* | 의수 검의 *물리 위력 보강* |
| **신경 가속** | *반응 속도 1.3~1.5배* | *공허검 3박자 호흡의 *멈춤 1박을 *0.3박으로 단축* |
| **호흡 안정** | *극한 상황에서 호흡 흐트러지지 않음* | *공허검 호흡 정통과 융합 — *비움 상태의 *지속성 강화* |
| **감각 예민화** | *주변 마나 흐름·살의·살기 감지력 2배* | *상대의 *공격 의도 예측·*검기 비키기의 *직관적 매개* |

### 2.2 한계

- **순간 위력 < 정통 마법사** — 내공은 *지속 화력 + 균등 효율*. *일격의 광역 위력은 *체외 마법보다 낮음*. 광역 학살·소드 엠퍼러 광역 검기 등 *대규모 공격 불가*
- **장기 운용은 신체 부담** — 1회 *5~10분 운용 후 *체내 마나 회복 1~2시간*. *장기전 시 카엘 패배 패턴*
- **잠재 정통 부분만 활성화** — *내공 자체의 *상위 단계는 *공허검 + 의수 4단계 통합 후* 본격 활성화. Ch2 P1~3 단계는 *기초 운용만*

## 4. 일반 마법사·기사·마검사가 *못 따라하는 이유*

### 4.1 마법사 — *체내 회로의 분리*
- 정통 마법사는 *마법 회로 = 외부 시전 전용. 신체 회로 = 일상 운동 전용. 둘 분리*
- *마법 시전 봉인 시 (예: 마나석 부재·신체 일부 결손) 마법 회로 자체가 *비활성화*. *체내 마나가 *근육·신경에 *섞일 매개 없음*
- *내공 변환 시도 가능하나 *효율 0* — *생리적 매개 부재*

### 4.2 기사 — *마나량 자체 부족*
- 정통 기사는 *체내 마나 = *검기 두르기용 소량*. *내공 변환에 필요한 *체내 잉여 마나 0*
- *소드 엠퍼러급* 은 *내공 정통 *유사 효과 발현 가능* (별도 정통 — *기 운용*). 단 *앤트로퍼 내공과는 *원리 다름*

### 4.3 마검사 — *마법사의 변종, 회로는 동일*
- 마검사도 *마법 회로 = 외부 시전 전용*. *카엘 = 앤트로퍼 마검사의 *특이 정통*. 일반 마검사 = *카엘처럼 못 함*

### 4.4 **앤트로퍼 약 발현·미발현 = *제한적 가능***
- 약 발현 캐리어 (알린·이렐라 후예 일부) = *내공 변환 *부분 가능*. 그러나 *체내 마나량 자체가 적어 *실용 효과 미미*
- 강 발현 (카엘) 만 *실용적 수준 도달 가능* — *본작 정통의 카엘 고유성*
