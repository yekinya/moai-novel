# 신성마법 체계

> 르세프 계열 마나를 *정렬·결박· 봉인* 하는 기법군. 본작에서 *카르샤 왕가의 균열의 빛* 이 이 체계의 정점이다.

상위 체계: [`magic-system.md`](./magic-system.md) 6.성직계
법칙 배경: [`../world/pantheon.md`](../world/pantheon.md) 르세프 계열
8왕국 원류: [`../world/eight-kingdoms-ancient.md`](../world/eight-kingdoms-ancient.md) #4 루멘 신성제국

## 1. 기본 원리

신성마법은 *외부에서 끌어오는 신의 힘* 이 아니다. **르세프 계열 마나(=결합·구조화 방향성의 마나)** 를 *시전자의 신앙심을 매개로 정렬* 하는 기법이다.

- 매개 = *신앙심* 의 강도. 신앙심이 약하면 시전 자체가 불가능.
- 시전자 본인의 마나 회로보다 *주변 르세프 마나* 를 정렬해 끌어와 사용 — 그래서 *자기 마나 고갈은 적지만 환경 의존도가 높다*.
- 무스페름(지하·어둠) 근처에서는 거의 작동하지 않는다. 마계 균열 지역에서도 마찬가지.
- *신좌침묵 전쟁* 때 신성마법이 평소만큼 작동하지 않은 이유 — 마나 평형 자체가 흔들리면서 *르세프 정렬* 이 불가능했기 때문.

## 2. 영창 구조

신성마법은 다른 마법과 달리 *영창 자체가 마법의 본체* 다. 룬·스펠 같은 매개 단축이 거의 안 통한다.

### 영창의 3요소

1. **호명(呼名)**: 해당 르세프 하위 법칙의 이름을 부른다 (예: *엘라*, *에필링*, *잉그리드*).
2. **귀의(歸依)**: 시전자가 자신을 *법칙의 매개* 로 선언한다. 이 단계에서 *자기 마나 회로 일부* 를 *법칙에 봉헌* 한다.
3. **발현(發現)**: 정렬된 마나를 *효과로 외화* 한다.

### 시전 시간

- 1써클 (광명·축복 등): 영창 1~2초.
- 5써클 (광역 결계·중상 치유): 영창 30초.
- 9써클 (균열 봉인·대규모 정화): 영창 *수 분 ~ 수십 분*.
- **균열의 빛 (왕가 비전)**: 영창 *수 시간*. 전투 중 즉발 불가 — 미리 *자세를 잡고 의식을 진행* 한 상태에서만 발동 가능.

### 자세

신성마법은 *자세* 가 영창의 일부다.

- **두 손 모음 자세**: 카르샤 왕가의 *손 모음 인사* 의 원형. 일반 신성마법의 *귀의 단계 표준 자세*.
- **양팔 벌림**: 광역 결계의 시작 자세.
- **무릎 꿇음 + 두 손 모음**: 대규모 봉인 의식의 자세. 알레이언 6세가 프롤로그 4합에서 취한 자세.

## 3. 등급 체계

신성마법은 마법 클래스(써클) 체계와 별개로 *5단계 신앙 등급* 으로 분류된다.

| 등급 | 통칭 | 자격 | 주 마법 |
|------|------|------|--------|
| 1 | 평신도 | 누구나 | 축복, 정화 (소) |
| 2 | 정식 사제 | 5년 수련 | 광명, 치유 (경상), 소결계 |
| 3 | 고위 사제 | 10년 + 9써클 마법사 동급 감응력 | 정화 (대), 치유 (중상), 광역 결계 |
| 4 | 대주교 / 성기사단장 | 30년 + 종교 정치 위계 | 영혼 정화, 대규모 결계, 마계 균열 *지연* |
| 5 | 신탁자 / 왕가 비전 보유자 | 혈통 + 평생 봉헌 | 균열 *봉합 시도*, 균열의 빛, 신성안 (앤트로퍼 식별) |

본작 알레이언 6세 = **5등급 + 카르샤 왕가 비전 보유자**. 대륙 내 동등 등급은 *3~5명* 추정.

## 4. 신성안 (Holy Sight / 神聖眼)

5등급 시전자가 발동 가능한 *심층 투시* 마법.

### 효과

- 시전자의 시야에 *대상의 마나 구조* 가 직접 보인다.
- 일반 마나 감지(셉터 능력)와 달리 *심장에 새겨진 마법적 결박* 까지 식별 가능.
- **앤트로퍼 숙주의 심장에 감긴 *붉은 사슬 7~9마디*** 를 명확히 본다.
- 단점: 발동 중 시전자는 *물리 시야가 잠시 흐려진다*. 전투 중 발동은 큰 위험.

### 본작 사용

- 프롤로그 3합 직후, 알레이언이 *전투 중* 신성안을 발동해 주인공의 *7마디 사슬* 을 확정 식별. 이 직후 *균열의 빛* 으로 결단을 옮겼다.
- 다른 5등급 시전자도 이론상 발동 가능하지만, *전투 중 사용* 한 사례는 알레이언이 유일.

## 5. 균열의 빛 (Rift Light / 龜裂의 光) — 카르샤 왕가 비전

본작 *최고위* 신성마법. 사프란의 7제자 중 *코르넬리스(루멘 → 카르샤 1대 왕)* 가 보존한 비전의 핵심.

### 원리

- 르세프 마나를 *극한까지 정렬* 해 *한 점으로 압축한 빛* 으로 발사.
- 대상 = *앤트로퍼 숙주의 심장 붉은 사슬*.
- 효과: 사슬의 *한 마디에 미세 균열* 을 낸다. *완전 해제는 불가능* — 사슬의 *분비량을 일시 감소* 시키는 효과만.
- 균열은 *영구적* (자연 회복 불가). 단, 미세 마기 분비량은 *시간이 지나면 회복* 한다 — 균열을 통해서도 *대부분의 마기는 계속 분비* 됨.

### 8왕국 봉인 의식의 *절반*

- 사프란의 *원형 의식* 은 **균열의 빛 + 회수 회로** 의 *2단 구조* 였다.
- *균열의 빛* = 사슬에 균열을 낸다 (카르샤 보존).
- *회수 회로* = 균열을 통해 *씨앗(앤트로퍼) 자체를 추출* 한다 (마녀령 흑혈 가문 보존, 그러나 9혈 중 4혈 멸문으로 *불완전 보존*).
- **두 절반이 모이면 *완전 해제* 가 가능**. 카르샤가 7대를 기다린 이유.

### 시전 조건

1. 시전자가 5등급 신성마법 + 카르샤 왕가 *혈통 + 비전 훈련*.
2. 시전 자세 = 무릎 꿇음 + 두 손 모음.
3. 영창 시간 = *최소 3시간*. 단, *생명을 매개로 한 즉발판* 가능 — 시전자의 *모든 마나와 수명을 즉시 봉헌* 하면 *수 초 ~ 수십 초* 로 단축. 대신 시전자는 즉시 *사망 또는 회로 완전 붕괴*.
4. 알레이언 6세가 프롤로그 4합에서 사용한 방식 = **즉발판**. 검을 내려놓고 두 손을 모은 자세가 *시전자의 모든 것을 봉헌하는 자세*.

### 일반 신성마법과의 차이

| 항목 | 일반 신성마법 | 균열의 빛 |
|------|--------------|----------|
| 표적 | 외부 결계·치유·정화 | *내부 결박 구조* |
| 매개 | 신앙심 + 환경 르세프 | *시전자의 생명 + 왕가 혈통* |
| 효과 | 가역 | *비가역* |
| 알려진 시전자 | 다수 | *역사상 4명* (코르넬리스 외 카르샤 2~3대 시도 실패, 알레이언 6세가 *첫 성공*) |

## 6. 르세프 하위 법칙별 신성마법 분류

| 법칙 | 주 효과 | 본작 등장 |
|------|--------|----------|
| 엘라 (조화) | 결계, 균형 회복 | 엘프 사제 (간접) |
| 에필링 (치유) | 상처 치유, 해독, 부활 (실패율 극히 높음) | 일반 치유 사제 |
| 잉그리드 (대지) | 광역 정화, 흙·돌 봉인 | 잉그리드 사원 사제 |
| 아샤스 (영광) | 사기 진작, 신체 강화 | 카랑크/강철제국 검기계와 결합 |
| 이그너스 (순결) | 영혼 정화, 마계 오염 제거 | 루멘 신관 (멸절). 카르샤에 잔존 |
| 로렌 (장미와 피) | 혈맹 결박, 서약 봉인 | 카르샤 왕가 혼인 의례 |
| 저그넛 (정의) | 진실 식별, 거짓 폭로 | 카르샤 왕실 재판 |
| 룔 (환생) | 영혼 보존, 사후 인도 | 모든 장례 의식 |

본작 *카르샤 왕가* 는 주로 **잉그리드 + 이그너스 + 저그넛** 의 3축을 융합한 *루멘 정통* 을 계승. *균열의 빛* 은 4축(+로렌의 혈맹) 모두를 결합한 *최고위 합성 마법*.

## 7. 신성마법의 정치적 위치

- 신성마법은 *환경 의존* + *영창 시간 김* + *전투 즉발 불가* 라는 한계 때문에 *대량 전쟁에서는 약함*.
- 그러나 *왕권 정통성·외교 의례·국가 행사* 에서는 *최고의 권위*.
- 카르샤는 *작은 영토 + 약한 군사력* 임에도 *균열의 빛 보유* 라는 단 하나의 카드로 *7대를 견뎌왔다*.
- 본작 4부에서 카르샤가 망한 이유: *균열의 빛은 앤트로퍼에게만 유효* 하다 — 일반 군사 침공(주인공의 광역 마법)에는 무력했다.

## 8. 본작 활용 체크리스트

- [ ] 프롤로그 3합 신성안 = 알레이언이 *전투 중 발동* 한 *대단한 위험 부담*. 묘사에서 *시야가 잠시 흐려짐* 디테일 누락 금지.
- [ ] 프롤로그 5합 균열의 빛 = *즉발판*. 영창 시간 단축의 대가가 *시전자의 죽음* 임이 후일 4-4에서 명시되어야 함.
- [ ] 4-5 카르샤 평화 협정 작성일 = *균열의 빛 시전 결단 직전*. 알레이언은 *전쟁을 피하려 끝까지 노력했다*.
- [ ] 5-2 카르엔 봉인 문서 = *회수 회로의 절반* 이 마녀령 흑혈 가문에 있음을 명시. 6부 이후 마녀령 진입의 필연성.
- [ ] 손 모음 인사 = *균열의 빛 시전 자세의 민중화*. 카르샤 시민이 *모두 약한 균열의 빛 흉내를 매일 하는* 것과 같음.
