## 1. 사회 — 일상의 마법

### 1.1 도시 인프라

- **마나 가로등** (셀라리아·미라돈) — 광명 마법을 *마나 코어 보관 결정* 에 결박. *야간 도시 조명*
- **마나 펌프** — 우물·하수도. 미라돈 항만이 가장 발달
- **결계 성벽** — 큰 도시의 외벽이 *약한 결계* 자동 발동
- **마나 시계** — 광장 중앙. 매 *각* (시간)마다 종소리 + 빛 신호
- **순간이동 우체통** (미라돈만) — 도시 내 *작은 메시지 전송*. 정해진 우체통 간

### 1.2 의료

- **수녀 치유원** — 모든 도시에 1개 이상. 평민도 *기초 치유* 접근 가능
- **마약·진통제** — 연금계 학파. 양귀비 + 마나 결합 *마비제*
- **마법 산파** — 출산 시 *모자 모두 안전 결계*. 카르샤 신앙권에서 발달
- **잉그리드 사원의 마비술** — 외과 수술 시 *국소 마비*. 마도공학 시술 표준

### 1.3 농업·축산

- **농지 결계** — *해충·짐승 방지* 약한 마법. 평민 가격
- **마나 비료** — 토양 결박 강화. 데미안 농가 표준
- **물 정제 마법** — 변경 부족이 *수녀 1명 고용* 으로 우물물 정화

### 1.4 가정용

- **광명 등불** — 양초 대체. 도시 평민 가격
- **결계 자물쇠** — 귀족 저택 표준
- **마나 화로** — 부유층. *불 없이* 음식 가열
- **결계 항아리** — 음식 *수개월 보관*. 미라돈 무역품

## 2. 문화·예술·연예

### 2.1 음악

- **음률계 마법** — 소리에 *마나 흐름 결합*. 청중에게 *감정 증폭·기억 자극·치유 효과*
- **마나 악기** — 류트·하프에 마나 결박. *음 1개로 광역 효과*
- **노래 의식** — 카르샤 손 모음 의례, 엘프 무녀의 침묵 노래

### 2.2 환술 공연 — *환각계 셉터의 합법 응용*

- 도시 광장에서 *공개 환각 쇼* — 청중에게 *집단 환각* 시전
- *바이서스 카멜라 4세 즉위 기념* 환술 공연 = 행커만의 시연
- 일부는 *현실 분간 어려운 정밀 환각* — 윤리 논쟁 대상

### 2.3 그림·조각

- **마나 안료** — 그림이 *살아 움직이는 듯한* 효과. 단 보관·복원 불가 (마나 휘발)
- **잉그리드 조각술** — 돌에 *마나를 결박* 해 *반(半)생명* 부여. 잉그리드 사원의 비전
- **마법 초상화** — 인물을 그리면 *그 인물의 마나 잔향* 까지 담김. 사후 *추모 의식* 도구

### 2.4 문학·기록

- **마나 잉크** — 거짓을 적으면 *글자가 사라짐*. 카르샤 신성 재판에서 사용
- **녹음 결정** — 음성 *수시간 보관*. 데미안 상인 계약 표준
- **자동 필기 깃펜** — 시전자의 사고를 글로 옮김. 학자 표준 도구. *카르엔 사이엔 서재에 다수 보유*

## 3. 스포츠·오락

### 3.1 마법 검투 — *검사형 셉터 정통 종목*

- 도시 경기장에서 정기 시합
- 마법 + 검 결합 자유. 살상 금지 (보호 결계 + 의무 치유)
- 챔피언은 *준왕족 대우*. 데미안 미라돈이 최대 흥행

### 3.2 점성 도박 — *재믹 신앙 정통*

- 데미안 재믹 거래소 = 도박장. 자혈 점성 마녀가 *결과 예언*
- *원칙적으로 점성은 짧은 미래만 보임* — 도박 결과는 *반은 점성, 반은 운*
- 부유층 오락 + 노름 중독자 양산

### 3.3 환술 카드 게임

- 카드 자체가 *환각 매개*. 패가 *공중에 나타남*
- 미라돈 + 셀라리아 귀족 살롱 표준

### 3.4 점멸 경주 — *기록 단위 0.1초 단위*

- 점멸 양피지를 사용한 *단거리 경주*
- 데미안 항만에서 시작. 셀라리아·미라돈 양도시 *프로 경주 리그* 운영
- 점멸 양피지 *제한 수* 으로 우승자 결정 = 마나 + 양피지 효율의 경쟁

### 3.5 오우거 환각 모험 — *엘더스톰 칠흑숲 관광*

- 일반인에게 *오우거 환각* 을 *통제된 강도* 로 체험시키는 *모험 관광*
- 비용 폭등. 부유층 모험 취미

## 4. 정치

### 4.1 마법사의 정치 지위

- **9써클 마스터** = 작은 영지 수준 정치 권력
- **시니어 셉터** = 귀족 동급 외교 영향력
- 마법사가 *모든 왕실 최고 고문직* 보유 (예외: 강철제국)
- **카르엔 사이엔** = 8써클 + 학파 시조 = 헤븐즈 외곽 연구소 비공식 학장

### 4.2 마법 외교

- **균열의 빛** 같은 *왕가 비전 마법* 보유 = *외교 카드*
- 카르샤가 *작고 가난해도 천 년 견딘 이유* = 균열의 빛 카드
- 마법사 *교환* = 평화 협정의 일환. 7-8써클 마법사 1명 = 외교 결혼 동등 가치

### 4.3 마법 재판

- **저그넛 신성마법** — 거짓 발언 시 *혀가 마비*
- **마나 잉크** — 위증 시 *글자가 사라짐*
- **점성 마녀** — 자혈 마녀의 *과거 회상 마법* — 사건 현장 *복원*. 단 마녀는 *왕실 재판 공식 출입 불가* (마녀령 협력 시 *비공식 사용*)

### 4.4 마법 첩보

- **청혈 환각 첩보** — 적국 사령관 *꿈* 침투
- **점멸 첩보** — 데미안 백색 사냥단의 정통
- **음률 도청** — 음률계 셉터가 *수십 미터 떨어진 대화* 청취

## 5. 경제·교역

### 5.1 무역품 — *대륙 표준 마법 상품*

| 상품 | 산지 | 가격 |
|------|------|------|
| 마나 코어 (작은 결정) | 데미안 | 평민 1년 수입 |
| 마나 코어 (큰 결정) | 데미안·드워프 광산 | 9써클 마법사 1년 수입 |
| 점멸 양피지 (1매) | 데미안 | 평민 한 달 수입 |
| 점멸 반지 (1일 한정) | 데미안·헤븐즈 | 평민 평생 수입 |
| 결계 부적 | 카르샤 신관단 | 평민 한 달 수입 |
| 의수 (마도공학) | 그라스·미라돈 | 평민 10년 수입 |
| 마나 펌프 | 미라돈 | 도시 시장 운영 단위 |
| 룬 갑주 | 미라돈·강철제국 | 기사 1대 가문 자산 |
| 마검 (마검사용) | 마녀령·미라돈 | 평민 평생 수입 + |
| 신성검 (루멘브링거급) | 카르샤 (생산 중단) | 왕가 보물 |

### 5.2 화폐

- **금화 (Gold Mark)** — 대륙 표준. 데미안 미라돈 발행
- **은화 (Silver Mark)** — 평민 일상
- **동화 (Copper Coin)** — 농촌·시장
- 환율: 1금 = 10은 = 100동

### 5.3 마도공학 산업

- **데미안 미라돈** = 대륙 1위 마도공학 생산 도시. 5대 길드 운영
- 연 생산: 마나 코어 *수십만 개* / 점멸 양피지 *수백만 매* / 룬 갑주 *수천 벌*
- 길드별 비밀 레시피 보유 — *상업 비밀* 로 보호

### 5.4 마나 농도와 경제

- 도시 인구 +1만 → 엔트로픽 농도 +0.2%
- 미라돈 (95만) = *대륙 농도 최대 기여* — 다음 리셋 가속의 주범
- *카르엔 학파* 가 *경제 성장 자제* 를 비공식 권유했으나 데미안 황실 *무시*

## 6. 외교

### 6.1 외교 관행

- **사절단 구성**: 정사 1 + 부사 1 + 호위 5 + 신관 1 + 기록관 1
- **선물 의례**: 마나 코어 1개 + 자국 특산물 + 점멸 양피지 5매 (긴급 통신용)
- **결투 외교**: 양국 합의 시 *분쟁을 1:1 결투로 해결* — 자주 사용되지 않음

### 6.2 본작 3-1 카르샤 사절단

- 카일렌 베르하르트 경 (정사, 신성검 부단장)
- 베르난 류엔 자작 (부사, 외무관)
- 카르샤 신성검단 호위 5명
- 셀라엔 대주교 직속 신관 1명
- 기록관 1명
- *총 9명* — 표준 사절단 구성

### 6.3 외교 충돌의 본작 원인

- 카르샤의 *손 모음 인사* 를 헤븐즈에서 *주술 자세* 로 인식 (헤븐즈 르세프 본원교의 *이단 분류*)
- 라이세린 공작의 *위조 첩보* 는 외교 의례를 *명분* 으로 활용

## 7. 과학

### 7.1 마나 측정학

- **마나 측정기** — 슐로브스 마레의 발명. 대기 마나 농도 측정
- **엔트로픽 농도 측정기** — 상아탑 비밀 기술. *카르엔 학파* 도 일부 보유
- **세프 입자 분리기** — 데미안 마도공학원. 세프 알갱이 *단위 분석*

### 7.2 마법 화학·연금술

- **연금술 + 마나 결합** — 데미안 정통. 약물·독·촉매 대량 생산
- **마나 변환 합금** — 아다만티움 + 일반 강철 혼합. 데미안 화기의 핵심
- **수은 + 마나 결합** — 마나 코어 제작의 핵심 원료

### 7.3 천문·점성

- **자혈 마녀 점성** = 사실상 *천문학*. 별의 위치 + 마나 흐름 상관관계
- **상아탑 천문관** = 셀라리아 호수 섬. 야간 별 관측 + 마나 농도 변화 기록

### 7.4 *과학적 발견* 의 본질

> *"우리는 마법을 하면서 *과학을 했다*. 다만 우리는 그것을 *마법이라 불렀다*."*
> — 슐로브스 마레, 강의록

- 리투니아인은 *과학 용어* 를 쓰지 않는다 — *마나 입자, 마나 결, 진동, 배열, 흐름, 농도, 잔향* 등 *마법 용어* 만 사용
- 그러나 *실질적으로는 화학·물리·생물학* 에 해당하는 학문이 *마법 학파 안에서* 발달

## 8. 사피엔의 응용 — *통념을 벗어난 학습*

대부분 피조물은 *기초 원리 + 수련* 의 정통을 따르나, 사피엔은 *전승·응용·학습* 으로 *통념을 벗어난 특이 응용성* 을 보인다.

### 사피엔 고유 응용 (재방문)

- **마도공학 화기** (§5.3) — 마법 재능 없는 평민도 사용
- **점멸 양피지·반지** (마법 매개의 *압축 보관*)
- **마법 첩보·암살** (잔혹 응용 — `magic-system.md` §7.4 참조)
- **마법 산업** — 마법을 *상품화*. 미라돈 길드 경제
- **마법 스포츠** — 마법 검투·점멸 경주
- **마법 외교** — 외교 결혼처럼 *마법사 교환*

### 사피엔의 *과잉 응용* 의 대가

- 마도공학 산업화 = 엔트로픽 농도 *3배 가속*
- 마법 무기 평민화 = *전쟁의 일상화*
- 마법 산업 = *마나 결박 누적의 산업 규모 가속*
- **이 모든 응용이 다음 리셋을 *수백 년 앞당기는* 진짜 원인**

## 9. 본작 활용 — 일상 묘사 5원칙

1. **도시 묘사 시 *마나 인프라 1~2개* 자연스럽게 언급** — 마나 가로등 / 결계 성벽 등
2. **인물의 *직업·신분이 마법 응용으로 표현*** — 평민 산파 = 수녀 / 도시 관리 = 마나 시계 관리
3. **카엘이 *마법 응용을 일상으로 사용*** 하는 묘사 — 1~3부에 마나 가로등 통과·점멸 양피지 사용 등 자연스럽게
4. **데미안 미라돈 등장 시 *상업 + 마법 산업* 의 압도적 풍경** — 5대 길드·항만·5개 등대
5. **마법의 *대가 (농도 상승)* 를 가끔 무게감 있게 언급** — *카르엔이 한숨* 같은 작은 디테일로
