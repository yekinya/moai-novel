# 📖 STORY MAP — 리투니아 설계 문서 내비게이션

> **목적**: 글을 쓰거나 세계관을 점검할 때, 링크를 따라가며 *필요한 정보만* 빠르게 도달하기 위한 단일 진입점.
>
> **사용법**: 아래 *진입 동선* 중 하나를 골라 위에서 아래로 클릭만 따라가면 된다.

---

## 🎯 3가지 진입 동선

```
┌──────────────────────────────────────────────────────────────┐
│  A. 처음 보는 사람 — "이 세계가 뭐야?"     → A 동선          │
│  B. 글 쓸 때 — "이 회차의 배경 정보가 필요해" → B 동선        │
│  C. 설정 점검 — "이 설정이 일관성 있나?"   → C 동선          │
└──────────────────────────────────────────────────────────────┘
```

## 🗺 인과 다이어그램 — 한 장으로 보는 본작

```
[태초/거인족]
     │
     ▼
[수만 년 반복된 리셋들]──[지층에 마나 잔향 누적]
     │
     ▼
[산발 현자들의 단편적 시도] ────────────┐
     │                                  │
     ▼                                  │
[CC -15  사프란이 8왕국을 모음]         │
     │                                  │
     ▼                                  │
[CC -3   신좌침묵 전쟁 발발]            │
     │                                  ▼
     ▼                          [드래곤 로드 분파 결성]
[CC 0    8왕 중 5왕 사망 ─ 의식 성공]
     │                                  │
     ▼                                  │
[리투니아 협정 + 앤트로퍼 시스템 + 협정력 시작]
     │
     ▼
[CC 4    사프란 사망 → 7제자 비전 분할]
     │
     ▼
[CC 30   8왕국 분열 → 정치 지도 확정]
     │
     ▼
[CC 30~700  700년 암흑기 + 앤트로퍼 변질 (3차 표적 편향)]
     │
     ▼
[CC 700~1000  300년 급속 문명화 = 영웅 시대 + 마도공학]
     │              │
     │              ▼
     │      [엔트로픽 농도 폭증 = 다음 리셋 임박]
     ▼
[CC 980  16대 앤트로퍼 = 주인공 출생]
     │
     ▼
[CC 999  카르샤-바이서스 외교 파탄]
     │
     ▼
[CC 1000  본작 시작]
     │
     ├─ 1~3부: 변질된 3차 표적 학살 (주인공은 자각 없음)
     ├─ 4부:   2차 표적 (알레이언) 살해 + 균열의 빛 자각
     ├─ 5부:   1차 표적 (이실 = 드래곤 로드 자객) 해주 시도 실패
     ├─ 6부:   잠복·재기·결전 예고
     └─ 중·후반부 (예정):
            마왕 강림 → 진짜 속죄전 → 사프란 의도 복원 →
            드래곤 로드와의 사상 충돌 → 자결 최종 카드 선택지
```
