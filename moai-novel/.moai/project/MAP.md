## 1. 본작 한눈에 보기

### 로그라인
> 마나 역류가 시작된 리투니아 대륙에서, **카엘**(앤트로퍼 소서러)은 자신이 악당과 마계의 창조물을 처단한다고 믿으며 전쟁에 뛰어든다. 그러나 그의 적과 전쟁은 *앤트로퍼의 저주가 만든 오해* 였고, 그는 자신이 인간들을 학살해왔다는 진실을 깨닫는다. 이후 그는 죄책감을 안고 *대현자 사프란의 진짜 의도* 를 이해하며, 이번 생명 초기화를 막기 위해 *실제 마계의 창조물·드래곤·정령왕* 을 죽이는 길을 선택한다.

### 핵심 약속
- 주인공은 **카엘** — 통칭 *앤트로퍼 소서러*.
- 초반 적대 구도는 주인공의 *왜곡된 인식* 에서 비롯된다.
- 30% 지점까지 *권선징악 판타지* 로 읽힌다. 그 후 학살의 진실이 재해석된다.
- 후반 목표: *인간 학살이 아니라 마계의 창조물·고마나 응축체* 사냥.

### 중심 질문
> *오해로 악행을 저지른 자가, 진실을 깨달은 뒤에도 세계를 구할 자격이 있는가?*

## 2. 세계관 핵심 — 마나·리셋·앤트로퍼

### 마나와 엔트로픽 농도

- 세계는 *마나* (학술명: 세프)로 구성된다.
- 모든 생명은 자유 마나를 자신의 육체·영혼·기억·문명에 *결박* 한다.
- 결박이 누적된 정도 = **엔트로픽 농도**.
- 농도가 임계를 넘으면 세계는 *생명을 해체* 해 평형을 회복한다 → **생명 초기화 (리셋)**.
- 리셋의 표면 현상 = *마계 침공* + *언데드 출현* + *마나 역류*.
- 본질 = *자연의 면역반응*. 악의 침공이 아니다.

심화: [`world/mana-and-entropic-density.md`](./world/mana-and-entropic-density.md) · [`world/life-reset-and-demon-war.md`](./world/life-reset-and-demon-war.md)

### 앤트로퍼 시스템

- 본세대 대현자 **사프란** 이 천 년 전 신좌침묵 직후 설계한 *생명 감축 장치*.
- 형태: *씨앗* 이 사피엔 영아에게 깃든다. 외관 식별 불가, 오직 **심장에 감긴 붉은 사슬 7~9마디** 로만 식별.
- 작동: 심장에서 *미세 마기* 가 혈류로 분비 → 뇌의 *해석 회로* 에 개입 → 외부 자극을 일관되게 *적의·기만·위협* 으로 번역.
- 이득: 마나 효율 ↑, 기술 탐구 ↑, 고통 역치 ↑, 살상 직관 ↑.
- 충동: *강해지지 않으면 불안* — 야망·정의감으로 합리화되어 *살인 갈망* 으로 외화.

심화: [`world/great-sage-and-entroper.md`](./world/great-sage-and-entroper.md)

### 사프란의 *진짜 의도* — 변질의 비극

사프란이 의도한 표적 우선순위:

1. **1차** — 드래곤·마왕·정령왕·고룡 등 *고마나 응축 단일 개체* (1체 = 왕국 1~2개 분량 해소)
2. **2차** — 인간 *왕·고위직 중 고응축 개체*
3. **3차** — 일반 인구 (1·2차로 부족할 때만)

8왕국이 분열하면서 후대 앤트로퍼는 *3차 표적 일반 학살* 만 수행하도록 *변질* 됐다. 본작 카엘의 1~3부 학살이 이 변질의 정점.

## 3. 통합 연표 요약 (CC 기준)

> **CC = 협정력 (Concord Calendar)**. 원년 = 신좌침묵 종전 + 8왕 리투니아 협정.
> **현재 = CC 1000**. 일반 시민은 *기념일* 로 알지만 고위 지식인은 *다음 리셋까지의 카운트다운* 으로 읽는다.

| 시기 | 사건 |
|------|------|
| CC -약 50,000 | 거인족 멸종 = 최초 대규모 리셋. 리펠리아 대륙 침몰. |
| CC -50,000 ~ -3,000 | 수십 차례 리셋 반복. 산발 현자 출현하나 *조직적 시도 없음*. |
| CC -3,000 ~ -500 | 일부 지식인이 *지층 마나 잔향* 으로 *과거 리셋* 추정 시작. |
| CC -82 | **사프란** 출생 (상아탑 변경). |
| CC -30 | 사프란이 다음 리셋이 *50년 내* 라는 추산 도달. |
| CC -15 | 사프란 *8왕국 비밀 순방* 시작. |
| CC -3 | 신좌침묵 전쟁 발발. |
| CC 0 | 8왕 연합 의식 → 5왕 사망 / 3왕 생존. **리투니아 협정 체결 = 협정력 원년.** |
| CC 4 | 앤트로퍼 1호 *모나르* 기동 직후 사프란 사망. 7제자 비전 분할. |
| CC 30 | 8왕국 분열 → 현재 정치 지형 골격 형성. |
| CC 30 ~ 700 | **700년 암흑기**. 지식 대부분 분실. 앤트로퍼 시스템 변질 시작. |
| CC ~700 | 옛 사프란 학파 후예가 지층 조사 기록 재발견. |
| CC 700 ~ 1000 | **300년 급속 문명화**. 모티어스·행커만·일레인·스펠른·마레·스틸만 활동. |
| CC ~850 | 행커만(변신 드래곤)의 바이서스 총사령관 시대. **바이서스 카멜라 4세**. |
| CC ~870 | 카르마서스(블랙 드래곤) 미드그레이드 공격 → 일레인 패퇴. |
| CC ~875 | 칠흑성 전투 — 일레인이 행커만의 정체 폭로. |
| CC ~950 | 마도공학 산업화 가속. 엔트로픽 농도 급상승. |
| CC 980 | **카엘** (주인공, 16대 앤트로퍼) 출생 — 노이엔 변경. |
| CC 998 | 카엘 바이서스 진입 (2부 시작). |
| CC 999 가을 | 카르샤-바이서스 외교 파탄 → 선전포고 (3-6). |
| **CC 1000 봄** | **본작 프롤로그 = 옥좌의 방**. |
| CC 1000~1001 | 5~6부 진행. |

## 4. 명명 규칙

### 호칭 체계

| 신분 | 표기 형식 | 예시 |
|------|----------|------|
| 왕가 정식명 | `{성} {이름} {N세}` (성-이름-세) | 카르샤 알레이언 6세, 바이서스 라카르 9세 |
| 왕가 약식 | `{이름} {N세} 폐하` 또는 `{성} 폐하` | 알레이언 6세 폐하, 라카르 폐하 |
| 귀족 | `{성} {이름} {작위}` | 라이세린 데오릭 공작 |
| 평민 영웅 (역사) | `{이름} {성}` (given-first) | 일레인 맥필렌, 데미안 스펠른 |
| 평민 일반 | `{이름}` 만 | 카엘, 사프란, 리엔, 단 |
| 평민이 작위 받음 | `{이름} {출신지명} 경/백작` | 카엘 노이엔 경 |
| 드래곤 (위장) | 가명 사용 — 통상 출신 도시명 surname | 카이엔 오르하임 |
| 학파 후예 | `{이름} {학파 surname}` | 카르엔 사이엔, 세파 류르 |

### 왕조 성씨 원칙

- **성은 보통 왕국 이름** (예: 바이서스, 데미안, 카르샤, 스틸만).
- 예외: 외래 입혼 가문 (마녀령 모르간 가문 등).
- **왕가의 성은 아무나 가질 수 없다** — 분파 인물도 작위 형태로 우회 (예: 스틸만 방계 = 세이른 스틸만).
- 부를 때는 성을 부른다. 성이 길거나 모호하면 *세대 번호 + 폐하*.

### 평민에 surname이 없는 이유

- 농촌·변경 사회는 *성을 가지지 않는다*. 가까운 마을·가족 관계만으로 식별.
- 도시 평민은 직업·출신지를 ad hoc surname처럼 사용 (예: *대장간의 베르*).
- 작위·관직을 받으면 *정식 surname* 부여 — 본인 선택 또는 왕실 지정.

### 왕조별 시조 → 현재까지 핵심 인물

#### 바이서스 (Vyserus) 왕가 — 헤븐즈 킹덤
- **1대 바이서스 카이론 1세** (Kairon I, CC ~5): 알마리스 8왕 *아르카논 4세* 의 외손. 협정 후 새 surname 채택.
- **4대 바이서스 카멜라 4세** (Carmela IV, 여, CC ~840~880): 행커만 시대. 칠흑성 전투 직후 연로하여 양위.
- **9대 바이서스 라카르 9세** (Lakar IX, CC 990~ ): 본작 현 국왕. 50대 후반. 라이세린 공작이 *외교 전권* 을 위임받은 상태.
- 차기 후계: 1왕녀 **바이서스 카리엔** (Karien, 30대 초반).

#### 엘드라이트 (Eldraite) 왕가 — 엘더스톰
- **1대 엘드라이트 일레안 1세** (Ilean I, CC ~30): 벨라리온 멸망 왕 *일레안 7세* 의 사촌 손자. 망명·재건.
- 영웅 시대 인물 **일레인 맥필렌** (Elain MacPhilen)은 엘드라이트 황실이 아닌 *엘더스톰 평민 마법사*. 4대 왕의 군사 고문 역임.
- **5대 엘드라이트 베이렌 5세** (Beiren V, CC 960~): 본작 현 국왕. 60대. 헤븐즈 라카르와는 *천 년 라이벌리* 의 마지막 세대.

#### 스틸만 (Steelman) 황가 — 강철제국
- **1대 스틸만 베른하르트 1세** (Bernhardt I, CC ~50): 베른하르트 강철왕의 차남. 부친 사망 후 즉위.
- 영웅 시대 **아라곤 스틸만** (Aragon Steelman, CC ~800)은 황가 방계 + 영웅. 4대 황제와 사촌.
- **6대 스틸만 그라이언 6세** (Graien VI, CC 970~): 본작 현 황제. 40대. 군사주의자. 카르샤-바이서스 전쟁에 *중립* — 그러나 후일 개입 가능성.
- *세이른 스틸만* (Seirn Steelman) = 황가 방계, 마법 거부로 추방. 본작 6-4 도장의 노 마검사. 60대.

#### 카르샤 (Carsha) 왕가 — 카르샤 왕국 ★ 본작 핵심

## 7. 대현자 사프란과 7제자 계보

- **사프란** = 본세대 대현자. *성을 가지지 않는다* — 상아탑 변경 평민 전통. 본명은 단일 음절 *사프란* 으로 평생 사용. 외관은 평범, 다만 *갈색 눈에 미세한 황금 반점*.
- 8왕국 *비밀 회담* (CC -3) 의 설계자. 신좌침묵 의식의 *중심 조율자*.
- CC 4년 앤트로퍼 1호 *모나르* 기동 직후 자기 영혼을 매개로 사망.
- 시신은 *상아탑 호수 섬* 에 봉인. 7제자가 비전을 7조각으로 분할 보존.

## 9. 반대 세력 4종

### 9.1 자연 수용파 — *고요한 손* (Silent Hand)
- 본거지: 남부 사막 침묵의 사원
- 시조: 상아탑 분열의 *반대측* 2현인 중 1명 **헬리오스 (Helios)**
- 주장: *리셋은 자연의 선택. 인간이 미루는 것은 오만*
- 행동: 설득형. 카엘에게 *너의 사슬을 풀지 마라. 그저 죽어라*
- 위협도: 낮음

### 9.2 표적 전환파 — *백색 사냥단* (White Hunt)
- 본거지: 제국 데미안 + 헤븐즈 일부
- 후원자: **라이세린 데오릭 공작**, 데미안 황가 비공식
- 주장: *리셋되어야 할 것이 꼭 인간이라는 법은 없다*
- 행동: 드래곤·엘프·마녀 학자 암살. 변신 드래곤 색출
- 위협도: 높음 (암살형)

### 9.3 키메라 생산파 — *다섯 모루 결사* (Five Anvils)
- 본거지: 제국 데미안 지하 + *그라스* 인근 비밀 공방
- 주장: *고마나 응축체를 *제작* 해 도살하자*
- 행동: 키메라 시제품 제작. *통제 불능 폭주 위험*
- 위협도: 매우 높음 (광역 피해)

### 9.4 드래곤 로드 진영 — *황금 비늘의 노룡*
- 본거지: 드래곤 영역 + 인간 사회 폴리모프 잠복
- 지도자: **드래곤 로드** (본명 미상. 추정 수명 3,000년+)
- 주장: *인간이 해법을 손에 쥐는 것 자체가 위험. 반드시 변질한다*
- 행동: 폴리모프 자객으로 앤트로퍼 운영자·숙주 제거
- 본작 자객: **카이엔 오르하임 (=이실)**
- 위협도: 극도로 높음 (대륙 단위)

> **드래곤족 내부 분열**: 방관파 60% / 로드 추종파 30% / 인간 친화파 10% (행커만이 이 분파).
