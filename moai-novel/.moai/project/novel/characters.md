## 주인공 — 카엘 (Kael)

- **호칭 변화**:
  - 1~2부: *카엘* (평민)
  - 2-6 임명: *카엘 노이엔 경* / 약칭 *고문 노이엔 경*
  - 3-6 출정: *노이엔 사령관*
  - 4부: *나* (1인칭)
  - 5~6부 잠복: 다시 *카엘* (작위 폐기)
  - 별칭 (역사적 호명): *앤트로퍼 소서러*
- **출신**: 노이엔 변경 (헤븐즈 킹덤 외곽). 평민. surname 없음
- **출생**: CC 980, 16대 앤트로퍼 숙주
- **표면 목표**: 악당과 마계 세력을 처단하고 전쟁에서 살아남는다
- **실제 상태**: 앤트로퍼 *붉은 사슬 7마디* — 미세 마기 분비로 모든 외부 자극을 적의로 번역
- **균열**: 프롤로그에서 알레이언 6세의 *균열의 빛 즉발판* 으로 사슬 1마디 미세 균열. *가장자리 이중상* 영구 잔류
- **신체**: 5-5에서 양팔 손목 아래 상실, 마나 회로 7개 중 5개 단절. 6-2 마도 의수 (타르모리스 그라가르드 룬 후예 제작)
- **외모**: 키 180cm 초반, 검은 머리(어깨에 닿는 길이), *옅은 회청색 눈*, 단정한 이목구비 — *상당히 준수한 외양*. 본인은 *추방 트라우마로 외모적 자존감 잠식* — *자기가 잘생긴 줄 모름*. 3부 출세기 *영웅의 외모* 가 *학살의 정당화 동력* 으로 작동. 6부 잠복기 *수염·모자* 로 외모 죽이는 무의식. 종결부 회복기에 *수염·머리 정리* = *자아 회복의 시각적 환유*. 상세: [`narrative-devices.md §9.5`](./narrative-devices.md#95-카엘의-외모--준수한-외양과-그-부담)
- **검술 전환**: 1~3부 검사형 셉터 (마검 흉내) → 6-4 세이른의 공허검 입문
- **충동**: 4-5 *손의 떨림* 으로 신체화. *왜곡 없는 시야 vs 살인 갈구 신체* 의 모순
- **핵심 감정**: 사명감 → 균열 → 죄책감 → 속죄적 결의

- **야망**: *외곽 연구소를 왕실 직속 권력축으로 복권*. 그러려면 *세계가 인정할 루키 마법사* 필요 → 카엘 = *맞춤형 1회성 도구*

- **카엘과의 관계**: 2-2 첫 만남에서 카엘의 *역마나 감각* 알아봄. 차 한 잔의 호의를 카엘은 *독* 으로 봄. 5년간 *유일한 정기 사사자* — 카엘에게는 *유일한 인간적 접촉*
- **추천장 비공개 문구**: *"이 아이는 위험하지만 더 위험한 것을 막을 수 있다"* — 진짜 의미: *카엘의 위험을 *내가 통제할 수 있다* 는 자기 과신*

- **죽음 직전 미완성 경고 서한**: *"네가 보는 세계가 세계의 전부가 아니다"* — 자기 죽음을 *예감* 한 마지막 한 줄
- **유산**: 봉인 문서 (5-2 핵심). 사프란·붉은 사슬·이득·충동의 명명 출처. *사프란 추모 무명 사당* (서재 깊은 방)

- **카르샤 왕비 엘라리아** (Eleria): 공주 세파의 친모 + 알레이언 6세의 미망인. Ch1 P5 §10 망명 시작 후 *카르샤 외부 망명지에서 키얀 왕자와 함께 외부 도주*. **v9 재배치**: ~~Ch2 P1 1-3 직면~~ → Ch2 P2 본격 직면. *침묵의 분노 + 짧은 한 마디 — 당신이 데려왔군요. 우리 딸을*. *왕비 엘라리아의 조용한 결단 — 비공식 동맹 (공식 발표 없음)*
- **셀라엔 대주교** (Selaen): 카르샤 신성 신앙 최고 권위. *공식 종교적 단죄 입장 — 너는 이 신성한 땅에 발을 들였다*. 공주의 *단호한 보호 진술*로 즉결 처분 보류. 키얀 측 *비공식 동맹의 단호한 반대자*. **v9: Ch2 P2 진입**
- **카일렌 베르하르트의 처남 1인**: 카르샤 정찰병 출신. 카르샤 외부 망명지 키얀 왕자 호위. *카엘에 대한 직접 적의 + 공주 호위 결단*. **v9: Ch2 P2 진입**

### Ch2 21단계 신규 인물 (v9 sync)

- **카르샤 망명지 형제들 + 소드 마스터들** (Ch2 P2 §8): 신성제국 잔존 *기사단·셉터 합동 망명자 약 10~15명*. *강철제국의 검과 검술 정통 공유 매개*. 일부 *공허검 정통 정보 시드* (모르간 가문과의 비공식 교류 매개)
- **대도시 마법 장인** (Ch2 P1 1-4 단편): 변경 대도시의 *강철 의수 제작자*. 작은 인물 단편. *마나 회로 미세 결각 정통 보유*
- **대도시 비공식 무기 대장간 장인** (Ch2 P1 1-6 단편): *카엘의 *강철 대포 의수 비밀 제작* 의뢰 수행. *무기 비밀 의뢰의 *비공식 통로*. 작은 인물 단편 (이름 미정)
- **앨더스톰 비공식 마법사 1인** (Ch2 P5~P6 §16·§18): *전도율 진단 + 행커만 과거 폭로 + 바이서스 악행 공유 + 다양한 마법 기술·지식 양도 (현재 구현 불가, 미래 시드)*. *바이서스 결전 시 도움 약속*. 정확한 작명 = *후속 결정*
- **드워프 지하세계 장인** (Ch2 P5 §17): *아다만티움 + 미스릴 마력 대포 의수 제작자*. *드워프 부족의 *비공식 거래 + 대가 양도 (마나석·고대 정통 정보·미래 약속) 매개*. 정확한 작명 = *후속 결정*
- **바이서스 왕 라카르 9세 + 자식들** (Ch2 P7 §19-20): 척살 대상. **자식들 중 일부 = 라인하르츠 행커만 (폴리모프 블랙드래곤) 후예**. *반인반룡 혈통, 외관은 인간, 마나량은 일반 인간의 3~5배*. *본인들 자기 핏줄 모름*. 카엘이 *행커만 핏줄임을 모르고 척살* → *드래곤 종족 적대 발화*. 정확한 자식 명단 = [`legendary-figures.md`](../world/legendary-figures.md) v2 보강 항목 참조
- **드래곤 로드** (Ch2 P7~P8 §20-21 매개): *드래곤 종족 최고 위계 (3,000세 이상)*. *행커만 자식 죽음 인지 후 *행커만 기행 우려 → 카자르드에게 카엘 척살 의뢰* 결단. *본 Ch2 직접 등장 없음 — 의뢰의 매개로만 작동*. Ch3 후속 직접 등장 시드

### 3부

- **에리언** (Erien): 카르샤 변경 농부. 갈색 머리. 3-3 사주 학살에서 *두 손 들고* 살려달라 외쳤으나 카엘이 베어버림
- **마하** (Maha): 에리언의 어머니. 3-5 교차 시점 주인공. 평범한 농부. *우리 아이가 미워한 사람이 없게 해주세요* 기도. 6부 한 컷 등장 옵션 (살아남음)
- **셋째 동생**: 에리언의 막내 동생. 3-5 단편

### 5부

- **오릴**: 변경 약초꾼. 5-6 카엘과 세파의 구조자. 첫 식사 *손 모음 인사*. 카엘이 *처음으로 손 모음을 위협으로 보지 않은* 순간의 배경
