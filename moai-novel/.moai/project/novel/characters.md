# 인물 설계

## 주인공 — 앤트로퍼 소서러

- 역할: 주인공. 앤트로퍼의 씨앗을 품은 소서러.
- 표면 목표: 악당과 마계 세력을 처단하고 전쟁에서 살아남는다.
- 실제 상태: 앤트로퍼의 감각 왜곡 때문에 인간/세력을 악마적 대상으로 오인한다.
- 죄: 오해 속에서 수많은 죽음을 만든다.
- 전환점: 자신이 죽인 대상이 실제 악당이 아니라 인간이었음을 깨닫는다.
- 후반 목표: 대현자의 의도를 이해하고, 이번 리셋을 막기 위해 실제 마계의 창조물을 죽인다.
- 핵심 감정: 사명감 → 균열 → 죄책감 → 속죄적 결의.

## 대현자

- 역할: 과거의 구원자이자 현재 비극의 설계자.
- 동기: 생명 초기화 재발 방지.
- 모순: 세계를 살리기 위해 인간을 학살자로 만들었다.
- 주인공과의 관계: 저주의 원인이자, 후반부에 이해해야 할 사상적 기준점.

## 초반의 적 — 악당으로 인식되는 세력

- 역할: 주인공이 악으로 믿고 싸우는 대상.
- 실제 기능: 앤트로퍼의 왜곡을 드러내는 장치.
- 설계 원칙: 완전한 선인은 아니어도, 주인공이 믿는 정도의 악은 아니어야 한다.
- 반전 효과: 주인공의 정의감이 학살의 명분이었음을 드러낸다.

## 마계의 창조물

- 역할: 후반의 실제 제거 대상.
- 기능: 이번 생명 초기화를 진행시키는 해체 개체.
- 차별점: 초반에 주인공이 오인해 죽인 인간들과 명확히 구분되어야 한다.

## 미정 인물

- 주인공의 오해를 깨뜨리는 생존자.
- 대현자의 기록을 해석하는 조력자.
- 주인공을 끝까지 처단하려는 추적자.
- 마계 창조물과 인간 사이의 경계 사례.
