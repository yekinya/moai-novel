## 회차 비트

### 5-1 — 룬의 도시

- **공간**: 강철제국 본가 → 앨더스톰 (도보·말 3~4주)
- **시점**: 카엘 1인칭
- **비트**: 앨더스톰 진입 + *룬으로 빛나는 첨탑 + 고대 마법 도서관 직접 견식* + 도시 외곽의 비공식 룬 마법사 거주지

### 5-2 — 마르덴 라온

- **공간**: 마르덴 라온의 첨탑 입구
- **시점**: 카엘 1인칭 (직면)
- **비트**:
  - **마르덴 라온 (70대 후반, 앨더스톰 룬 마법사 대원로 + 대마법사 7써클급) 직면**
  - 세이른 추천장 매개 + 카엘 정통 직접 시연 (강화강철 의수 + 팔 검기 70% 시연) + **비공식 사사 허락**
  - 마르덴의 한 마디 — *세이른이 보낸 자는 *시간 낭비하지 않는다. 너의 답을 *내가 안다*

### 5-3 — 전도율의 진단

- **공간**: 마르덴의 도서관
- **시점**: 카엘 1인칭 (진단 + 결단)
- **비트**:
  - 마르덴의 *전도율 진단* — *세이른 진단 결론 재확인*
  - *아다만티움 = 인간 일부 제련 가능 (조잡) + 세밀 가공 불가*
  - *미스릴 = 인간 완전 제련 불가 — 드워프 지하세계 비전 기술 전용*
  - 두 재질 *모두 마나 감응 특화 + 돈으로 살 수 없는 재료*
  - 마르덴 = *드워프 부족과 비공식 거래 매개의 유일한 인간 통로* (조상의 *큰 은혜 양도* 역사적 채무 매개)

### 5-4 — 드워프의 문

- **공간**: 앨더스톰 마르덴 첨탑 → 드워프 부족 사절단 도착장
- **시점**: 카엘 1인칭 (의뢰 협상)
- **비트**:
  - 마르덴 매개로 *드워프 부족의 비공식 거래 매개 연결*
  - **부족장 카르틴 그라엔의 *공식 의뢰 조건 3중 통보***:
    1. **마나석 양도** — 카엘의 *바이서스 시기 자기 제작 아티팩트 잔존 + 마검사 검 부서진 부품의 최고급 마나석 1~2개*
    2. **고대 정통 정보 양도** — *카르엔 학파 잠적 분파 비전 + 마녀령 룬 지식 + 그라스 노 장인 3대 양도 정통 + 사프란 비망록 마지막 페이지 복사본*
    3. **미래 약속** — *카엘이 미래에 *드워프 부족의 어떤 의뢰를 *받으면 거절하지 않음*. *조건 없음*
  - 카엘이 *팔찌를 한 번 본다*. *조용 — 붉은 빛 없음*
  - **3중 조건 모두 수락 결단**

### 5-5 — 지하의 모루

- **공간**: 앨더스톰 산령의 비공개 통로 → 드워프 지하세계
- **시점**: 카엘 1인칭 (깊은 침묵 + 근본적 자각)
- **비트**:
  - 비공개 통로 *3일 도보* + 룬으로 빛나는 지하 도시 진입
  - **드워프 부족 시험**:
    - *검 1합* — 드워프 검사가 카엘의 강화강철 검기 시연 평가. 공허검 호흡 인지 + 진지한 평가
    - *마법 1점* — 드워프 룬 장인이 카엘의 마력→내공 + 의수 검기 시연 평가. 내공 4중 회로 인지 + 짧은 칭송
    - *1마디 답변* — 부족장 질문 — *너는 왜 강해지려 하는가*. 카엘 답 — *지키기 위해서*. 부족장 *짧은 묵례 — 통과*
  - 지하 도시 직접 견식 — *인간이 결코 따라잡지 못하는 정통의 깊이*
  - 카엘의 *깊은 자각 — 내가 강해진다 해도 *드워프와는 *근본 다른 길*

### 5-6 — 아다만티움의 손가락

- **공간**: 드워프 본가 의수 작업장
- **시점**: 카엘 1인칭 (극심한 통증 + 조용한 노동)
- **비트**:
  - **콘드라 그라드네 (드워프 본가 의수 라인 노 장인, 카르틴의 사촌) 의 직접 작업 매개**
  - *약 2개월 의수 작업* — 부족 장인 6주 + 카엘 회로 조율 2주
  - *재료 = 아다만티움 본체 + 미스릴 정밀 관절 + 다중 룬 결각 + 마력 회로 통합*
  - 카엘의 *극심한 통증 + 정신력 견딤*

### 5-7 — 마력 대포의 완성

- **공간**: 드워프 작업장 → 외곽 시험장
- **시점**: 카엘 1인칭 (깊은 명료성)
- **비트**:
  - **아다만티움 + 미스릴 마력 대포 의수 1조 완성**
  - *마나 감응 최고 효율 (일반 강철의 3~5배)*
  - *검기 운용 = 팔 전체 검기 두께 일반 검의 100~120% 도달*
  - *룬 결각 전체 표면 + 다중 아티팩트화*
  - *마력포 무한 발사 (마나 충전 자동, 1초 간격 연속)*
  - *손가락 촉각 감응 첫 부분 가능*
  - *세밀 작업 정상*
  - 카엘의 한 마디 — *…이제 *시작이다*

### 5-8 — 부족의 작별

- **공간**: 드워프 본가 부족장의 거처 → 앨더스톰 복귀 직전
- **시점**: 카엘 1인칭 (조용한 발걸음)
- **비트**:
  - 카르틴 그라엔의 짧은 작별 — *미래 약속 매개. 우리는 *너를 기억한다*
  - *부족의 일부 룬 정통 추가 양도* — Ch3+ 자기 룬 제작 매개
  - 콘드라 그라드네의 침묵 + 짧은 묵례
  - *드워프 지하세계 → 앨더스톰 복귀 *지하 통로 도보 3일*
  - 본 Part 종결 = *앨더스톰 복귀 직전 — 마르덴 라온과의 *재회 + 다양한 마법 지식 양도는 Ch2 P6 본격*
- **단서**: *팔찌 조용 — 본성의 결단 안정*

## 종결 후크

> 앨더스톰의 *룬 첨탑 그림자가 *카엘의 등 뒤로 *길게 늘어진다*. 카엘이 *아다만티움 의수의 손가락을 한 번 펴고, 접는다*. *손가락 끝의 *마나가 *명료하게 흐른다*.
>
> 카엘의 내적 한 줄 — *…이제 *마법사 마르덴에게 *물어볼 것이 있다. 행커만은 *누구였는가*.
>
> 멀리 — *마르덴 라온이 *룬 첨탑의 *문 앞에서 *카엘을 기다리고 있다*. *그의 손에는 *오래된 양피지 1장이 *반쯤 펼쳐져 있다*.
