## 작업 규약

1. **회차 단위 파일 추가**: 각 부 파일은 회차 요약 + 핵심 비트만 담는다. 본문 집필이 시작되면 `chapters/manuscripts/{part}/{episode}.md`에 따로 저장한다.
2. **단서 무결성**: `plot.md` "단서 배치 원칙"에 따라, 초반 30%(=1~3부) 내 모든 핵심 단서가 들어가야 한다. 각 회차 비트에 `[단서]` 태그로 명시.
3. **신뢰할 수 없는 화자**: 1~4-3회까지 주인공 시점 묘사는 *그가 본 세계* 만 기술한다. 객관 사실은 외부 인물의 반응·시체·물건의 디테일로만 흘린다.
4. **시점 교차**: `[교차]` 표시가 붙은 회차는 적국·드래곤 등 다른 시점. 분량은 본 회차의 30% 이하 단편으로 권장.
5. **회차 분량 기준**: 본문 약 5,000~6,500자(웹소설 1회차 표준).
