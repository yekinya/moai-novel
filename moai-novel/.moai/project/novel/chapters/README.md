## 파일 명명 규칙

- **파일명**: `part-{매크로 N}-{서브 M}.md` — *매크로* = 시놉시스 대분류(1~4) / *서브* = 매크로 안의 챕터 그룹
- **내부 회차 번호**: 본문 내 *N-M* 표기는 *서브 챕터 그룹 안의 회차* (예: `5-2` = part-1-6.md 안의 두 번째 회차)

> 내부 회차 표기(1-1 ~ 6-7)는 *기존 설계 추적성* 을 위해 유지. 파일명만 매크로 Part 체계로 정렬.

## 회차 통합 가이드 (narrative-devices.md 적용)

각 회차에 다음 8축을 *밀도 있게* 배치:

1. **재미 4축** ([§2](../narrative-devices.md#2-본작-핵심-재미-요소--4축)) — 기술 창의성 / 신체·도구·자아 성장 / 아이템 사이클 / 상징물(2개 한정)
2. **잔향** ([§1.7](../narrative-devices.md#17-금제-마나-잔향--주변인이-느끼는-까닭-모를-불쾌감)) — 주변인의 *까닭 모를 어색한 거리감*. *잔향* 단어 본문 등장은 5-2부터
3. **카엘 외모** ([§9.5](../narrative-devices.md#95-카엘의-외모--준수한-외양과-그-부담)) — 준수한 외양 + 본인 무자각. *외모 + 잔향* 의 모순이 주변 혼란
4. **짠내 전투** ([§9.1](../narrative-devices.md#91-짠내나는-전투의-4원칙)) — 1~2부 *진짜 죽음 위협*. 부상 지속 / 자원 부족 / 승리의 씁쓸함
5. **의식 분리** ([§1](../narrative-devices.md#1-의식-분리--3인칭-자아-방관-self-dissociation-as-observer)) — 1-1 *짧은 섬광* 시드 → 4-5 본격 발동 → 6부 의식적 통제
6. **상징물 2개** ([§2.4](../narrative-devices.md#24-인물-관계-상징물)) — 부두인형(단의 손바느질 인형, 5-5 발견, 6-7 무덤에 놓음) / 붉은 머리끈(5-2 충동 구매, 6-2 의수 봉인 회로)
7. **여성 캐스팅** — 에르나(1.5부 소모성 동행), 미라(2-5 단편), 모르간(6-3 황녀). *비도구화·로맨스 라인 아님*
8. **알레고리** ([§8](../narrative-devices.md#8-현실-세계-알레고리--판타지-외피-안의-동시대-진단)) — 본문에 *명시적 설교 금지*. 현대 용어(자본·악플·SNS) *절대 등장 금지*. 4축은 *세계관 자연어로만*

## 작업 규약

1. **회차 단위 파일 추가**: 각 부 파일은 회차 요약 + 핵심 비트만 담는다. 본문 집필이 시작되면 `chapters/manuscripts/{part}/{episode}.md`에 따로 저장한다.
2. **단서 무결성**: `plot.md` "단서 배치 원칙"에 따라, 초반 30%(=part-1-3·1-4) 내 모든 핵심 단서가 들어가야 한다. 각 회차 비트에 `[단서]` 태그로 명시.
3. **신뢰할 수 없는 화자**: 1~4-3회까지 주인공 시점 묘사는 *그가 본 세계* 만 기술한다. 객관 사실은 외부 인물의 반응·시체·물건의 디테일로만 흘린다.
4. **시점 교차**: `[교차]` 표시가 붙은 회차는 적국·드래곤 등 다른 시점. 분량은 본 회차의 30% 이하 단편으로 권장.
