# Ch2 P7 — 바이서스 결전: 왕·자식·행커만 후예 척살

> Ch2 P6 종결 직후. 바이서스 변경 진입 → 셀라리아 → 왕궁 대전당 → 옥좌의 방. 라카르 9세 5합 결투 (역방향 거울) + 자식 척살 (행커만 후예 포함) + 드래곤 종족 적대 발화 시드 점화.
> 시점: **1인칭 (카엘)**. 10회차.
>
> spec: [`../specs/spec-chapter-2-part-7.md`](../specs/spec-chapter-2-part-7.md) v3 (21단계 §19-20)

## 부 단위 약속

- **단서 누설 비율**: *행커만 후예 척살 = 드래곤 종족 적대 발화 시드 점화*. 시드만 보임 — 본격 발현은 Ch2 P8 §21
- **금지**: *카엘의 *외적 분노 폭발* 없음. *조용한 결단 + 체계적 무거움 + 결단 단문*
- **회상 대응**: Ch1 P5 4-4 알레이언 5합 결투의 *역방향 거울 정통*. *알레이언 = 자기 희생 / 라카르 9세 = 자기 보존·비명·도주*

## narrative-devices 적용

- **5합 결투의 역방향 거울 정통** [§5 v2]: Ch1 알레이언 5합 결투를 *Ch2 라카르 9세 5합 결투로 거울로 비춤*. *알레이언 = 명예 / 라카르 9세 = 비명*
- **카엘의 결단 변하지 않음** [§4 v3]: *팔찌 부분 붉은 빛에도 *나의 결단은 *내가 책임진다 — 드래곤이 추격해도 받아들인다*
- **자식 척살의 *공평 결단*** [§6 v3]: *행커만 후예 + 일반 인간 혈통 모두 *공평 척살*. *카라엔 (5공주, 13세, 일반 인간 혈통) 의 *가장 무거운 단일 척살*
- **팔 전체 검기 첫 본격 발현** [`world/prosthetics.md`]: *아다만티움 의수 + 공허검 호흡 + 마력→내공 + 룬 결각 전체 = 100~120% 두께 첫 결전 본격*

## 회차 비트

### 7-1 — 변경의 새벽

- **공간**: 앨더스톰 → 바이서스 변경 진입
- **시점**: 카엘 1인칭
- **비트**:
  - *앨더스톰 → 바이서스 변경 진입 + 세렌 마르덴·베라트 알린 비공식 동행*
  - *카르샤 망명지 협공 시드 활성화* — 세파 + 키얀 + 신성제국 형제·소드 마스터들의 *바이서스 외곽 동쪽 동시 압박*
  - 변경의 *조용한 새벽 + 짠내 침투 톤*

### 7-2 — 외곽의 정찰

- **공간**: 바이서스 외곽 산악·변경 도시
- **시점**: 카엘 1인칭
- **비트**:
  - *마르덴 양도 5왕국 비공식 인맥 라인 활용*
  - *변경 정찰병 5~10인 *소리 없이 격퇴* — 공허검 2원리 + 의수 검기 정통의 완벽한 침투력
  - 세렌 마르덴의 *룬 마법으로 *원거리 시야 차단 + 정찰병 위치 미세 추적*

### 7-3 — 셀라리아의 잔향

- **공간**: 바이서스 외곽 → 셀라리아 진입
- **시점**: 카엘 1인칭 (깊은 회상)
- **비트**:
  - 셀라리아 진입 + Ch1 P3 *5년 학습한 외곽 연구소 + 광장·도서관·식당의 *깊은 회상 단편*
  - **외곽 연구소 잔존 인물 1~2명 (카르엔 사이엔 사후 잠적 분파 조용한 잔존자) 의 비공식 협력**
  - *왕궁 대전당 진입 매개 — 마법학자 자격 위장 + 마르덴 비공식 외교 기록 활용*

### 7-4 — 왕궁의 문

- **공간**: 셀라리아 왕궁 대전당 외곽 → 진입
- **시점**: 카엘 1인칭 (조용한 위엄)
- **비트**:
  - *왕궁 호위 일부 무력화 — 마르덴 양도 비공식 외교 기록 매개*
  - 세렌 마르덴·베라트 알린의 *룬 마법 외부 매개 지원*
  - **카엘의 *팔 전체 검기 첫 본격 발현*** — *아다만티움 의수 + 공허검 호흡 + 마력→내공 4중 회로 + 룬 결각 전체 = 100~120% 두께*
  - *왕궁 호위 마법사 + 기사 다수 *동시 무력화*

### 7-5 — 자식의 어깨

- **공간**: 왕궁 대전당 1층 회랑
- **시점**: 카엘 1인칭 (조용한 결단)
- **비트**:
  - **로카르 (1왕자, 30대 후반, 반인반룡 혈통 의심 최상위) 척살**
  - *9써클 직전 마법 자질 잠재 — 본인 왜 그렇게 빠르게 학습되는지 모름*
  - 로카르가 *광역 결계 시전 시도* — 카엘 *공허검 한 점 침으로 척살*
  - 로카르의 마지막 짧은 단문 — *"너는… *누구의 아들이지?"*
  - 카엘 답 없음. *팔찌를 한 번 본다 — 붉은 빛 없음*

### 7-6 — 호스반의 응답

- **공간**: 왕궁 대전당 2층 회랑
- **시점**: 카엘 1인칭 (호각 결투의 압축된 단문)
- **비트**:
  - **호스반 셰이드런 직면**
  - 호스반의 짧은 평가 — *"공허는 *어디서 배웠나."*
  - **호각 결투** — 호스반의 광역 8써클 마법 vs 카엘의 마검사 변형 + 의수 마력포 + 공허검 호흡 + 마력→내공
  - 호스반 = 광역 위력 압도적. 카엘 = *공허검 1원리 (불교) 로 광역 마법 비키게 함*
  - *결정타 — 카엘의 바람의 길 응용 + 마력포 = 호스반 옆구리 직격 척살*

### 7-7 — 라이세린의 마지막

- **공간**: 왕궁 비공개 거처
- **시점**: 카엘 1인칭 (짧은 응답)
- **비트**:
  - **라이세린 데오릭 직면** — *잠적 후 비공식 권력 라인 운영. 결전 시점에 왕궁 비공개 거처 잠시 복귀*
  - 카엘의 조용한 단문 — *"너의 *위조 첩보는 *우리 어머니의 *침묵보다 무거웠지."*
  - 라이세린의 응답 — *"너는 결국 돌아왔구나. 내가 너를 *키운 것이 *내 잘못이었다."*
  - *공허검 한 점 침으로 척살*. *비공식 권력 라인 측근 다수 잠적·항복*

### 7-8 — 옥좌의 피

- **공간**: 옥좌의 방
- **시점**: 카엘 1인칭 (역방향 거울 정통)
- **비트**:
  - **라카르 9세 5합 결투** — Ch1 P5 4-4 알레이언 5합 결투의 *역방향 거울 정통*
  - **알레이언 = 자기 희생 / 라카르 9세 = 자기 보존 + 비명 + 도망치려 함**
  - **5합 흐름**:
    1. 라카르 9세의 *근위 마법사 1인 보조 시전* — 카엘 공허검 1원리로 비킴
    2. 라카르 9세의 *광역 결계 시전* — 카엘 의수 마력포로 결계 분쇄
    3. 라카르 9세의 *도주 시도* — 카엘 *바람의 길 응용 (Ch1 카자르드 용언 응용 시드)* 으로 추격
    4. 라카르 9세의 *광역 1회성 조커 카드 시전 시도* — 카엘 공허검 2원리 (공출) 로 비움 + 빈 자리 침
    5. 라카르 9세의 *마지막 한 마디 — 너는 *나의 *도구가 아니었다* + 카엘의 척살

### 7-9 — 자식들의 끝 ★

- **공간**: 옥좌의 방 → 인접 방 (왕가 거처)
- **시점**: 카엘 1인칭 (반복적 무거움 + 결단 단문)
- **비트**:
  - **남은 자식 4명 척살** — *카엘이 *행커만 후예 가능성을 *알고도 결단 변하지 않음*
  - **데이렌 (2왕자, 30대 초반, 의심 부분) 척살** — *마검사 변형 정통으로 짧은 일격*
  - **하르엔 (3왕자, 20대 후반, 의심 강함, 조용한 학자형) 척살** — *공허검 한 점 침*. 마지막 시선 — *"너는… *누구인가?"*
  - **리에네 (4공주, 20대 초반, 의심 강함) 척살** — *짧은 신성 시전 시도 직전 — 카엘 의수 마력포 직격*
  - **셀리언 (5왕자, 14세, 의심 부분, 어린 학자형) 척살** — *마검사 변형 짧은 일격*
  - **카라엔 (5공주, 13세, 일반 인간 혈통) 척살** — *팔찌 *부분 붉은 빛 → 카엘의 *짧은 망설임 → 비뚤린 시야 우세*. *카라엔의 마지막 어린 단문 — 왜 우리야?*. *척살. 본 회차의 *가장 무거운 단일 척살*
  - 카엘의 짧은 단문 — *"…나의 결단은 *내가 책임진다."*

### 7-10 — 옥좌의 침묵

- **공간**: 옥좌의 방
- **시점**: 카엘 1인칭 (침묵 + 종결)
- **비트**:
  - 결전 종결 + 카엘이 *왕좌의 방 한가운데에 *조용히 서 있다*
  - 주변 = 라카르 9세 + 자식들 + 호스반 + 라이세린 + 호위 마법사·기사 다수의 *시신*
  - 카엘의 의수 손가락 끝에서 *마지막 공허검 호흡 *조용히 머문다*
  - **드래곤 종족 적대 발화의 *시드 점화*** — *행커만 후예 3~4명 동시 사망 → 드래곤 종족적 인지 즉시 발화 (Ch2 P8 본격)*
  - 카엘의 *셀라리아 출발* — *세렌·베라트 비공식 정보망 통해 철수*
  - 카르샤 망명지 협공 측 외곽 동쪽 *조용한 철수* — 임무 완수 시그널
- **단서**: *팔찌 조용 — 본성의 결단 안정*. *카엘의 내적 한 줄 — …드래곤이 온다. 받아들인다*

## 종결 후크

> 옥좌의 방 한가운데에 *카엘이 조용히 서 있다*. *주변의 *시신 풍경이 *낮은 햇빛 아래에서 *고요하다*.
>
> 카엘의 의수 손가락 끝에서 *마지막 공허검 호흡이 *조용히 멈춘다*. *팔찌가 *조용 — 붉은 빛 없음*.
>
> 카엘의 내적 한 줄 — *…드래곤이 *온다. *나는 *그것도 *받아들인다*.
>
> 멀리 — *셀라리아의 *낮은 종소리가 *왕의 죽음을 *알리고 있다*. *그러나 카엘의 귀에는 *그 종소리 뒤에 *다른 무엇이 들린다 — *드래곤의 *날갯짓 같은 *바람의 잔향*.
