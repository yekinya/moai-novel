# 문체·서사 가이드

> *스피디한 페이싱 + 중간중간 유머 + 극적 서사·대사* 의 3원칙을 본문 집필에 적용하는 구체 사양.
>
> 회차 톤 매트릭스: [`../MAP.md §16`](../MAP.md#16-집필-톤문체-가이드)
> 프롤로그 어휘 가이드: [`specs/spec-chapter-1-prologue.md`](./specs/spec-chapter-1-prologue.md) — 본 spec 내 *어휘·문체 가이드* 섹션 (chapters/ 흡수 후 2026-05-30)

## 1. 작품 전체 톤 — 3대 축

### 1.1 스피디 (Speed)

본작은 *느린 묘사형 판타지가 아니라 빠른 회전형 판타지*. 한 회차 = 한 호흡으로 읽힌다.

#### 회차 비트 구조

- **회차당 3~5 비트** — 한 비트 = 약 1,000~1,500자
- 각 비트는 *장면 단위* — 공간 이동 / 시간 비약 / 인물 등장 변화 중 하나가 일어남
- 한 비트 안에서 *3분 이상의 정적 묘사 금지*. 인물이 움직이거나 말을 한다

#### 페이싱 원칙

| 묘사 종류 | 비중 |
|----------|------|
| 행동 / 대사 | **60%** |
| 내면 묘사 | 25% |
| 풍경 / 배경 | 10% |
| 회상 / 사상 | 5% |

#### 금기

- 한 문단이 *5줄 이상* 이어지면 *반드시 분단*
- *상황 설명만으로 1페이지 채우기* 금지 — 항상 *인물의 행동·대사*에 묻혀 흐름
- *주변 환경 묘사 1줄, 인물 행동 2줄* 의 리듬 유지

#### 예시 — 좋은 페이싱

```
바이서스의 아침은 늘 회색이다. 카엘은 평소보다 일찍 일어났다.
"또 그 책인가." 카르엔이 차를 따르며 말했다.
카엘은 답하지 않고 책장을 넘겼다. 손가락이 떨렸다. 모르겠다.
이 떨림이 흥분인지 두려움인지.
```

→ 4문장. 환경 1, 행동 2, 내면 1. 비트 시작으로 적정.

#### 예시 — 나쁜 페이싱 (금지)

```
바이서스의 아침은 늘 회색이었다. 마법사들의 도시답게 안개가
짙게 끼었고, 그 안개는 셉터들이 매일 시전하는 결계의 부산물
이라는 것이 학자들의 정설이었다. 카멜라 4세 시대부터 이어진
이 현상은 도시의 정체성을 형성했다. ... (계속 5줄)
```

→ 인물 없음. 정보 덤프. **금지**.

### 1.2 유머 (Humor) — 중간중간

본작은 *어둠과 죄책감* 의 무거운 서사. 그러나 **무거움 사이에 유머가 있어야 호흡한다**.

#### 유머의 시기

| 시기 | 유머 가능성 |
|------|-----------|
| 프롤로그 | **불가** (극적 죽음의 무게) |
| 1부 | 거의 불가 (카엘의 어두운 유년) — *마을 사람의 한 줄 농담* 1~2회 |
| 2부 | 가능 — 카르엔의 *부드러운 비꼼*. 미라의 *어색한 친절* |
| 3부 | 불가 (학살의 무게) |
| 4부 | 불가 (각성의 비극) |
| 5부 | 시작 — 세파의 *날카로운 잔소리*. 오릴의 *시골 단호함* |
| **6부 후반** | **본격** — 카이엔의 *인간 감각 호기심* / 세이른의 *무뚝뚝 농담* / 세파-카엘의 *티격태격* |

#### 유머의 종류 (본작 4가지)

1. **잔소리형** (세파) — *각하 아니에요. 학살자예요.* 같은 비웃음 + 정확한 지적
2. **순진한 호기심형** (카이엔 6부) — *이 술이라는 것은 왜 어지러운가?* 같은 어른의 어린이같은 질문
3. **무뚝뚝 농담형** (세이른) — *검은 막는 것이 먼저다. 너는 어차피 사람도 못 막지만.* 같은 깐깐한 노인의 농담
4. **자기 비하형** (5~6부 카엘) — *나는 마법사인데 검을 배운다. 이상한 인생이다.* 본인이 본인을 비웃기 시작

#### 유머 금기

- *대량 학살 직후* 유머 금지 (3부·4부 본문)
- *주변 인물의 죽음 직후* 유머 금지 (카르엔 사망·카일렌 사망 등)
- *유머가 인물 가벼움을 의미하지 않음* — 세파는 *심각한 사람이 무거운 상황에서 한 줄 농담* 하는 사람

#### 유머 한도

회차당 *유머 비트 0~2개*. 6부 후반은 *3~4개까지*. 무거운 회차(프롤로그·3-5·4-4·5-5)는 *0개*.

### 1.3 극적 서사·대사 (Drama)

본작의 *심장*. 모든 회차에 *1개 이상의 *주먹질 장면*** — 독자가 멈춰서 다시 읽는 순간.

#### 주먹질 장면의 5종

1. **결정적 한마디** — 알레이언의 *"너는… 무엇을 보고 있는가."* / 카르엔의 미완성 서한 / 세파의 *그건 당신이 정할 일이 아니에요*
2. **물리적 충격** — 5-5 양팔 상실 / 4-4 왕의 무릎 / 카르엔 죽음 발견
3. **재해석 순간** — 4-5 거울 속 옅어진 일렁임 / 6-2 부친 책의 출처 회수 / 5-2 사프란의 이름
4. **시점 충돌** — 3-5 마하의 기도 / 6-6 카엘-카이엔 5초 어긋남 / 6-3 카이엔의 인간 매혹
5. **선언** — 6-7 *나는 여기에 있다* / 1-5 *세상이 잘못되었다면, 내가 바꾼다* / 4-5 *이제부터 본다*

#### 극적 대사의 원칙

- **짧다** — 1~2문장 한도. 길면 무게가 빠진다
- **반복 없음** — 같은 의미를 다른 말로 풀지 않는다
- **설명 없음** — 대사가 *왜 무거운지* 본문이 풀면 안 됨. 대사 자체가 무게
- **타이밍** — 비트 끝에 배치. 다음 비트가 *바로 화면 전환*

#### 예시 — 좋은 극적 대사

```
알레이언의 입이 열렸다.
"너는… 무엇을 보고 있는가."
나는 답하지 않았다. 답할 수 없었다.
화면이 검게 꺼졌다.
```

→ 4문장. 대사 1, 침묵 1, 단호. 비트 종결.

#### 예시 — 나쁜 극적 대사 (금지)

```
알레이언의 입이 열렸다.
"너는 무엇을 보고 있는가. 내가 너에게 보내는 마지막 질문이며,
이는 너의 시야가 진정으로 정직한가에 대한 근원적 물음이고..."
```

→ 길고 설명적. *무게가 풀어짐*. **금지**.

## 3. 시점 운용 세부

### 3.1 시점 전환 규칙

- **회차 도중 시점 전환 금지** — 한 회차 = 한 시점
- 예외: *교차 단편* (3-5 마하 / 6 홀수 카이엔) — 명시적 *회차 분리*
- 시점 전환 시 *회차 첫 줄에 화자 명시* — *카이엔의 아침은 차가웠다* 같은 즉시 식별

### 3.2 1인칭 vs 3인칭 근접

**1인칭 (프롤로그·4~6부 카엘/카이엔)**:
- *나* / *내가* / *나의* 사용
- 내면 직접 진술 가능
- 다른 인물의 내면 *불가* — *그는 ~ 했다* 까지만

**3인칭 근접 (1~3부 카엘)**:
- *카엘은* / *그는* 사용
- *카엘의 시각으로 보이는 세계* 만 묘사
- 다른 인물의 내면 *불가*. 표정·행동·대사로만

### 3.3 신뢰할 수 없는 화자 — 핵심 기법

- 1~3부: 카엘의 시각이 *명백히 왜곡되어 있음* 을 *독자가 먼저 알아채야 함*
- 기법:
  - 카엘이 *"적의로 보았다"* 같은 명시적 진술
  - 다른 인물의 *반응이 카엘의 해석과 어긋남* (예: 카엘은 *비웃음으로 들었으나* 옆 아낙이 *곱게 웃는다 한다*)
  - *물건의 객관 디테일* — 카르샤 농가에 *어린아이의 머리카락* 같은 진실 단서
- 4-5 자각 후: *과거형 자기 의심* — *그때 나는 그것을 영창으로 들었다* 같은 *재진술*

## 4. 대사 작성 원칙

### 4.1 대사 길이

- 일상 대사: 1~2문장
- 극적 대사: 1문장 (짧을수록 무거움)
- 설명 대사 (불가피한 정보 전달): 3~4문장 한도 — *대화 도중 끼워넣기*. 일방적 강의 금지

### 4.2 대사 태그

- 표준: *"대사."* + 대사자 행동 + (필요 시) 표정
- *말했다* 의 반복 회피 — *답했다 / 중얼거렸다 / 끄덕였다 / 입을 다물었다* 등
- **단** 1인칭에서 *말했다* 가 *적절히 단호* 할 때는 사용
- 형용사 부사 *남용 금지* — *부드럽게 말했다* 보다 *부드러운 손짓으로 말했다* (행동으로 표현)

### 4.3 대사 톤 (인물별)

| 인물 | 대사 특징 | 예시 |
|------|----------|------|
| 카엘 (1~3부) | 자기 정당화 + 거만 + 격식체 | *"이 자가 나를 모욕하려 하는군."* |
| 카엘 (4부) | 단호 + 균열 시 *흐려짐* | *"… 그는 검을 내려놓는다. 나는 멈출 수 없다."* |
| 카엘 (5~6부) | 짧고 건조 + 자조 | *"이상한 인생이지."* |
| 알레이언 6세 | 부드럽고 단호 — *왕의 무게* | *"네 죄가 아니다. 그러나 너 외에는 멈출 자가 없다."* |
| 라이세린 | 정중하면서 *뱀처럼 매끄러움* | *"자네 같은 사람을 기다렸다, 노이엔 경."* |
| 카르엔 사이엔 | 노 마법사의 *부드러운 시*. 직접 충고 회피 | *"세상에는 두 종류의 차가 있네. 마시는 차와… 마시지 않는 차."* |
| 세파 류르 | **날카롭다. 격식 거부.** 한 줄 잔소리 | *"각하 아니에요. 학살자예요."* |
| 카이엔 (이실) | 인간 외양 *불완전 적응*. 형용사 *어색하게 풍부* | *"이 술이라는 것은… 어지럽다. 어지러움이 즐겁다. 인간들은 이것을 매일 마신단 말인가?"* |
| 모르간 린이엔 | 황녀 격식 + 검사 단호 | *"검은 거짓말하지 않습니다. 사람만 합니다."* |
| 세이른 스틸만 | 무뚝뚝 + 단호 + 깐깐한 농담 | *"검은 막는 것이 먼저다. 너는 어차피 사람도 못 막지만."* |
| 카일렌 베르하르트 경 | 정중 + 짧음 | *"왕이여, 도망치소서."* |
| 라이세린의 측근들 | 비밀스러움 + 정중 | (대사 거의 없음) |

### 4.4 침묵의 활용

- *말하지 않는 것* 이 *말하는 것보다 무거울 때* 가 자주 있다
- 형식: 침묵을 *문장으로* 명시 — *나는 답하지 않았다* / *그는 말을 멈췄다*
- 침묵 후 *3초 행동* — *그는 컵을 들었다. 차는 식어 있었다.* 같은 *시간의 진행 표시*

## 5. 묘사 원칙

### 5.1 감각의 우선순위

본작은 *시각보다 후각·청각·촉각이 더 자주 등장*. 이유: *왜곡된 시야* 의 화자 + *4-5 가장자리 이중상* 효과.

| 감각 | 비중 | 본작 활용 |
|------|------|----------|
| 후각 | **30%** | 피·향·양초·금속 / 카이엔의 마나 잔향 추적 |
| 청각 | **30%** | 영창·비명·*얇은 금속음* / 침묵 / 발걸음 |
| 촉각 | **20%** | 자상 잔열 / *심장 한 점의 차가움* / 의수의 무게 |
| 시각 | 15% | *흐리게* (1~3부) / *가장자리만 선명* (4부 이후) |
| 미각 | 5% | 카이엔의 *맛의 충격* / 카엘의 *재처럼 마른 입* |

### 5.2 풍경 묘사

- 풍경은 *인물 감정의 거울*. 객관 풍경 묘사 금지
- 셀라리아 = *회색 도시* (안개) — 카엘의 모호한 인식
- 카르샤 세이렌 = *흰색 + 빛* — 카엘이 *모욕적 깨끗함* 으로 봄
- 빙벽 협곡 = *고요* — 카엘의 *비어 있음 직전*
- 그라스 = *기름 냄새 + 망치 소리* — 카엘의 *재기 의지*

### 5.3 시간 묘사

- *각/분/초* 단위로 명시할 때 = *극적 비트*
- *몇 시간 후 / 다음 날 / 일주일 후* = 일상 비트
- 회상 = *과거형 안에 현재형* (프롤로그 회상 컷 *마을 우물가에서 누군가가 웃고 있었다 — 그 웃음은 비웃음이었다(고 나는 믿었다)*)

## 6. 회차 구조 표준

### 6.1 회차 길이

- 표준: **5,000~6,500자** (웹소설 1회차 표준)
- 프롤로그: 6,000자 내외
- 짧은 회차 (3-5 교차 등): 1,500~2,000자

### 6.2 회차 구조 — *5비트 표준*

```
[비트 1] 시작 — 공간·시간 설정 (500~800자)
[비트 2] 발단 — 첫 사건·인물 등장 (1,000~1,500자)
[비트 3] 전개 — 갈등 전개 / 정보 노출 (1,000~1,500자)
[비트 4] 충돌 — 주먹질 장면 (1,500~2,000자)
[비트 5] 마무리 — 결과 + 다음 회차 갈고리 (500~800자)
```

### 6.3 회차 시작 — 첫 줄의 무게

- **첫 줄은 *독자를 즉시 끌어들이는 한 문장*** — 풍경·대사·행동 중 1
- 예: *바이서스의 아침은 늘 회색이다.* / *"또 그 책인가." 카르엔이 차를 따랐다.* / *카엘은 양피지를 손에 쥔 채 일어서지 못했다.*

### 6.4 회차 종결 — 갈고리 (Hook)

- 다음 회차로 이어지는 *질문 / 충격 / 약속*
- 형식: 마지막 1~2문장이 *다음 회차를 읽지 않을 수 없게* 만듦
- 예시 (좋음): *그날 밤, 폐허의 가장 깊은 방에서, 그는 처음으로 마나를 점화했다.* — 1-5 종결
- 예시 (좋음): *"나는 여기에 있다. 폐허 마탑. 당신이 본 적 있는 모든 것을 다시 보러 오라."* — 6-7 종결

## 7. 어휘 가이드

### 7.1 금지 어휘 (본문 사용 금지)

- **외래어 마법명** — 라이트·파이어볼·메테오·블링크·텔레포트·폴리모프 등. 한국어 (광명·화염구·운석강하·점멸·순간이동·변신술) 사용. [AGENTS.md HARD RULE]
- **시제 혼란** — 1인칭 과거형 회차 안에 *현재형* 무분별 사용 금지 (회상 컷은 예외)
- **앤트로퍼 / 저주 / 붉은 사슬** — 본문 4-5 이전 직접 사용 금지
- **정의를 위해 / 악을 처단** — 1~3부 카엘의 *대사·내면 독백에만* 허용. 서술자 진술 금지

### 7.2 권장 어휘

- *감응* (느낌) / *결박* (결합) / *해체* (분리) — 마나 용어. 일상 동사로도 활용
- *얇은 금속음* / *심장 한 점* / *가장자리 이중상* — 균열의 신체적 표지 (4-5 이전 암시 전용)
- *호각* / *결착* / *합* — 검술 대화

### 7.3 사용자 선호 어휘 풀 (스피디·유머·극적)

**스피디**: 즉시·일순간·찰나·금방·금세·잠시·곧·바로
**유머**: 쯧·흠·아·이런·뭐랄까·말하자면·하긴
**극적**: 침묵·정적·무게·그러나·다만·결국·마침내

## 8. 대표 장면 — 첫 회차 샘플 (1-1 도입부 30줄)

> *집필 진입 시 첫 시도용 샘플*. 이 톤으로 본문 작성.

```
산파가 도망쳤다.
아무도 모르는 사이에, 산통이 채 끝나기도 전에.
어머니는 그날 밤 죽었다. 사람들은 출혈이라 했고, 사람들은 *아이의
탓이라* 고 수군거렸다.
아버지 알린은 갓 태어난 아이를 안고 마을 변두리 집으로 들어갔다.
그 안에서 7년을, 그는 누구에게도 아이를 보여주지 않았다.

어느 날 마을 최고령자가 죽기 직전에 한마디 중얼거렸다.
"엔트로퍼의 씨를 본 적이 있다…"
손자가 노망이라며 그 말을 묻었다.
다른 누구도 듣지 못했다.

아이의 이름은 카엘이었다.
아버지가 지어준 이름이었다.
그 이름이 어디서 왔는지, 아버지는 끝까지 답하지 않았다.

카엘이 처음 말을 한 것은 두 살 때였다.
"나 좋아하는 사람이 없어."
알린은 답하지 않았다. 그저 아이의 머리를 쓰다듬었다.
손이 떨렸다.

아홉 살이 되던 해.
마을 광장 우물가에서, 한 소녀가 사과를 내밀었다.
"먹어볼래?"
카엘은 그 사과를 받아 들었다.
그리고 떨어뜨렸다.
소녀를 밀쳤다.
"썩은 부분을 골라줬구나."
소녀가 무릎을 까이고 울었다. 옆의 아낙이 말했다.
"애가 곱게 웃는데 왜 그러니."
카엘은 그 말을 듣지 않았다. 그는 이미 등을 돌리고 있었다.
```

→ 30줄. 비트 1~2 통과. 페이싱 빠름. 단서 (*엔트로퍼의 씨* + *곱게 웃는데* 대비) 심음. 첫 줄 *산파가 도망쳤다* 가 후크.
