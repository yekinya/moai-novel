# 문체 예시집 (Prose Examples) — 집필용 상시 참조 파일

> 본 파일은 [`style-guide.md`](./style-guide.md)의 *원칙*을 *집필 직전 즉시 꺼내 쓸 수 있는 묘사 예시*로 구체화한 것이다. 매 회차 집필 전 반드시 참조한다.

## 0. 4대 묘사 원칙 (요약)

1. **배경 묘사** — 배경 변경 시 *짧고 인상적으로*. 객관 정보 덤프 금지. *주인공의 시선·후각·청각에 자연스럽게 묻혀* 흐른다.
2. **인물 묘사** — 한 인물 = 한 가지 *지배 특징* + 한 가지 *모순적 디테일*. 머리부터 발끝까지 훑지 않는다.
3. **전투 묘사** — 생동감 + 전황 + 심리. *적의 내면은 쓰지 않는다*. 주인공의 *판단·의도·결단*만. 한 합 한 합을 그리지 않고, *결정적 순간 3컷* 으로 자른다.
4. **주인공 말투** — 위트 있고 핵심 위주. *말이 적고 생각이 많다*. 대사 1줄 / 내면 2~3줄의 리듬.

## 0.1 표기 규칙 [HARD]

본문 집필 시 다음 표기 규칙을 *예외 없이* 적용한다.

| 종류 | 표기 | 예시 |
|---|---|---|
| **인물의 생각·내면 독백** | `'생각 내용.'` (홑따옴표) | `'…피곤하군.'` |
| **인물의 대사** | `"대사 내용."` (큰따옴표) | `"부탁이에요."` |
| **굵게 강조 (`**...**`)** | **자제 / 사용 금지** | (사용하지 않음) |
| **기울임 강조 (`*...*`)** | 본문 prose 안에서는 *자제*. 회상 도입 등 *서사적 장치*에 한해 최소 사용 | 회상 4컷의 *— ...* 도입 정도 |

### 적용 원칙

- **생각과 대사의 시각적 구분이 명확해야 한다** — 독자가 한눈에 *말한 것*과 *생각한 것*을 분간할 수 있어야 한다.
- **굵은 글씨로 비트·합·장면 구분 금지** — `**1합.**` `**결착.**` 같은 마크다운 굵게 표시를 본문에 쓰지 않는다. 비트 구분이 필요하면 *빈 줄* 또는 `---` 수평선으로만 처리한다.
- **본문 narrator emphasis의 기울임 자제** — 키워드를 강조하고 싶을 때는 *문장 구조*로 표현한다. 꼭 필요한 경우(예: 회상 4컷 도입의 `— ...` 라인)에 한해 보조 사용.
- **신호어·전문용어**(예: 얇은 금속음, 가장자리 이중상, 신성안)는 별도 강조 없이 *그대로 한국어 텍스트로* 쓴다. 반복 등장 자체가 신호 기능을 한다.

### 예시 — 적용 전/후

```
[적용 전 — 금지]
*사악한 의식의 잔재로군.*
**1합.**
손끝에서 *작은 화염*이 튀었다.

[적용 후 — 정통]
'사악한 의식의 잔재로군.'
1합.
손끝에서 작은 화염이 튀었다.
```

## 1. 배경 묘사 (Setting Description)

### 1.1 원칙

- **배경 변경 시 = 3~5문장으로 끝낸다**. 그 이상은 페이싱을 죽인다.
- **카엘의 시선·감각으로** 들어간다. *공기·냄새·소리* 가 시각보다 먼저 온다 (잔향·왜곡된 시야 화자이므로).
- *그 공간의 *지배적 한 가지*만 잡는다*. 회색·기름·금속·짠내 등.
- **인물의 행동과 섞어 흐른다** — 단독 풍경 문단 금지. 첫 1~2문장이 배경, 3~5문장에서 인물이 들어온다.

### 1.2 예시 — 좋음 (변경 도로 객점 진입, Ch1 P2 1.5-3 노이른)

```
노이른 객점에 들어서자 익숙한 짠내가 났다.
젖은 짚단, 식어버린 기름, 이미 마셔버린 누군가의 두 번째 술잔.
카엘은 문턱에서 잠시 멈췄다. 도박장 구석의 작은 그림자들이 판돈을 챙기며 황급히 자리에서 일어나는 것이 보였다. 키가 절반인 자들.
한 자는 모자를 깊이 눌렀고, 한 자는 카엘의 손목 언저리를 힐끗 보았다가
시선을 거뒀다. 카엘은 그것이 경계 인지 *경멸* 인지 분간하지 못했다.
```

→ 5문장. *짠내·기름·짚단* 후각 우선. 하플링은 *키가 절반인 자들* 로 간접 명시 (races.md §7 권장 처리). 카엘의 *왜곡된 해석* (겁인지 경멸인지)이 자동으로 따라 붙는다.

### 1.3 예시 — 좋음 (대도시 진입, Ch2 P1 1-4 시르베인)

```
시르베인은 망치의 도시였다.
변경 마을의 적막을 일주일 걸어온 카엘의 귀에, 둔중한 금속음이 사정없이 내리치는 듯했다.
둔중하고 일정한 금속음사이로 키 낮은 누군가가 망치를 짚고 이쪽을 힐끔 내비칠때
세파가 부축을 풀고 한 발 비켰다. "여기서부터는 당신 다리로 걸어요."
```

→ 5문장. *망치 소리*가 도시를 정의한다. 드워프는 *키 낮은 누군가*로 익명 처리 (Ch2 P1 races.md §6 지하세계 강령 반영). 세파의 한 줄로 인물 행동이 들어와 정적 묘사가 닫힌다.

### 1.4 예시 — 좋음 (약초 상점, Ch1 P2 1.5-6 카르엔)

```
어두운 약초상점의 말린 풀의 쓴 냄새가 천장 들보에 매여 있었다. 시장의 소란이 문턱에서
끊겼다. 늙은 인간 상인의 옆에서, 유달리 큰 눈의 점원이 약초 다발을 묶고 있었다.
점원은 카엘이 들어오는 것을 보았으나 시선을 피하지 않았다.
시장의 다른 모든 사람이 피한 그의 시선이었다.
"청년."
점원이 말했다. 손은 여전히 약초를 묶고 있었다.
"예로부터 붉은 내음을 조심하라고 했는데.."
잠시 약초를 마저 묶은 점원이 다시 돌아보며 말했다.
"그래서 뭐가 필요하다구?"
```

→ 8문장. *말린 풀의 쓴 냄새*가 공간을 잡는다. 백색 엘프는 *유달리 큰 눈의 점원*으로 종족명 없이 처리. *시선을 피하지 않았다*는 한 문장이 잔향 작동의 예외성을 자동으로 알린다.

### 1.5 예시 — 나쁨 (금지)

```
시르베인은 헤븐즈 8왕국 변경의 중심 도시로, 데미안 제국과 강철제국
사이의 지정학적 중립 지대였다. 인구는 약 5만 명에 달했으며, 무기와
약품, 정보의 비공식 교역이 활발히 이루어지는 곳이었다. 도시의 거리는
바이서스의 그것과 달리...
```

→ 위키 항목. *카엘이 없음*. 정보 덤프. **금지**.

### 1.6 빠른 전환 (장면-장면 사이)

- 시간 비약: *다음 날 새벽 / 사흘 뒤 / 일주일을 걸었다*
- 공간 비약: *그가 그 객점을 나섰을 때, 시르베인은 이미 닷새 거리 뒤에 있었다.*
- **장면 사이 풍경 묘사 1줄로 충분**. *길은 점점 좁아졌다.* / *산은 가까워졌다.* / *바람의 방향이 바뀌었다.*

## 2. 인물 묘사 (Character Description)

### 2.1 원칙

- **한 인물 = 한가지 지배적 특징 + 한가지 모순적 디테일**.
  - 글라렌: *늙은 검객의 단호함* + *목소리만은 부드럽다*
  - 엘: *햇볕에 그을린 거친 모습* + *잿빛 눈은 묘하게 차분하다*
  - 세파: *왕녀의 격식* + *욕설같은 짧은 잔소리*
- **머리부터 발끝까지 훑지 않는다**. 카엘의 시선이 *멈춘 한 군데*만 묘사한다.
- **외양은 *행동의 중간에* 흘려보낸다**. 단독 외양 문단 금지.
- **반복 등장 인물의 외양은 새 회차에서 *한 디테일만 새로 갱신***. 같은 외양을 두 번 풀지 않는다.

### 2.2 예시 — 첫 등장 (엘, Ch1 P2 1.5-3)

```
옆방의 여자가 술잔을 들고 카엘 쪽을 보았다.
햇볕에 그을린 피부. 광대뼈의 흉터. 잿빛 눈.
준수하다고 하기엔 거칠었고, 거칠다고 하기엔 시선이 너무 차분했다.
"앞자리 비었나."
그녀가 말했다. 대답을 기다리지도 않고 앉았다.
```

→ *지배 특징* = 거친 외양. *모순적 디테일* = 차분한 시선. 3문장 안에 외양 완결. 곧바로 행동(앉음)으로 이어 정적이 깨진다.

### 2.3 예시 — 첫 등장 (세파, Ch1 P6 / Ch2 P1)

```
공주의 손가락은 가늘었다.
그런데 그 가는 손가락이 카엘의 갈비뼈를 어림없이 누르고 있었다.
"숨 쉬어요." 그녀가 말했다.
명령이었다. 부탁이 아니었다.
```

→ *지배 특징* = 왕녀의 가는 외양. *모순적 디테일* = 의료인의 단호함. 외양·행동·대사가 4문장에 동시 발화.

### 2.4 예시 — 첫 등장 (배경 종족, 드워프)

```
지하로 내려가는 계단 끝에, 거대한 망치가 서 있었다. 
다부진어깨에 허리까지 오는 키. 손등의 굳은살이 다른 어떤 것보다 두꺼웠다.
그는 카엘을 보지도 않았다. 보지 않으면서도 카엘을 *지나가게 두지 않을 것*이라고 온몸으로 말하고 있다.
```

→ 4문장. *지배 특징* = 망치를 짚은 정적. *모순적 디테일* = 시선 없이도 차단 의지가 읽힘. 드워프라는 단어 없이 종족 인지 가능.

### 2.5 예시 — 반복 등장 시 갱신 (세파, Ch2 P1 1-4 시르베인)

```
세파는 시르베인의 좁은 골목 안에서 키가 더 작아 보였다.
바이서스의 햇빛 아래서는 그렇지 않았다. 도시의 그림자가 그녀의 어깨에
한 단계 더 얹혀 있었다.
```

→ 외양을 새로 풀지 않는다. *키가 더 작아 보였다* 한 줄로 공간과 인물을 동시에 갱신.

### 2.6 예시 — 나쁨 (금지)

```
세파 류르 카르샤 공주는 21세였다. 그녀는 카르샤 신성제국의 둘째 공주로,
어머니를 닮아 가는 골격에 흰 피부, 깊은 청색 눈동자를 가졌다. 머리는
은빛이 도는 금발로 허리까지 길렀고, 신성마법사 특유의 길고 가는 손가락이
인상적이었다. 그녀가 입은 옷은...
```

→ 정보 덤프. 인물 카드. **금지**. 같은 정보를 위 2.3처럼 *4문장에 행동과 함께* 푼다.

### 2.7 첫 등장 외양 분량 가이드

| 인물 비중 | 첫 등장 외양 분량 |
|---|---|
| 메인 (카엘·세파·카이엔·알레이언) | 회차에 분산하여 *누적 6~8문장*. 한 컷에 몰지 않음 |
| 서브 (엘·글라렌·델란·라이세린) | 첫 등장 *3~4문장*에 외양 완결 |
| 배경 (하플링·오크·드워프·점원 엘프) | 첫 등장 *1~2문장*. 종족명 직접 노출 선택 |
| 1회용 (도적·아낙·노인) | *0~1문장*. 행동만으로 처리 |

## 3. 전투 묘사 (Combat Description)

### 3.1 원칙 [HARD]

- **전체 전황을 먼저 잡는다**. *3대 5* / *카엘 1인 vs 도적 8인* — 첫 1~2문장에 수와 구도가 보인다.
- **결정적 순간 3컷으로 자른다**. 첫 합·중간 위기·마무리. 한 합 한 합 그리지 않는다.
- **적의 내면 묘사 절대 금지** [HARD]. *그가 무엇을 노렸는지*는 *카엘의 추측·판단*으로만 들어간다. *그는 카엘이 어리다고 얕보았다* 같은 적 시점 진술 금지.
- **카엘의 판단·의도·결단을 짧게 끼운다**. *한 자 — 화염. 두 자 — 검. 세 번째는 보지 않았다.* 같은 절제된 내면 비트.
- **호흡이 길어지면 자른다**. 한 합 묘사가 3문장 넘어가면 잘못 쓴 것.
- **승리의 짠내**를 남긴다. 카엘이 이겨도 *부상 / 자원 소모 / 시체 뒤지기* 중 하나가 따라온다.

### 3.2 전투 구조 표준

```
[A] 전황 (1~2문장) — 수, 구도, 거리, 지형
[B] 첫 합 (3~5문장) — 카엘의 첫 판단 + 첫 일격
[C] 중간 위기 (3~5문장) — 부상 또는 자원 한계
[D] 결단 (2~3문장) — 카엘의 *비뚤린 결단* 또는 *변칙 마법*
[E] 마무리 (2~4문장) — 끝난 풍경 + 짠내 1줄
```

총 13~19문장. 5,000자 회차의 *주먹질 장면*에 해당.

### 3.3 예시 — 좋음 (Ch1 P2 1.5-2 도적 5인 학살)

```
도적은 다섯이었다.
여관 지붕 위의 카엘에게, 그들은 일렬로 늘어선 *과녁* 처럼 보였다.

한 자가 노파의 머리채를 쥐고 있었다. 카엘은 그 손목을 먼저 골랐다.
손끝에서 작은 화염이 튀었다. 정통 시전이 아니었다. 그저 '그곳에 있어야 한다'는
감각이 손가락 끝을 데웠다.
도적의 손목이 검게 꺾였다.

두 번째 자가 칼을 들고 돌아섰다. 카엘은 이미 다음 손가락을 그쪽으로 향하고 있었다.
세 번째와 네 번째는 거의 동시에 무너졌다. 그가 다섯 번째를 봤을 때, 그 자는
도망치고 있었다.
도망치는 자의 등에도 그는 손가락을 들었다.

마지막 화염이 꺼지고, 지붕 위에 카엘만 남았다.
숨이 차지 않았다. 그 사실이 카엘 자신보다 먼저 '이상하다'고 느꼈다.
시체 다섯 구의 주머니에서 동전 네 잎과 작은 단검 한 자루가 나왔다.
손이 떨렸다. 그러나 순식간에 떨림은 잦아 들었다.
손이 왜 떨렸는지 알 수 없었다. 
```

→ 16문장. A(2)·B(4)·C(0 — 카엘이 압도하므로 위기 생략)·D(3)·E(5). 적의 내면 0회. 카엘의 내면 3회 (과녁·이상하다·왜 떨리는지). 짠내 = 시체 뒤지기.

### 3.4 예시 — 좋음 (Ch1 P2 1.5-3 객점 도적 8인, 위기 포함)

```
도적은 여덟이었다. 객점 안은 좁았다.

엘이 등을 카엘의 등에 붙였다. "왼쪽 셋, 내가 받는다."
카엘은 오른쪽 다섯을 보았다. 다섯은 너무 많았다.
언제든 화염을 튕기술 있도록 손가락 끝에 마나를 모았다. 한 자, 둘, 셋 — 세 번째에서 손이 떨리기 시작했다.
네 번째 자의 검이 카엘의 옆구리를 그었다. 갈비뼈 두 대가 *얇은 금속음* 처럼 울렸다.

카엘은 그 자리에서 무너지지 않았다. 무너지면 곧 목으로 칼이 들이친다는 것을 *몸이* 알았다.
다섯 번째 자에게 그는 *손가락이 아니라 손바닥*을 향했다. 화염이 두 배로 튀었다.
산소가 모자랐다. 불은 작고 빠르게 꺼졌다. 그러나 직격이었다.

엘의 단검이 등 뒤에서 날아와 카엘이 미처 보지 못한 여섯 번째의 목을 꿰었다.
일곱 번째와 여덟 번째는 무너지는 동료를 보고 멈췄다. 그것이 그들의 마지막
판단이었다.

객점은 다시 조용해졌다.
카엘은 옆구리를 짚고 의자에 앉았다. 엘이 단검을 회수하러 다가왔다.
한 발 떨어진 거리에서, 그녀는 한 발 더 물러섰다.
이유는 묻지 않았다.
카엘도 묻지 않았다.
```

→ 18문장. A(2)·B(4)·C(3 — 갈비뼈 부상)·D(3)·E(6 — 잔향 작동 포함). *얇은 금속음*은 균열의 신체적 표지 (style-guide §7.2). 적의 내면 진술 0회.

### 3.5 예시 — 나쁨 (금지: 적의 내면)

```
도적의 우두머리는 카엘의 가는 체격을 보고 '이 자는 별것 아니겠다'고
판단했다. 그는 칼을 빼들며 동료들에게 신호를 보냈고, 카엘을 산 채로
잡아 몸값을 받을 생각이었다.
```

→ *적이 무엇을 생각했는지*. **금지**. 같은 정보를 *카엘의 추측*으로 바꾼다:

```
우두머리가 카엘을 위아래로 훑었다. 칼을 곧장 뽑지 않은 것이, 카엘에게는
'살려서 끌고 갈 셈이군'으로 읽혔다.
```

### 3.6 예시 — 나쁨 (금지: 한 합 한 합)

```
도적이 검을 위에서 아래로 내리쳤다. 카엘은 왼쪽으로 반보 비켜섰다.
도적이 검을 회수하며 옆으로 베어왔다. 카엘은 뒤로 한 보 물러섰다.
도적이 다시 찌르기를 시도했다. 카엘은 오른쪽으로 비켜서며...
```

→ 검술 합보 일지. **금지**. *한 자 — 화염. 두 자 — 검. 세 번째는 보지 않았다.* 식으로 압축.

### 3.7 마법 시전 묘사 [HARD]

- 외래어 절대 금지: 파이어볼 / 라이트 / 텔레포트 → 화염구 / 광명 / 순간이동 (AGENTS.md HARD RULE)
- 정통 시전 = *3단계* 명시. *마나 회전 → 마찰 흡입 → 화염 발현*. 카엘의 *자기 직관 시전*은 *손가락 끝* / *손바닥*에서 *튀어나오는* 식으로 짧게.
- 시전 중 카엘의 *왜 이게 되는지 모름*을 1줄로 끼운다.

## 4. 주인공 (카엘) 말투 (Protagonist Voice)

### 4.1 원칙

- **위트 있되 짧다**. 카엘은 *말이 많은 인물이 아니다*.
- **자기 비하는 5~6부부터**. 1~3부는 *자기 정당화 + 거만한 격식체*.
- **내면 독백이 대사보다 많다**. 한 회차에 카엘 대사가 *10줄을 넘기지 않는다*.
- **핵심 말 하나, 침묵 하나, 행동 하나** — 카엘의 회차당 평균 발화 리듬.
- **농담은 자기 자신을 향한다** (4부 이후). 타인 조롱 형 농담 금지 (그건 1~3부 카엘의 *비뚤린 시선*과 구분되지 않는다).

### 4.2 시기별 대사 톤

| 시기 | 톤 | 예시 |
|---|---|---|
| 1~3부 (어리고 비뚤림) | 자기 정당화 + 격식체 + 거만 | *"이 자가 나를 모욕하려 하는군."* |
| Ch1 P6 (양팔 상실 직후) | 단호 + 짠내 | *"멀리도 날아왔네. 아직도 뱃속이 뒤집히는 거 같아."* |
| Ch2 P1~ (회복기) | 짧고 건조 + 첫 자조 | *"…내가 1년 만든 게, 다시 1년 살릴 돈으로 돌아오는군."* |
| 4부 (균열 자각) | 흐려지는 단호함 | *"… 그는 검을 내려놓는다. 나는 멈출 수 없다."* |
| 5~6부 (재정립) | 위트 + 자조 | *"나는 마법사인데 검을 배운다. 이상한 인생이다."* |

### 4.3 예시 — 좋음 (위트 + 핵심)

```
세파가 의수의 손가락을 시험 삼아 폈다 접었다 했다.
"움직이긴 하네요."
"칭찬으로 들리지 않는데."
"칭찬 아니에요."
카엘이 픽 웃다가 윽하면서 아파한다. 강철 의수의 검지가 그 박자에 한 번 굳었다.
"…웃지도 못하게 만드는군."
```

→ 카엘 대사 2줄. 둘 다 짧고 위트. *자조 톤* (강철 의수가 굳는 것을 자기 농담의 결과로 받음).

### 4.4 예시 — 좋음 (말 적고 생각 많음)

```
시르베인의 좁은 골목사이로 어디선가 망치 소리가 들렸다.
카엘은 저 소리는 지하에서 온다고 생각했다. '지하에 누가 있는가봐요.'
세파가 옆에서 뭐라 말했다. 그는 듣지 않았다.
'그리고 내 팔에는 지하에서 온 부품이 박혀 있다.'
"…뭐라고 했소?"
"아무것도 아니에요. 그냥 골목이 좁다고요."
"내 귀도 좁은가 보군.."
```

→ 카엘 대사 2줄(둘 다 1단어~1문장). 내면 3줄. *말이 적고 생각이 많다*의 표본.

### 4.5 예시 — 좋음 (극적 대사, 비트 종결)

```
세파가 그를 보았다. "당신은 마법사가 아니지만 검사도 아닌것 같아요."
카엘은 한참 말하지 않았다.
의수의 강철이 햇빛에 한 번 반짝였다.
"…" 카엘의 머릿속은 이미 인간들이 만든 직업의 경계는 관심조차 없었다.
```

→ 카엘 대사 1줄. 침묵 1, 행동 1, 단호 1. style-guide §1.3 *극적 대사 원칙* 정확히 준수.

### 4.6 예시 — 나쁨 (금지)

```
"하하, 이게 무슨 일이람! 두 팔을 잃었는데 의수까지 차게 되다니, 정말이지
인생이란 알 수 없는 것이군요. 세파, 당신은 어떻게 생각하시오? 우리가 이렇게
시르베인까지 오게 될 줄 누가 알았겠소? 나는 정말로..."
```

→ 카엘이 *말이 많다*. 농담이 *길다*. 자기 비하가 *과장됨*. **금지**.

같은 의미를 압축:

```
"의수 차고 시르베인이라. 한 달 전엔 못 했을 농담이오."
```

→ 1줄. 위트. 자조. 핵심.

### 4.7 카엘 대사 분량 가이드

- 회차당 카엘 직접 대사: **5~10줄** (한 줄 = 1~2문장)
- 회차당 카엘 내면 독백: **15~25줄** (한 줄 = 1문장)
- 비율: 대사 1 : 내면 2~3

### 4.8 대사 끝맺음

- *…* (말줄임표) — 카엘이 *말 끝을 흐릴 때*. 4부 균열 자각 후 빈도 증가
- *—* (긴 줄표) — *말 끊김 / 침묵 전환*. 1~3부 카엘의 *비뚤린 단호함* 표지
- *.* (마침표) — 평상시. 짧고 단정

## 5. 통합 예시 — 한 비트 전체 (Ch2 P1 1-4 시르베인 강철 의수 시술 도입)

> 위 1~4 모든 원칙이 한 비트(약 1,200자) 안에서 어떻게 함께 작동하는지의 표본.

```
시르베인은 망치의 도시였다.
변경 마을의 적막을 일주일 걸어온 카엘의 귀에, 그 소리는 차라리 폭력처럼 쏟아져 내렸다. 둔중하고 일정한 금속음 사이로 세파가 부축을 풀고 한 발 비켜섰다.
"여기서부터는 당신 다리로 걸어요."
"명령이오?"
"부탁이에요. 명령은 다음 골목에서 할 거예요."
카엘이 픽 웃다가 윽 하고 옆구리를 짚었다. 갈비뼈가 한 박자 늦게 시큰거렸다.
"…아직은 다음 골목까지가 멀군."

마법 장인의 공방은 좁고 낮았다. 늙은 장인은 손가락이 짧고 단단했고, 그가 카엘의 두 단면을 한 번씩 짚을 때마다 시선은 단면 위에 묶여 있었다.
"강철로 갈 거요. 마나 회로 미세 결각은 내가 직접 박지."
"부품은."
"…지하에서 온 것이오. 자세한 건 묻지 마시오."

카엘은 더 묻지 않았다.
'묻지 않는 게 예의인 곳이군.'
'지하에서 왔다라.. 그러면 지하에는 누가 또 있는가 봐요.'
세파가 옆에서 가방을 풀었다. 월광석 보석 장갑을 판 돈이 거기에 있었다.
'1년을 만들어 팔 한 짝을 잃고, 그 돈이 다시 1년 살릴 돈으로 돌아왔군.'
그 생각을 입 밖에 내지는 않았다.
대신 의수가 박힐 두 단면을, 강철 받침대 위에 가만히 올렸다.
```

→ 약 1,100자. 배경 묘사 3문장(망치의 도시·공방의 좁고 낮음)·인물 묘사 2문장(늙은 장인 손가락·시선이 단면 위에 묶여 있다)·대사 7줄(카엘 3 / 세파 2 / 장인 2)·내면 3줄. 적·전투 없음.

이 한 비트 안에 본 가이드의 1·2·4번 원칙이 모두 작동한다.

## 6. 회차 집필 직전 체크리스트 (4대 원칙 통과)

집필 시작 전 자문:

- [ ] **배경**: 배경 변경 시 묘사를 3~5문장으로 제한했는가? 카엘의 감각을 통과했는가?
- [ ] **인물**: 첫 등장 외양을 *지배 특징 + 모순 디테일*로 압축했는가? 머리부터 발끝까지 훑지 않았는가?
- [ ] **전투**: 전황 → 첫 합 → 위기 → 결단 → 마무리 5단 구조를 지켰는가? 적의 내면을 단 한 줄도 쓰지 않았는가?
- [ ] **카엘 말투**: 회차당 대사 5~10줄·내면 15~25줄 비율인가? 자기 비하 톤은 시기에 맞는가?

집필 직후 자문:

- [ ] 한 비트가 5줄 단독 풍경으로 시작되지는 않았는가?
- [ ] 외양 묘사가 *행동 없는 단독 문단*으로 떨어진 곳이 있는가?
- [ ] 전투 중 *합보 일지* 형식으로 늘어진 곳이 있는가?
- [ ] 카엘이 *5줄 이상 연속 발화*한 곳이 있는가? (4부 이전이면 금지)

위 4개 중 *예*가 하나라도 있으면 *재집필*. 4개 모두 *아니오*여야 통과.
