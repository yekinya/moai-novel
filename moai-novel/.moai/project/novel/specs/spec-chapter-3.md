## 1. Chapter 3 한 줄 시놉시스

> Chapter 2 종결 직후 — *카엘이 *셀라엔 망명지에서 회복기 진입*. 그 사이 *카자르드 (가명 카이엔 오르하임) 가 *변경 도시 인간계 유랑 + 카엘 행적 추적*. 두 사람의 양방향 추격이 *변경에서 천천히 좁혀짐*. **카엘이 동부마녀연맹에 *9혈 외부 손님 8번째로 공식 등록* + 모르간 린이엔과의 *외부 마검술 변종 사사 본격 진행***. 그 사이 *카르샤 변경에서 *첫 마계 영지 균열 핵 1체 발견* — *카엘·카르샤 잔존·마녀령·앨더스톰의 *5중 비공식 동맹 임시 활성화*. **카엘 vs 카자르드 첫 직면 + 첫 결전 호각** — *결정타 없음*. 그 와중에 **첫 마왕 강림** — *카엘의 *4단계 의수 + 공허검 + 마력→내공의 정점 시도 + 5중 동맹의 합산 의식* 으로 *간신히 마왕 처치*. *카엘 의수 첫 파괴 + 회복 수개월 시드*. 그리고 **행커만 깨어남 가능성의 첫 시드** — Chapter 4 진입의 *세계관적 전환점*.

## 2. Chapter 3 핵심 서사 축 (5개)

### 축 A — 카엘 vs 카자르드 *양방향 추격*

- **A1**: Ch3 P1 — 카자르드 인간계 풍문 첫 등장 (변경 도시 객점·시장에 *수염 없는 30~40대 남성 검사 + 왼쪽 눈 황금 반점* 풍문)
- **A2**: Ch3 P2~P3 — 카엘이 *마녀령·카르샤 변경 활동 중 *카자르드 동선 추정 정보 점진 누적*
- **A3**: Ch3 P4 — **카엘 vs 카자르드 (카이엔 오르하임) 첫 직면**. 변경 도시 객점에서 *짧은 대화 + 결투 회피*. 카엘의 *팔찌 부분 붉은 빛 발현 + 자기 정체 확인 매개*. *서로의 정체를 *알면서도 *짧게 헤어짐*
- **A4**: Ch3 P5 — **첫 결전 호각**. *카엘 4단계 의수 + 공허검 정점 + 마력→내공 vs 카자르드 그린드래곤 용언 3종 + 독 정통 첫 본격*. *결정타 없음 → 둘 다 짧은 부상 후 분리*
- **A5**: Ch3 P7~P8 — 카자르드가 *마왕 강림 시점에 *짧게 카엘 측 매개로 우연한 합류*. *임무 우선이나 *마왕의 위협 앞에 *짧은 협공 단편*. 단 *Ch3 종결 시점에는 *임무 미완 상태로 이별*

### 축 B — *9혈 원탁 외부 손님 8번째 공식 등록*

- **B1**: Ch3 P2 — *모르간 솔테의 공식 등록 결단*. 카엘이 *9혈 외부 손님 정통의 8번째*. Ch2 P3 시드의 본격 회수
- **B2**: Ch3 P2 — 모르간 린이엔과의 *외부 마검술 변종 사사 본격*. *흑혈 + 백혈 + 황실 + 카엘의 마검사 변형의 4중 정통 매개*. 린이엔의 *조용한 관찰 + 호감 + 거리 두기의 복합 감정 진화*
- **B3**: Ch3 P6 — 흑혈 도장 추가 사사 + *공허검 3원리 (허) 진입 시도*. *카엘의 비뚤린 시야 50%가 *60%로 약간 회복 후 *허 1단계 시도 가능 도달*

### 축 C — *카르샤 잔존 공식 동맹 활성화*

- **C1**: Ch3 P3 — *Ch2 P8 §21 비공식 → 공식 전환 시드*가 *카르샤 변경 마계 균열 핵 발견 매개로 본격 활성화*
- **C2**: Ch3 P3 — *왕비 엘라리아·셀라엔 대주교·키얀 왕자 (10세) + 카르샤 잔존 신성검 호위의 *공식 동맹 라인*
- **C3**: Ch3 P6 — *5중 동맹 활성화* — 카엘 + 카르샤 + 마녀령 + 앨더스톰 + 강철제국 (세이른 라인) + 드워프 (콘드라 그라드네의 짧은 매개)
- **C4**: Ch3 P8 — 마왕 처치 후 *공식 동맹 정착*. *Ch4 본격 활성화의 매개*

### 축 D — *마계 영지 균열 핵 + 첫 마왕 강림*

- **D1**: Ch3 P1 — 카르샤 변경 첫 마계 영지 균열 핵 풍문 (Ch2 P8 §21 시드의 본격 확인)
- **D2**: Ch3 P3 — *균열 핵 1체 위치 확정 — 카르샤 변경 외곽 마을 (작명 후속, 임시 *카르덴 마르*)*
- **D3**: Ch3 P6 — *처치 의식 매개 — 균열의 빛 + 회수 회로 결합 의식 + 카르샤 신성 신앙 + 마녀령 흑혈 정통 + 카엘의 4단계 의수 + 공허검 정점 + 마력→내공 = 5중 합산*
- **D4**: Ch3 P7 — **첫 마왕 강림** — *균열 핵 위에서 압도적 외형 + 본격 위협*. *마왕 = 마계 영지 군주 + 균열 핵의 *통합체*. 외형 = *키 약 5m + 다중 팔 + 검은 갑옷 + 붉은 눈 + 마계 잔향*
- **D5**: Ch3 P7~P8 — *간신히 처치* — *5중 동맹 + 의식 매개*. 카엘 의수 첫 파괴 + 회복 수개월

### 축 E — *행커만 깨어남 가능성 시드*

- **E1**: Ch3 P8 종결 — *드래곤 종족의 *느슨한 인지 + 행커만 깨어남 첫 미세 신호*. 드래곤 로드의 *짧은 동요*
- **E2**: Ch3 P8 종결 — 카엘의 *조용한 인지 — 다음 위협이 *드래곤 본태 일지도*
