# Chapter 1 Part 5 Spec: 왕을 죽인 날 — *50% 각성*

## Summary

세번 강 회전 → 세이렌 함락 → 옥좌의 방 진입 → **프롤로그 완전판 (왕 살해)** → 평화 협정 발견 + 각성. 본작 *50% 시점*. 시점 1인칭 본격 전환.

## 회차 구성 (5회차)

| 회차 | 제목 | 비트 |
|------|------|------|
| 4-1 | 마법으로 그은 선 | 세번 강 광역 학살 1,400명 + *형태 해체* |
| 4-2 | 함락 | 세이렌 외성벽 + 항복 사절단 *예방적 섬멸* + **머리카락 단서 회수** |
| 4-3 | 옥좌의 방 | 백색 회랑 진입 + 카일렌 *왕이여, 도망치소서* + 첫 *과거형 자기 의심* |
| 4-4 | **너는 무엇을 보고 있는가** | **프롤로그 완전판** — 5합 결투 / 알레이언 정식명 / *7대 기다림* / **키얀 왕자 도주** |
| 4-5 | 회랑의 거울 | 평화 협정 발견 + *앤트로퍼* 단어 첫 등장 + **손의 떨림** 신체화 |

## 핵심 비트

1. **시점 1인칭 전환** — 본작 *지배 시점 본격화*. 프롤로그 1회뿐이었던 1인칭이 이제 정주
2. **광역 마법 정점** — 4-1 압축 화염구 + 중력 응용 + *형태 해체* 의 9써클급 정점
3. **프롤로그와의 합류** (4-4) — 모든 단서 회수. *서술 톤 동일 + 새 디테일 (정식명·7대 기다림·키얀 왕자)*
4. **앤트로퍼 단어 첫 등장** (4-5) — 본문에 *앤트로퍼* 단어가 *처음* 직접 등장. 그 전까지는 *얇은 금속음·이중상·붉은 사슬* 으로만 암시
5. **각성의 신체화** — 손의 떨림 = *살의가 풀리지 않은 잔압*. 앤트로퍼 *충동* 의 첫 자각

## 시점·문체

- **1인칭** (카엘) — Part 5 전체
- 4-3까지는 *왜곡 유지*. 4-3 회랑에서 *첫 과거형 자기 의심* (*그때 나는 그것을 영창으로 들었다*)
- 4-4는 *프롤로그와 동일 텍스트가 절반 + 새 디테일 절반*. *재진술의 무게*
- 4-5는 *균열 이후* — 가장자리 이중상이 *중앙으로 들어옴*

## ★ Review points

1. **4-4 프롤로그 완전판 vs 프롤로그의 *재진술* 정도**:
   - 현재 안: 동일 사건 + 새 디테일 (정식명·7대·키얀). 5합 결투는 *완전 재진술*
   - 대안 A: 5합 결투 *짧게 압축* — *프롤로그를 다시 읽지 않는다* 정통
   - 대안 B: 5합 결투 *완전 재진술* — *프롤로그가 끝나고 다시 시작*
2. **4-5 *손의 떨림*** 명시 강도 — 현재 안: 한 줄 명시 *손이 떨렸다 — 두려움이 아니라 살의가 풀리지 않은 잔압*. 더 미묘하게? 더 강하게?
3. **키얀 왕자 도주 (4-4)** — 알레이언이 *마지막 힘으로 출구 쪽으로 밀어줌*. 9세 키얀이 *완전히 도주* vs *카엘의 시야 가장자리에서 본 흐릿한 그림자만*?
4. **4-2 어린아이 화환잡이 사망** — 머리카락 목걸이 회수 매개. 본문 분량? 1줄 / 1문단 / 1페이지?
