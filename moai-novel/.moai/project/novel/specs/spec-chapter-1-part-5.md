## Summary

**바이서스 제국 (공격측)** 의 *카르샤 제국 (방어측)* 침공 전쟁. 세번 강 회전 → 세이렌 함락 → 옥좌의 방 진입 → **프롤로그 완전판 (왕 살해)** → 평화 협정 발견 + 각성.

본 Part 의 정치 본질 — **카엘은 라이세린·바이서스 왕실의 *1회성 조커 카드***. 왕실 직속 *8써클 전쟁 참모 호스반 셰이드런 경* 이 *승리의 판은 깔되 절대 드러나지 않는 선* 에서 카엘을 감시·지원·최종 제거 임무. 알레이언 6세는 *카엘이 연구하던 금지 비기의 독* 으로 옥좌 진입 전 *이미 중독 상태* — 카엘 제거의 씨드 (왕의 죽음이 *독 누명* 으로 카엘에게 돌아옴). *50% 시점*. 1인칭 본격 전환.

## 핵심 비트 (사용자 보강 7항 통합)

### 1. 첫 전쟁 진입 시 카엘 상태 — *미숙 + 정의감* + *비정상 승리*

- 카엘의 실제 마법 숙련 = **3등급 정통 + 비상식 응용** (Part 4 캐논). 정통 5써클 시전은 *극히 일부만* 가능
- 첫 *대규모 전쟁 + 단신 광역 학살* 은 본인에게도 *미숙·낯섦*. *내가 이정도였나* 의 충격이 4-1 핵심
- **정의감 존재** — 카엘은 *바이서스 = 정의·카르샤 = 부정의* 의 라이세린·호스반 서사를 *그대로 믿음*. 의식 표면에서 *내가 정의를 집행한다* 의 확신
- *정상적 방식 (정통 5써클 + 군 지휘) 으로는 승리 불가* — 카엘이 *모든 수단 (정통 + 비상식 응용 + 금지 비기 + 아티팩트 + 회복 약 + 살상 직관) 을 모두 동원* 해야 가능한 *비정상 승리*
- 이 비정상성이 *후일 잔혹함의 소문·정의롭지 못함 비판* 의 정통 근원

### 2. 정치 획책 — *카엘 = 1회성 조커 카드*

#### 카엘의 직급 전환

- Part 4 종결: *외곽 연구소 비정규 계약직 → 외교실 자문* 흡수 완료
- 본 Part 출정 시점: **바이서스 *왕실 직속 전선 주력 마법사*** 로 다시 한 단계 이동. *공식 칭호 — 5써클 인증 + 라카르 9세 직속 *전선 자문 마법사**
- 의미: *왕실 권위 명목 + 풍족한 자원 + 후일 책임 떠넘기기 용이성*. 라이세린의 *흡수 → 라카르 9세에게 *기증*** 의 형태

#### 풍족한 지원 (1회성 조커 활용 정통)

- *왕실 직속 전선 자문* 자격으로 *바이서스 왕실 보급창 자유 출입권*. 일반 마법사는 *허가증 필요·재고 한정* 이지만 카엘은 *원하는 만큼 가져갈 수 있음*
- 보급창 항목:
  - **마나 결정** (시전 보조) 다량
  - **회복 약** (성소 성수·마나 의료원 약·고급 치유 영약) 사실상 무한
  - **마법 양피지** + 마법 자료 (도서관 외부 반출 가능)
  - **아티팩트 시제품** (왕실 마도공학 연구소 시제품 — 외부 미공개)
  - **금지 분야 자료** (라이세린이 *눈감음*)
- **라이세린·호스반·라카르 9세의 의도** — *카엘을 *원하는 만큼 부풀린 도구* 로 만든 후 *전쟁 종결과 함께 폐기**. 풍족함이 *후일 책임 떠넘기기의 명분*

### 3. 카엘의 *특이한 연구 습성* (풍족한 지원 활용)

- 정통 마법사는 *마법 연구만* 하지만 카엘은 **마법 + 아티팩트·아이템 연구 동시 진행**:
  - **마법 연구**: 정통 3~5등급 학습 지속 + 비상식 응용 확장 + 금지 분야 *엔트로피 역연산·바람 통로 + 독·살상 직관 시전* 시도
  - **아티팩트 연구**: 바이서스 마도공학 연구소 시제품을 *분해·재조립·자기 마나 회로에 맞춘 개조*. 본 Part 종결 시 *카엘 전용 마나 보조 팔찌 + 비상 회복 캡슐 + 살상 직관 보조 단검* 의 *초기 자기 제작 아티팩트 3종* 보유. **6부 의수 정통 시드** (두 팔 상실 후 *자기 제작 의수* 의 정통)
- *회복 약·자기 안위 아이템* 적극 획득 — *1.5부 떠돌이 시기의 *부상 누적 트라우마*** 가 *야지 생활 + 전쟁* 에서 *과보상* 형태로 발현. *카엘 = 회복 약 다량 휴대* + *야영지에서 자기 약품 정리 정통*
- 정통 학자들은 카엘의 연구 습성을 *학자답지 못함·잡상인* 으로 비판. 카엘은 *살아남는 자가 옳다* 의 무의식 확신

### 4. 호스반 셰이드런 경 — *8써클 전쟁 참모 + 감시자 + 제거자*

#### 정체

- **호스반 셰이드런 경** (Hosban Sheidren) — 바이서스 왕실 직속 *8써클 전쟁 참모*
- 50대 후반. 라카르 9세 직속 수하. *공식 신분은 *전선 자문 마법사*** 이나 실은 *왕실의 그림자 손*
- 본 Part 이전 캐논에서 단편 언급 (전쟁 참모) → **본 Part 에서 정식 인물화**

#### 임무 (3중)

1. **카엘 감시** — 카엘의 *비상식 응용·금지 비기·아티팩트 연구* 의 모든 정황을 *라카르 9세에게 정기 보고*. 위험 임계 시 *즉시 제거 권한* 보유
2. **전쟁 도움** — 8써클 시전 능력 + 전선 경험 으로 *카엘이 보지 못하는 *후방 전략 판** 을 깔아 줌. 카엘이 *내가 정의로 이긴다* 고 믿는 동안 호스반이 *실제 승리의 뼈대* 를 깔고 있음
3. **최종 제거** — 전쟁 종결 직전 *카엘 제거*. 단 **절대 드러나지 않는 선** — *카엘이 *자기 책임·자기 행동* 으로 죽는 정황 연출*

#### 호스반의 *판 깔기*

- 4-1 세번 강: *카엘의 광역 학살이 *전군 정면 돌파* 의 정통성을 갖도록 후방 정찰·기상 조작·적 보급 차단* 을 *동시 진행*. 카엘은 *내 마법만으로 승리* 라 믿음
- 4-2 세이렌: *항복 사절단의 *진짜 의도가 정찰 + 암살*** 정황 *조작* 으로 카엘에게 흘림. 카엘의 *예방적 섬멸 정당화*
- 4-3 회랑: *알레이언이 이미 중독 상태로 만들어 둠* (4 항)
- 4-5 평화 협정: 카엘이 *진실을 보기 직전* 호스반이 *제거 시도*. 실패

#### 호스반의 한계 — *드러나지 않는 선*

- 호스반의 모든 작전은 *외부에서 보면 카엘 단독 행위* 여야 함. *조작의 흔적이 남으면 호스반·라이세린·라카르 9세 모두 *책임 떠안기 불가**
- 이 *드러나지 않는 선의 어려움* 이 호스반의 *실수 가능성*. 4-5 *카엘 제거 시도 실패* 는 호스반의 *처음으로 드러난 흔적*. 카엘이 *후일 회수* (Part 7 결전 편지)

#### 호스반의 카엘 인식

- *비뚤린 천재 + 통제 불가능한 잠재력*. *살려두면 *바이서스에게도 칼날*** 의 직감
- 그러나 카엘의 *살상 효율* 은 *전쟁에 필수*. *쓴 후 버린다* 의 정통 1회성 조커 인식

### 5. 알레이언의 *중독 상태* — *카엘 제거 씨드*

#### 사건 정황

본 Part 시작 *훨씬 이전부터* 호스반이 *카엘의 금지 비기 연구 — *바람 통로 + 독*** 자료를 *흘려서 알레이언 측 어의에게 도달*. 알레이언의 *어의 1명이 매수* 되어 *왕의 일상 차에 미량 투여*.

- **독의 정체**: 카엘이 외곽 연구소 5년 차 시기 *카르엔 도서관 금지 자료* 에서 발견한 *바람 통로 + 신경 마비 독*. *마나 회로 흐름 둔화 + 검 휘두름 속도 30% 저하* 효과. 즉발성 아닌 *수개월 누적형*
- **투여 기간**: 본 Part 6개월 전 ~ 본 Part 시점. 알레이언의 *몸 둔함의 자각 + 의구심* 은 4-3 카일렌의 *왕이여, 도망치소서* 의 진짜 근거 (카일렌만 어의 매수 정황 일부 의심)
- **알레이언 4-4 마지막 대사**: 새 추가 — ***"너희가 무엇을 내게 먹였는지 알고 있는가"*** + *5합에서 점점 검이 무거워지는 자각*. 카엘이 *처음 듣는 정황*. 4-5 평화 협정 발견 시 *이 한마디가 메아리* 로 돌아옴

#### 정치적 의도

- **카엘이 *진짜 5합으로 알레이언을 이겼다는 환상*** 을 깨지 않도록 외부에는 *카엘의 광역 마법 + 5합 결투 승리* 로 포장
- 그러나 *왕의 사인이 *독 누명* 으로 카엘에게 돌아옴* — 후일 *바이서스 왕실이 카엘을 *제거할 명분*** 으로 활용
- **카엘 제거 씨드** — 본 Part 에서 *발아 (호스반이 알레이언 시신에서 *독 흔적 증거 채취*)*. 회수 = Part 6·7 *카엘 추방·결전 편지의 정통 명분*

#### 카엘까지 죽일 계획 — *실패*

- 호스반의 4-5 *제거 시도* — 카엘이 평화 협정을 발견한 순간 *마지막 약점 노출*. 호스반이 *백색 회랑 후방에서 마법 일격* 시도
- 카엘의 *살상 직관* 이 무의식적으로 *방어 시전*. 호스반의 일격이 *카엘 옆 기둥에 빗나감*
- 호스반 *즉시 후퇴 + 작전 위장* — *바이서스 후방 정찰병 1인이 *카르샤 잔당과 교전* 으로 위장*
- 카엘은 *희미하게 이상함을 느끼나 곧 잊음* — 평화 협정 충격 + 손의 떨림이 *모든 의식을 잠식*

### 6. 군사 자원 — *바이서스 압도적*

| 사기 | *변경 정세 격화 명분 + 라카르 9세 친정* | *방어 + 가족 보호 + 신성검단 의지* |

→ **바이서스 군이 *카르샤보다 모든 면에서 압도적***. 정상적 전쟁이라면 *바이서스의 점진적 승리* 가 예상되나 *전쟁이 빠르게 끝나야 하는 정치 일정* 이 있음 (라카르 9세의 *재위 30주년 기념 + 라이세린의 외교실 권력 공고화*).

#### 카엘의 역할

- *압도적 양에도 불구하고 *전쟁 빠른 종결을 위해 카엘의 광역 학살 필수**. 단 *카엘 없이도 바이서스가 결국 이긴다*. 카엘은 *시간 단축의 도구*
- 이 *카엘 없어도 되는 정황* 이 *1회성 조커 활용의 정통 근거*. 카엘이 *과대평가된 자기 인식* 으로 *내가 이긴 전쟁* 이라 믿는 것의 *실제 진실*

### 7. *카엘의 모든 수단 동원 + 잔혹 소문 + 정의롭지 못함 비판*

#### 모든 수단

- 정통 1~5등급 + 비상식 응용 + 금지 비기 (바람 통로 + 독·살상 직관 시전·엔트로피 역연산) + 아티팩트 보조 + 회복 약 무한 + 살상 충동 *자기 정당화*
- **본 Part 가 *카엘의 모든 도구 첫 본격 동원***. 4-1 광역 학살이 *카엘 본인도 충격받는 위력*
- 4-2 항복 사절단 *예방적 섬멸* = *전쟁 윤리 명백한 위반*. 그러나 카엘은 *호스반이 흘린 *암살 위장 의심***에 *정당화*

#### 잔혹 소문 시작

- **바이서스 군 내부 소문**: *그 자는 마법사가 아니라 *학살자*. *바이서스 군 4-2 진입 시 *카엘 옆에 서지 않으려 함***
- **카르샤 측 인식**: *바이서스의 *흑마법사·악마 화신*. 카르샤 망명자들 (왕비 엘라리아·셀라엔 대주교) 이 *외부로 카엘의 잔혹성* 전파
- 본 Part 종결 시 카엘은 *바이서스 내부에서도 *불편한 존재*. *전쟁의 영웅 + 인간으로서는 가까이 하기 싫음*** 의 모순 위치

#### 정의롭지 못함 비판

- 4-4 알레이언의 *너는 무엇을 보고 있는가* — 본 Part 의 *정의롭지 못함 비판 의 정통 시점*
- 4-5 평화 협정 발견 후 카엘 자신의 *처음으로 자기 정당성 의심*. 그러나 *즉시 자기 부정 — 그래도 바이서스가 옳다·내가 옳았다*

#### 독 누명

- **경제적 제재**:

  - **자원 수탈** — 카르샤 *마나 광산 2곳 + 곡창지 1곳 의 바이서스 직속 운영*

- **종교·문화 탄압**:

  - *카르샤 신성 신앙 격하* — 바이서스 *주신 신앙으로 점차 전환 유도*
  - *카르샤 전통 의례 + 명절 금지* (10년간)

#### 카르샤 잔당 + 외부 망명 정통

## 카엘의 *옥좌의 방 진입 정통* (4-3·4-4)

이전 캐논 = *전권 사령관 지휘권으로 직접 진입* → **폐기**.

새 정통 = **라이세린·호스반의 *비밀 단신 작전 명령***:

- 4-1 세번 강 회전 = *카엘 단신 광역 마법* 으로 카르샤 1열 격파 (정통 5써클 + 앤트로퍼 9써클급 위력 + *호스반 후방 판 깔기*). 카엘 본인도 *내가 이정도였나* 충격
- 4-2 세이렌 포위 = 정규 사령관 *카엘론 군 총사령관* 지휘. 카엘은 *결계 분쇄 광역 마법 + 첫 진입로 개척* 담당. *호스반의 항복 사절단 정찰·암살 의심 흘림*
- 4-3 회랑 진입 = **라이세린·호스반의 *비밀 단독 작전 명령*** — *왕을 직접 살해. 모든 정황은 본인이 책임진다.* 카엘이 *5써클 + 앤트로퍼 본능적 광역 살상 직관* 으로 단신 침투. 신성검단 30명 학살 단신 가능. *알레이언은 이미 중독 상태*
- 4-4 옥좌의 방 = 카엘 단신 + 알레이언 6세 1:1. *프롤로그 완전판*. *독으로 둔해진 왕* 의 *5합 시간 단축*
- 4-5 회랑 거울 = 카엘이 평화 협정 발견 + *호스반의 카엘 제거 시도 실패*
- 즉 카엘의 *옥좌의 방 진입은 *공식 사령관 지휘권* 이 아닌 *라이세린·호스반의 음모 작전*** 의 결과. 본 캐논의 *진짜 음모성* 강조
