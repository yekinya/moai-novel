## Summary

**바이서스 제국 (공격측)** 의 *카르샤 제국 (방어측)* 침공 전쟁. 세번 강 회전 → 세이렌 함락 → 옥좌의 방 진입 → **프롤로그 완전판 (왕 살해)** → 평화 협정 발견 + 각성.

본 Part 의 정치 본질 — **카엘은 라이세린·바이서스 왕실의 *1회성 조커 카드***. 왕실 직속 *8써클 전쟁 참모 호스반 셰이드런 경* 이 *승리의 판은 깔되 절대 드러나지 않는 선* 에서 카엘을 감시·지원·최종 제거 임무.

**가장 중요한 정통 1** — 카엘은 *바람 통로 + 독 비기를 연구만 했지 실전에서는 직접 사용하지 않음*. 그러나 호스반이 *카엘의 비기 자료를 무단 합성* + *카르샤 주요 강·식수원에 다량 독 투입* (바이서스 군 일부 군사 실수로 가장). 카르샤 군·시민·왕실 *광범위 중독*. 세이렌 외부 봉쇄 + 보급·의약 차단 상황에서 **알레이언 6세가 *자기 해독 물품까지 모두 군사들에게 분배* + 자신은 *해독 거부*** — 4-4 5합의 *진짜 둔함의 정통* + 카르샤 측 *카엘 = 학살자* 호칭의 정통 근원.

**가장 중요한 정통 2** — *카엘의 광역 학살은 외관상 그렇게 보일 뿐 실제는 합작*. 카엘의 마법은 *실제 3등급 + 비상식 응용* 으로 *광역 학살 1,400명을 단독으로 수행할 위력이 절대 아님* (사용자 HARD — *개연성을 해칠 정도의 강함 묘사 금지*). 실제는 **호스반의 *광범위 중독 + 마도공학 공성기 + 독 화살촉 + 결계 분쇄 폭열탄* 합작 + 카엘의 *4중 효율 우수성 + 백병전·짠내·응용*** 의 *합산 효과*. 외부에 *카엘의 단독 광역 학살로 보이도록 호스반이 시각 효과 조작*. *50% 시점*. 1인칭 본격 전환.

## 핵심 비트 (사용자 보강 7항 통합)

### 1. 첫 전쟁 진입 시 카엘 상태 — *미숙 + 정의감 + 마검사 전투 스타일*

- 카엘의 실제 마법 숙련 = **3등급 정통 + 비상식 응용** (Part 4 캐논). 정통 5써클 시전은 *극히 일부만* 가능
- **광역 학살 1,400명의 단독 수행은 *위력적으로 절대 불가능*** (사용자 HARD — *개연성을 해칠 정도의 강함 묘사 금지*)
- 첫 *대규모 전쟁* 은 본인에게도 *미숙·낯섦*. *내가 이정도였나* 의 충격이 4-1 핵심 — 단 이 충격의 *진짜 정통* = *내가 한 게 아닌데 내가 한 것처럼 보임* 의 *미세한 위화감 + 자기 부풀림 동시 작용*
- **정의감 존재** — 카엘은 *바이서스 = 정의·카르샤 = 부정의* 의 라이세린·호스반 서사를 *그대로 믿음*. 의식 표면에서 *내가 정의를 집행한다* 의 확신
- *정상적 방식 (정통 5써클 + 군 지휘) 으로는 승리 불가* — **그래서 호스반의 *합작 + 시각 효과 조작* 이 필요**
- 이 비정상성이 *후일 잔혹함의 소문·정의롭지 못함 비판* 의 정통 근원

### 1-2. 카엘의 *실제 능력* — 4중 효율 우수성 (사용자 HARD)

카엘의 공식 등급 = 5써클 (카르엔 부풀림). 실제 = 3등급 + 비상식 응용. *최대 위력은 5써클을 넘지 않음*. 그러나 *4중 효율 측면에서 정통 동급 마법사보다 우수*:

| 효율 항목 | 카엘 수치 | 정통 5써클 동급 | 비고 |
|---------|----------|----------------|------|
| **마나의 양 (총량)** | 일반 5써클의 *1.5배* | 1.0 | 앤트로퍼 *역마나 흡수* 자동 효율. 자체 회로의 *마나 저장 용량 ↑* |
| **마나 회복 속도** | 일반의 *1.5배* | 1.0 | 회복 약 + 자기 직관 회로의 *짧은 휴식 효율 ↑*. 4-2 진입 시 *5분 회복으로 다음 광역 시전 가능* |
| **스킬 효율성 (마나당 위력)** | 일반의 *1.3배* | 1.0 | 비상식 응용의 *불필요한 절차 단축*. 발바닥 빙판이 광역 빙판의 *마나 1/5* 로 동일 효과 |
| **스킬 시전 속도** | 일반의 *1.5배* | 1.0 | 1단계 직관 시전 (정통 5단계 시전 → 카엘의 1단계 시전). 정통 마법사가 *영창 5초* 를 카엘은 *직관 1초* |

→ **카엘의 진짜 강점** = *지속 화력 + 빠른 회복 + 효율 + 속도*. *단 최대 위력은 5써클 수준 (실제는 3등급)* — 이 정통이 *카엘이 *광역 학살의 외관 효과* 만 단독으로 만들 수 있고 *실제 사살 수는 호스반의 합작 결과* 의 인과 근거*

#### 용병 습관 + 정통 기사단 못마땅함

- 카엘의 검술은 *1.5부 떠돌이 시기의 *짠내 용병 생활* 습관 + 외곽 연구소 3년 학습* 의 *모자이크*
- *정통 기사단의 기사 = 못마땅* — 카엘은 *기사 명예·기사 정신·전통 검술 자세를 *답답함·허세*** 로 인식. *살아남는 것이 옳다* 의 용병 철학 우위
- 정통 기사가 *명예의 한 합* 을 시도할 때 카엘은 *빙판 활주로 회피 + 등 뒤에서 단검* 의 *비기사적 응답*. 정통 기사가 *비겁하다* 비판하면 카엘은 *살아남았으니 옳다* 일축

#### 카엘의 *특이 검술 5종* (사용자 보강 — 핵심 묘사 자산)

검술 기초(상) + 응용 마법 접목 = *정통 검사가 짐작 불가능한 변칙 검술*. **본 Part 의 신선한 묘사 정점**:

1. **바람 손날 — 갑옷 자상**
   - 검을 휘두름과 동시에 *바람 마법 1단계* 시전 → *검 끝에 얇은 바람 칼날 부착*
   - 정통 기사 갑옷의 *접합부 (목·겨드랑이·관절)* 에 *얇은 자상* 을 남김. *치명상은 아니나 누적되면 마비*
   - 정통 기사 시각 = *검만으로 이런 자상이 나올 수 없다* 의 *혼란*

2. **검 막은 상대 화염 폭발**
   - 카엘의 검을 *상대가 받아쳐 막는 순간* — 검 면에서 *1초 시전된 화염구 폭발 발동*
   - 상대 검을 *손에서 놓치거나·얼굴 화상·갑옷 손상*. 카엘 본인은 *내열 마법 결계로 보호*
   - 정통 기사 시각 = *검 받침이 *공격이 되는* 변칙*. 두번째 검 받침을 망설임 → 카엘의 진입 기회

3. **빙판 변칙 보법**
   - 발바닥에 *빙판 마법 1초 시전* → *적의 발 앞 30cm 빙판 즉석 생성*
   - 적이 *발 디딤 실패 + 균형 무너짐* → 카엘이 *역방향 단검 진입*
   - 또는 *카엘 본인 발바닥 빙판 활주* 로 *예측 불가능한 보법* 발현. 정통 기사의 *발 디딤 예측 검술* 무력화

  - **카엘이 *동시성 제어 아티팩트* 사용** — *1년 전부터 준비한 1회 한정 아티팩트*. 본 작전을 위해 *Part 4 종결 시점에 완성*. 카엘이 가진 *가장 강한 마법인 5써클 광역 화염* 에 *1차동 동시 시전 마법* 을 더해 *압축률 극대화*. *총 4가지 마법 효과 (광역 화염·바람 폭풍·중력 응축·광명 직격) 응용* 으로 폭발력 증가
  - **현재 카엘은 동시성 시전이 어려운 한계** — 정통 5써클 마법사도 *2개 마법 동시 시전이 한계*. 4개 동시는 *7써클 이상에서나 가능*. 카엘은 이 한계를 *1회 한정 동시성 제어 아티팩트* 로 *기계적 보완*
  - **카엘의 준비로 인한 광역 폭발이 *생각외로 강력함* 을 확인** — 본인도 놀람. *내가 이정도였나* 의 충격이 *부분적으로는 진짜* (압축률 + 4종 응용의 본인 천재성 일부 인정)
  - **호스반이 *광역 6써클 마법 몰래 동시 시전*** — 카엘 모름. 카엘 폭발 시각 효과 안으로 합성. *합쳐서 거의 7~8써클 초반 위력* (참고: *메테오 = 9써클 중반*)
  - 강 독살로 *이미 중독된 카르샤 1열* 이 *방어력·기동력 저하* — 같은 위력 마법에 *3~4배 피해*
  - 외부에서는 *카엘의 광역 마법 한 방으로 1,400명 학살* 로 보임
  - 카엘 본인은 *내가 한 것* 으로 오인 + *내가 이정도였나* 의 충격 — *미세한 위화감* 만 의식 표면 아래에 (전체 폭발의 50% 정도는 *호스반 합작* 으로 의식 깊은 곳에서 의심)
  - **카엘의 *5써클 마법 + 동시성 아티팩트 단발* 후 카엘 본인 마나 거의 고갈** — 4-1 후반부 카엘은 *백병전·짠내·응용 마검사 모드* 로 전환. 광역 학살은 *단 1회만* (이후 광역 학살 외관은 *호스반 단독 작전 + 카엘 백병전 동시 묘사* 로 외부 인식)
  - 외부 정찰·기상 조작·적 보급 차단도 *동시 진행*
- **4-2 세이렌**: *항복 사절단의 *진짜 의도가 정찰 + 암살*** 정황 *조작* 으로 카엘에게 흘림. 카엘의 *예방적 섬멸 정당화*. 동시에 *호스반의 결계 분쇄 폭열탄* 으로 세이렌 외성벽 결계 분쇄 → *카엘의 진입로 개척 외관 부여*
- **4-3 회랑**: *알레이언이 이미 강 독살로 중증 중독 상태로 만들어 둠* (§5 참조)
- **4-5 평화 협정**: 카엘이 *진실을 보기 직전* 호스반이 *제거 시도*. 실패

#### 호스반의 한계 — *드러나지 않는 선*

- 호스반의 모든 작전은 *외부에서 보면 카엘 단독 행위* 여야 함. *조작의 흔적이 남으면 호스반·라이세린·라카르 9세 모두 *책임 떠안기 불가**
- 이 *드러나지 않는 선의 어려움* 이 호스반의 *실수 가능성*. 4-5 *카엘 제거 시도 실패* 는 호스반의 *처음으로 드러난 흔적*. 카엘이 *후일 회수* (Part 7 결전 편지)

#### 호스반의 카엘 인식

- *비뚤린 천재 + 통제 불가능한 잠재력*. *살려두면 *바이서스에게도 칼날*** 의 직감
- 그러나 카엘의 *살상 효율* 은 *전쟁에 필수*. *쓴 후 버린다* 의 정통 1회성 조커 인식

### 5. 알레이언·카르샤 중독 정통 — *호스반의 강 독살 + 왕의 자기 희생* (v2 재구성)

#### 핵심 정통 — 카엘은 *실제 전쟁에서 금지 비기·독을 직접 사용하지 않음* ★

- **카엘이 보유한 *바람 통로 + 독* 비기** = 외곽 연구소 5년차 *카르엔 도서관 금지 자료* 연구품. **카엘은 *연구만 했지 실전 사용 의도 없음***
- 카엘 본인의 의식 = *비기는 *마지막 카드*. 실전에서는 정통 마법 + 비상식 응용만으로 충분*. 본 Part 전체에 걸쳐 *카엘이 *직접 독·금지 비기를 시전하지 않음***
- 그러나 후일 *카르샤 측에서 카엘을 *학살자* 로 부르는 호칭의 정통* 이 본 Part 에서 발아 — **카엘이 실제로는 안 썼는데도 *모든 책임이 카엘에게 돌아오는 정황***. *비뚤린 시야 너머의 카엘은 정의로운 본성* (Chapter 1 사용자 HARD 룰) 의 *외부 비대칭 인식* 정통

#### 사건 정황 — 호스반의 *강 독살 뒷 획책*

본 Part 시작 *훨씬 이전* — 4-1 세번 강 회전 직전 + 4-2 세이렌 포위 진행 중:

1. **호스반의 *카엘 비기 자료 흘림 + 자체 합성***:
   - 호스반이 *카엘의 *바람 통로 + 독* 연구 자료를 *몰래 복사* + *바이서스 왕실 마도공학 연구소에서 *대량 합성***
   - 합성된 독은 *마나 회로 흐름 둔화 + 검 휘두름 속도 30% 저하 + 마법 시전 영창 길어짐* 의 *수일 누적형*
   - **카엘은 *합성 사실도 모름*** — 카엘의 비기 자료를 호스반이 *기록물 보관실 자유 출입권* 으로 무단 복사

2. **강 독살 작전 (4-1 직후 ~ 4-2 진행 중)**:
   - 호스반이 *일부 바이서스 군사 4~5명* 을 *작전 의도 알려주지 않고* 동원. *카르샤 영내 주요 식수원 강 3곳 + 세이렌 성내 식수 저장지 1곳* 에 *대량 독 투입*
   - 명목 — *바이서스 군 야영지의 *오염된 폐수 처리 실수***. 군사들은 *진짜 의도 모름*. 작전 종료 후 호스반이 *해당 군사들을 다른 전선으로 분산 배치 + 일부 *전사 처리***
   - **외부 인식 정황** — *바이서스 군의 일부 군사 실수로 인한 오염*. *고의성 단서 거의 없음*

3. **카르샤 내부 *뒤늦은 자각***:
   - 강물·식수를 마신 *카르샤 군사·시민·왕실 다수* 가 *수일 내 일제히 동일 증상* 발현 — 몸 둔함·시야 흐림·기력 저하
   - 카르샤 어의들이 *처음에는 전쟁 스트레스·기근* 으로 오진. 4-2 진행 중 *동일 증상의 광범위함* 으로 *식수 오염 의심* 확인
   - **이미 카르샤 군 + 시민 + 왕실 거의 전원 *경증 ~ 중증 중독*** 상태. 알레이언 본인도 *왕실 식수 사용으로 중증 중독*

#### 알레이언의 *자기 희생* — *해독 물품 군사 분배*

본 Part 의 **가장 중요한 정통 — 카르샤 *학살자* 호칭의 진짜 근원**:

1. **세이렌 외부 봉쇄 (4-2 진행 중)**:
   - 바이서스 군이 세이렌 *완전 포위*. 외부 의약·식량 보급 *완전 차단*. 카르샤가 *외부 동맹·외부 의료 지원 모두 불가*
   - 카르샤 *왕실 비축 해독제·고급 치유 영약* 만이 *유일한 자원*. 카르샤 군의 *7써클급 마법사 1명* 이 *추가 합성 시도* 하나 *원자재 부족 + 시간 압박* 으로 *공급 미달*

2. **알레이언의 결단 (4-2 후반 ~ 4-3 사이)**:
   - 알레이언 본인이 *왕실 비축 해독제 + 자기 개인용 치유 영약까지 *모두 아낌없이* 자기 군사·시민들에게 분배*
   - *나는 왕이다. 왕은 마지막에 마신다.* — 알레이언의 결정 한마디. **알레이언 본인은 *해독 거부*** — *내가 마지막이다*
   - **결과** — 카르샤 군 + 시민 다수가 *경증 회복*. 알레이언 본인은 *중증 중독 그대로 유지*

3. **알레이언 4-4 5합의 *진짜 둔함의 정통***:
   - 4-4 5합 결투에서 *알레이언의 검 속도가 점점 무거워지는 자각* + *마법 영창 미세 지연* = **자기 결단의 결과**
   - 카엘이 *5합에서 알레이언을 이긴 진짜 이유* = *카엘의 결투 실력이 아닌 *알레이언이 자기 군사를 위해 자기 해독을 포기한 결과***
   - **카엘이 *진짜 알레이언과 대등하게 싸우지 않았다*** 의 정통

4. **알레이언 4-4 마지막 대사** (사용자 보강):
   - ***"너희가 무엇을 내게 먹였는지 알고 있는가"*** — 단순한 *나를 죽인 독* 의 한마디가 아니라 **나의 *백성에게* 너희가 무엇을 먹였는지 알고 있는가** 의 의미
   - 카엘이 *처음 듣는 정황*. 4-5 평화 협정 발견 시 *이 한마디가 메아리* 로 돌아옴 — 카엘이 *내가 무엇을 한 것인가* 의 *균열 시드*
   - **알레이언의 시선** — 카엘에 대한 *원망보다 큰 *연민*. *너는 이것이 너의 것이 아니라는 것을 모르는구나*

#### 카르샤의 *카엘 = 학살자* 호칭의 정통

- 카르샤 측의 사후 인식:
  1. *바이서스 군의 강 독살 = 카엘의 *바람 통로 + 독* 비기 명백*
  2. *알레이언의 자기 희생 = 카엘의 책임*
  3. *세이렌 함락 + 왕 살해 = 카엘이 직접*
  → *카르샤 측에서 카엘 = *흑마법사·학살자* 의 정통 호칭 형성. 일부 군사·시민이 *카엘을 *학살자* 로 부름*
- **실제 사실 — 카엘은 *강 독살을 직접 하지 않았으며 *진실도 모름***. 호스반의 뒷 획책이 *카엘의 비기를 빌려 누명까지 씌움*
- 카엘이 *진실의 일부를 처음 보는 시점* = 4-5 평화 협정 발견 + *알레이언의 마지막 한마디 메아리*. *완전 자각은 Part 6·7*

#### 정치적 의도 (호스반·라이세린·라카르 9세)

- **표면 외부 공식 입장**:
  - 알레이언 사인 = *5합 결투 패배로 인한 *명예로운 죽음**
  - 카르샤 군·시민 중독 = *바이서스 군 일부 실수로 인한 *불행한 사고**
- **실제 의도**:
  1. *카엘이 *진짜 5합으로 알레이언을 이겼다는 환상*** 유지 (카엘 본인도 안 깨짐)
  2. *왕의 사인이 *독 누명* 으로 카엘에게 돌아옴* — 후일 *바이서스 왕실이 카엘을 *제거할 명분*** 활용
  3. *카르샤 측의 *카엘=학살자* 호칭이 *바이서스 측의 흑마법 처분 명분* 의 정통 근거*
- **카엘 제거 씨드** — 본 Part 에서 *발아*. 호스반이 *알레이언 시신에서 *독 흔적 증거 채취* + 강 독살 작전의 *모든 흔적을 카엘 비기와 동일하게 정리** 해 *증거 보존*. 회수 = Part 6·7 *카엘 추방·결전 편지의 정통 명분*

#### 카엘까지 죽일 계획 — *실패* (검토 4 결정: 대안 A 채용)

- 호스반의 4-5 *제거 시도* — 카엘이 평화 협정을 발견한 순간 *마지막 약점 노출*. 호스반이 *백색 회랑 후방에서 마법 일격* 시도
- **카엘은 *엔트로퍼로서 감응력이 높음*** — 외부 마나 흐름·살의를 *공기 진동·잔향 변화* 로 감지. 더군다나 호스반도 *마법사* 라서 *기습 시전 직전의 마나 응축이 카엘의 감응에 즉시 포착*. **기습이 쉽지 않음**
- 카엘이 *즉시 알아챔* — *살상 직관 방어 시전 + 1보 측면 회피*. 호스반의 일격이 *카엘 옆 기둥에 빗나감*
- **카엘 vs 호스반 첫 의심 교환** (4-5 핵심 신규 비트):
  - 카엘이 회랑 후방을 *돌아봄*. 호스반이 *천천히 등장* — *과묵·간결 표정*
  - 카엘: "*방금 그것은 무엇이었나*"
  - 호스반: 짧은 멈춤. "*카르샤 잔당의 시도였소. 마침 처리했소.*"
  - 카엘은 *호스반의 말을 *완전히 믿지 않음***. 그러나 *증거도 없음*. *호스반에 대한 *처음으로 명확한 의심*** 의 시드 — Part 7 결전 편지 시점에 회수
- 호스반의 *작전 위장* — *바이서스 후방 정찰병 1인을 *카르샤 잔당과 교전* 으로 위장*. *카엘 앞에 위장 시신 노출*
- 카엘은 *명확히 의심* 하나 *증거 부재 + 평화 협정 충격 + 알레이언 한마디 메아리 + 손의 떨림 + 학살자 호칭의 외부 잔향* 이 *의식의 다수 점유*. *의심을 표면에서 한 단계 아래로 내림* — Part 6·7 회수

→ **바이서스 군이 *카르샤보다 모든 면에서 압도적***. 정상적 전쟁이라면 *바이서스의 점진적 승리* 가 예상되나 *전쟁이 빠르게 끝나야 하는 정치 일정* 이 있음 (라카르 9세의 *재위 30주년 기념 + 라이세린의 외교실 권력 공고화*).

#### 카엘의 역할

- *압도적 양에도 불구하고 *전쟁 빠른 종결을 위해 카엘의 광역 학살 필수**. 단 *카엘 없이도 바이서스가 결국 이긴다*. 카엘은 *시간 단축의 도구*
- 이 *카엘 없어도 되는 정황* 이 *1회성 조커 활용의 정통 근거*. 카엘이 *과대평가된 자기 인식* 으로 *내가 이긴 전쟁* 이라 믿는 것의 *실제 진실*

### 7. *카엘의 모든 수단 동원 + 잔혹 소문 + 정의롭지 못함 비판*

#### 모든 수단 — *카엘 실제 행위 vs 외관* + *비기·독은 실전 미사용*

- **카엘이 실제 전쟁에서 동원한 수단**:
  - 정통 1~3등급 숙련 + 4~5등급 일부 비상식 응용
  - *비상식 응용* — 발바닥 빙판·공기 칼날 일점 응축·광명 눈동자 직격·마나 직접 분사·광역 마나 방류
  - *살상 직관 시전* (1단계 직관 + 앤트로퍼 살상 본능)
  - *엔트로피 역연산* (Part 4 시도 → 본 Part 부분 활용)
  - **백병전·짠내·응용** — *근거리 단검 + 회복 약 + 즉석 응용 패턴 변경*. *4중 효율 우수성* 기반 (§1-2)
  - *자기 제작 아티팩트 3종* + *회복 약 무한 휴대* + *살상 충동 자기 정당화*
- **카엘이 동원하지 않은 수단** ★:
  - **바람 통로 + 독 비기 — 실전 미사용**. 본 Part 전체에 걸쳐 카엘이 *직접 독·금지 비기를 시전하지 않음*. 비기는 *연구품·마지막 카드* 로만 보유
- **카엘이 *동원했다고 외부가 인식하나 실제는 호스반의 합작* 수단**:
  - *광역 학살 1,400명 → 실제는 카엘 직접 사살 약 200명. 나머지 = 호스반의 마도공학 + 8써클 광역 마법 + 강 독살 중독 피해*
  - *세이렌 외성벽 결계 분쇄 → 실제는 호스반의 결계 분쇄 폭열탄*
  - *왕의 5합 결투 둔함 → 실제는 알레이언의 자기 희생 + 강 독살 누적*
  - *카르샤 항복 사절단 학살 정당화 → 실제는 호스반의 의심 정황 흘림 + 카엘의 살상 직관 응답*
- **본 Part 가 *카엘의 모든 *허용 도구* 첫 본격 동원* + *호스반의 합작 외관 조작 절정***. 4-1 광역 학살이 *카엘 본인도 충격받는 위력* — 단 충격의 진짜 정통은 *내가 한 게 아닌데 내가 한 것처럼 보임* 의 *미세한 위화감 + 자기 부풀림 동시 작용*
- 4-2 항복 사절단 *예방적 섬멸* = *전쟁 윤리 명백한 위반*. 카엘이 *직접 손* 으로 실행한 행위. 호스반이 흘린 *암살 위장 의심* + 카엘의 *살상 직관 응답* 의 합작 결과
- *사용자 HARD — 카엘의 강함은 *4중 효율 + 신선 응용 + 백병전 짠내* 에 그쳐야 하며 *압도적 단독 위력* 으로 묘사해서는 안 됨*

#### 잔혹 소문 시작

- **바이서스 군 내부 소문**: *그 자는 마법사가 아니라 *학살자*. *바이서스 군 4-2 진입 시 *카엘 옆에 서지 않으려 함***
- **카르샤 측 인식**: *바이서스의 *흑마법사·학살자 화신*. 카르샤 망명자들 (왕비 엘라리아·셀라엔 대주교) 이 *외부로 카엘의 잔혹성* 전파
- 본 Part 종결 시 카엘은 *바이서스 내부에서도 *불편한 존재*. *전쟁의 영웅 + 인간으로서는 가까이 하기 싫음*** 의 모순 위치

#### 정의롭지 못함 비판

- 4-4 알레이언의 *너는 무엇을 보고 있는가* — 본 Part 의 *정의롭지 못함 비판 의 정통 시점*
- 4-5 평화 협정 발견 후 카엘 자신의 *처음으로 자기 정당성 의심*. 그러나 *즉시 자기 부정 — 그래도 바이서스가 옳다·내가 옳았다*
- 후일 Part 6·7 회수 (자각 본격화)

#### 독 누명 — *카엘이 직접 안 썼는데도 모두가 카엘의 책임으로 인식*

- **알레이언의 사인 공식 발표** (바이서스 측 공식 입장):
  - *5합 결투 패배로 인한 *명예로운 죽음**
  - 실제: *호스반의 강 독살 + 알레이언의 자기 희생 → 5합 둔함*
- **카엘이 직접 사용하지 않은 *바람 통로 + 독* 비기**가 *호스반의 강 독살 작전 흔적* 으로 *카엘에게 귀속*:
  - 카르샤 측 어의들이 독 분석 → *바람 통로 + 신경 마비 독* 특성 확인 → *카엘의 비기 연구* 정황과 *완전 일치*
  - **카르샤 측은 *카엘이 직접 강을 독살했다고 확신***. 카엘=학살자 호칭의 정통 근거
- **바이서스 측 일부 측근** 사이에서도 *알레이언이 독으로 죽었다*·*카엘이 강을 독살했다* 의 소문 시작 — *호스반의 흘림* 이 *바이서스 내부에까지 침투*. 카엘에 대한 *불편함 + 두려움* 정통
- **카엘 본인은 *진실의 *호스반 측 뒷 획책* 정황을 모름*** — 4-5 평화 협정 발견 + 알레이언 한마디 메아리로 *처음 균열*. *진실의 완전 자각은 Part 6·7*
- **카엘 제거 명분의 정통 시드** — Part 6 두 팔 상실의 *공식 명분 = 흑마법 처분 + 강 독살 책임*. Part 7 결전 편지의 *카엘 처형 명령 명분*

### 8. 카르샤 제국 — *패전 후 상태* (사용자 보강)

#### 패전 직후 상태 (Chapter 1 종결 시점, 부드러운 처우 반영)

- **왕실 명목 유지 + 외부 망명**: 알레이언 6세 사망. *카르샤 왕가 정통 계승자 = 키얀 왕자 (9세, 외부 망명)*. 왕비 엘라리아 + 셀라엔 대주교가 *키얀과 함께 외부 망명* — *카르샤 왕실의 정통이 외부로 옮겨감*. *명목상 섭정 체제* (왕자 부재 중 통치)
- **수도 세이렌**: 바이서스 군 *단기 진주* 후 점진적 철수. *바이서스 외교 자문관 1인 상주 + 호위 기사단 약 1,000명*. 카르샤 왕실 시설 일부 사용
- **국토**: *카르샤 영토 전체 명목 자치*. 단 *바이서스 외교 자문관의 권고 권한* + *경제 협정 강제*. 남부 변경 일부 산간은 *카르샤 잔당 게릴라* 잔존 (정통성 명분 = *외부 망명 왕자 충성*)
- **인구 + 경제**: 전쟁 사상자 *민간 포함 약 4만* (현 안 8만→4만 완화). 수도 + 주요 도시 *건물 일부 파괴*. 농지 *20% 황폐화* — *2~3년 회복기* 진입

#### 바이서스의 *속국 처우* (검토 5 결정: 대안 A 채용 — 부드러운 처우 + 외부 망명)

- **정치적 관여 (명목 자치 유지)**:
  - 카르샤 *왕실 명목 자치 유지* — 카르샤 *임시 통치 위원회* 가 *명목상 카르샤 귀족 출신* 으로 구성 (실은 *바이서스 친화 인사*)
  - 바이서스 *외교 자문관 1인* 만 세이렌 상주 (직접 결재권 *없음* — 권고 권한만)
  - 카르샤 *왕실 명목 유지* — *키얀 왕자 부재 중* 의 *섭정 체제* 명목. 실제 키얀은 외부 망명
- **경제적 제재 (집중)**:
  - **전쟁 배상금** — 카르샤 *국가 세입의 25% 를 20년간 바이서스 왕실 지불* (현 안 40%→25% 완화)
  - **무역 협정** — 카르샤 항구 *바이서스 우선 관세 면제* + *카르샤 상인의 외부 무역은 자유 (단 바이서스 거쳐야 함)*
  - **자원 협력** — 카르샤 *마나 광산 1곳* 의 *바이서스-카르샤 공동 운영* (50:50). *곡창지 직접 운영 폐지*
  - **인구 이동 자유 보장** — 카르샤 *장인·마법사·학자의 바이서스 *자발적 이주*** 길 열어둠. *강제 이주 폐지*
- **종교·문화 *경공*** (경비 + 공존):
  - *카르샤 신성검단 명목 해산* — 실은 *바이서스 호위 기사단 일부로 재편입*. 검은 *해당 기사 휴대*. **루멘브링거만 *바이서스 보물고 봉인*** (왕의 검이라는 상징성)
  - *카르샤 신성 신앙 유지* — 바이서스 *주신 신앙 강제 전환 없음*
  - *카르샤 전통 의례·명절 유지* — *바이서스 왕실 기념일 추가 의무화* 만
- **사용자 의도**: *카르샤 속국 처우는 너무 가혹하지 않음*. *카르샤 잔당 외부 망명의 정통성 + 후일 적대 라인의 *복수 정당화 정도* 가 *극단적이 아닌 *왕실 자존심 회복* 수준** 으로 유지

#### 카르샤 잔당 + 외부 망명 정통

- **키얀 왕자 망명 (9세)** — 왕비 엘라리아 + 셀라엔 대주교 + 카일렌 베르하르트의 *처남 1인* 호위. 외부 *7대 (7대륙 또는 7세대) 망명* 정통 (Chapter 1 *7대 기다림* 의 의미)

- **복수의 명분** = *왕의 자기 희생 + 학살자 카엘* 의 *2중 의리*. *극단적 영토 수탈이 아닌 *왕실 자존심* 의 *명예 복수*** 정도의 의미. 후일 적대 라인의 *질투·증오가 아닌 *원한* 의 정통*
