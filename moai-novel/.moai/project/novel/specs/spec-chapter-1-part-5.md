## Summary

**바이서스 제국 (공격측)** 의 *카르샤 제국 (방어측)* 침공 전쟁. 세번 강 회전 → 세이렌 함락 → 옥좌의 방 진입 → **프롤로그 완전판 (왕 살해)** → 평화 협정 발견 + 각성.

본 Part 의 정치 본질 — **카엘은 라이세린·바이서스 왕실의 *1회성 조커 카드***. 왕실 직속 *8써클 전쟁 참모 호스반 셰이드런 경* 이 *승리의 판은 깔되 절대 드러나지 않는 선* 에서 카엘을 감시·지원·최종 제거 임무.

**가장 중요한 정통 1** — 카엘은 *바람 통로 + 독 비기를 연구만 했지 실전에서는 직접 사용하지 않음*. 그러나 호스반이 *카엘의 비기 자료를 무단 합성* + *카르샤 주요 강·식수원에 다량 독 투입* (바이서스 군 일부 군사 실수로 가장). 카르샤 군·시민·왕실 *광범위 중독*. 세이렌 외부 봉쇄 + 보급·의약 차단 상황에서 **알레이언 6세가 *자기 해독 물품까지 모두 군사들에게 분배* + 자신은 *해독 거부*** — 4-4 5합의 *진짜 둔함의 정통* + 카르샤 측 *카엘 = 학살자* 호칭의 정통 근원.

**가장 중요한 정통 2** — *카엘의 광역 학살은 외관상 그렇게 보일 뿐 실제는 합작*. 카엘의 마법은 *실제 3등급 + 비상식 응용* 으로 *광역 학살 1,400명을 단독으로 수행할 위력이 절대 아님* (사용자 HARD — *개연성을 해칠 정도의 강함 묘사 금지*). 실제는 **호스반의 *광범위 중독 + 마도공학 공성기 + 독 화살촉 + 결계 분쇄 폭열탄* 합작 + 카엘의 *4중 효율 우수성 + 백병전·짠내·응용*** 의 *합산 효과*. 외부에 *카엘의 단독 광역 학살로 보이도록 호스반이 시각 효과 조작*. *50% 시점*. 1인칭 본격 전환.

### 1-2. 카엘의 *실제 능력* — 4중 효율 우수성 (사용자 HARD)

카엘의 공식 등급 = 5써클 (카르엔 부풀림). 실제 = 3등급 + 비상식 응용. *최대 위력은 5써클을 넘지 않음*. 그러나 *4중 효율 측면에서 정통 동급 마법사보다 우수*:

| **마나의 양 (총량)** | 일반 5써클의 *1.5배* | 1.0 | 앤트로퍼 *역마나 흡수* 자동 효율. 자체 회로의 *마나 저장 용량 ↑* |
| **마나 회복 속도** | 일반의 *1.5배* | 1.0 | 회복 약 + 자기 직관 회로의 *짧은 휴식 효율 ↑*. 4-2 진입 시 *5분 회복으로 다음 광역 시전 가능* |
| **스킬 효율성 (마나당 위력)** | 일반의 *1.3배* | 1.0 | 비상식 응용의 *불필요한 절차 단축*. 발바닥 빙판이 광역 빙판의 *마나 1/5* 로 동일 효과 |
| **스킬 시전 속도** | 일반의 *1.5배* | 1.0 | 1단계 직관 시전 (정통 5단계 시전 → 카엘의 1단계 시전). 정통 마법사가 *영창 5초* 를 카엘은 *직관 1초* |

→ **카엘의 진짜 강점** = *지속 화력 + 빠른 회복 + 효율 + 속도*. *단 최대 위력은 5써클 수준 (실제는 3등급)* — 이 정통이 *카엘이 *광역 학살의 외관 효과* 만 단독으로 만들 수 있고 *실제 사살 수는 호스반의 합작 결과* 의 인과 근거*

### 1-3. 카엘 *마검사 전투 스타일* — 5종 특이 검술 + 짠내·응용 (사용자 보강)

본 Part 의 *전쟁 묘사의 신선함* 정통:

#### 정통 마법사 vs 정통 기사 vs 카엘 (마검사)

| 항목 | 정통 마법사 | 정통 기사 | **카엘 (마검사)** |
|------|-----------|----------|------------------|
| 위치 | 후방 영창대 | 전열 | **전열 내부 깊숙이** |
| 보호 | 호위 기사 | 갑옷·방패 | **자기 보호 자기 — 빙판 활주·도약** |

| 체력 | 낮음 (후방) | 높음 | **체력 소모 大, 회복력 大** |

#### 용병 습관 + 정통 기사단 못마땅함

- 카엘의 검술은 *1.5부 떠돌이 시기의 *짠내 용병 생활* 습관 + 외곽 연구소 3년 학습* 의 *모자이크*
- *정통 기사단의 기사 = 못마땅* — 카엘은 *기사 명예·기사 정신·전통 검술 자세를 *답답함·허세*** 로 인식. *살아남는 것이 옳다* 의 용병 철학 우위
- 정통 기사가 *명예의 한 합* 을 시도할 때 카엘은 *빙판 활주로 회피 + 등 뒤에서 단검* 의 *비기사적 응답*. 정통 기사가 *비겁하다* 비판하면 카엘은 *살아남았으니 옳다* 일축

#### 카엘의 *특이 검술 5종* (사용자 보강 — 핵심 묘사 자산)

검술 기초(상) + 응용 마법 접목 = *정통 검사가 짐작 불가능한 변칙 검술*. **본 Part 의 신선한 묘사 정점**:

1. **바람 손날 — 갑옷 자상**
   - 검을 휘두름과 동시에 *바람 마법 1단계* 시전 → *검 끝에 얇은 바람 칼날 부착*
   - 정통 기사 갑옷의 *접합부 (목·겨드랑이·관절)* 에 *얇은 자상* 을 남김. *치명상은 아니나 누적되면 마비*
   - 정통 기사 시각 = *검만으로 이런 자상이 나올 수 없다* 의 *혼란*

2. **검 막은 상대 화염 폭발**
   - 카엘의 검을 *상대가 받아쳐 막는 순간* — 검 면에서 *1초 시전된 화염구 폭발 발동*
   - 상대 검을 *손에서 놓치거나·얼굴 화상·갑옷 손상*. 카엘 본인은 *내열 마법 결계로 보호*
   - 정통 기사 시각 = *검 받침이 *공격이 되는* 변칙*. 두번째 검 받침을 망설임 → 카엘의 진입 기회

3. **빙판 변칙 보법**
   - 발바닥에 *빙판 마법 1초 시전* → *적의 발 앞 30cm 빙판 즉석 생성*
   - 적이 *발 디딤 실패 + 균형 무너짐* → 카엘이 *역방향 단검 진입*
   - 또는 *카엘 본인 발바닥 빙판 활주* 로 *예측 불가능한 보법* 발현. 정통 기사의 *발 디딤 예측 검술* 무력화

4. **바람 마법 응용 도약**
   - 바람 마법 일부 응용 — *발바닥 아래 바람 폭발* → *수직 3m·수평 5m 도약 가능*
   - 정통 기사의 *횡적 검술 거리감* 완전 무력화. *공중에서 단검 투척 + 착지 후 빙판 활주* 의 연계
   - 단점 = *시전 직후 1초 무방비*. 적이 *대응 빠르면 격추 가능*

5. **맞댄 검 전기 충격**
   - 카엘 검과 적 검이 *맞닿는 순간* — *번개 마법 1단계* 즉석 시전 → *검을 매개로 전기 충격 전달*
   - 적은 *순간 마비 + 헛점 노출*. 카엘이 *0.5초 진입* 으로 *치명타* 가능
   - 적의 *마법 방어구 / 절연 갑옷* 이 있으면 무효. 카엘은 *맞붙기 전 적 갑옷 재질 빠른 판단* 필요

#### 체력·회복력·순간 근력

- 카엘의 검술은 *체력 상당 소모* — 정통 기사의 *조절된 검술* 과 달리 *변칙·도약·즉석 시전* 의 *과부하* 가 누적
- 그러나 **대단한 회복력** — 앤트로퍼 마나 흡수 + 회복 약 + 4중 효율 우수성 (마나 회복 1.5배) 합산. *짧은 휴식 5~10분* 으로 *다음 전투 즉시 가능*
- **순간 근력 증가** — 정통 근력은 *일반 평민 수준* 이나 *단발 마나 → 근력 직접 변환* 으로 *순간 검 휘두름 속도·도약 거리·검 압력* 가 *정통 중급 기사 수준*. *지속력은 5~10초*

#### 기사단 검사들의 카엘 평가 (사용자 보강 핵심)

- **정통 기사단 검사들이 카엘의 검술 수위를 *짐작하지 못함***:
  - 정통 검술 기준 *기초(상)* → *하급 검사도 정면 한 합으로 이길 수준*
  - 그러나 *응용 마법 + 변칙 5종 접목* 시 *상급 기사도 농락당함*
  - 결과 — *카엘과의 비무가 *예측 불가능*. *어떨 때는 허무하게 이기고 어떨 때는 말도 안되게 농락당함**
- **바이서스 군 일반 기사들의 인식**:
  - *저 자는 마법사인지 검사인지 알 수 없다*
  - *방심하면 죽고 신중하면 이긴다 — 그러나 신중하는 법을 모르겠다*
  - 카엘에 대한 *혼란 + 두려움 + 학자답지 못함* 의 모자이크 인식
- **소드 마스터 레벨 비무 미경험 (HARD 캡)**:
  - **카엘은 본 Part 시점까지 *왕실 고위 소드 마스터 레벨과 비무 경험 없음***
  - 카엘의 *실제 검술 수위* 는 *기초(상) + 응용 마법* 으로 *변칙성에 의존*. *진정한 검술 강자* 와 맞붙으면 *어떻게 되는지 본인도 모름*
  - 후일 Part 6·7 *드래곤·9혈 마녀 의회 등의 강자* 와의 첫 진검 비무에서 *카엘의 한계 노출*. *Chapter 1 종결 시까지 소드 마스터 비무 0회*

#### 짠내 전투 묘사 시드

- 카엘이 *자주 부상*. 검에 베이고·화살에 맞고·결계 폭발 파편에 다침. *순간 근력 증가의 부작용* 으로 *근육 미세 파열* 빈발
- 회복 약 *즉시 사용 → 다음 전투 즉시 진입* 의 반복. *야영지에서 약품 정리 + 다음 전선 즉시 출정* 의 정통
- 1.5부 떠돌이 시기의 *짠내* 가 본 Part 에서 *전쟁 짠내* 로 진화. 다른 마법사가 *우아한 영창대*에 서 있는 동안 카엘만 *피·먼지·약품 냄새·검 진동* 의 마검사

#### 응용력 묘사 시드

- 카엘이 *즉석으로 마법 활용 패턴 변경*. 적의 검술이 *빙판에 익숙* 해지면 *바람 손날* 로 교체. *전기 충격 무효한 절연 갑옷 적* 에게 *화염 폭발* 로 우회
- 정통 마법사는 *학습한 마법만 사용*. 카엘은 *전장 즉석에서 새 응용 발명* — 본 Part 전체에 걸쳐 *위 5종 + 즉석 응용 변형 3~4개* 노출

#### 신선한 방식 묘사 톤

- 1인칭 시점이 *광역 마법 영창 후 결과* 가 아닌 *백병전 한가운데의 호흡·먼지·근육의 긴장·약품 냄새·단검 무게·검 진동·전기 잔열·바람 폭발 후 청각 일시 손실* 같은 *신체 감각 직접 묘사*
- 정통 마법 전투의 *공중 영창대 묘사* 와 *완전히 다른 톤*. *진흙·피·약·금속음·전기·화염·바람의 모자이크 1인칭*

→ **사용자 HARD — 절대 *개연성을 해칠 정도의 강함 묘사 금지***. 카엘은 *4중 효율 + 변칙 검술 5종 + 응용 마법사 + 소드 마스터 미경험* 의 *변칙 마검사*. *압도적 단독 강자가 아님*

### 2. 정치 획책 — *카엘 = 1회성 조커 카드*

- 본 Part 출정 시점: **바이서스 *왕실 직속 전선 주력 마법사*** 로 다시 한 단계 이동. *공식 칭호 — 5써클 인증 + 라카르 9세 직속 *전선 자문 마법사**
- 의미: *왕실 권위 명목 + 풍족한 자원 + 후일 책임 떠넘기기 용이성*. 라이세린의 *흡수 → 라카르 9세에게 *기증*** 의 형태

- 보급창 항목:
  - **마나 결정** (시전 보조) 다량
  - **회복 약** (성소 성수·마나 의료원 약·고급 치유 영약) 사실상 무한
  - **마법 양피지** + 마법 자료 (도서관 외부 반출 가능)
  - **아티팩트 시제품** (왕실 마도공학 연구소 시제품 — 외부 미공개)

### 3. 카엘의 *특이한 연구 습성* (풍족한 지원 활용)

- 정통 마법사는 *마법 연구만* 하지만 카엘은 **마법 + 아티팩트·아이템 연구 동시 진행**:
  - **마법 연구**: 정통 3~5등급 학습 지속 + 비상식 응용 확장 + 금지 분야 *엔트로피 역연산·바람 통로 + 독·살상 직관 시전* 시도
  - **아티팩트 연구**: 바이서스 마도공학 연구소 시제품을 *분해·재조립·자기 마나 회로에 맞춘 개조*. 본 Part 종결 시 *카엘 전용 마나 보조 팔찌 + 비상 회복 캡슐 + 살상 직관 보조 단검* 의 *초기 자기 제작 아티팩트 3종* 보유. **6부 의수 정통 시드** (두 팔 상실 후 *자기 제작 의수* 의 정통)
- *회복 약·자기 안위 아이템* 적극 획득 — *1.5부 떠돌이 시기의 *부상 누적 트라우마*** 가 *야지 생활 + 전쟁* 에서 *과보상* 형태로 발현. *카엘 = 회복 약 다량 휴대* + *야영지에서 자기 약품 정리 정통*
- 정통 학자들은 카엘의 연구 습성을 *학자답지 못함·잡상인* 으로 비판. 카엘은 *살아남는 자가 옳다* 의 무의식 확신

### 4. 호스반 셰이드런 경 — *8써클 전쟁 참모 + 감시자 + 제거자*

#### 정체

- **호스반 셰이드런 경** (Hosban Sheidren) — 바이서스 왕실 직속 *8써클 전쟁 참모*
- 50대 후반. 라카르 9세 직속 수하. *공식 신분은 *전선 자문 마법사*** 이나 실은 *왕실의 그림자 손*
- 본 Part 이전 캐논에서 단편 언급 (전쟁 참모) → **본 Part 에서 정식 인물화**

#### 호스반 캐릭터 디테일 (결정 완료)

- **성격 4중**: *음흉 + 과묵 + 간결하고 똑똑함 + 일부 죄책감이 있으나 단호함*
- **음흉**: 모든 말에 *두 겹의 의미*. 표면은 정중·공식 자문, 이면은 *제거·악행 위임 계산*
- **과묵**: 필요 이상의 한 마디 안 함. 회의에서도 *결정적 한 문장* 만 발화. *침묵의 위압*
- **간결하고 똑똑함**: 8써클의 두뇌. 모든 작전을 *짧은 문장으로 본질 압축*. 부하들에게 *단호한 명령 한 문장* 만 내림
- **일부 죄책감이 있으나 단호함**: 알레이언의 자기 희생을 *내심 인정*. *카르샤 시민의 광범위 중독을 본 후 *짧은 머뭇거림*** 1회 (4-2). 그러나 *왕실 명령 + 정치 임무* 앞에서 단호하게 진행. *죄책감을 자기 안에 묻는 자*
- **외형 디테일**: 왼쪽 눈에 옛 *마나 화상 흉터* (라카르 9세 호위 시기). 회색 수염 짧게 정돈. *목소리 낮고 건조*
- **말투 디테일**: *문장 끝마다 짧은 멈춤*. 동의 구함 *없음* (사용자의 *그렇지요?* 패턴은 회수 — *과묵·간결* 특성에 더 맞음). 호칭은 *그대 + 직책명*

  - 카엘 본인은 *내가 한 것* 으로 오인 + *내가 이정도였나* 의 충격 — *미세한 위화감* 만 의식 표면 아래에 (전체 폭발의 50% 정도는 *호스반 합작* 으로 의식 깊은 곳에서 의심)

- **4-2 세이렌**: *항복 사절단의 *진짜 의도가 정찰 + 암살*** 정황 *조작* 으로 카엘에게 흘림. 카엘의 *예방적 섬멸 정당화*. 동시에 *호스반의 결계 분쇄 폭열탄* 으로 세이렌 외성벽 결계 분쇄 → *카엘의 진입로 개척 외관 부여*

### 5. 알레이언·카르샤 중독 정통 — *호스반의 강 독살 + 왕의 자기 희생* (v2 재구성)

#### 핵심 정통 — 카엘은 *실제 전쟁에서 금지 비기·독을 직접 사용하지 않음* ★

- **카엘이 보유한 *바람 통로 + 독* 비기** = 외곽 연구소 5년차 *카르엔 도서관 금지 자료* 연구품. **카엘은 *연구만 했지 실전 사용 의도 없음***
- 카엘 본인의 의식 = *비기는 *마지막 카드*. 실전에서는 정통 마법 + 비상식 응용만으로 충분*. 본 Part 전체에 걸쳐 *카엘이 *직접 독·금지 비기를 시전하지 않음***
- 그러나 후일 *카르샤 측에서 카엘을 *학살자* 로 부르는 호칭의 정통* 이 본 Part 에서 발아 — **카엘이 실제로는 안 썼는데도 *모든 책임이 카엘에게 돌아오는 정황***. *비뚤린 시야 너머의 카엘은 정의로운 본성* (Chapter 1 사용자 HARD 룰) 의 *외부 비대칭 인식* 정통

#### 사건 정황 — 호스반의 *강 독살 뒷 획책*

본 Part 시작 *훨씬 이전* — 4-1 세번 강 회전 직전 + 4-2 세이렌 포위 진행 중:

1. **호스반의 *카엘 비기 자료 흘림 + 자체 합성***:
   - 호스반이 *카엘의 *바람 통로 + 독* 연구 자료를 *몰래 복사* + *바이서스 왕실 마도공학 연구소에서 *대량 합성***
   - 합성된 독은 *마나 회로 흐름 둔화 + 검 휘두름 속도 30% 저하 + 마법 시전 영창 길어짐* 의 *수일 누적형*
   - **카엘은 *합성 사실도 모름*** — 카엘의 비기 자료를 호스반이 *기록물 보관실 자유 출입권* 으로 무단 복사

2. **강 독살 작전 (4-1 직후 ~ 4-2 진행 중)**:
   - 호스반이 *일부 바이서스 군사 4~5명* 을 *작전 의도 알려주지 않고* 동원. *카르샤 영내 주요 식수원 강 3곳 + 세이렌 성내 식수 저장지 1곳* 에 *대량 독 투입*
   - 명목 — *바이서스 군 야영지의 *오염된 폐수 처리 실수***. 군사들은 *진짜 의도 모름*. 작전 종료 후 호스반이 *해당 군사들을 다른 전선으로 분산 배치 + 일부 *전사 처리***
   - **외부 인식 정황** — *바이서스 군의 일부 군사 실수로 인한 오염*. *고의성 단서 거의 없음*

3. **카르샤 내부 *뒤늦은 자각***:
   - 강물·식수를 마신 *카르샤 군사·시민·왕실 다수* 가 *수일 내 일제히 동일 증상* 발현 — 몸 둔함·시야 흐림·기력 저하
   - 카르샤 어의들이 *처음에는 전쟁 스트레스·기근* 으로 오진. 4-2 진행 중 *동일 증상의 광범위함* 으로 *식수 오염 의심* 확인
   - **이미 카르샤 군 + 시민 + 왕실 거의 전원 *경증 ~ 중증 중독*** 상태. 알레이언 본인도 *왕실 식수 사용으로 중증 중독*

#### 알레이언의 *자기 희생* — *해독 물품 군사 분배*

본 Part 의 **가장 중요한 정통 — 카르샤 *학살자* 호칭의 진짜 근원**:

1. **세이렌 외부 봉쇄 (4-2 진행 중)**:
   - 바이서스 군이 세이렌 *완전 포위*. 외부 의약·식량 보급 *완전 차단*. 카르샤가 *외부 동맹·외부 의료 지원 모두 불가*
   - 카르샤 *왕실 비축 해독제·고급 치유 영약* 만이 *유일한 자원*. 카르샤 군의 *7써클급 마법사 1명* 이 *추가 합성 시도* 하나 *원자재 부족 + 시간 압박* 으로 *공급 미달*

2. **알레이언의 결단 (4-2 후반 ~ 4-3 사이)**:
   - 알레이언 본인이 *왕실 비축 해독제 + 자기 개인용 치유 영약까지 *모두 아낌없이* 자기 군사·시민들에게 분배*
   - *나는 왕이다. 왕은 마지막에 마신다.* — 알레이언의 결정 한마디. **알레이언 본인은 *해독 거부*** — *내가 마지막이다*
   - **결과** — 카르샤 군 + 시민 다수가 *경증 회복*. 알레이언 본인은 *중증 중독 그대로 유지*

3. **알레이언 4-4 5합의 *진짜 둔함의 정통***:
   - 4-4 5합 결투에서 *알레이언의 검 속도가 점점 무거워지는 자각* + *마법 영창 미세 지연* = **자기 결단의 결과**
   - 카엘이 *5합에서 알레이언을 이긴 진짜 이유* = *카엘의 결투 실력이 아닌 *알레이언이 자기 군사를 위해 자기 해독을 포기한 결과***
   - **카엘이 *진짜 알레이언과 대등하게 싸우지 않았다*** 의 정통

4. **알레이언 4-4 마지막 대사** (사용자 보강):
   - ***"너희가 무엇을 내게 먹였는지 알고 있는가"*** — 단순한 *나를 죽인 독* 의 한마디가 아니라 **나의 *백성에게* 너희가 무엇을 먹였는지 알고 있는가** 의 의미
   - 카엘이 *처음 듣는 정황*. 4-5 평화 협정 발견 시 *이 한마디가 메아리* 로 돌아옴 — 카엘이 *내가 무엇을 한 것인가* 의 *균열 시드*
   - **알레이언의 시선** — 카엘에 대한 *원망보다 큰 *연민*. *너는 이것이 너의 것이 아니라는 것을 모르는구나*

#### 카르샤의 *카엘 = 학살자* 호칭의 정통

- 카르샤 측의 사후 인식:
  1. *바이서스 군의 강 독살 = 카엘의 *바람 통로 + 독* 비기 명백*
  2. *알레이언의 자기 희생 = 카엘의 책임*
  3. *세이렌 함락 + 왕 살해 = 카엘이 직접*
  → *카르샤 측에서 카엘 = *흑마법사·학살자* 의 정통 호칭 형성. 일부 군사·시민이 *카엘을 *학살자* 로 부름*
- **실제 사실 — 카엘은 *강 독살을 직접 하지 않았으며 *진실도 모름***. 호스반의 뒷 획책이 *카엘의 비기를 빌려 누명까지 씌움*
- 카엘이 *진실의 일부를 처음 보는 시점* = 4-5 평화 협정 발견 + *알레이언의 마지막 한마디 메아리*. *완전 자각은 Part 6·7*

#### 정치적 의도 (호스반·라이세린·라카르 9세)

- **표면 외부 공식 입장**:
  - 알레이언 사인 = *5합 결투 패배로 인한 *명예로운 죽음**
  - 카르샤 군·시민 중독 = *바이서스 군 일부 실수로 인한 *불행한 사고**
- **실제 의도**:
  1. *카엘이 *진짜 5합으로 알레이언을 이겼다는 환상*** 유지 (카엘 본인도 안 깨짐)
  2. *왕의 사인이 *독 누명* 으로 카엘에게 돌아옴* — 후일 *바이서스 왕실이 카엘을 *제거할 명분*** 활용
  3. *카르샤 측의 *카엘=학살자* 호칭이 *바이서스 측의 흑마법 처분 명분* 의 정통 근거*
- **카엘 제거 씨드** — 본 Part 에서 *발아*. 호스반이 *알레이언 시신에서 *독 흔적 증거 채취* + 강 독살 작전의 *모든 흔적을 카엘 비기와 동일하게 정리** 해 *증거 보존*. 회수 = Part 6·7 *카엘 추방·결전 편지의 정통 명분*

#### 카엘까지 죽일 계획 — *실패* (검토 4 결정: 대안 A 채용 / v7 보강: *왕 사후* 시점 + *공주 추격 회항* 동기)

**v7 변경 — 기습 타이밍 명시 + 호스반의 *공주 추격 후 회항* 동기 추가**:

- 호스반의 4-4 진행 중 *별행 분기* — 알레이언이 *딸을 뒷길로 탈출시킨다* 는 정황을 외부 시야로 포착하고 *조용히 공주 탈출로로 이동* (§9-B). 이때 카엘은 *옥좌의 방에서 알레이언과 5합 결투* 중. 호스반의 부재 = *공주 격리 작전 동시 진행*
- **호스반의 카엘 기습은 *왕이 죽은 후*** — 알레이언 사망 + 평화 협정 발견 직후, 호스반이 *공주 격리 완료 후 회랑으로 *되돌아옴*. 카엘이 *평화 협정으로 인한 약점 노출* 의 순간을 노려 *기습 일격* 시도
- **카엘의 회피 — 3중 효율로 기습 무력화** (사용자 보강):
  1. **빠른 회복력** — 4-4 결투 직후의 *생체 마나 회복 1.5배 효율* (§1-2) 로 *기습 시점 마나 회로 이미 안정*. 정통 마법사라면 *결투 직후 무방비 시간 30초* 가 있으나 카엘은 *5초 안에 회복*
  2. **마법사 감응** — 호스반도 *마법사* 라서 *기습 시전 직전의 마나 응축이 *공기 진동 + 잔향 변화* 로 카엘의 감응에 즉시 포착*. 정통 검사의 기습이라면 더 어려웠으나 *마법사 vs 마법사* 의 *마나 흐름 상호 가시성* 이 호스반의 단점
  3. **엔트로퍼 이득** — *역마나 흡수의 *위협 감지 본능*** 이 외부 살의를 *공기 진동* 으로 자동 변환. 카엘 본인은 *왜 알아챘는지 모름* — 의식 표면에서는 *우연히 돌아봄* 으로 인식
- 결과 — 카엘이 *살상 직관 방어 시전 + 1보 측면 회피*. 호스반의 일격이 *카엘 옆 기둥에 빗나감* — *돌가루 한 줌 떨어짐* (종결 후크 시드)
- **카엘 vs 호스반 첫 의심 교환** (4-5 핵심 신규 비트):
  - 카엘이 회랑 후방을 *돌아봄*. 호스반이 *천천히 등장* — *과묵·간결 표정*
  - 카엘: "*방금 그것은 무엇이었나*"
  - 호스반: 짧은 멈춤. "*카르샤 잔당의 시도였소. 마침 처리했소.*"
  - 호스반의 *위장 시신* — *바이서스 후방 정찰병 1인을 *카르샤 잔당과 교전* 으로 위장* 한 시신을 *복도 끝에 노출*. 호스반의 *위장 작전 능숙도* 노출
  - 카엘은 *호스반의 말을 *완전히 믿지 않음***. *왕이 죽은 직후* 호스반이 *어디에서 *되돌아왔는가* + *왜 하필 그 방향에서 자기를 노렸는가*** 의 *2중 의심 시드* — Part 7 결전 편지 시점에 회수
- 카엘은 *명확히 의심* 하나 *증거 부재 + 평화 협정 충격 + 알레이언 한마디 메아리 + 손의 떨림 + 학살자 호칭의 외부 잔향* 이 *의식의 다수 점유*. *의심을 표면에서 한 단계 아래로 내림* — Part 6·7 회수
- **카엘이 *공주 탈출* 을 *4-4 에서 묵인했음* 의 사실은 *의심의 후일 회수* 시점에 *호스반의 진짜 동기 = 공주 격리* 였음을 *깨닫게 되는 정통 시드*** (§9 + §10 연결)

### 6. 군사 자원 — *바이서스 압도적*

#### 양적 비교

| 항목 | 바이서스 (공격측) | 카르샤 (방어측) |
|------|-----------------|-----------------|
| 총 병력 | 약 18만 | 약 6만 (선전포고 후 동원 한계) |
| 정예 기사단 | 12개 (각 1,500명 규모) | 4개 (각 1,500명 규모 — 신성검단 포함) |
| 마법사 | 7써클 이상 8명 + 5써클 이하 200명 + *카엘* | 6써클 1명 + 5써클 이하 50명 |
| 군량·보급 | *왕실 보급창 +라이세린 외교실 지원* — 사실상 무한 | *외부 동맹 차단됨 + 자국 1년 비축* |
| 사기 | *변경 정세 격화 명분 + 라카르 9세 친정* | *방어 + 가족 보호 + 신성검단 의지* |

#### 카엘의 역할

- *압도적 양에도 불구하고 *전쟁 빠른 종결을 위해 카엘의 광역 학살 필수**. 단 *카엘 없이도 바이서스가 결국 이긴다*. 카엘은 *시간 단축의 도구*
- 이 *카엘 없어도 되는 정황* 이 *1회성 조커 활용의 정통 근거*. 카엘이 *과대평가된 자기 인식* 으로 *내가 이긴 전쟁* 이라 믿는 것의 *실제 진실*

### 7. *카엘의 모든 수단 동원 + 잔혹 소문 + 정의롭지 못함 비판*

#### 모든 수단 — *카엘 실제 행위 vs 외관* + *비기·독은 실전 미사용*

- **카엘이 실제 전쟁에서 동원한 수단**:
  - 정통 1~3등급 숙련 + 4~5등급 일부 비상식 응용
  - *비상식 응용* — 발바닥 빙판·공기 칼날 일점 응축·광명 눈동자 직격·마나 직접 분사·광역 마나 방류
  - *살상 직관 시전* (1단계 직관 + 앤트로퍼 살상 본능)
  - *엔트로피 역연산* (Part 4 시도 → 본 Part 부분 활용)
  - **백병전·짠내·응용** — *근거리 단검 + 회복 약 + 즉석 응용 패턴 변경*. *4중 효율 우수성* 기반 (§1-2)
  - *자기 제작 아티팩트 3종* + *회복 약 무한 휴대* + *살상 충동 자기 정당화*
- **카엘이 동원하지 않은 수단** ★:
  - **바람 통로 + 독 비기 — 실전 미사용**. 본 Part 전체에 걸쳐 카엘이 *직접 독·금지 비기를 시전하지 않음*. 비기는 *연구품·마지막 카드* 로만 보유
- **카엘이 *동원했다고 외부가 인식하나 실제는 호스반의 합작* 수단**:
  - *광역 학살 1,400명 → 실제는 카엘 직접 사살 약 200명. 나머지 = 호스반의 마도공학 + 8써클 광역 마법 + 강 독살 중독 피해*
  - *세이렌 외성벽 결계 분쇄 → 실제는 호스반의 결계 분쇄 폭열탄*
  - *왕의 5합 결투 둔함 → 실제는 알레이언의 자기 희생 + 강 독살 누적*
  - *카르샤 항복 사절단 학살 정당화 → 실제는 호스반의 의심 정황 흘림 + 카엘의 살상 직관 응답*
- **본 Part 가 *카엘의 모든 *허용 도구* 첫 본격 동원* + *호스반의 합작 외관 조작 절정***. 4-1 광역 학살이 *카엘 본인도 충격받는 위력* — 단 충격의 진짜 정통은 *내가 한 게 아닌데 내가 한 것처럼 보임* 의 *미세한 위화감 + 자기 부풀림 동시 작용*
- 4-2 항복 사절단 *예방적 섬멸* = *전쟁 윤리 명백한 위반*. 카엘이 *직접 손* 으로 실행한 행위. 호스반이 흘린 *암살 위장 의심* + 카엘의 *살상 직관 응답* 의 합작 결과
- *사용자 HARD — 카엘의 강함은 *4중 효율 + 신선 응용 + 백병전 짠내* 에 그쳐야 하며 *압도적 단독 위력* 으로 묘사해서는 안 됨*

#### 잔혹 소문 시작

- **바이서스 군 내부 소문**: *그 자는 마법사가 아니라 *학살자*. *바이서스 군 4-2 진입 시 *카엘 옆에 서지 않으려 함***
- **카르샤 측 인식**: *바이서스의 *흑마법사·학살자 화신*. 카르샤 망명자들 (왕비 엘라리아·셀라엔 대주교) 이 *외부로 카엘의 잔혹성* 전파
- 본 Part 종결 시 카엘은 *바이서스 내부에서도 *불편한 존재*. *전쟁의 영웅 + 인간으로서는 가까이 하기 싫음*** 의 모순 위치

#### 정의롭지 못함 비판

- 4-4 알레이언의 *너는 무엇을 보고 있는가* — 본 Part 의 *정의롭지 못함 비판 의 정통 시점*
- 4-5 평화 협정 발견 후 카엘 자신의 *처음으로 자기 정당성 의심*. 그러나 *즉시 자기 부정 — 그래도 바이서스가 옳다·내가 옳았다*
- 후일 Part 6·7 회수 (자각 본격화)

#### 독 누명 — *카엘이 직접 안 썼는데도 모두가 카엘의 책임으로 인식*

- **알레이언의 사인 공식 발표** (바이서스 측 공식 입장):
  - *5합 결투 패배로 인한 *명예로운 죽음**
  - 실제: *호스반의 강 독살 + 알레이언의 자기 희생 → 5합 둔함*
- **카엘이 직접 사용하지 않은 *바람 통로 + 독* 비기**가 *호스반의 강 독살 작전 흔적* 으로 *카엘에게 귀속*:
  - 카르샤 측 어의들이 독 분석 → *바람 통로 + 신경 마비 독* 특성 확인 → *카엘의 비기 연구* 정황과 *완전 일치*
  - **카르샤 측은 *카엘이 직접 강을 독살했다고 확신***. 카엘=학살자 호칭의 정통 근거
- **바이서스 측 일부 측근** 사이에서도 *알레이언이 독으로 죽었다*·*카엘이 강을 독살했다* 의 소문 시작 — *호스반의 흘림* 이 *바이서스 내부에까지 침투*. 카엘에 대한 *불편함 + 두려움* 정통
- **카엘 본인은 *진실의 *호스반 측 뒷 획책* 정황을 모름*** — 4-5 평화 협정 발견 + 알레이언 한마디 메아리로 *처음 균열*. *진실의 완전 자각은 Part 6·7*
- **카엘 제거 명분의 정통 시드** — Part 6 두 팔 상실의 *공식 명분 = 흑마법 처분 + 강 독살 책임*. Part 7 결전 편지의 *카엘 처형 명령 명분*

### 8. 카르샤 제국 — *패전 후 상태* (사용자 보강)

#### 패전 직후 상태 (Chapter 1 종결 시점, 부드러운 처우 반영)

- **왕실 명목 유지 + 외부 망명**: 알레이언 6세 사망. *카르샤 왕가 정통 계승자 = 키얀 왕자 (9세, 외부 망명)*. 왕비 엘라리아 + 셀라엔 대주교가 *키얀과 함께 외부 망명* — *카르샤 왕실의 정통이 외부로 옮겨감*. *명목상 섭정 체제* (왕자 부재 중 통치)
- **수도 세이렌**: 바이서스 군 *단기 진주* 후 점진적 철수. *바이서스 외교 자문관 1인 상주 + 호위 기사단 약 1,000명*. 카르샤 왕실 시설 일부 사용
- **국토**: *카르샤 영토 전체 명목 자치*. 단 *바이서스 외교 자문관의 권고 권한* + *경제 협정 강제*. 남부 변경 일부 산간은 *카르샤 잔당 게릴라* 잔존 (정통성 명분 = *외부 망명 왕자 충성*)
- **인구 + 경제**: 전쟁 사상자 *민간 포함 약 4만* (현 안 8만→4만 완화). 수도 + 주요 도시 *건물 일부 파괴*. 농지 *20% 황폐화* — *2~3년 회복기* 진입

#### 바이서스의 *속국 처우* (검토 5 결정: 대안 A 채용 — 부드러운 처우 + 외부 망명)

- **정치적 관여 (명목 자치 유지)**:
  - 카르샤 *왕실 명목 자치 유지* — 카르샤 *임시 통치 위원회* 가 *명목상 카르샤 귀족 출신* 으로 구성 (실은 *바이서스 친화 인사*)
  - 바이서스 *외교 자문관 1인* 만 세이렌 상주 (직접 결재권 *없음* — 권고 권한만)
  - 카르샤 *왕실 명목 유지* — *키얀 왕자 부재 중* 의 *섭정 체제* 명목. 실제 키얀은 외부 망명

## 9. 4-4 옥좌의 방 결전 — *왕의 딸 분기* ★ (v7 신설)

본 절은 *4-4 회차 내부 장면 순서* 를 명시한다. 기존 *프롤로그 완전판 5합 결투* 는 유지하되 그 *진입 직전 + 동시 진행* 으로 새 분기 두 줄기를 끼워 넣는다.

### 9-A. 알레이언의 *내실 귀환 + 딸 보호 분기*

#### 외부 전쟁 → 내실 귀환의 *개연성 해결* (사용자 보강)

기존 캐논의 *개연성 균열* — *알레이언이 외부 전쟁장에서 어떻게 옥좌의 방에 들어와 있는가*. v7 해결:

1. **4-1·4-2 시점** — 알레이언은 *세번 강 전선·세이렌 외성 방어선* 에 *친정 출진*. 정통 *카르샤 왕가의 *친정 전통*** (왕이 군 앞장).
2. **4-3 초반** — 카엘이 *바이서스 비밀 단신 작전* 으로 *세이렌 외성 결계 분쇄 후 백색 회랑 침투*. 신성검단 학살. 카르샤 군은 *수도 외부 방어선* 에 묶여 있어 *내부 응전 부재*.
3. **4-3 후반 — *전갈 도착*** — 알레이언이 *외부 전선* 에 있을 때 *수도 내부 호위 책임자 카일렌 베르하르트* 의 *긴급 전갈* 도착: *적의 단신 마법사가 *내실로 진격 중 — 공주가 위험합니다*. 알레이언이 *전선을 부장(副將)에게 위임 + 친위 호위 12인 + 카일렌 합류 → 비밀 통로로 *수도 귀환***.
4. **4-4 시작** — 알레이언이 *내실 도착*. 시간상 *카엘이 백색 회랑을 거의 끝낸 시점*. **알레이언 = *옥좌의 방이 아니라 내실 (왕녀의 거처) 에 먼저 도착***.

→ *외부 전쟁장에 있던 왕이 옥좌의 방에 *우연히* 들어와 있는 *균열* 이 *딸 보호 명분으로 자연스럽게 해소***

#### 내실에서의 짧은 격렬 — *바이서스 군 일부의 *눈먼 공격*** 격퇴

- 카엘의 단신 침투와 *별개로* — 바이서스 *후방 정찰 부대 일부 5~6인* 이 *성공에 눈먼 상태* 로 *내실까지 무단 진입* 시도. *공식 작전 명령은 *왕실 핵심 인물 생포·해치지 말 것**(라이세린의 *공주 외교 가치 보존* 명령) 이었으나 *현장 군사들의 *전쟁 성공 충동 + 약탈 욕*** 이 *명령 무시*
- 공주 *세파* + 시녀 2인 + 호위 기사 1인이 *내실에서 협박당함*. 알레이언이 *친위 호위 12인과 함께 도착* → *바이서스 군 5~6인 즉시 격퇴*. *알레이언 = 5써클 후반 + 신성검 *루멘브링거* 의 직격* 으로 *짧은 격렬 + 무력감 거의 없는 격퇴*
- *이 짧은 격렬* 이 *알레이언의 *왕다움* 마지막 노출*. *결투 시 둔함* 의 정통 (강 독살 + 자기 희생) 과 *대조적*. 단 *이 격퇴 직후 알레이언의 *재차 둔해짐*** 이 *독의 잔여* 신체화

#### 부녀의 짧은 포옹 + 뒷길 탈출 지시

- 알레이언이 공주를 *짧게 안아준 후* — *시간이 없다. 뒷길로 가라. 키얀이 *외부에서 기다린다***. 부녀의 *마지막 대면*. 알레이언의 *왕가 정통이 외부로 이어진다는 결단*
- 공주 *세파* 의 반응 — *아버지께서는*. 알레이언: *나는 마지막이다*. *시녀 1인 + 부상 호위 기사 1인과 뒷길 진입*. **공주는 *왕의 사망을 모르고 떠남***
- 알레이언이 *내실 → 옥좌의 방* 으로 이동. *친위 호위 12인은 백색 회랑 잔존 카엘 진로 차단으로 분산*. 알레이언이 *옥좌의 방에서 카엘을 *기다림***

### 9-B. 카엘의 *공주 탈출 묵인* + 호스반의 *공주 추격 분기*

#### 카엘의 *방관* — *정의 명분의 빈자리* 시드

  - *카엘의 비뚤린 시야* — *왕만 죽이면 끝이다* 의 *임무 단순화*. *공주는 왕가의 *어린 사람* 일 뿐 — 적이 아님* 의 무의식
  - *마검사 직관* — *피로 누적 + 마나 거의 고갈* 상태에서 *추가 표적 추격은 *비효율***. *카엘의 짠내 합리화* 가 *우연히 공주에게 유리*
  - **정통적 의미** — 본 묵인은 *비뚤린 시야 너머의 *카엘의 본래 정의로움이 잠재함***(Chapter 1 HARD 룰)의 *희미한 흔적*. *4-5 호스반 의심 시드 + Part 6·7 자각의 정통 근원*

#### 호스반의 *조용한 공주 추격* (카엘 모름)

- 호스반이 *카엘의 5합 결투 진입 직전* — *바이서스 후방에서 *공주 일행의 뒷길 진로* 를 외부 시야로 포착*. *호스반 8써클 + 정보 수집 직속 부대* 의 *자동 가동*

  - 시녀 1인 *방어 시도 중 사망*

  2. 4-4 옥좌의 방 5합 결투 (*카엘 vs 알레이언*) + *동시* 뒷길에서 호스반 *공주 격리*

  5. 4-5 종결 후 — 공주는 *이미 격리 시설*. 카엘 모름

### 9-C. *호스반 동기 재구성* — *왕 사후 기습* 의 진짜 이유

기존 v6 캐논 = *호스반의 카엘 제거 시도가 *평화 협정 발견 직후 약점 노출* 을 이유로 발동*. v7 보강 = *제거 시도의 *직접 동기* 가 *왕의 사망 + 공주 격리 완료* 의 *2중 정리 완료 시점*에 정합*:

- **호스반 입장에서 *왕 사망 전 카엘 제거는 불가*** — 5합 결투를 카엘이 완수해야 *왕 사인 = 카엘 단독 책임* 의 명분이 성립. 결투 도중 카엘이 죽으면 *왕 사망 책임이 흩어짐*
- **공주 격리 완료 전에도 불가** — 공주 격리는 *비살상 + 비밀* 작전. 카엘이 *호스반 부재 사실* 을 의식 표면에 올리면 *공주 분기 노출 위험*. 호스반은 *카엘이 5합 결투 + 평화 협정 발견의 *충격*** 에 묶인 *짧은 약점 창* 을 기다림
- 따라서 *왕 사망 → 공주 격리 완료 → 호스반 회항 → 카엘 기습* 의 *3단 시퀀스* 가 *호스반 작전 설계의 최적 순서*. v7에서 시퀀스를 명시한다

### 10-A. 공주 *세파* 의 유폐 — *고위층 별실 + 외부 정보 누출*

- **유폐 위치** — 바이서스 수도 셀라리아 *외교실 관할 *고위층 거주 구역*** 의 *별실 한 채*. 라이세린의 *외교실 차관 직권* 으로 *바이서스 왕실 시야 일부에서 보호 + 라이세린 직속 시녀·경비 배치*

  2. *공주 외모 가치 활용* — *바이서스 왕실 행사·외교 자리에 *제한적 노출*** 으로 *바이서스 관용 외관* 연출

  - 공주는 *바이서스 측이 자기를 *카르샤 왕녀 신분 그대로 객인 대우* 한다 믿는 외관 + 자기를 *경계하지 않는 잡담의 빈틈*** 을 *역이용*

### 10-B. 공주가 흡수한 *2건의 핵심 정보*

#### 정보 1 — *카엘 = 학살자·독 사건의 *전쟁범죄 재판 대상***

- 카엘이 *세번 강 1,400명 광역 학살 + 카르샤 강 독살 + 왕 살해* 의 *전쟁범죄* 죄목으로 *바이서스 왕실 비밀 재판* 회부 예정
- *대부분의 죄를 카엘 단독으로 뒤집어쓰는 형태* — 호스반의 합작·라이세린의 명령은 *공식 기록에서 삭제*
- *비밀 재판* — 카르샤 측 망명자 + 외부 8왕국 외교관 *비공개*. *재판 결과만 외부 발표 — 흑마법 처분 명분 확보 용도*
- *결과 = 사형 또는 추방* 의 *2선택*. 라이세린·호스반·라카르 9세의 *최종 결정 보류 상태*. *공식 처형이 외부 명분에 유리 vs 추방이 *카엘의 추가 활용 여지** 의 *2안 저울질*

#### 정보 2 — *카엘 = 자기 아버지의 *직접 살해자***

- 공주가 *바이서스 외교관의 잡담* 에서 *알레이언의 사인 = 카엘 5합 결투 패배* 의 *공식 입장* 확인
- *내부 사정 — 실은 강 독살 + 자기 희생 둔함 + 카엘 결투* 의 합작 은 *공주에게 도달하지 않음*. 공주는 *카엘 = 아버지의 *직접 손에 의한 살해자*** 로 *확신*
- **공주의 *복수 동기 형성*** — 17세 *왕녀로서의 절제* 안에서도 *아버지의 직접 살해자에 대한 *원망 + 분노*** 가 *불을 지피는 시간*

### 10-C. 공주의 *재판 증인 출석 요청* — *복수 명분으로 라이세린에 청원*

- 공주가 *유폐 한 달 시점* 에서 *라이세린에게 *서면 + 대면 청원*** — *전쟁범죄 재판의 *결정적 증인으로 출석하고 싶다*. 아버지의 *직접 살해자를 *내 입으로 죄인으로 만들고 싶다***
- **라이세린의 반응** — *처음에는 *공주의 외교 카드 가치 손상* 우려 + *재판 비밀성 훼손* 우려로 *유보***. 그러나 *공주의 *카르샤 왕가 정통성을 *바이서스 재판의 *정통성 보강*** 으로 활용 가능* 의 *2차 계산* 으로 *수락*
- **재판 출석의 *공식 명목*** — *카르샤 왕가 직계 성년 직접 증언 = 카엘 죄 확정의 *결정적 증거**. 공주의 출석으로 *재판 결과의 *외부 명분이 강화***
- **공주의 *진짜 동기 — *2중 의도***:
  1. *의식 표면* — 복수. *아버지를 죽인 자를 내 손으로 죄인으로 만든다*
  2. *의식 깊은 곳* — **카엘과의 *직접 대면 + 탈출 협력 시드 형성***. 공주는 *재판장 = 자기가 *카엘에게 *접근할 수 있는 유일한 공식 자리** 임을 *전략적 계산* (단 *공주 본인도 *탈출 협력의 정확한 그림* 은 아직 안 잡힘. *카엘의 처지를 먼저 보고 판단* 의 *유보 전략*)

### 10-D. 비밀 재판 + 카엘의 *조용한 관망* + *깨달음*

- **재판 진행** — 바이서스 왕궁 *외교실 깊은 비공개 법정*. 라카르 9세·라이세린·호스반·왕실 사법관 + 카르샤 공주 *결정적 증인* + 카엘 *피고인*
- **공주의 증언** — *아버지가 카엘의 손에 죽은 정통 + 카르샤 군·시민 광범위 중독의 책임이 카엘에게 있다는 외부 인식* 를 *왕녀로서 *왕실 정통성 + 신성성을 담아 진술***. 공주의 *외모 + 왕가 표지의 시각적 압도력* 이 *법정 분위기를 카엘 단죄로 굳힘*
- **카엘의 반응** — *분노 + 강한 깨달음* (사용자 보강 HARD):
  - 카엘이 *재판장에서 *자기가 *처음으로 이용당했음을 명확히 인지***. *호스반의 합작·라이세린의 1회성 조커·라카르 9세의 폐기 계획* 의 *총합이 자기에게 돌아옴* 의 *세상의 원리 깨달음*
  - **분노** — *그러나 *카엘이 폭발하지 않음***. *조용히 관망*. 4-5 손의 떨림 + 알레이언 한마디 메아리 + 평화 협정 충격이 *이미 의식 깊은 곳에 누적*. *재판이 그 마지막 못을 박는 형태*
  - **세상의 원리 깨달음** — *정의·악의 이분법이 무의미. 사람은 사람을 이용한다. 내가 정의의 도구로 *부려졌고* 이제 *폐기될 때가 되니 죄인이 된다***. **카엘의 *비뚤린 시야의 진짜 붕괴 시점*** — Part 6 자각의 *직전 단계*
  - **카엘의 짧은 한 줄 응답** (재판 마지막 진술) — *"…그렇군. 알겠다."* (3음절). 외부는 *체념* 으로 인식. 실은 *모든 것을 본 자의 *조용한 응시***
- **재판 결과** — *카엘에게 *흑마법 처분 + 사형* 선고. *공식 명분 = 전쟁범죄 + 왕 살해 + 강 독살*. *형 집행 = 7일 후*

### 10-E. 카엘의 *탈출* — *마법 + 아티팩트 술 + 공주의 결정적 협력*

#### 카엘의 탈출 준비

- 카엘이 *재판 ~ 형 집행 7일 사이* — *바이서스 왕실 *지하 사형 대기실*** 에 *마나 봉인 결박 + 마법 결계* 로 감금
- **카엘의 *자기 제작 아티팩트 3종*** (§ 아티팩트·단서 — 마나 보조 팔찌·비상 회복 캡슐·살상 직관 보조 단검) 이 *체포 시 압수*. 그러나 *카엘의 *진짜 비장*** = **체내 마나 회로에 *결박 무효화 미세 아티팩트* 1점 미리 *피하 봉합*** (Part 4 자기 의술 학습의 *짠내 결과물*. 라이세린·호스반도 모름)
- *마나 봉인 결박은 *카엘의 외부 마나 출력만 차단*. *체내 마나 회로의 *자체 회복 + 미세 아티팩트 가동* 은 *시간을 들이면 우회 가능***. 카엘이 *7일간 *조용히 회로 우회 작업***
- *피하 미세 아티팩트* 가 *결박 마나 결계의 *한 점 균열을 누적적으로 뚫는* 방식. *형 집행 직전 *짧은 자유 마나 출력 창* 을 *1회 한정 제공* 으로 설계

#### 형 집행 당일 + 공주의 결정적 협력

- *형 집행 = 바이서스 왕궁 외교실 *비공개 처형장***. 라카르 9세·라이세린·호스반·왕실 사법관 + *카르샤 공주 세파 *증인 참관 자격으로 동석***
- 공주는 *한 달간 카엘의 처지 + 재판 + 사형 결정의 흐름을 관찰* — *마지막 단계에서 *자기 판단 결정*. **공주의 진짜 선택**:
  - *이 자를 *내 손으로 죽이는 것이 아버지의 복수인가*. 아니면 *이 자를 *살려 데려가는 것이 *진짜 카르샤 왕실의 도구가 되는가**
  - *바이서스가 카엘을 *그렇게 단죄하려 한다* = *카엘이 그들에게도 *위협*. 카엘을 *카르샤가 *역이용*** 하면 *바이서스에 대한 진짜 복수*
  - **결정 = *카엘 살려서 함께 탈출***
- **탈출 시퀀스**:
  1. 카엘의 *피하 미세 아티팩트가 형 집행 직전 *짧은 자유 마나 출력 창* 발동 — *마나 결박 1점 균열 + 결박 결계 일시 무효*
  2. 카엘이 *살상 직관 + 마검사 변칙 5종 중 *전기 충격·바람 폭발 도약* 즉석 시전* — 처형장 호위 마법사 2인 *짧은 마비*. 카엘이 *처형대에서 1초 만에 이탈*
  3. **결정적 위기** — 호스반이 *8써클 *압살* 즉시 시전* 으로 카엘을 *공기 응축 일점 폭발 직격* 시도. 카엘의 *기력 거의 고갈 + 회복 약 없음* — 정상 상태라면 *직격 회피 불가*
  4. **공주의 결정적 협력** — 공주가 *카르샤 왕가 비전 *신성 결계 봉지 1매*** 를 *품에서 꺼내 카엘 앞에 던짐* (외부 망명 직전 *아버지에게 받은 마지막 호신용*). *신성 결계가 호스반의 *압살을 *0.5초 분쇄*** — 카엘의 *회피 시간 확보*
  5. 카엘이 *공주를 한 팔로 끌어안고 *바람 폭발 도약* + *처형장 천장 결계 약점 직격*** — *천장 결계의 *카엘의 *바람 통로 비기 응용* 으로 *분쇄*. 사용자 HARD: *비기를 실전 사용 0회* 캐논 보존을 위해 *바람 통로 = 비기가 아닌 *기초 바람 마법의 변칙 응용*** 으로 묘사. *비기 본체는 여전히 미사용*
  6. **탈출 성공** — 카엘 + 세파가 *왕궁 외곽 *카르샤 외부 망명 라인의 *내응 인사 (셀라엔 대주교 측 셀라리아 잠입 신부 1인) 의 *비밀 도움***으로 *바이서스 수도 외곽 *비밀 안전 가옥*** 으로 이동. *키얀 왕자 외부 망명 진영 = 세파의 *생환을 *전국 비밀 망에 통보** + *카엘 동행은 *예상 밖의 정황으로 즉시 보고**

#### 카엘의 *공주에 대한 미세한 죄책감*

- 탈출 직후 *안전 가옥* 에서 *카엘이 공주를 처음 *제대로 봄***. *17세·아버지의 얼굴이 닮은 윤곽·왕녀의 절제 + 복수를 미루고 자기를 살린 결단의 무게*
- 카엘 내적 — *내가 *이 아이의 아버지를 죽였다*. *알레이언의 한마디가 다시 *메아리* — *너희가 무엇을 내게 먹였는지 알고 있는가*. 그 *너희*에 *내가 *들어 있다***
- **카엘의 *희미한 죄책감*** — *비뚤린 시야의 마지막 잔여* 너머 *처음으로 *명확한 죄책감*** 이 *생긴다*. *공주에게 *직접 사과는 하지 않음*. 그러나 *공주를 *조용히 보호*** 의 *결단 형성*
- **공주의 *처음 의도와의 분기*** — 공주는 *카엘을 *역이용* 으로 *살린* 것* 이었으나 *탈출 과정에서 *카엘이 *호스반의 압살에 자기를 내어주는 자세* 로 *공주를 우선 보호** 한 것을 *목격*. 공주의 *카엘에 대한 *복수 → 모호* 의 *1차 분기***

### 10-F. *Part 6 진입 시드* — *두 추방자의 *동행***

- Part 5 종결 후 *카엘 + 세파 카르샤 = *바이서스 왕실의 공식 적*. *카르샤 외부 망명 측의 *공주 구원 시도가 갑자기 *카엘과 동행한 형태로 외부 노출***
- 카엘 = *흑마법 처분 도주자*. 세파 = *바이서스 왕실 볼모 도주*. **두 추방자가 *동일 노선에 묶임***
- §11 (Chapter 1 극적 종결) 으로 진입 — *카엘 + 세파 의 *2인 도주 라인 + 드래곤 조우 + 두 팔 상실 + 절벽 낙하* 의 *Chapter 1 클라이맥스*. 기존 Part 6 두 팔 상실 정통이 *Part 5 종결부 §11 로 흡수*
- **카엘의 *조건 없는 가까움 회복 정통*** = §11 절벽 종결에서 *세파의 *복수 미루고 카엘을 살리는 결단*** 으로 *진짜 가족 후보의 첫 결단 형성*. Part 3 카르엔 환상 → §11 회복 시드

### 10-G. *균형 조항* — 사용자 HARD 캡 보존

- *카엘이 *바이서스 왕실 + 호스반의 8써클* 을 *단신으로 돌파해 탈출하는 묘사가 *개연성 균열* 이 되지 않도록* 다음을 유지:
  - 카엘의 *피하 미세 아티팩트* = *1회 한정 + 결박 결계의 한 점 균열만*. *호스반의 8써클 압살을 카엘 단독으로 방어 불가능* 의 한계 유지
  - **공주의 *신성 결계 봉지* 가 *호스반 압살을 분쇄하는 결정적 사건***. *카엘이 *조력자 없이 단독으로 탈출했다는 묘사 금지*
  - *세파 측의 *비밀 안전 가옥* = *외부 도주의 *조력자 없이 카엘이 어디로 가는가* 의 *균열 메우기*
- *호스반·라이세린의 *추격 시작*** — Part 6 시작 시 *바이서스 왕실의 *전국 흑마법 처분 명령*** 의 *명분 정통이 본 §10 의 *재판 결과*** 에서 발생

### 10-H. 호스반의 *전투 보고* — *강력한 추적자 미투입 개연성* ★

탈출 직후 호스반이 라이세린에게 *카엘 전투 능력 보고* 를 올림. 이 짧은 회상 장면 (4-1·4-3 정찰 결과 + 탈출 시점 묘사 총합) 이 *바이서스 왕실이 *카엘 추격에 *강력한 추적자 (8써클·소드 마스터급)* 를 보내지 않는 정통 개연성* 으로 작동.

**대화 장면 — 라이세린의 집무실, 호스반의 보고**:

> 라이세린이 *마나 측정기 옆에 놓인 카엘의 *5써클 인증서* 를 손가락으로 두드린다*. 호스반이 *왼쪽 눈 흉터를 천천히 한 번 깜빡인다*.
>
> 라이세린: *"흠. 정말 신기한 자로군. 끽해야 5써클 마법사가 7써클급 마법을 시전하는데 전투는 마검사 스타일이라니… 게다가 마검사와 많이 다르다고."*
>
> 호스반: *"하지만 그 정도입니다."*
>
> 라이세린이 *침묵 한 박자*. 그리고 *어깨를 가볍게 으쓱한다*. *마나 측정기 옆 인증서를 *밀어 옆으로 치움*.
>
> 라이세린: *"…그렇다면 굳이."* — *(말을 끝내지 않는다.)*

**정통 함의**:
- *호스반의 평가 = *공식 5써클 + 일회성 동시성 아티팩트 + 마검사 변칙*. *세번 강 7~8써클 효과는 호스반 본인의 6써클 합작 + 카엘의 압축 아티팩트 합산 결과* — *카엘 단독으로는 5써클이 최대*. 외부 정찰병이 보면 *7써클급으로 보이나 실제는 그 정도가 아님*
- *라이세린의 결단 — 강력한 추적자 미투입*. *전국 흑마법 처분 명령은 발령*하나 *고위 추적자 (8써클·소드 마스터급) 는 보내지 않음*. *일반 추격대 (정찰 마법사 + 기사 5~10인 규모) 만 동원*. *카엘이 *5써클 수준에 그친다면 일반 추격대로 충분* 판단
- 이 결정이 **Part 6 도주에서 카엘·세파가 *일반 추격대 회피만 하면 됨* 의 정통 근거**. *바이서스 8써클·소드 마스터 급 추적자가 등장하지 않는 이유*
- *동시에 *드래곤 둥지 방향 도주의 정통 근거 보강** — *일반 추격대조차 드래곤 영역에는 들어오지 않음*. 카엘이 *둥지 방향을 선택한 진짜 이유 = 추적 회피*

**라이세린의 *말 안 한 한 마디***:
- *…그렇다면 굳이* — *호스반에게 *직접 처리하라* 의 위임도 안 함*. *추격은 명목상 진행*. 라이세린의 *진짜 관심은 이미 다음 정치 사안* — *카르샤 잔당 처리 + 키얀 망명지 추적 + 외교실 권력 공고화*. *카엘은 *조커 카드가 다 쓰인 후의 잔재*. 정치적 명분만 닫히면 *방치 가능한 변수*
- 호스반의 *왼쪽 눈 흉터의 한 번 깜빡임* = *4-5 카엘 제거 시도 실패의 회한* + *라이세린의 결정에 *짧은 침묵의 동의*. *호스반 본인도 *카엘이 5써클 수준이라면 굳이 자기가 8써클 압살을 직접 시전하는 명분이 약함* 의 인정

**Part 6 으로의 정통 흐름**:
- 카엘·세파가 *드래곤 둥지 방향으로 도주* = *일반 추격대 회피 + 영역 진입 후 추격 단절*
- 그러나 *드래곤이 깨어있을 경우 카엘·세파 본인에게는 *추격대보다 훨씬 큰 위협*. *카엘의 도주로 설정의 *치명적 오판***
- 이 정통이 **Part 6 §1 도주 동선의 *드래곤 둥지 선택* 의 *카엘의 합리적 오판* 의 정통**
