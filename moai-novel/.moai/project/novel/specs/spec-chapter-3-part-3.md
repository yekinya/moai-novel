## Summary

그라스 인근 폐광에서 *키메라 *루크라* 통제 불능 폭주*. 변경 마을 3개 학살. 카엘이 우연히 처치 → 다섯 모루 결사 본거 (미라돈 지하) 잠입 → 카르덴 스펠른 비밀 후원 정황 발견 → 본거 파괴.

## 회차 구성 (9회차)

| 회차 | 비트 |
|------|------|
| 3-2-1 | 키메라 루크라 폭주 시작. 변경 마을 1차 학살 |
| 3-2-2 | 카엘·카이엔의 우연한 합류 처치. 카이엔의 *드래곤 본태 비늘 일부* 를 검에 결박 — *9써클 + 드래곤 본태 첫 융합* |
| 3-2-3 | 폭주 사건 추적. 미라돈 지하 진입 단서 |
| 3-2-4 | 다섯 모루 결사 본거 잠입 (카엘·세파·카이엔 3인 비밀 작전) |
| 3-2-5 | 본거 내부 — *카르덴 스펠른 비밀 후원 정황 + 데미안 황실 묵인 증거* 발견 |
| 3-2-6 | 본거 일부 파괴 시도 vs 내부 정예 키메라 + 백색 사냥단 합동 |
| 3-2-7 | 카르덴 스펠른과의 *대면 한마디* — *나는 자네를 미워하지 않는다. 단지 *세상이 자네 같은 사람을 두면 망한다*는 것을 알 뿐* |
| 3-2-8 | 본거 부분 파괴 + 정수 결정체 2개 확보. 결사 *남은 잔재 추적 라인* 발생 |
| 3-2-9 | 데미안 황실의 *공식 부인* + 카루린의 *베르카르 추가 폭로* |

## 핵심 비트

1. **본작 첫 *키메라 폭주 실전*** — 마법공학의 *통제 불능* 첫 가시화
2. **카이엔 + 카엘의 첫 합동 전투** — *드래곤 본태 + 9써클 융합* 의 첫 시연
3. **다섯 모루 결사 본거 부분 파괴** — Chapter 4까지 잔재 추적 라인
4. **카르덴 스펠른과의 대면** — *순수 악역이 아닌 *광신적 자본 합리주의자*** 톤
5. **데미안 황실 분열 가속**

## 인물

- 카엘 / 세파 / 카이엔 / **키메라 루크라** (보스) / **카르덴 스펠른** (마도공학원 원장 65세, 비밀 후원자) / 데미안 정예 키메라·백색 사냥단 (단편)

## 아티팩트·단서

- 키메라 정수 결정체 2개 (회수)
- 카르덴 스펠른의 *연구 노트 1권* (카엘이 부분 회수)
- 다섯 모루 결사 *조직도 한 페이지*
