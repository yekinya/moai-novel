## Summary

Ch2 P7 종결 (옥좌의 방 결전 후 카엘 셀라리아 외곽 출발 직전) 직후 — *두 시점 교차 진행*. **(드래곤 측)** *바이서스 라카르 9세 자식 척살 직후 드래곤 종족적 인지 즉시 발화. 로드 드래곤이 행커만 후예 죽음을 짧은 감지 — 행커만이 깨어나 또 기행할 것을 우려*. *로드의 종족 회의 결단 — 카자르드에게 카엘 척살 공식 의뢰*. 카자르드의 *둥지에서 짧은 결단 — 수락. 자기 영역 정통 미완 + 앤트로퍼 기색 확인 동기에 로드의 공식 임무가 합산 — 임무로 격상*. 카자르드의 *폴리모프 인간계 진입 — 30~40대 남성 검사 외형 + 왼쪽 눈 미세 황금 반점 + 가명 카이엔 오르하임*. *변경 도시 진입 + 카엘 행적 정보 수집 시작*. **(카엘 측)** *셀라리아 외곽 → 카르샤 외부 망명지 복귀*. *세파와 재회 — 짧은 침묵 + 팔찌의 부분 붉은 빛 → 안정으로 가라앉음*. *키얀의 짧은 직면 — 너는 돌아왔구나*. *카르샤 망명자들의 공식 동맹 인정 시드* (Ch3 본격). 본 Part 종결 = *Chapter 3 진입 시드 — 카엘이 드래곤이 자기를 추격하는 것을 조용히 자각 + 회복기 진입 + 마계 영지 군주 출현 첫 풍문*.

## 핵심 비트

1. **카엘의 셀라리아 외곽 출발 + 짧은 회상** (8-1) — *카엘이 셀라리아의 낮은 종소리를 등 뒤로 조용히 외곽으로 빠져나옴*. *카르샤 망명지 협공 측의 외곽 동쪽 철수와 조용히 합류 직전*. 짧은 회상 단편 — *라카르 9세의 비명·자기 보존 / 호스반의 짧은 평가 / 라이세린의 너는 결국 돌아왔구나 / 로카르의 너는… 누구의 아들이지? / 하르엔의 너는… 누구인가? / 카라엔의 왜 우리야?*. *공통 정통 — 체계적 무거움의 내면 정통 정착*. 카엘의 내적 한 줄 — *…나는 내 결단을 책임진다. 드래곤이 와도 받아들인다*
2. **로드 드래곤의 즉시 감지** (8-2) ★ — *드래곤 종족의 느슨한 인지 매개가 행커만 후예 3~4명 동시 사망에 즉시 전체 종족 경고로 전환*. *드래곤 로드 (3,000세 이상, 본작 정통 유일 1체) 가 동면 중에서 짧은 각성 — 즉시 회의 결단*
3. **종족 회의 결단** (8-2) — *고룡 (1,000세 이상) 약 50체 중 깨어 있는 5~7체 짧은 회의 참여 (대부분 동면 중)*. 회의 결단 — *행커만 본인은 수면 중. 후예 죽음을 깨어나 인지하면 또 기행 우려 매우 큼. 로드 직접 강림은 과한 매개. 처리 자체는 공식 임무로 격상*
4. **카자르드 공식 의뢰 결정** (8-2) — *영역 정통 미완 + 앤트로퍼 기색 확인 동기 보유의 최적 후보*. *임무 완수 시 드래곤 종족 내 위계 1단계 향상 + 부족 기여 인정*
5. **카자르드의 둥지에서의 결단** (8-3) ★ — *Ch1 P6 5-5 카엘 발길질 직후 영역으로 돌아갔으나 시체 미확인 미완으로 조그만 가시 같은 불편함 + 앤트로퍼 기색 정체 불명의 둥지 정적 중의 되살아남이 Ch2 P1~P7 동안 깊어짐*. *로드의 공식 의뢰 도착 직후 — 짧은 결단 — 수락. 자기 동기에 로드 임무 합산 = 공식 임무로 격상*
6. **폴리모프 인간 변신** (8-3) ★ — *Ch1 P6 수염 덮수룩한 중년 나무꾼 변신은 영역 내 단발*. *Ch2 P8 = 인간계 본격 진입을 위한 새 폴리모프*. **새 외형**:
   - *30~40대 남성 인간. 수염 없음. 평범한 떠돌이 검사 또는 약초꾼 외형*
   - **왼쪽 눈동자 미세 황금 반점 — 촛불에 반사되어 한 번 짧게 빛남** (드래곤 흔적)
   - **가명 = 카이엔 오르하임** (legacy 이름 재활용 — 오르하임은 우연히 들른 도시명 가져옴)
7. **변경 도시 진입** (8-3) — *카자르드 둥지 영역에서 인간계 변경 도시 1곳 진입 (작명 후속)*. *카엘 행적 정보 수집 시작 — 바이서스 셀라리아 결전 풍문 + 카엘의 변화한 무력 풍문이 변경 도시까지 천천히 전파됨*. *카자르드의 침착한 추적 정통 — Ch1 P6 v3 동기 그대로 + 로드 임무의 공식 격상이 매개*
8. **카엘 망명지 도착 + 호위 안내인 매개** (8-4) — *카엘이 카르샤 외부 망명지 외곽 도착*. *호위 안내인 1인 (Ch2 P2 동일 인물) 매개로 조용한 진입*
9. **세파와 재회** (8-4) ★ — *세파가 수도원 외곽에서 카엘을 기다리고 있음 — 의수와 함께 변화한 카엘의 팔 + 몸의 조용한 변화 직접 인지*. *짧은 침묵* — *팔찌가 세파의 접근으로 부분 붉은 빛 발현 → 카엘이 조용히 측정 → 안정으로 가라앉음*. 세파 — *"…돌아오셨군요."* 카엘 — *"…돌아왔습니다."*
10. **키얀의 직면** (8-4) — *키얀 왕자 9세 → 본 Part 시점 약 10세*. *카엘 앞에 정면으로 섬*. 키얀 — *"너는 돌아왔구나. 바이서스 왕은… 죽었어?"*. 카엘 *한쪽 무릎 + 짧은 끄덕임 — 모두. 자식들도 죽었습니다*. 키얀의 *조용한 침묵 → 짧은 끄덕임 — 그래. 우리 아버지가 그것을 알면 — 슬퍼하겠지. 그러나 그것이 지금의 우리의 길이다*
11. **공식 동맹 인정 시드** (8-5) — *왕비 엘라리아의 조용한 묵례 + 짧은 한 마디 — 당신은 우리의 약속을 지켰습니다*. *셀라엔 대주교의 공식 단죄 입장 부분 완화 — 너는 우리의 신앙 안에서는 여전히 단죄의 대상이다. 그러나 우리의 정치 안에서는 공식 동맹으로 받아들인다*. *비공식 → 공식 전환 시드 정착* (Ch3 본격). 카엘의 짧은 답 — *나는 돌아온다는 약속을 지켰습니다*
12. **드래곤 추격 자각** (8-6) — *카엘이 수도원 외곽 작은 방에서 조용히 의수 손가락을 펴고 접는다*. 내적 한 줄 — *…드래곤이 온다. 카자르드 본인이. 그것을 나는 Ch1 둥지에서 이미 알고 있었다. 이제 공식 임무로 나에게 온다*
13. **회복기 진입** (8-6) — *Ch2 전체 정통 — 무력 회복 + 정통 학습 + 결전 + 종결 시드 — 가 카엘의 깊은 내적 통합 단계 도달*. *Ch3 진입 시드 — 회복기 + 카자르드 양방향 추격 + 새 무대 (마계 영지 군주 출현 첫 풍문)*
14. **마계 영지 군주 출현 첫 풍문** (8-6) — *변경 떠도는 풍문 — 어느 외곽 영지에서 기괴한 마계 군주 같은 존재 출현. 정확한 정체 불명*. *세파의 조용한 감지 — 신성마법으로 외곽 마나 평형이 흔들리는 것을 느낌*. *Ch3 진입의 세계관 시드 활성화 — 본격 마계 위협의 첫 풍문*
15. **본 Part 종결 + Chapter 3 진입 시드 점화** (8-6) — 카엘의 내적 결단 — *회복하고, 카자르드를 기다린다. 마계 군주는 그 다음에 본다. 지금은 세파·키얀·망명지를 지킨다*. *팔찌 조용 — 본성의 결단*

- **금지 1**: *행커만 깨어남* — 본 Part 절대 안 일어남. *로드의 우려만 본문*. 본격 깨어남은 Ch3+ 결정
- **금지 2**: *카엘과 카자르드의 직접 직면* 본 Part 잠금 (Ch3 본격)
- **회상 대응**: Ch1 P6 5-5 카자르드 발길질 → Ch2 P1~P7 정통 누적 → Ch2 P8 *공식 임무 격상*의 *완결된 인과 곡선*

## narrative-devices

- **시점 교차**: 본 Part = *카엘 1인칭 + 드래곤 로드 3인칭 + 카자르드 3인칭의 3중 교차*. Chapter 2 종결의 *세계관적 전환 매개*
- **카자르드 새 폴리모프 시드**: *왼쪽 눈동자 미세 황금 반점이 촛불에 반사되어 한 번 짧게 빛남* — Ch3 카엘 직면 매개

## 회차 비트

### 8-1 — 옥좌의 그림자

- **공간**: 셀라리아 외곽 → 카르샤 망명지 방향 산악
- **시점**: 카엘 1인칭 (조용한 발걸음 + 회상)
- **비트**:
  - 카엘의 셀라리아 외곽 출발 + 카르샤 망명지 협공 측 외곽 동쪽 *조용한 철수와 짧은 합류 직전*
  - **짧은 회상 단편**: 라카르 9세의 비명·자기 보존 / 호스반의 *공허는 어디서 배웠나* / 라이세린의 *너는 결국 돌아왔구나* / 로카르의 *너는… 누구의 아들이지?* / 하르엔의 *너는… 누구인가?* / 카라엔의 *왜 우리야?*
  - *공통 정통 — 체계적 무거움의 내면 정통 정착*
  - 카엘의 내적 한 줄 — *…나는 내 결단을 책임진다. 드래곤이 와도 받아들인다*

### 8-2 — 종족의 회의 ★ *(드래곤 로드 3인칭 시점)*

- **공간**: 드래곤 종족 회의장 (정확한 위치 비공개 — 대륙 어느 깊은 산령)
- **시점**: **드래곤 로드 3인칭 짧은 시점**
- **비트**:
  - *드래곤 종족의 느슨한 인지 매개가 행커만 후예 3~4명 동시 사망에 즉시 전체 종족 경고로 전환*
  - 드래곤 로드 (3,000세 이상, 본작 정통 유일 1체) 동면 중에서 *짧은 각성 — 즉시 회의 결단*
  - **종족 회의** — 고룡 깨어 있는 5~7체 짧은 회의 참여
  - 회의 결단 — *행커만 본인 수면 중. 후예 죽음 인지하면 또 기행 우려 매우 큼. 로드 직접 강림은 과한 매개. 처리는 공식 임무로 격상*
  - **카자르드 의뢰 결정** — *영역 정통 미완 + 앤트로퍼 기색 확인 동기 보유의 최적 후보. 임무 완수 시 종족 내 위계 1단계 향상 + 부족 기여 인정*

### 8-3 — 둥지의 결단 ★ *(카자르드 3인칭 시점)*

- **공간**: 카자르드 둥지 영역 (Ch1 P6 무대)
- **시점**: **카자르드 3인칭 짧은 시점**
- **비트**:
  - Ch1 P6 5-5 카엘 발길질 직후 *영역 복귀했으나 시체 미확인 미완 + 앤트로퍼 기색 정체 불명*의 *둥지 정적 중의 되살아남이 Ch2 P1~P7 동안 깊어짐*
  - **로드의 공식 의뢰 도착 직후 — 짧은 결단 — 수락**. *자기 동기에 로드 임무 합산 = 공식 임무로 격상*
  - **폴리모프 인간 변신** — 새 외형 (30~40대 남성 검사 + 왼쪽 눈 황금 반점 + 가명 카이엔 오르하임)
  - 카자르드의 *침착한 추적 정통* — Ch1 P6 v3 동기 그대로 + 로드 임무 공식 격상 매개
  - **변경 도시 진입 + 카엘 행적 정보 수집 시작** — 바이서스 셀라리아 결전 풍문 천천히 전파됨

### 8-4 — 망명지의 문

- **공간**: 카르샤 외부 망명지 (셀라엔 망명지) 외곽 → 본관
- **시점**: 카엘 1인칭 (조용한 안도)
- **비트**:
  - 카엘의 망명지 외곽 도착 + 호위 안내인 1인 (Ch2 P2 동일 인물) 매개로 *조용한 진입*
  - **세파와 재회** — *수도원 외곽에서 세파가 카엘을 기다리고 있음*
    - 의수와 함께 변화한 카엘의 *팔 + 몸의 조용한 변화 직접 인지*
    - 짧은 침묵 — *팔찌가 세파의 접근으로 부분 붉은 빛 발현 → 카엘이 조용히 측정 → 안정으로 가라앉음*
    - 세파 — *"…돌아오셨군요."*
    - 카엘 — *"…돌아왔습니다."*
  - **키얀의 직면** (10세, Ch2 P2 9세 → 약 1년 경과)
    - 키얀: *"너는 돌아왔구나. 바이서스 왕은… 죽었어?"*
    - 카엘 *한쪽 무릎 + 짧은 끄덕임 — 모두. 자식들도 죽었습니다*
    - 키얀의 *조용한 침묵 → 짧은 끄덕임 — 그래. 우리 아버지가 그것을 알면 — 슬퍼하겠지. 그러나 그것이 지금의 우리의 길이다*

### 8-5 — 동맹의 시드

- **공간**: 망명지 본관 + 셀라엔 대주교의 거처
- **시점**: 카엘 1인칭 (조용한 명료성)
- **비트**:
  - **카르샤 망명자들의 공식 동맹 인정 시드**:
    - 왕비 엘라리아의 조용한 묵례 + 짧은 한 마디 — *당신은 우리의 약속을 지켰습니다*
    - 셀라엔 대주교의 *공식 단죄 입장 부분 완화* — *너는 우리의 신앙 안에서는 여전히 단죄의 대상이다. 그러나 우리의 정치 안에서는 공식 동맹으로 받아들인다*
    - *비공식 → 공식 전환 시드 정착*
  - 카엘의 짧은 답 — *나는 돌아온다는 약속을 지켰습니다*
  - 팔찌 조용 — *붉은 빛 없음. 본성의 결단*

### 8-6 — 추격의 그림자 *(Chapter 3 진입 시드)*

- **공간**: 수도원 외곽 작은 방
- **시점**: 카엘 1인칭 (회복기 + 결단)
- **비트**:
  - 카엘이 *수도원 외곽 작은 방에서 조용히 의수 손가락을 펴고 접는다*
  - 내적 한 줄 — *…드래곤이 온다. 카자르드 본인이. 그것을 나는 Ch1 둥지에서 이미 알고 있었다. 이제 공식 임무로 나에게 온다*
  - **회복기 진입** — Ch2 전체 정통 *카엘의 깊은 내적 통합 단계 도달*
  - **마계 영지 군주 출현 첫 풍문**:
    - *변경 떠도는 풍문 — 어느 외곽 영지에서 기괴한 마계 군주 같은 존재 출현*. 정확한 정체 불명
    - 세파의 조용한 감지 — 신성마법으로 *외곽 마나 평형이 흔들리는 것을 느낌*
    - *Ch3 진입의 세계관 시드 활성화 — 본격 마계 위협의 첫 풍문*
  - **본 Part 종결**:
    - 카엘의 *내적 결단* — *회복하고, 카자르드를 기다린다. 마계 군주는 그 다음에 본다. 지금은 세파·키얀·망명지를 지킨다*
    - 팔찌 조용 — 본성의 결단
    - **Chapter 3 진입 시드 점화**
