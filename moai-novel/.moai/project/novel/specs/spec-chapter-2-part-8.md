## Summary

Ch2 P7 종결 직후 — *바이서스 군사 비상령 본격 발화*. *라카르 9세 + 왕실 직속 호위 + 잔존 정규 사령부가 *왕실 직속 5인 소드 마스터 + 정규 소드 마스터 80~90명+ 즉시 비상 소집 + 길드 용병 약 2~3만 추가 동원 + 셀라리아 전역 봉쇄*. **카엘 vs 소드 마스터 수십명 협공 — 1차 격돌**. *카엘의 아다만티움 의수 + 공허검 + 마력→내공 + 마력포 = 일반 마스터 1대 1 우위*. 그러나 *동시 협공 10~15명에 *압도당함*. *후퇴 + 은신 + 룬 마법사 2인 매개로 *전술 재정비 약 1주 + 비밀 거점 이동 + 마스터 협공 사슬 약점 분석 → 2차 재격돌*. **2차 재격돌 = 카엘의 *분산 유인 + 마력 대포 함정 매개 + 공허검 1·2원리 합성 정통* → 소드 마스터 상당수 (약 30~40명) 척살 + 카엘 본인 누적 부상·정신 피로 최대점*. *왕궁 외곽 진입 + 왕실 직속 호위 5인 + 엠퍼러급 1명 격돌*. **9써클 대마도사의 부재 — 왕가 안전 최우선으로 *왕성 깊은 곳 결계 보호로 후퇴*** (본작 정통 — 9써클은 *왕실 안전 최후 보장*. 라카르 9세가 *비상령 발화와 동시에 9써클에게 자식들·왕성 결계 우선 명령*). **엠퍼러급 1명 격돌 = 카엘의 *마력 대포 총동원 + 자기 무기 전부 소진 + 룬 회로 부분 균열 + 의수 일부 균열* → 가까스로 척살**. *옥좌의 방 진입*. **라카르 9세 직면 — *힘이 없는 왕*. *그러나 *과거 수많은 리셋의 진실을 알고 있는 왕가의 마지막 *경전 보유자***. *카엘이 그를 죽이기 전 *대현자 시대 이후 수많은 리셋의 진실 + 앤트로퍼와 사피엔의 영원한 순환 + 카엘 본인이 *몇 번이고 다시 태어나 같은 결단을 반복하는 자라는 정통 단편*을 마주함*. *라카르 9세 척살 + 남은 자식 일부 척살 (행커만 후예 포함)*. **9써클 대마도사는 결국 왕실 결계 보호를 끝까지 유지 + 카엘과 직접 직면 거부 → 카엘의 왕성 출발 시점에 *결계 해제 + 자기 거처로 잠적*** (대륙 정통 — 9써클은 *왕실의 결단보다 자기 정통적 결단 우선*). 결전 종결. *바이서스 왕권 다툼 즉시 발화 — 라카르 9세 형제·남은 왕가 직계 분파 다수 + 라이세린 사적 사슬 잔존 후원 자식들의 *공식 후원자 부재 상태에서 각자 자기 권력 다툼 본격*. *바이서스 = 멸망 안 함 + 헤븐즈 8왕국 종주 체계 사실상 와해 + 주변국 빠른 이권 침탈 + **카르샤 주권·국가 복원 즉시 시작***.

## 핵심 비트

1. **군사 비상령 본격 발화** (8-1) ★ — *라카르 9세 + 잔존 정규 사령부 결단*. *왕실 직속 5인 소드 마스터 + 정규 소드 마스터 80~90명+ 즉시 비상 소집 + 길드 용병 약 2~3만 추가 동원 + 셀라리아 전역 봉쇄*. *9써클 대마도사 = 비상령 결단과 동시에 *왕가 자식들·왕성 깊은 결계 보호로 후퇴 발령 (왕실 안전 최후 보장의 정통 직무)*. 카엘 측 = *세렌 마르덴·베라트 알린의 시야 차단 + 외곽 연구소 잔존 인물 매개 + 비밀 거점 이동 준비*

2. **1차 협공 + 카엘의 무력 한계 노출** (8-2) ★ — *셀라리아 시내 어느 광장에 카엘 진입 시점 — 정규 소드 마스터 약 10~15명 동시 협공*. *카엘의 4단계 의수 + 공허검 + 마력→내공 + 마력포 = 1대 1 마스터 우위 (마스터 5명까지는 부분 우위)*. *그러나 10~15명 동시 협공의 *검기 사슬 압박 + 광역 합성 진형 + 카엘 회피 공간 제거*에 *압도당함*. *카엘 본인 의수 미세 균열 1차 + 갈비뼈 균열 + 마력 회로 일시 마비*. **결단 — 후퇴**. *룬 마법사 2인의 *원거리 시야 차단 + 광역 결계 분쇄 보조 매개*로 *겨우 탈출*

3. **후퇴와 재정비** (8-3) — *비밀 거점 이동* (셀라리아 외곽 별개 안전 가옥 — 카르샤 망명지 협공 측 외곽 동쪽 라인의 *비공식 보조 거점*). *약 1주 회복 + 룬 마법사 2인 매개 + 마스터 협공 사슬 약점 분석*. *분석 결과 — *마스터 협공의 사슬은 *진형 중심부 무력화로 일시 해체 가능 + 분산 유인으로 사슬 자체를 *카엘 함정 안에 들어오게 만들 수 있음*

4. **2차 재격돌 — 마스터 상당수 척살** (8-4) ★ — *카엘의 *분산 유인 + 마력 대포 함정 + 공허검 1·2원리 합성 정통*. *셀라리아 외곽 산악 좁은 골짜기·왕궁 진입 회랑·셀라리아 시내 좁은 골목 등 *카엘이 사전 조사한 지형 함정*에 *마스터 협공 사슬을 분산 유인*. **소드 마스터 약 30~40명 척살** (정규 100명+ 중 1/3~1/2 사망). *카엘 본인 누적 부상·정신 피로 최대점* — *의수 일부 균열·갈비뼈 다수 균열·왼쪽 어깨 부분 마비·룬 회로 30% 손상·마력 회로 60% 소진*

5. **왕궁의 문 + 9써클 부재 자각** (8-5) ★ — *왕궁 외곽 진입*. *왕실 직속 호위 5인 마스터 + 엠퍼러급 1명 직면*. **카엘이 *9써클 대마도사가 보이지 않음을 즉시 자각***. *왕성 깊은 결계의 *마나 압력 패턴이 9써클 광역 차단 결계*임을 인지 — *왕가 자식들 + 왕성 핵심 보호로 후퇴*. *카엘의 결단 — 9써클은 끝까지 안 나온다. 그것이 그의 정통이다. 그러면 나는 *엠퍼러급 + 5인 마스터만 처리하면 된다*

  - **카르난 발도르 6중 정통 동시 발현** — 온몸 검기화 + 카보나이트 갑옷 + 1초 시전 차단 + 검기 단검 던지기 + 검은 검 + 5인 마스터 보조
  - **카엘 6중 대응 전술** — 마력 대포 총동원 + 충전식 폭열탄 + 룬 결각 광역 함정 + 신성 결계 봉지 1매 + 공허검 1·2원리 합성 + 룬 마법사 2인 시야 차단 매개
  - 호위 마스터 5인 차례차례 척살 (약 30분~1시간 격투)
  - *카르난의 검기 단검 던지기를 룬 마법사 2인 시야 차단 매개로 부분 회피*
  - *공허검 1원리 (불교) 비키기 시도 — 그러나 카르난의 응축 일격은 광역 마법이 아닌 근접 정점 검기로 비키기 매개 약함*

  - **결정타** — *공허검 2원리 (공출) + 마력포 잔량 + 신체 직접 검기 합성 일격*. 카르난의 검은 검 직격 직전 시점에 *공허검 공출 매개로 검 궤도 비킴 + 마력포 잔량 일격 = 카르난 옆구리 + 카보나이트 갑옷 균열 + 카르난 본체 직격*
  - **카르난 가까스로 척살** — 마지막 — *"…너는… 처음 보는 길이군. 내가 평생 알던 검과 완전히 다른 길. 나의 길이 너의 길에 졌다."*
  - **카엘의 결정적 자각** — *카르난 같은 엠퍼러급이 2명이었다면 나는 결단 불가능했다. 두 정점이 동시 직격했다면 — 나는 죽었다*
