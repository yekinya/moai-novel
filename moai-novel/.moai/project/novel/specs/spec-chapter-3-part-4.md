# Chapter 3 Part 4 Spec: 위험한 조각

> Chapter 3 overview: [`./spec-chapter-3.md`](./spec-chapter-3.md)

## Summary

7비전 4조각이 카엘 주변에 모이며 카르엔 학파 원로 *세이언 라모르 학장* (국립상아탑) 이 비밀 사절로 접촉. 사프란 비망록 *3페이지 추가 공개* — *7조각을 모으지 마라* + 가설 D (즉각 리셋) 위험.

## 회차 구성 (8회차)

| 회차 | 비트 |
|------|------|
| 3-4-1 | 카엘이 *7비전 4조각 모임* 의식. 세파 + 카이엔과의 *위치 점검* |
| 3-4-2 | 셀라엔의 *균열의 빛 절반 추가 의식 도해* 송부 |
| 3-4-3 | 그라스 노 장인의 *사프란 비망록 마지막 페이지 + 추가 1페이지* 공개 |
| 3-4-4 | 라모르 학장의 *비밀 사절* 도착. *7조각을 모으지 마라 — 모이면 *내가 의도하지 않은 것* 이 작동할지도 모른다* |
| 3-4-5 | 가설 4종 검토 — A (DNA 정화) B (균열 영구 봉인) C (사프란 부활) D (즉각 리셋) |
| 3-4-6 | 카엘의 갈등 — *4조각도 위험할 수 있다* vs *마왕을 막으려면 더 필요* |
| 3-4-7 | 라모르의 한마디 — *내가 자네에게 줄 답은 *나도 모른다* 뿐이네*. 학장의 침묵 |
| 3-4-8 | 결단 보류. *4조각 부분 결합 의식만* 사용하기로. Chapter 3 클라이맥스 마왕 처치에만 사용 |

## 핵심 비트

1. **사프란 비망록 추가 페이지** — *위험* 의 의미 본격 공개
2. **가설 D (즉각 리셋) 위험 40%** — 본작 *7비전 결합 의식의 진짜 무게*
3. **라모르 학장의 *침묵*** — 본작 *모르겠다는 학자적 정직함* 의 첫 사례
4. **결단 보류** — Chapter 3 클라이맥스에 *부분 결합만* 사용. *전체 결합은 Chapter 4*
