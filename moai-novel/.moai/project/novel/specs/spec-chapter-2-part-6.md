## Summary

## 핵심 비트

### 1. 첨탑의 도서관 + 비공개 자료 직면 (6-1)

**마르덴의 첨탑**:
- *앨더스톰의 *고대 마법 도서관 본관 + 부속 첨탑*. 마르덴의 거주지
- *비공개 보관 자료 = 행커만 정통 + 바이서스 정치 비공식 기록 + 다른 5왕국 비공식 외교 단편*

**행커만 단편 첫 직면**:
- 마르덴이 *오래된 양피지 1장을 *카엘 앞에 *천천히 펼침*
- *양피지 = 칠흑성 전투 직후 일레인 맥필렌이 직접 작성한 *행커만 폴리모프 폭로 기록의 *유일한 원본 사본*
- 마르덴의 한 마디 — *"너의 결단의 매개가 *여기에 있다."*

### 2. 행커만 폴리모프 과거 공식 공개 (6-2)

**칠흑성 전투 + 정체 폭로**:
- *CC 870 칠흑성 전투* — *일레인 맥필렌이 행커만의 *드래곤 정체 폭로*
- *행커만이 시전한 *운석강하를 *일레인이 *중력장 + 압축 얼음창으로 공중에서 분해해 막아냈다*
- *부상 + 변신술 강제 해제 성공*. *행커만은 *드래곤 본태로 돌아가며 도주 → 동면 상태로 진입*. *현재 본작 시점 (CC 1000~1001) 까지 깨어나지 않음*

**행커만 본인 진술 (역사적 발언)**:
- *드래곤들의 응용력은 형편없다. 그래서 인간의 모습으로 응용력을 배우기 위해 세상을 주유했다*
- *마법학계 = 행커만의 정통 = *고대마법이론 (392쇄 발행, 사피엔 마법서 정석)* 가 *드래곤의 학습 결과*

**마르덴의 짧은 평가**:
- *행커만은 *드래곤이었으나 *그의 정통이 *인간 마법학을 *어떻게 바꿨는지는 *우리가 그에게 빚지고 있다. 정체와 별개로*

### 3. 바이서스 왕가 후예 + 반인반룡 정통 (6-3) ★

**행커만의 인간 위장 시기**:
- *바이서스 카멜라 4세 시대 = 행커만이 *바이서스 총사령관 (마법 총사령관) 임명*
- *임명 시기 동안 *바이서스 왕가에 *인간 모습으로 합류*. *결혼·자식 양육의 정확한 방식은 *역사적으로 영구 모호*
- *후예 자식들은 *반인반룡 혈통* — *외관은 인간 + 마나량은 일반 인간의 3~5배 + 드래곤 비전 미세 잔존 (예: 본인 모르게 일부 마법 시전 효율 향상)*. **본인들도 자기 핏줄 모름** — 행커만이 *왕가에 어떻게 합류했는지는 *역사적 비공개*

**현 시점 후예**:
- *카멜라 4세 → 라카르 9세까지 약 130년. 4~5대 후예 라인*
- *라카르 9세의 *형제·자식 일부 = 행커만 후예 가능성이 매우 높음*
- *마르덴의 추정 명단 — 라카르 9세의 *3왕자 하르엔 + 2왕자 데이렌 + 4공주 리에네 (이름은 모두 후속 결정·검토) 등 3~5명이 *반인반룡 혈통 의심*

**드래곤 종족적 보호 의식**:
- *행커만 본인은 *수면 중이나 *자식들의 *반인반룡 핏줄은 *드래곤 종족이 *느슨하게 인지 + 보호 의식 존재*
- *드래곤 로드의 *간헐적 감시* — *후예가 *위협받지 않는 한 직접 개입 안 함*
- *카엘이 *후예를 척살하면 *드래곤 종족적 적대 즉시 발화*

**카엘의 *조용한 자각***:
- 카엘이 *팔찌를 한 번 봄*. *팔찌 조용 — 붉은 빛 없음*
- *카엘의 내적 한 줄 — …내 결단은 *그것을 *알고도 *변하지 않을 것이다*

### 4. 바이서스 악행 공유 (6-4)

**4중 악행 정리**:
1. **라이세린 위조 첩보** — *카르샤 평화 사절단 도래 직후 *외교 파탄을 위한 위조 첩보 작성. *카르샤 악마화 강연 + 선전포고의 정치적 매개*
2. **호스반 광범위 독살** — *카르샤 주요 강·식수원에 다량 독 투입. *바이서스 군 실수로 가장*. *카르샤 시민·왕실 광범위 중독*
3. **알레이언 6세 자기 희생** — *해독 물품 모두 군사들에게 분배 + 자기 해독 거부*. *4-4 5합 결투의 진짜 둔함의 정통*. *카엘 = 학살자 호칭의 근원*
4. **평화 협정 위조 + 카엘 1회성 조커 카드** — *카엘은 *라이세린·호스반의 *1회성 무기*. *호스반이 카엘의 비기 자료 무단 합성 + 광역 학살 시각 효과 조작 + 왕 사후 즉결 처분 임무*. *카엘 본인이 *정의의 도구로 *오해받은 *외부 정치 본질*

**카엘의 *분노의 응축***:
- *Ch1 4-5 자각 이후 처음으로 *공식 정보가 *체계적으로 정리됨*
- *체외 분노 발현 없음 — 카엘의 내공 회로가 *분노를 *깊이 응축*. *팔찌가 *조용히 *부분 붉은 빛*
- 카엘의 한 마디 — *"…알고 있었다. 그러나 *공식 단편으로 *처음 본다."*

### 5. 앨더스톰 비공식 동맹 (6-5)

**마르덴의 결단**:
- *행커만의 *과거 + 바이서스의 악행은 *우리가 *오래 보고 있던 것*. 너의 결단을 *돕는다*
- **앨더스톰의 *비공식 동맹 결단 — 바이서스 공격 시 도움 약속***
- *조건 = 카엘이 *바이서스 결전 시 *앨더스톰의 *룬 마법사 1~2명 비공식 지원 + 행커만 후예 척살 시 *드래곤 종족 적대 발화에 대한 *공동 대응 준비*

**다른 5왕국 비공식 인맥 라인**:
- *마르덴이 *데미안 (마도공학 일부 분파) + 강철제국 (세이른 라인) + 마녀령 (모르간 가문 외곽) + 카르샤 망명자 (셀라엔 대주교 신앙 라인) + 드워프 (앨더스톰 본가 매개) 의 *5중 비공식 인맥 라인을 *카엘에게 공유*
- *공식 동맹 아닌 *카엘 결단의 *조용한 지지 매개*

### 6. 다양한 마법 지식 양도 + 결전 결단 (6-6)

**현재 구현 불가 미래 시드**:
- *마르덴이 카엘에게 *다양한 고대 마법 기술과 지식 양도* — *현재 카엘의 마법 봉인 상태로는 직접 구현 불가*
- *미래 시드* — Ch3+ 카엘의 *마법력 총량 향상 가능성 + 의수 진화 + 룬 정통 확장 시점에 *부분 구현 가능*
- 주요 양도 정통:
  - *고대마법이론 발췌 정통* (행커만 원본 일부)
  - *룬 결각 고급 정통* (드워프 지하세계 지식과 상호 보완)
  - *카르엔 학파 잠적 분파의 *외부 단편* (카엘 친조부 알리스 정통과 연결)
  - *마녀령 9혈 외부 손님 정통의 *심화 단편*
  - *바이서스 왕가 비공식 외교 기록* (결전 시 활용 가능)

**카엘의 *바이서스 결전 결단***:
- *모든 정보 + 동맹 + 무력 + 정통이 *한 점에 모임*
- 카엘의 결단 — *나는 *바이서스를 *치고 *돌아오지 않을 것이다. 그러나 *행커만 후예가 *내 손에 죽는다면 — *드래곤 종족이 *나를 추격할 것이다. *그것도 *받아들인다*
- 팔찌 조용 — *붉은 빛 없음. 본성의 결단*

**결전 준비 진입**:
- *마르덴이 *바이서스 결전 동선 + 정보 + 동맹 매개를 *조용히 정리*
- *카엘의 *앨더스톰 출발 직전 종결*
