## Summary

칠흑숲 변경에 *마계 영지 군주 비카르나스* 첫 강림. 변경 마을 4개 학살. 카엘 + 셀라엔 + 모르간 솔테 + 카이엔의 *4축 부분 결합 의식* 으로 *간신히* 처치. 카이엔 정체 *완전 노출*. 의수 1단계 부서짐.

## 회차 구성 (9회차 — 제안)

| 회차 | 제목 (가제) | 비트 |
|------|-----------|------|
| 2-5-1 | 균열의 전조 | 카엘이 그라스 도착. 백색 사냥단 정예 3명을 *카이엔이 우연히 처치* (카이엔의 비공식 합류 신호) |
| 2-5-2 | 첫 학살 | 비카르나스 강림 — 변경 마을 1차 학살. 카엘 *직접 도착하나 너무 늦음* |
| 2-5-3 | 그의 이름 | 비카르나스 인물 묘사 — 키 2.5m + 검은 갑주 + 박쥐 날개 + 인간형. *마계 방언으로 카엘에게 말함*. *너의 사슬을 본다* |
| 2-5-4 | 의식의 4축 | 카엘 + 셀라엔 (소환 + 신성마법 + 균열의 빛 부분) + 모르간 솔테 (회수 회로 부분) + 카이엔 (드래곤 본태 마나) — *4축 결합 의식* 준비 |
| 2-5-5 | 의식의 시작 | 8시간 영창 + 비카르나스 *마을 5번째 공격 도중 의식 표적화* — 의식 *방어 결계* 동시 운용 |
| 2-5-6 | 정체의 노출 | 카이엔의 *왼쪽 눈 황금빛 완전 노출* + 셀라엔 *경악 + 즉시 수용*. 모르간 솔테가 *린이엔이 알았을 것* 추정 |
| 2-5-7 | 비카르나스 결착 | 균열의 빛 부분 + 회수 회로 부분 + 카이엔 본태 마나 + 카엘 사슬 일부 균열의 *합* — *비카르나스 결박 분해*. 카엘 *의수 1단계 부서짐* |
| 2-5-8 | 마계 진주 | 처치 자리에 *마계 진주 2개* 형성. 베르난 호산 (금혈) 정중히 회수. 후일 *마왕 협상 자원* 시드 |
| 2-5-9 | 영지 군주의 의미 | 사프란 비망록 *추가 페이지* — *영지 군주 출현 = 1년 내 마왕*. 카엘이 *처음으로 등에 식은땀* |

## 핵심 비트

1. **본작 첫 마계 영지 군주 등장** — 외형 + 마계 방언 + *카엘의 사슬을 본태 시야로 인지*
2. **4축 부분 결합 의식** — Chapter 1에서 *7비전 7조각* 의 *부분 활용 첫 사례*
3. **카이엔의 정체 *공식 노출*** — 외부 (셀라엔·모르간) 에 첫 공개. 의외로 *경악 후 즉시 수용*
4. **의수 1단계 부서짐** — 4축 의식 매개로 사용한 결과. 2단계 시술 필요 시드
5. **카엘이 처음으로 *진짜 적* 을 만남** — 비카르나스의 *마계 방언 인사* + *사슬 인지* 의 무게
6. **마왕 1년 카운트다운 시작** — Chapter 3 진입 전조

## 인물

- 카엘 / 세파 / **카이엔** (그라스 인근 *우연 등장*) / **셀라엔 대주교** / **모르간 솔테** (마녀령에서 직접 동원) / **베르난 호산** (금혈, 마계 진주 회수) / **비카르나스** (마계 영지 군주) / 그라스 노 장인 (백색 사냥단 살해 위협 직후 카엘과 재회) / 변경 마을 평민 다수 (학살 묘사)
