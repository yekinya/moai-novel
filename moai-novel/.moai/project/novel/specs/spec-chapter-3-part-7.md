# Chapter 3 Part 7 Spec: 소드 엠퍼러의 의미

> Chapter 3 overview: [`./spec-chapter-3.md`](./spec-chapter-3.md)

## Summary

오르가트 처치 후 1주일. 카엘이 *9써클·소드 엠퍼러급 공식 인정*. 그러나 *충분히 강하지 않음* 자각. 부수 피해 추모. 카엘이 *진짜 적은 그 다음* 임을 처음 인식.

## 회차 구성 (7회차)

| 회차 | 비트 |
|------|------|
| 3-7-1 | 처치 직후 *연합군 해산 + 전사자 명단 작성*. 변경 마을 3개 학살 명단 |
| 3-7-2 | 드래곤 1체 추모 — 카이엔이 *동족 익명 사망에 대한 짧은 침묵 의례* |
| 3-7-3 | 8왕국 *학자 회의* — *카엘 9써클·소드 엠퍼러급 공식 인정* 결의 |
| 3-7-4 | 라카르 9세의 *비공개 친서* — *짐은 자네를 더 이상 도구로 보지 않네. 자네가 *짐의 후예* 가 되는 길도 있네* (카엘이 거절) |
| 3-7-5 | 카엘이 *드래곤 1체 사망지* 직접 방문. 카이엔과 함께 *침묵의 추모* |
| 3-7-6 | 카엘의 *진짜 적은 그 다음* 자각 — 드래곤·정령왕·고룡의 무게 첫 직시 |
| 3-7-7 | 의수 4단계 도면 *최종 확정* — *아다만티움 + 양팔 비대칭 + 메크 인터페이스 1차*. 드워프 산 방문 결단 *재확정* |

## 핵심 비트

1. **공식 인정** — 본작 *영웅 칭호 + 위험 응축체* 의 모순 시작
2. **추모 의례** — 본작 *영웅의 무게의 첫 신체화*
3. **라카르 9세의 *후예 제안*** — 본작 *왕가 정통성 흡수 제안*. 카엘 거절 = *학살자의 정체성을 *왕족화* 로 세탁하지 않음*
4. **드래곤 추모** — 카이엔의 *내적 분열 정점* 시드
5. **의수 4단계 최종 도면** — Chapter 4 진입 직전

## 인물

- 카엘 / 세파 / 카이엔 / 셀라엔 (회복 중, 간접) / 8왕국 학자 회의 (간접 묘사) / 라카르 9세 (친서로만) / 키얀 일행 (그림자 추적 유지)

## 아티팩트·단서

- 전사자 명단 (마을 3개 + 군 다수 + 드래곤 1체)
- 8왕국 학자 회의 결의문 (카엘 공식 인정)
- 라카르 9세 비공개 친서 (후예 제안)
- 의수 4단계 *최종 도면*

## 지명

- 칠흑숲 중심 (해산지) / 드래곤 1체 사망지 (북쪽 산악) / 그라스 (의수 도면 최종지)

## Tone

- *영웅의 외로움* + *부수 피해의 죄책감*
- 짠내 *낮음* — 단 *심리적 짠내 정점*
- 유머 *0*

## ★ Review points

1. **3-7-4 라카르 9세 *후예 제안*** — 본 Part 핵심 시드. *직접 친서* vs *간접 사절을 통한 제안*?
2. **3-7-5 드래곤 추모** — 카이엔의 *동족 침묵 의례* 분량 — 1회차 (현 안) vs 회차 일부?
3. **카엘의 *영웅 칭호 수용 vs 거절*** — 현 안: *수용도 거절도 아닌 침묵*. 더 명시?
4. **7회차 분량** — 5회차 압축 (3-7-1+3-7-2, 3-7-5+3-7-6 통합) 가능?
