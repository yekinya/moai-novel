## 1. Chapter 1 한 줄 시놉시스

> 변경에서 *저주받은 아이* 로 태어난 카엘이, *떠돌이* 와 *학자* 의 길을 거쳐 *왕실 마법 고문* 까지 올라가고, *적국 왕을 살해* 하는 순간 *자기 시야의 왜곡* 을 자각한다. 두 팔을 잃고 *공허검* 에 입문한 그가, 폐허 마탑에서 *나는 여기에 있다* 는 편지를 띄우며 첫 결전을 예고한다.

## 2. 8개 회차 파일 — 비트 통합 (집필 직전 최종 spec)

### 2.1 프롤로그 — 옥좌의 피 (1회)

- **시점**: 1인칭 (카엘) · **분량**: 6,000자
- **공간**: 카르샤 왕궁 옥좌의 방 (CC 1000 봄)
- **3막 구조**:
  1. (1막) 옥좌까지 — 백색 회랑·호위 시신·머리카락 목걸이 단서
  2. (2막) **호각의 5합 결투** — 알레이언이 신성안으로 *붉은 사슬 7마디* 확정 → 즉발판 균열의 빛 시전 → 자기 죽음 선택
  3. (3막) **0.5초 자각의 창** + *너는 무엇을 보고 있는가* + 회상 4컷
- **단서 5종 + α**: 머리카락 목걸이 / 도망치소서 비명 / 손 모음 문장 / 평화 협정 양피지 / 옅어진 일렁임 / *얇은 금속음* / *가장자리 이중상* / *7대를 기다린 자*
- **세부**: [`../chapters/00-prologue.md`](../chapters/00-prologue.md)
- **결말 후크**: *「20년 전, 변경의 어느 마을.」*

### 2.2 part-1-1 — 저주받은 아이 (5회, 1-1 ~ 1-5)

- **시점**: 3인칭 근접 (카엘) · **시기**: CC 980~996
- **공간**: 노이엔 변경 → 폐허 마탑 (사흘 거리)
- **핵심 비트**:
  - 1-1 *산파가 도망쳤다* / *엔트로퍼의 씨* 노인 한마디
  - 1-2 우물가 사과 사건 / *곱게 웃는데* 아낙의 객관 진술
  - 1-3 성년식 첫 살상 사고 (단의 다리)
  - 1-4 마을 회의 추방 / 부친 알린의 *책 + 무릎*
  - 1-5 폐허 마탑 첫 마나 점화 / *세상이 잘못되었다면 내가 바꾼다*
- **단서**: 친조부의 책 (= 그라스 카르엔 학파 잠적 분파 사본), 책 첫 문장 *세상을 미워하는 자는 마나를 다루지 마라*, 책 여백의 동일 문장 3중 옮겨 적힘
- **세부**: [`../chapters/part-1-1.md`](../chapters/part-1-1.md)

### 2.3 part-1-2 — 떠돌이의 길 (7회, 1.5-1 ~ 1.5-7)

- **시점**: 3인칭 근접 · **시기**: CC 996~998 (17~18세)
- **공간**: 변경 도로 → 가르엔 시장 → 노이른 객점 → 모닥불 야영지 → 카르엔 시장 → 미드그레이드
- **핵심 비트**:
  - 1.5-1 검객 글라렌 *눈은 죽은 자의 눈* / *호흡은 코로* 유머
  - 1.5-2 도적 5인 학살 **첫 의식적 살인** / *1단계 직관 시전* (정통 5단계 아님)
  - 1.5-3 객점 짠내 전투 + **엘 첫 만남** / 갈비뼈 부상 (지속) / *호흡은 코로* 2회차 반복
  - 1.5-4 학자 마린 *훔치는 자* 충고 / 5단계 정통 학습
  - 1.5-5 모닥불 + 엘의 *연민과 거리감* / 부두인형 흔적 (엘이 *침묵으로 알아챔*)
  - 1.5-6 시장 *시장 상인 부친 팔 부러뜨림* / **엘 떠남** + 단검 + 메모 + *호흡은 코로* 3중 반복
  - 1.5-7 변경 마법사 델란 / 카르엔 사이엔 앞 추천장 / *다른 무엇*
- **세부**: [`../chapters/part-1-2.md`](../chapters/part-1-2.md)
- **상징물 시드**: 부두인형 (엘이 침묵으로 알아챔)

### 2.4 part-1-3 — 바이서스의 탑 (6회, 2-1 ~ 2-6)

- **시점**: 3인칭 근접 · **시기**: CC 998~999 (**5년의 학습** 압축)
- **공간**: 셀라리아 외성문 → 카르엔 외곽 연구소 → 지하 금서고 → 왕립 연구원 → 공동 실험실 → 왕궁 대전당
- **핵심 비트**:
  - 2-1 셀라리아 진입 / 표식 등록 (외곽 연구소)
  - 2-2 카르엔 사이엔 사사 / *마시지 않은 차*
  - 2-3 지하 금서고 / 엔트로피 역연산 / *부작용 감춤*
  - 2-4 라이세린 데오릭 공작 천거 / *자네 같은 사람을 기다렸다*
  - 2-5 미라 베스턴 손목 골절 / *증거 인멸 시도* 오해
  - 2-6 왕실 마법 고문 임명 / 카르엔 사망 + 미완성 서한 소각
- **단서**: 카르엔 추천장의 어조 (1.5-7 델란 추천장과 동일), 미완성 서한, 자기 잔에도 따른 차
- **세부**: [`../chapters/part-1-3.md`](../chapters/part-1-3.md)

### 2.5 part-1-4 — 정의의 가면 (6회, 3-1 ~ 3-6) ★ **30% 분기점**

- **시점**: 3인칭 근접 + 1회 교차 (3-5 마하) · **시기**: CC 999
- **공간**: 영빈관 → 외교실 분석실 → 세번 강 변경 → 대학 강당 → 카르샤 농가 (교차) → 왕좌의 방
- **핵심 비트**:
  - 3-1 카일렌 베르하르트 경 사절단 / 손 모음 인사 *조롱 오인*
  - 3-2 라이세린 위조 첩보 분석 / *바이서스산 종이* 단서 흘림
  - 3-3 국경 사주 학살 / **에리언 사망**
  - 3-4 카르샤 악마화 강연 / 청중 1인 *증거 부족* 외침
  - 3-5 **[교차] 마하의 기도** — *우리 아이가 미워한 사람이 없게 해주세요*
  - 3-6 선전포고 / 왕의 망설임 / 사령관 출정
- **단서 회수 완료점**: Chapter 1 전반 30% 단서 모두 노출
- **세부**: [`../chapters/part-1-4.md`](../chapters/part-1-4.md)

### 2.6 part-1-5 — 왕을 죽인 날 (5회, 4-1 ~ 4-5) ★ **50% 각성**

- **시점**: 1인칭 (카엘) — **시점 전환** · **시기**: CC 999 가을 ~ CC 1000 봄
- **공간**: 세번 강 전장 → 세이렌 외성벽 → 백색 회랑 → 옥좌의 방 → 새벽 회랑
- **핵심 비트**:
  - 4-1 광역 학살 1,400명 (압축 화염구 / 형태 해체)
  - 4-2 세이렌 함락 / **머리카락 목걸이 단서 회수**
  - 4-3 회랑 진입 / 카일렌 *왕이여, 도망치소서* / 첫 *과거형 자기 의심*
  - 4-4 **프롤로그 완전판** — 5합 결투 / 알레이언 정식명 / *7대를 기다린 자* + 키얀 왕자 도주
  - 4-5 평화 협정 발견 / *앤트로퍼* 단어 첫 등장 / **손의 떨림** 신체화 + *이제부터 본다*
- **단서 회수**: 프롤로그 5종 단서 모두 회수
- **세부**: [`../chapters/part-1-5.md`](../chapters/part-1-5.md)

### 2.7 part-1-6 — 드래곤의 분노 (6회, 5-1 ~ 5-6) ★ **70% 상실**

- **시점**: 1인칭 (카엘) · **시기**: CC 1000 봄~여름
- **공간**: 점령된 세이렌 도서관 → 카르엔 외곽 연구소 → 칠흑숲 변경 → 변경 엘프 마을 → 잉그리드 사원 → 빙벽 협곡 → 오릴 마을
- **핵심 비트**:
  - 5-1 라이세린 잠적 / *자네는 너무 잘 보았어*
  - 5-2 **카르엔 봉인 문서** — 사프란·붉은 사슬·이득·충동 명명 / **세파 류르 합류**
  - 5-3 빙벽 협곡 여정 / 변경 엘프 마을 *침묵의 환대* / 잉그리드 몽크 한 줄
  - 5-4 이실의 의식 영창 비틀림 → 분노
  - 5-5 **양 팔 상실** / 마나 회로 5/7 단절 / 세파 다리 부상 / **부두인형 발견**
  - 5-6 오릴 마을 회복 / *손 모음을 처음으로 위협으로 보지 않음*
- **단서·신체 변화**: 잘 진행. 5-5가 *Chapter 1의 마지막 충격 분기점*
- **세부**: [`../chapters/part-1-6.md`](../chapters/part-1-6.md)

### 2.8 part-1-7 — 추격자 (7회, 6-1 ~ 6-7) ★ **결전 편지로 종결**

- **시점**: 듀얼 1인칭 (홀수=카이엔, 짝수=카엘) · **시기**: CC 1000 여름~겨울 → CC 1001 봄
- **공간**: 바엘 (마녀령 변경) → 그라스 (의수 시술) → 오르하임 (동부국 검술대회) → 세이른 산속 도장 → 그라스 시장 → 폐허 마탑 (결전 편지)
- **핵심 비트**:
  - 6-1 [이실] 카이엔 폴리모프 / 인간 감각 매혹
  - 6-2 카엘 의수 시술 / *부친 책의 출처 회수* (친조부 = 그라스 잔존자) / 사프란 비망록 마지막 페이지
  - 6-3 [이실] 카이엔 동부국 검술대회 우승 + 모르간 린이엔 사사
  - 6-4 카엘 공허검 입문 / *검은 막는 것이 먼저다* / 도장에 *황금빛 손님* 흔적
  - 6-5 [이실] 카이엔 정보망 카엘 추적 도달 / *내가 더 인간이 되기 전에*
  - 6-6 그라스 시장 *5초 어긋남* / 카엘이 *손 모음 인사를 위협으로 보지 않음*
  - 6-7 폐허 마탑 결전지 / **나는 여기에 있다** / 카이엔 *작별의 검*
- **세부**: [`../chapters/part-1-7.md`](../chapters/part-1-7.md)

## 6. 8축 — narrative-devices 통합 점검

- [ ] **재미 4축**: 1단계 직관 시전 (1.5-2), 의수 진화 (6-2~), 부친 책·부두인형·붉은 머리끈·카르엔 봉인 문서·사프란 비망록·의수, 부두인형 2회+ 붉은 머리끈 2회
- [ ] **잔향**: 산파 도주(1-1) → 마을(1-2~1-4) → 엘(1.5-3~1.5-6) → 시장 상인 부친(1.5-6) → 미라(2-5) → 변경 엘프 마을(5-3) → 오릴(5-6). 단어 자체는 5-2부터
- [ ] **카엘 외모**: 1.5-3 객점 주인 아낙 + 1.5-7 델란 짧은 시선 + 2-1 셀라리아 등록자 시선 + 5-3 변경 엘프 무녀 *준수한 어둠* 짧은 평
- [ ] **짠내 전투**: 1.5부 전체 + 4-3 회랑 진입 (마법사 vs 기사 근접 절대 불리의 *반대* — 광역 마법 우위) + 5-5 양 팔 상실
- [ ] **의식 분리**: 1-1 *짧은 섬광* 시드 → 1-3 시점에서 *자기를 멀리서 보는 한 줄* → 4-3 *과거형 자기 의심* → 4-5 본격 발동
- [ ] **상징물**: 부두인형 (1.5-5 시드 → 5-5 발견 → 6-7 폐허 마탑 무덤?) / 붉은 머리끈 (5-2 충동 구매 → 6-2 의수 봉인 회로 → 6-7 결전 직전 한 번 더 만짐)
- [ ] **여성 캐스팅**: 엘(1.5-3~1.5-6 떠남), 마하(3-5 단편), 미라(2-5 단편), 셀라엔 대주교(4-4 직후), 엘라리아 왕비(4-2 망명), 모르간 린이엔(6-3 황녀), 산파(1-1), 카르엔 외곽 연구소 동료 1인(2-3 단편)
- [ ] **알레고리**: 라이세린-카루린 외척 밀약 = *권력 외척 자본의 전형* / 백색 사냥단 = *비주류 박해 동조 권력* / 평민 학살 = *전쟁의 무명 희생*. 현대 용어 *직접 금지*

## 7. 톤·문체 — Chapter 1

- 프롤로그·4부·5부: **1인칭**. 4-3까지 *왜곡 노출*, 4-5 이후 *과거형 자기 의심*
- 1부·1.5부·2부·3부: **3인칭 근접** (3-5만 마하 시점 교차)
- 6부 듀얼: 홀수=카이엔 1인칭, 짝수=카엘 1인칭
- **유머 분포**: 1.5부 *떠돌이 막말 코미디* 4~5회 / 5-3·5-6 *잠적 후 부드러운 빛* 1~2회 / 6부 *카이엔의 인간 감각 + 세파의 잔소리* 5~7회

## 8. 결말 — Chapter 2 전환

- **마지막 회차** (6-7) *결전 편지*: *나는 여기에 있다. 폐허 마탑. 당신이 본 적 있는 모든 것을 다시 보러 오라.*
- **카이엔 측 동시**: 황녀에게 *작별의 검* 남기고 출발. *인간으로 살게 해줘서 고맙다*
- **Chapter 2 진입 조건**:
  - 카엘은 *공허검 입문 단계* (9써클·소드 엠퍼러급 *아직 아님*)
  - 의수 1단계 (강철·임시 사양)
  - 세파 동행 확정
  - 키얀 왕자 추적자 라인 *준비 완료*
  - 마녀령·카르샤 잔존과 *비공식 채널만* 형성 (아직 동맹 아님)

## 9. ★ 사용자 검토 포인트

다음 항목을 *집필 직전 검토* 해 주세요. 변경 시 본 spec + 회차 파일 동기 갱신 필요.

### 검토 1. 떠돌이 7회차 (part-1-2) 분량의 적정성

- 현재 7회차 = Chapter 1 전체의 16% (43회차 중)
- 사용자 선호: *카엘이 떠돌이 단계에서 *진짜 강해지지 못함* + *모든 만나는 자에게 막말 들음** — *기술 형편없는 상태* 명확
- **검토**: 7회차가 *너무 길지는 않은가*? 5회차로 압축 (1.5-5 모닥불 + 1.5-6 시장 결정타 통합 등) 가능

### 검토 2. 5년의 학습 (part-1-3) 압축의 정도

- 현재 6회차에 *5년 압축*
- 사용자 의도: *"중간 마법에 대한 진지한 학습과 상승의 과정이 필요. 쉽지 않아야 하고 몇 년이 걸려야 함"*
- **검토**: 6회차로 충분한지, 8~10회차로 확장이 필요한지

### 검토 3. 두 팔 상실 (5-5) 의 톤

- 현재: *이로서 네 마법사로의 인생은 끝이다 / 네 인생이 끝나지 않는다면 말이지* (이실의 무언 + 카엘의 무언 결의)
- **검토**: 이 대사가 *명시적으로 등장* 하길 원하는지, *무언의 톤* 으로만 처리할지

### 검토 4. 6-7 결전 편지의 *공개 vs 비공개*

- 현재: *공개 편지* 로 카이엔에게 발신 (대륙 단위 풍문 가능)
- **검토**: 카이엔 *비공개 친서* 가 더 적절한지, *공개* 의 정치적 파장이 의도된 것인지

### 검토 5. 1.5부 엘의 *생존 여부*

- 현재: 1.5-6에서 떠남. 후반부 *간접 흔적* (용병 길드 명부) 가능성만 시드
- **검토**: 엘이 *후반부 재등장* 할지, *영구 소모성 인물* 인지 (재등장 시 어떤 입장으로)

### 검토 6. 부두인형의 *기원*

- 현재: *단의 손바느질 인형* (1-3 단의 다리 사고와 연결)
- **검토**: 단이 *카엘에게 직접 만들어 준 인형* 인지 / *단 사후 카엘이 가져온* 인형인지 / 다른 기원인지

### 검토 7. 키얀 왕자의 *추적자 라인 본격화 시점*

- 현재: 6부 *준비만*. Chapter 2에서 본격 추적
- **검토**: Chapter 1 안에서 *직접 등장 (작은 컷)* 1회 추가가 좋을지 (예: 6-7 변경 풍문)
