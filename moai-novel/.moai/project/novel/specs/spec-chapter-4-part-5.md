## Summary

데미안 베르스 8세 + 헤븐즈 하르엔 왕자 + 청혈 가문 후원 + 마도공학 메크 *기계화 사단* 결성 — *카엘이 거대 표적 다 처치한 후 *카엘의 앤트로픽을 죽여 헷지* 음모*. 1차 격돌.

## 회차 구성 (8회차)

| 회차 | 비트 |
|------|------|
| 4-5-1 | 데미안 *마법공학 코어·화기 대량 생산 시대 공식 선언* — *데미안 패권 시대 개막* |
| 4-5-2 | 카엘의 *대중적 영웅 칭호 부여* + *대륙 최대 앤트로픽 응축체* 역설 상황 |
| 4-5-3 | 하르엔 + 데미안 *비밀 협약* 카엘 측 알아차림. 카리엔 왕녀의 *비공식 항의 사신* |

## 핵심 비트

1. **카엘 *영웅 칭호 + 위험 응축체* 역설** — 본작 *영웅의 비극* 가시화
2. **메크 직결 5단계 첫 시도** — 의수 6단계 진화 시드
3. **물량전 첫 본격** — 본작 *마법·검술이 부족한 적* 의 첫 등장
4. **라카르 9세 공식 침묵** — *왕도 이제 카엘을 막을 수 없음*

## 인물

- 카엘 / 세파 / 카이엔 / 키얀 일행 4인 / **데미안 베르스 8세** (선언) / **하르엔 왕자** / 청혈 가문 100명 / *데미안 메크 30체 + 군 1만 + 하르엔 직속 2천* / **카리엔 왕녀** (비공식 항의 사신) / 카르덴 스펠른 (메크 설계자, 간접)

## 아티팩트·단서

- 의수 4단계 *풀가동 + 부분 손상*
- *메크 직결 인터페이스* (메크 1체 *내부 잠입 + 마운트*)
- 점멸 양피지 다수 소모

## 지명

- 변경 평원 (1차 격돌 무대)
- 데미안 미라돈 (간접 — 메크 생산지)

## Tone

- *물량전 + 외교 + 잠입 + 5단계 시도* 의 *복합 작전*
- 짠내 *높음* — 카엘 측 *4인 vs 1만+30체* 의 극한 비대칭
- 유머 *0* — 단 세파의 *카엘이 *메크 안에 들어가는 게 *진짜 인생 최대 우스움* 이야* 한 줄

## ★ Review points

1. **8회차 분량** — Part 4-8 (본작 최대 전투) 의 *예고편* 으로 적정. 6회차 압축 vs 10회차 확장?

3. **4-5-8 *2차 진영 풍문***  — 본 Part 종결 시드 vs 다음 Part 진입?
4. **라카르 9세 *공식 침묵*** — 의도된 침묵 (헤븐즈 정치 균형) vs 무력 침묵 (왕이 막을 수 없음)?
