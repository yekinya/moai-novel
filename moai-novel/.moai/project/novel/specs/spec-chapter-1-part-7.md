## Summary

듀얼 시점 (홀수 카이엔 / 짝수 카엘). 카이엔의 인간화 + 카엘의 의수·공허검 입문 동시 진행. 6-6 *5초 어긋남*. 6-7 결전 편지 발신으로 Chapter 1 종결.

## 회차 구성 (7회차)

| 회차 | 시점 | 제목 | 비트 |
|------|------|------|------|
| 6-1 | 이실 | 인간의 맛 | 폴리모프 직후 바엘 도시 + 첫 식사·술·검술 시합 + 카이엔 호명 |
| 6-2 | 카엘 | 의수 | 그라스 의수 시술 + **부친 책 출처 회수** + 사프란 비망록 마지막 페이지 + **붉은 머리끈 봉인 회로** |
| 6-3 | 이실 | 동방의 검 | 동부국 검술대회 우승 + 모르간 린이엔 사사 |
| 6-4 | 카엘 | 이름 없는 제자 | 세이른 산속 도장 + *검은 막는 것이 먼저다* + 도장에 *황금빛 손님* 흔적 |
| 6-5 | 이실 | 마검사 이실 | 동부국 최고 마검사 칭호 + *내가 더 인간이 되기 전에* |
| 6-6 | 카엘 | 두 개의 그림자 | 그라스 시장 *5초 어긋남* + 카엘이 *손 모음 인사를 위협으로 보지 않음* |
| 6-7 | 듀얼 | 결전의 땅 | 폐허 마탑 결전지 + **나는 여기에 있다** + 카이엔 *작별의 검* |

## 핵심 비트

1. **듀얼 시점 본격화** — 홀수 카이엔 / 짝수 카엘
2. **6-2 부친 책 출처 회수** — 그라스 의수 시술자 = 카엘 친조부와 같은 가문 (카르엔 학파 잠적 분파). 책의 출처가 *친조부* 임이 명시
3. **사프란 비망록 마지막 페이지** (6-2) — *최종 카드 자결* 의 의미 첫 노출
4. **붉은 머리끈 봉인 회로** (6-2) — 5-2 충동 구매의 의미가 *의수 봉인 매개* 로 회수
5. **카이엔 인간화 본격** — 식사·술·검술 + 황녀 사사. 6-5에서 *내가 인간이 되어간다* 자각
6. **6-6 *5초 어긋남*** — 두 사람이 *같은 좌판 등 맞대고 5초*. 서로 못 알아챔. 카이엔의 *손 모음 인사* 가 카엘 시야 가장자리에 들어옴 — *위협으로도 평화로도 보지 못한 채 그냥 본다*. 본작 *각성 신체화 세 번째 단계*
7. **6-7 결전 편지** — *나는 여기에 있다. 폐허 마탑. 당신이 본 적 있는 모든 것을 다시 보러 오라*
8. **카이엔의 작별의 검** — 황녀에게 *인간으로 살게 해줘서 고맙다* 검에 새김

## 시점·문체

- **듀얼 1인칭** — 홀수 카이엔 / 짝수 카엘
- 카이엔: *어색하게 풍부한 형용사 + 인간 감각의 충격 묘사*. 유머 자주
- 카엘: *짧음 + 건조 + 자조*. 가끔 농담 시작 (재기의 신호)
- 회차 첫 줄에 *시점 화자 명시* (예: *카이엔의 아침은 차가웠다* / *카엘은 가방을 다시 쌌다*)

## 인물

- 카엘 (1인칭, 짝수) / **카이엔 오르하임** (=이실, 1인칭 홀수) / **세파 류르** (카엘 동행) / **세이른 스틸만** (6-4 도장 노 마검사) / **모르간 린이엔 황녀** (6-3 동부국 황실) / **그라스 의수 시술자** (60대 노 장인, 6-2) / 동부국 황녀 직속 무사 (6-3) / 변경 도시 *바엘* 객점 인물 (6-1) / 그라스 시장 인물 (6-6)

## 아티팩트·단서

- **카엘의 마도 의수 1단계** (6-2 시술 — 강철 + 룬 회로)
- **붉은 머리끈** (6-2 의수 봉인 매개 — 노 장인의 한마디 *네가 이걸 차고 있어 다행이군*)
- **사프란 비망록 마지막 페이지** (6-2 공개)
- **친조부의 책 출처 회수** (6-2 — 노 장인이 카엘 친조부와 같은 가문)
- **부두인형** (가방 깊이 — 5-5 발견 후 동행 중)
- **이실의 비늘 한 조각** (5-5에서 박힘. 6-2 시술 시 *제거 어려움* 으로 *그냥 둠*)
- **카이엔의 작별의 검** (6-7 황녀에게 — 흑색 강철 + 자수정 자루 + *인간으로 살게 해줘서 고맙다* 새김)
- **카엘의 공개 편지** (6-7)

## 지명

- **바엘** (마녀령 변경 도시, 6-1)
- **그라스** (데미안 마법 장인 도시, 6-2·6-6)
- **오르하임** (마녀령 수도, 6-3·6-5)
- **세이른의 산속 도장** (강철제국 변경, 6-4)
- **폐허 마탑** (6-7, 1-5와 동일 공간 — 원환)

## Tone

- 듀얼 시점의 *교차 리듬*. 카이엔=호기심·매혹·균열, 카엘=재기·결단·자조
- 유머 *증가* — 카이엔의 인간 외양 어색함 + 세파의 잔소리 + 세이른의 무뚝뚝 농담
- 짠내 *낮음* (잠복 중) — 단 *외부 자객 단편 격퇴* 1~2회 가능
- 결말 무게 *고조*

## ★ Review points

1. **6-1 카이엔 *인간 외양 첫 묘사*** — 30대 후반, 검은 머리, 깡마름, *왼쪽 눈 황금빛 미세*. 더 디테일? 신장? 옷차림?
2. **6-3 모르간 린이엔 사사의 *공식 절차*** — 동부국 외부 호칭 *황녀* 사용. 본 회차 내 정정 (린이엔이 *내부 호칭 흑혈 차기 종주* 라 자기 정정) vs 외부 호칭 유지?
3. **6-6 *5초 어긋남*** 분량 — 현재 안: 회차의 1/3 (약 2,000자). 더 길게 (3,000자, *극적 강조*) vs 더 짧게 (1,000자, *우연한 어긋남* 톤)?
4. **6-7 결전 편지의 *공개 vs 비공개*** — 사용자 검토 4 (Chapter 1 spec)에서 이미 제기됨. *공개 편지* (대륙 풍문 가능) vs *비공개 친서* (카이엔 단독 발신). 본 part-7 spec에서 결정 필요
5. **6-7 *카엘 + 카이엔 시점 합류*** — 한 회차 안에서 두 시점이 *교차* (현재 안) vs *카엘 단독* (카이엔의 출발만 마지막 1줄로 암시)?
6. **세파의 *6-7 떠나지 않음*** — 현재 안: *그건 당신이 정할 일이 아니에요*. 강한 결단. 적정한지?
