## Summary

## 핵심 비트 (사용자 보강 5항 통합)

### 1. 드래곤 둥지 방향 도주 — *추적 회피의 합리적 오판*

**도주 동선 결정**:
- 카엘이 *5-1 시점에 도주로 결정* — *바이서스 일반 추격대를 회피하려면 *추격대가 들어오지 못하는 영역* 진입 필요*
- 카엘의 *바이서스 외곽 연구소 5년 도서관 정독 기억* 에서 *드래곤 둥지 영역 = 영역 침범 시 드래곤의 자동 적의 반응 + 다수 몬스터 (오우거·트롤·와이번 등) 서식 = 인간 진입 자살 행위* 의 정보 떠올림
- **카엘의 합리적 판단** — *우리는 인간이 아니라 도주자 = 일반 인간보다 강함 + 마검사 5종 + 회복 약 = 몬스터 회피 가능*. *추격대는 영역 진입 자체를 시도 안 함 = 진입 후 추격 단절* 의 *합리적 선택*
- 라이세린의 *강력한 추적자 미투입 결단* (Part 5 §10-H) 이 이 정통의 *반대편 정통* — *추격이 일반 추격대 수준이라는 정황이 카엘의 *둥지 도주 결정의 합리성 강화*

**둥지 영역 특성**:
- *바이서스 북부 산령의 어느 산악 지대 — 천년 이상 인간 침입 0 의 영역*
- 다수 몬스터 서식: *오우거 5~7마리 / 트롤 3~4마리 / 와이번 10여 마리* + *기타 마수*
- 드래곤 본체 = *비교적 젊은 드래곤 — 약 1000살 내외*. 본작 정통 *고대 7대 드래곤* 보다는 *훨씬 어린 드래곤*. 임시 작명 — *적염룡 카자르드* 후보를 *유지 vs 변경* (사용자 검토)
- *젊기 때문에 잠들지 않은 경우 종종 있음* + *자기 둥지 침입자 제거 성향*. *고대 드래곤들이 *수백 년 단위 동면* 인 것과 달리 *카자르드는 깨어있을 확률이 30~50%* 수준

**카엘의 *치명적 오판***:
- *카엘은 *드래곤이 깨어있을 가능성을 *낮게 추정**. *도서관 기억의 표면 정보만 의존*. *깊은 정보 — *젊은 드래곤의 깨어있을 확률* 은 본 Part 직전 시점까지 *모름*
- 둥지 외곽 진입 후 *오우거·트롤·와이번 회피* 가 *예상보다 잘 풀림* — *카엘이 *내 판단이 옳았다* 의 자기 부풀림*
- *5-3 종결부터 *어느 순간 몬스터 흔적이 사라짐* 의 자각이 시작* — *드래곤이 인근 몬스터를 *전부 흩어놓은 정황*. *카엘이 *최악의 가능성을 *진지하게 고려*** — 그러나 *후회는 늦음*

### 2. 두 추방자의 도주 + 세파의 3회 살해 시도 + 금제 균열 4회

(Part 6 v2 의 기존 정통 유지 — §2·§3 와 합산 묘사)

**도주 동선 (수정)**:
- 셀라리아 외곽 비밀 안전 가옥 → *바이서스 북부 산악 지대 진입* → *드래곤 둥지 영역 외곽 (몬스터 영역)* 진입. *목적지 = 둥지 영역 통과 후 카르샤 외부 망명 라인*. *예상 도보·말 이동 약 2~3주*
- 동행 인원 = *카엘 + 세파 2인만*. 안전 가옥 내응 신부의 *동선 안내 + 식량·은신처 정보* 제공만

**세파의 3회 살해 시도** (Part 6 v2 §1 유지):
1. **1차 — 야영지 첫 밤** (5-1): 단검 + 금제 균열 1회 *세파… 미안하다* → 무산
2. **2차 — 강 도하** (5-2): 발 디딤 비틀어짐 + 금제 균열 2회 *손 내미는 비뚤림 없는 눈* → 무산
3. **3차 — 둥지 외곽 동굴** (5-3): 신성 단검 + *자기 의지로 포기* + *카엘이 알고 있음 자각*

**카엘의 *모든 시도 인지 + 대응 안 함*** (v2 §1 유지)
**엔트로퍼 *금제 균열* 4회 발현** (v2 §2 유지 + 5-4·5-5 추가):
- 1회: 5-1 *세파… 미안하다*
- 2회: 5-2 *손 내미는 비뚤림 없는 눈*
- 3회: 5-3 *침묵의 인정*
- 4회: 5-5 *공주 옷을 입으로 물기 직전 한 마디 — *너는 살아라**

### 3. 드래곤 *인간 변신 첫 등장* — *수염 덮수룩한 나무꾼*

**첫 등장 (5-4 시작)**:
- 둥지 영역 깊숙이 진입 직후 — *마치 약초꾼이나 산골 사냥꾼처럼 보이는 *수염 덮수룩한 중년 나무꾼* 이 길 모퉁이에서 나옴*. *맨손 + 가벼운 옷차림 + 손에 든 도끼·낫 없음*
- **드래곤 첫 한 마디**: ***"네놈들 뒤에 줄줄이 딸린 놈들은 네놈 때문이렷다."***
- 카엘이 *멈춤*. *세파가 카엘 등 뒤로 한 걸음*. *짧은 침묵*

**카엘의 *도서관 기억 떠오름 + 긴장 유지***:
- 카엘이 *바이서스 외곽 연구소 5년 도서관 정독* 에서 본 *드래곤 첫 등장의 정통 — *침입자와 같은 종족의 모습으로 변신하여 *대화로 정황 파악* 후 처분 결정*** 의 정보를 *떠올림*
- 카엘의 *내적 — *설마. 그러나 이 정보가 정확하다면 — 지금 눈앞의 자는 *드래곤이다*. 그리고 *내 도주 동선의 정황을 *이미 파악한 상태로 등장**
- *긴장을 놓지 않은 채 대화 시도* — *친절하게 대답하지도 않고 무시하지도 않음*. *최대한 시간을 끌면서 정보 수집 + 기습·도주 계획 계산*

**드래곤의 *앤트로퍼 전승 기억***:
- 본 드래곤은 *고대 7대 드래곤이 아닌 젊은 1000살 내외*. 그러나 *드래곤 종족 전체의 *고대 전승 기억* 으로 *앤트로퍼의 존재는 알고 있음*
- 단 *카엘이 *앤트로퍼인지는 모름*. *카엘의 *원인 모를 *특이한 오라* 만 즉시 감지* — *인간의 마나 흐름과 다른 *역마나 흡수 본능* + *살상 충동의 미세한 잔향*
- 드래곤의 *내적* — *이 자에게서 *흔치 않은 *기색* 이 난다. 그러나 그 정체는 *명확하지 않음*. *대화로 정황을 더 본다*

### 4. 카엘의 *몬스터 흔적 사라짐 자각* + 기습·도주 계획

**자각의 점진성** (5-3 후반 → 5-4 중반):
- 5-3 둥지 외곽 진입 시 *오우거·트롤·와이번 회피 잦음 — *추격 회피 너무 잘 풀림*. *5-3 말미에 카엘이 *그런데 어느 순간부터 *몬스터 흔적이 안 보인다*** 의 자각 시작
- 5-4 첫 회차 — *나무꾼 등장 직전 약 30분 동안 *완전한 정적*. *카엘이 *최악의 가능성을 본격 고려***
- 5-4 *나무꾼 첫 한 마디 + 도서관 기억 합산* 으로 *카엘의 정체 추정 거의 확정*. *대화로 시간 끌면서 *내심 *기습 + 도주 경로 계산*

**카엘의 기습 + 도주 계획**:
- *최악의 순간이라는 판단이 서자마자* — *마검사 5종 + 직관 시전 동시 동원 + 공주를 어깨에 메고 *둥지 영역 반대 방향으로 도주*
- *기습 = *드래곤이 *대화 모드인 짧은 순간* 을 노려 *마검사 변칙 5종 (바람 손날·검 막은 화염 폭발·빙판 보법·바람 도약·전기 충격) 중 *가장 빠른 *바람 도약 + 단검 투척* 시도*
- **카엘의 *뼈저린 후회*** — *도주로 설정을 *치명적으로 잘못 골랐다*. *바이서스 일반 추격대를 회피하느라 *드래곤이 깨어있는 영역에 진입한 자기 판단의 *비뚤린 시야 너머의 후회*

### 5. 드래곤의 *용언 3종 동시 시전* — 카엘의 *두 팔 상실*

**전투 흐름 (5-5)**:
- 카엘이 *기습 실행 — 바람 도약 + 단검 투척 + 빙판 보법 후퇴* 의 *0.5초 연계*
- **드래곤의 응답 — 용언 3종 동시 시전**:
  1. **풍압의 칼** — *공기를 칼날 형태로 압축하여 *원거리에서 정밀 절단*. *시전 시간 0.3초 + 효과 발동 즉시 0.1초*
  2. **바람의 길** — *공기 흐름을 통로화하여 *시전자가 임의 거리를 *즉시 이동*. *시전 시간 0.5초 + 이동 즉시*
  3. **잔상환술** — *시전자 본인의 잔상을 *그 자리에 남겨 *대상이 인지하는 시야 속에 환상으로 유지*. *시전 시간 0.3초 + 효과 발동 즉시 ~ 수 초 지속*
- **카엘의 인지 균열** — *잔상환술* 로 *드래곤이 *원래 자리에 가만히 서있다고 인지*. *그러나 실제 드래곤은 *바람의 길로 이미 카엘 옆구리에 있음*
- **풍압 칼날의 도착** — *카엘의 *양 어깨에서 *두 팔이 동시에 잘려나감*. *살이 비명을 지를 새도 없이 절단*
- 카엘이 *돌아보는 순간* — 드래곤이 *눈앞에 강림*. *맨발·맨손이나 *드래곤의 일격의 위력은 *물리 차원 거대*

**드래곤의 발길질**:
