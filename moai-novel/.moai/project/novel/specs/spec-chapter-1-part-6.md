# Chapter 1 Part 6 Spec: 드래곤의 분노 — *70% 상실*

> 회차 시나리오: [`../chapters/part-1-6.md`](../chapters/part-1-6.md)
> Chapter 1 overview: [`./spec-chapter-1.md`](./spec-chapter-1.md)

## Summary

각성한 카엘이 잠적 → 카르엔 봉인 문서 → 빙벽 협곡 의식 → **두 팔 상실** + 세파 부상. 본작 *70% 충격 분기점*. 부두인형 발견.

## 회차 구성 (6회차)

| 회차 | 제목 | 비트 |
|------|------|------|
| 5-1 | 죄의 무게 | 점령된 세이렌 도서관 + 전사자 명단 + **라이세린 잠적** + 마지막 편지 *너무 잘 보았어* |
| 5-2 | 금제의 이름 | 카르엔 외곽 연구소 + **봉인 문서** + 사프란·붉은 사슬·이득·충동 명명 + **세파 합류** + **붉은 머리끈 충동 구매** |
| 5-3 | 북쪽 산맥 | 칠흑숲 변경 + 엘프 마을 *침묵의 환대* + 잉그리드 사원 몽크 *드래곤은 신이다* |
| 5-4 | 깨움 | 빙벽 협곡 + 의식 영창 비틀림 + 이실의 첫 분노 |
| 5-5 | 비늘의 폭풍 | **양 팔 상실** + 마나 회로 5/7 단절 + 세파 다리 부상 + **부두인형 발견** |
| 5-6 | 살아남은 자 | 오릴 마을 회복 + *손 모음을 처음으로 위협으로 보지 않음* |

## 핵심 비트

1. **잔향 단어 본문 첫 등장** (5-2) — *얇은 금속음 = 붉은 사슬* 의 의미 확정
2. **세파 류르 합류** (5-2) — *각하 아니에요. 학살자예요*
3. **두 팔 상실** (5-5) — 본작 *Part 1 최대 충격*. *이로서 네 마법사로의 인생은 끝이다* (이실 무언) vs *네 인생이 끝나지 않는다면 말이지* (카엘 무언 결의)
4. **부두인형 발견** (5-5) — 세파가 카엘의 짐 가방에서 *낡은 부두인형* 발견. *물어보지 않음*. 1.5-5 에르나의 *침묵 발견* 과 대칭
5. **붉은 머리끈** (5-2 충동 구매) — *시장에서 한 시장 상인의 좌판에서 무심코 집어듬*. *왜 산지 본인도 모름*. 6-2 의수 봉인 회로 매개
6. **첫 *손 모음 인사 = 평화* 인지** (5-6) — 오릴의 식전 인사. 카엘이 *위협으로 보지 않는다*. 본작 *각성의 신체화 두 번째 단계*

## Tone

- 잠적 후 *침묵의 무게*
- 자기 응시 + 죄책감
- 짠내 *최고급* — 5-5 두 팔 상실. 세파 다리 부상
- 유머 *5-3 잉그리드 몽크 한 줄 *드래곤은 신이다*** (작은 쓴웃음) / 5-6 오릴의 단호한 시골 친절

## ★ Review points

1. **5-5 *이로서 네 마법사로의 인생은 끝이다 / 네 인생이 끝나지 않는다면 말이지*** — 사용자 요청 대사
   - 옵션 A: *명시 대사* (이실이 떨어지며 한마디, 카엘이 무릎 꿇은 채 답)
   - 옵션 B: *무언의 톤* (두 인물의 시선·자세로만 표현, 본문에는 *그것이 그의 무언의 통고처럼 느껴졌다* 같은 진술)
   - 옵션 C: 둘 다 (이실 명시 + 카엘 무언)
2. **세파 첫 만남 5-2** — 현재 안: *기다렸어요. 5년이나.* 한 줄로 시작. 더 부드럽게? 더 차갑게?
3. **붉은 머리끈 도입의 *동기 모호함*** — 5-2에서 카엘이 *왜 샀는지 본인도 모름*. 적정한지 (상징물 도입 정통) / 더 명시적 동기 부여 (예: 마하의 기도 회상)?
4. **부두인형 *5-5 발견 회수*** — 세파가 *물어보지 않음* 한 줄. 카엘이 *왜 가지고 있는지 본인도 모름* 잔향. 적정한지?
