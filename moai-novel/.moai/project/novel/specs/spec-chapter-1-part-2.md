# Chapter 1 Part 2 Spec: 떠돌이의 길

> 회차 시나리오: [`../chapters/part-1-2.md`](../chapters/part-1-2.md)
> Chapter 1 overview: [`./spec-chapter-1.md`](./spec-chapter-1.md)

## Summary

폐허 마탑을 떠난 카엘이 검객 그라엔·학자 마린·여성 검사 에르나·변경 시니어 셉터 데르엔 등을 만나며 *기술 형편없는 떠돌이* 의 1년을 보낸다. 모두 카엘에게 *막말* 을 남기고 떠난다. 첫 의식적 살인.

## 회차 구성 (7회차)

| 회차 | 제목 | 비트 |
|------|------|------|
| 1.5-1 | 떠돌이의 첫 길 | 검객 그라엔 *눈은 죽은 자의 눈* + *호흡은 코로* 유머 |
| 1.5-2 | 도적 격퇴 | 가르엔 시장 마을 + 도적 5인 학살 + **첫 의식적 살인** + *1단계 직관 시전* |
| 1.5-3 | 짠내 객점 | 노이른 객점 + 도적 8인 + **에르나 첫 만남** + 갈비뼈 부상 (지속) |
| 1.5-4 | 두 번째 여행자 | 학자 마린 5단계 정통 마법학 + *훔치는 자* 충고 |
| 1.5-5 | 야영의 모닥불 | 카엘의 추방 이야기 + 에르나의 절반 자기 이야기 + *부두인형 침묵 발견* |
| 1.5-6 | 시장의 상인 | 시장 상인 부친 팔 부러뜨림 + **에르나 떠남** + 단검 + 메모 |
| 1.5-7 | 변경 마법사 | 미드그레이드 + 데르엔 + 카르엔 사이엔 앞 추천장 + *다른 무엇* |

## 핵심 비트

1. **3중 *호흡은 코로* 반복** — 1.5-1 그라엔 / 1.5-3 에르나 / 1.5-6 메모. 셋이 같은 충고를 한다면 *진리* 라는 작은 유머
2. **카엘 비정통 시전** — *1단계 직관 시전* (정통 5단계 아님). 결과만 있고 절차 없음
3. **에르나의 *연민과 거리감*** — 잠시 손을 뻗으려다 거둠 (잔향 §1.7)
4. **부두인형 시드** — 1.5-5 에르나가 *침묵으로 알아챔*. 본격 의미는 5-5
5. **첫 의식적 살인 + 자기 합리화** — *그들은 악이었다. 나는 정의했다*

## 시점·문체

- **3인칭 근접** (카엘)
- 짠내 + 떠돌이 막말 유머 (1~2회 회차당)
- *길* 의 묘사 — 변경 도로·객점·시장·산속 야영지의 *짠내 분위기*

## 인물

- **그라엔** (검객, 30대 후반, 1.5-1 떠남)
- **마린** (학자, 50대, 1.5-4 떠남)
- **에르나** (여성 검사, 카엘 또래, 1.5-3~1.5-6 동행 후 떠남) — *소모성 여성 인물·비도구화·로맨스 없음*
- **데르엔** (변경 시니어 셉터, 40대, 1.5-7) — 카르엔 사이엔과 옛 학파 동료
- 도적 8+5명 (1.5-2, 1.5-3) / 객점 주인 아낙 / 시장 상인 가족 3인 (1.5-6) / 마을 어귀 노인 (1.5-2)

## 아티팩트·단서

- **부두인형** (1-3 시드 → 1.5-5 에르나 침묵 발견 → 5-5 본격)
- **마나 양피지 마지막 1매** (1.5-3 노잣돈 매개 — 짠내)
- **에르나의 단검** (1.5-3 도적 시체에서 회수 → 1.5-6 에르나가 카엘에게 *되돌려 줌*)
- **에르나의 메모** + *호흡은 코로 3중 반복 유머*
- **데르엔의 추천장** (1.5-7, 카엘 *내용 모름*. 어조가 2-4 카르엔 추천장과 동일)

## 지명

- 변경 도로 / **가르엔** 시장 마을 / **노이른** 객점 / 산속 야영지 (모닥불) / **카르엔** 시장 마을 / **미드그레이드** 외곽

## Tone

- 1년 시간 압축. 다섯 명의 만남·이별
- 유머 4~5회 (그라엔·에르나·마린의 막말 + 메모 마지막 유머)
- 짠내 *진짜 죽음 위협* — 1.5-3 갈비뼈 부상이 2부 시작까지 통증으로 남음

## ★ Review points

1. **7회차 분량의 적정성** — Part 1 전체 43회차 중 16%. 사용자가 *떠돌이 단계의 짠내·진짜 기술 없음·다섯 명의 막말* 을 강조하므로 7회차 유지 권장. 5회차 압축도 가능 (1.5-5+1.5-6 통합).
2. **에르나의 생존 여부** — 1.5-6 떠남 후 *영구 소모성* vs *후반부 간접 흔적 (용병 길드 명부)* 재등장 가능성
3. **카엘 외모 정면 대면 (1.5-3)** — 객점 주인 아낙의 *너처럼 잘생긴 청년이 왜 변경 도로를 떠도느냐* 첫 직접 외모 언급. 톤이 적정한지
