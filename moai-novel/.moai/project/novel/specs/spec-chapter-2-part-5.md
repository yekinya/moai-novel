## Summary

Ch2 P4 종결 (앨더스톰 진입 직전) 직후 — *카엘 앨더스톰 진입 + 마르덴 라온 (앨더스톰 룬 마법사, 마법학 대원로 1인) 직면*. *세이른의 추천장 + 카엘의 정통 직접 시연으로 *비공식 사사 허락*. **전도율 진단** — *마법력 총량 향상은 카엘의 정통상 어려움 + 전도율 향상이 *유일한 해결 길* + *아다만티움/미스릴 = 드워프 지하세계 전용*. 마르덴이 *드워프 부족의 *비공식 거래 매개 연결*. *부족장 (작명 후속) 의 *공식 의뢰 조건 — 카엘의 *마나석 양도 + 고대 정통 정보 양도 + 미래 약속*. 카엘 부족의 *시험 통과* + 약 2~3개월 의수 작업 + **아다만티움 + 미스릴 마력 대포 의수 완성**. *마나 감응 최고 효율 + 검기 운용 일반 강철의 3~5배 + 룬 결각 전체 표면 + 다중 아티팩트화 + 마력포 무한 발사*. *카엘의 팔 전체 검기 두께가 *일반 검의 100~120%* 도달. 본 Part 종결 시점 = *카엘이 아다만티움 의수 + 공허검 2원리 + 마력→내공 4중 회로 활성화 + 룬 결각 전체 + 마력포 = 본격 무력 회복*. *앨더스톰 복귀 직전 종결*.

## 핵심 비트

### 1. 앨더스톰 진입 + 마르덴 라온 (5-1·5-2)

**앨더스톰 정통**:
- *룬 마법사 + 대마법사 중심의 고대 마법 도시*. *데미안 영토 외곽 산령 입구*
- *룬 결각의 *대륙 최대 보고* + *데미안 마도공학과 비공식 협력 라인*
- *국립상아탑보다 *깊은 정통이나 *외부 노출은 *국립상아탑에 양보* — *조용한 비공식 권위*

**마르덴 라온 (작명 후속)**:
- *70대 후반. 앨더스톰 룬 마법사 대원로 1인 + 대마법사 7써클급*
- *세이른 스틸만의 옛 인연 (강철제국 본가 라인 + 앨더스톰 룬 라인의 *공허검 5인 마스터 중 1인의 *학문적 동맹*)*
- *카엘의 정통 직접 시연 + 추천장 매개* → *비공식 사사 허락*. *공식 제자는 아니나 *진단·지식 양도 매개*

### 2. 전도율 진단 (5-3) ★

**마르덴의 진단**:
- *세이른 진단 결론 재확인 — 마법력 총량 향상은 정통상 어려움 + 전도율 향상이 유일한 해결*
- *아다만티움 = 인간 일부 제련 가능 (조잡한 수준) + 세밀 가공 불가*
- *미스릴 = 인간 완전 제련 불가 — 드워프 지하세계 비전 기술 전용*
- **본작 카엘의 의수 = 두 재질 모두 + 드워프 지하세계 장인 전용 제작**

**대가의 정통**:
- *아다만티움·미스릴은 *돈으로 살 수 없는 재료*. *드워프 부족의 *비공식 거래·교환·은혜 양도만 가능*
- *마르덴 = 드워프 부족과의 *비공식 거래 매개의 *유일한 인간 통로*. *과거 마르덴 본인의 *조상이 드워프 부족에 *큰 은혜 양도*했던 *역사적 채무 매개*

### 3. 드워프 부족의 의뢰 조건 (5-4)

**부족장 (작명 후속, 임시 *카르틴 그라엔* 후보)**:
- *드워프 부족 본가의 *중년 부족장 + 룬 장인 정점 1인*
- *마르덴의 매개로 카엘과 *간접 협상 — 공식 의뢰 조건 통보*

**공식 의뢰 조건 3중**:
1. **마나석 양도** — *카엘의 *바이서스 시기 자기 제작 아티팩트 잔존 + 마검사 검 부서진 부품의 *최고급 마나석 1~2개*. *드워프 부족의 *룬 결각 매개로 사용*
2. **고대 정통 정보 양도** — *카엘의 *카르엔 학파 잠적 분파 비전 + 마녀령 룬 지식 + 그라스 노 장인의 *3대 양도 정통 + 사프란 비망록 마지막 페이지 *복사본*. *드워프 부족의 *마법학 정보 보고에 추가*
3. **미래 약속** — *카엘이 미래에 *드워프 부족의 *어떤 의뢰를 *받으면 *거절하지 않음*. *조건 없음. 부족의 결단에 따름*

**카엘의 결단**:
- *카엘이 *팔찌를 한 번 본다*. *팔찌 조용 — 붉은 빛 없음*
- *3중 조건 모두 수락 결단*
- *마르덴의 *조용한 묵례*

### 4. 드워프 지하세계 진입 + 시험 (5-5)

**지하세계 진입**:
- *마르덴 + 드워프 부족 사절 매개로 *앨더스톰 산령의 *비공개 통로 진입*
- *통로 = 작은 동굴 입구 → 약 *3일 도보 + 룬으로 빛나는 지하 도시 진입*

**부족의 시험**:
- *드워프 부족 정통 시험 — 검 1합 + 마법 1점 시연 + 1마디 답변*
- *검 1합 — 카엘의 강화강철 검기 시연. 드워프 검사가 *공허검 호흡 인지 + 진지한 평가*
- *마법 1점 — 카엘의 *마력→내공 + 의수 검기 시연. 드워프 룬 장인이 *내공의 4중 회로 인지 + 짧은 칭송*
- *1마디 답변 — 부족장의 질문 — 너는 *왜 강해지려 하는가*. 카엘의 답 — *지키기 위해서*. 부족장 *짧은 묵례 — 통과*

**지하 도시 직접 견식**:
- *모루 + 룬 + 광물 정통의 *대륙 최고 보고 직접 견식*. *인간이 결코 따라잡지 못하는 정통의 깊이*
- 카엘의 *깊은 자각 — *내가 강해진다 해도 *드워프와는 *근본 다른 길*

### 5. 아다만티움 의수 작업 + 완성 (5-6·5-7) ★

**작업 기간**:
- *약 2개월 의수 작업* — *부족 장인 1인 (작명 후속) + 부족장 보조 + 카엘의 *자기 마나 매개로 의수 회로 *조율*

**작업 정통**:
- *재료 = 아다만티움 본체 + 미스릴 정밀 관절 + 다중 룬 결각 + 마력 회로 통합*
- *시술 시간 = 의수 자체는 *부족 장인 작업 6주 + 카엘 회로 조율 2주 = 약 2개월*
- *카엘의 시술 통증 + 정신력 견딤* — *마나 감응이 *체내 마나와 *직접 결합하는 매개 작업의 *극심한 통증*

**완성 (5-7)**:
- **아다만티움 + 미스릴 마력 대포 의수 1조**:
  - *마나 감응 최고 효율 (일반 강철의 3~5배)*
  - *검기 운용 = 팔 전체 검기 두께 일반 검의 100~120% 도달*
  - *룬 결각 전체 표면 + 다중 아티팩트화*
  - *마력포 무한 발사 (마나 충전 자동, 1초 간격 연속 발사 가능)*
  - *손가락 촉각 감응 첫 부분 가능*
  - *세밀 작업 정상* (도자기 잡기·바늘질·악기 연주까지)
  - *무게 = 일반 인간 팔의 1.3배 (합금 구조 + 마나 부유 결각으로 체감 무게 상쇄)*
- 정통 사양: [`../../world/prosthetics.md`](../../world/prosthetics.md) 4단계

**카엘의 *깊은 명료성***:
- *팔 전체 검기 시연 — 두께 100~120%*. *처음으로 *결정타 가능 수준*

- 팔찌 조용 — *붉은 빛 없음. 본성의 결단*
