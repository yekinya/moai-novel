## Summary

드래곤 로드 추종파 자객 3체가 카엘 표적. 카이엔이 *동족 1체와 결투 → 동족 살해* → 드래곤족 *추방 결정* 시드. 카엘이 고룡 *카르마이그* 동면 사냥 — 도덕적 무게.

## 회차 구성 (6회차)

| 회차 | 비트 |
|------|------|
| 4-3-1 | 드래곤 로드 자객 3체 동시 추적 신호. 카이엔이 *동족 본태 시야* 로 감지 |
| 4-3-2 | 자객 1체 *카엘 직격 시도*. 카이엔이 *동족 살해 — 자기 손으로*. 본작 최대 도덕적 무게 |
| 4-3-3 | 자객 2체 합동 공격. 카엘 + 카이엔 *연합 처치*. 카이엔의 *둘째 살해* + 카엘의 *첫 본태 드래곤 직격 처치* |
| 4-3-4 | 카이엔의 *드래곤족 추방 결정* 임박 — 드래곤 로드 *침묵의 응답* |
| 4-3-5 | 고룡 카르마이그 동면지 추적. 강철제국 북부 화산지대 |
| 4-3-6 | 카르마이그 *동면 중* 사냥 — 카엘 9써클·소드 엠퍼러급 + 카이엔 본태 마나 + 세파 회수 회로 절반 보조. *깨우면 즉시 패배*. *간신히* 처치. 카이엔이 *평생 죄책감* |

## 핵심 비트

1. **카이엔의 *동족 살해 3건*** — 본작 *카이엔 아크 가장 어두운 순간*
2. **카엘의 *첫 본태 드래곤 직격 처치*** — Chapter 1 5-5 *이실 본태에 패배* 의 정반대
3. **고룡 동면 사냥의 *도덕성*** — *유일한 가능 방법* + *카이엔 평생 죄책감*
