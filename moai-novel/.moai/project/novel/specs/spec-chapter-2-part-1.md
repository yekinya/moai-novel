## Summary

Chapter 1 종결 (공중 발길질 직후 암전) 직후 — *절벽 아래 골짜기로 낙하 + 카엘 두 팔 절단 + 척추·두부 손상 + 의식 거의 소실. 공주 다리 한쪽 뒤틀림*. *세파의 신성마법 응급 처치 + 엔트로퍼 마나 흡수 비정상 효율로 *겨우 숨만 쉬는 회복***. 둥지 외곽 도주 3~5일 → 변경 마을 의료시설에서 **나무 의수 1조 임시 장착**. *세파의 돌봄 + 망명 친지들 길드 경유 소식 + 2주 대기 → 망명지 위치 파악 → 이동 결단*. 시르베인 대도시 경유 → 강철 의수 제작 + **카엘 *마법 시전 완전 봉인 자각* + 강철 의수 검술 훈련 + 마력을 내공으로 활용하는 앤트로퍼 특전 발견**. *세파의 엔트로퍼 진단 + 붉은 보석 팔찌 양도 + 카엘 이중 생각 자기 의심 시드*. *강철 의수로는 자기·세파 보호 불충분 판단 → 세파 몰래 대장간 의뢰 → **왼손 강철 대포 의수 비밀 제작*** (1회 한정·입 트리거·뚜껑 열림). 본 Part 종결 = *망명지 외곽 진입 직전*. **두 팔 없는 몸으로 의수 2단계 + 마력→내공 + 비밀 대포 = 첫 자기 보호 무력 회복**. *비뚤린 시야 vs 본성의 50% 회복 정착 + 세파의 카엘 = 엔트로퍼 자각 + 붉은 보석 팔찌 양도 + 카엘 이중 생각 시드*.

## 핵심 비트

1. **낙하 + 생환 3중 메커니즘** — (1) 엔트로퍼 이득 (정신 안 잃음 + 정신력으로 고통 상쇄 + 빠른 회복력) (2) 바이서스 시기 챙긴 보호구 3종 작동 (가슴 보호구 = 척추 충격 30~40% 완화·균열 후 부분 사용 불능 / 신발 부스터 부츠 = Ch1 P6 5-5 기습 시 사용·24시간 충전 중·낙하 사용 0 / 블링크 반지 = 풍압 칼날에 *팔과 함께 영구 소실*) (3) 세파의 신성마법 보조 + 엔트로퍼 마나 흡수 비정상 효율 = *치유 영약 10배 효율*. **(+ 왼팔에 결합돼 있던 *마나응축 보석 장갑*은 절단된 왼팔과 함께 골짜기 아래로 분리됨 — 1-2 회수 매개)**

1a. **팔 회수 + 마나응축 보석 장갑 회수·판매** ★ (사용자 추가 보강 — 1-1 종결 직후 ~ 1-2 초반) — *카엘 의식 명료 회복 직후 세파에게 자기 팔 수색 요청*. *세파의 신성마법 미세 탐지 응용으로 두 팔 회수 + 마나응축 보석 장갑 무사 회수 (보석 미세 균열·판매 가치 유지) + 드래곤 발톱 비늘 한 조각 동반 챙김*. *변경 마을 진입 직전 비공식 아티팩트 중개상에 보석 장갑 판매 → 의료비 + 강철 의수 보증금 + 1~2개월 야지 생활비 매개*. **카엘의 1년 자기 제작 노동이 팔과 함께 다른 형태의 생존 자금으로 환원되는 짠내**

1b. **품 안쪽 금 재산 무사 확인** ★ (사용자 추가 보강) — *Ch1 P5 §10-E 환전·갈무리의 직접 회수*. 카엘이 가슴 옷솔기 안쪽을 *의수 없는 단면으로 어색하게 더듬어 확인 — 세파 도움 매개 + 금괴·금화 갈무리 무사*. **이 금 + 보석 장갑 판매 대금 = Ch2 P2 이후 야지 생활의 본격 자금원**. 카엘 짧은 한 줄 — *…남았군. 이게 우리 노자돈이다*
2. **카엘 *멀리도 날아왔네*** (1-1) — 의식 명료 첫 순간. 세파의 *살아남으시오, 다리는 내가 알아서 합니다*. 카엘의 *미안하오. 당신의 다리가 그 정도로 다쳤는데… 내 회복부터* + *멀리도 날아왔네. 아직도 뱃속이 뒤집히는 거 같아*
3. **그린드래곤 영역 외곽 *미세 독성 식물·이끼·균사*** (1-2) — 카자르드 영역 정통 일부. *카엘 절단된 두 팔 단면 *미세 마비 잔여*. *세파의 카르샤 왕가 비전 신성 해독*이 결정적 매개. *식수·식량 채집 시 세파 신성 해독 필수*
4. **나무 의수 1단계 장착** (1-2 종결) — 단단한 목재 + 가죽 끈 + 청동 못. *자세 잡기 + 음식 그릇 받침 + 짐 끈 걸치기 정도*. 마나 회로 0
5. **세파의 *엔트로퍼 자각 시드*** (1-3) — 1-1·1-2 시점 카엘 옆에서 *불쾌감 전무*. 회복기에 *조금씩 살아남*. *생리적 자동 작동임을 자각 → 카엘이 의도한 게 아님 확신*. 본격 진단 = 1-5
6. **시르베인 강철 의수 제작** (1-4) — 강철 부분 (손바닥 + 검지 끝 + 회전 관절 3축) + 목재 보강. 마나 회로 미세 결각. 제작 약 3~5일
7. **카엘의 *마법 시전 완전 봉인 자각*** (1-4) — 바람 손날·발바닥 빙판·화염구 모두 *시전 출수 통로 0*. *나는 마법사가 아니다. 마검사도 아니다. 지금의 나는 그저 두 팔 잃은 자다*. Ch2 재정체화의 *생리적 시작점*
8. **마력 → 내공 운용 *첫 발견* (앤트로퍼 특전)** ★ (1-5) — *우연한 발견* — 체내 마나가 *손가락 끝에 머무르는 감각*. 검을 휘둘렀더니 *물리 위력 1.2배*. 반복 시도 → *근육 강화 회로 1번 + 신경 가속 회로 일부 활성화*. 카엘 본인도 *왜 이게 되는지 모름 — 몸이 이렇게 하라고 부른다*. 세파 — *당신은 마법사가 아닌 다른 무엇이 되고 있는 것 같아요*
9. **세파의 *엔트로퍼 진단*** (1-5) — 진단 절차 (질문 시리즈 + 신성제국 의료·진단 기술). **카엘 본인이 처음으로 자기 특성 정확히 인지** — *나는 엔트로퍼다*. *역마나 흡수 + 살상 충동의 금제 마기 + 비뚤린 시야의 정통이 내 안에 공식적으로 존재한다*
10. **세파의 *금제 균열 첫 인식*** (1-5) — 카엘의 금제에 *이미 균열이 가 있음 감지*. *균열 = Ch1 누적 사건의 흔적*. **세파의 깨달음 — *이 균열은 내 아버지(알레이언 6세)의 뜻에 의한 것*** — 4-4 알레이언 한마디 + 자기 희생 + 평화 협정 의미가 *카엘 균열의 정통 근원*. *복수의 그림자가 처음으로 옅어지는 순간*
11. **붉은 보석 팔찌 양도** (1-5) — *카르샤 왕가 비전 호신 아티팩트*. *주변 인물의 악의 기운·악한 생각·살의 의도를 붉은 빛으로 알려줌*. 세파의 어릴 때부터 호신용. *세파: 이 팔찌는 저보다 카엘이 더 필요할 것 같군요*. *당신의 금제가 영향을 미칠 때는 팔찌가 알려줄 거에요*
12. **카엘의 *이중 생각 시드*** (1-5) — *죽여! (비뚤린 시야·금제 마기) vs 멈춰! (50% 회복된 본성·균열 너머 잠재 의식)*. *나의 진짜 생각은 어느 거지?* + 팔찌의 *붉은 빛이 답*. **Ch2~Ch3 카엘 인격 곡선의 결정적 도구**
13. **왼손 강철 대포 의수 비밀 제작** (1-6) — *오른손잡이 = 검을 오른손*. 왼손 강철 의수에 *대포 1회 한정 + 입 트리거 + 손부분 뚜껑 개폐*. 의뢰비 = *가슴 보호구 남은 부품 + 마검사 검 1자루 남은 부품 + 세파 호신 아티팩트 추가 1점*. 외형 = 기존 의수와 거의 동일. *세파 모름*
14. **카엘의 *비밀 결단의 도덕적 부담*** — 대장간 의뢰 시점 *팔찌 붉은 빛 발현*. 카엘 자기 질문 — *이게 나의 본성인가 금제 영향인가*. 결단 — *둘 다일 수 있다. 그러나 세파를 지키지 못하면 둘 다 의미 없다*. 비밀 유지

## 개연성 — 부 단위 약속

- **단서 누설 비율**: *카엘의 마법 시전 완전 봉인 자각*은 1-4에서, *마력→내공 발견*은 1-5에서 본문 명시. *그 전까지는 모름 + 답답함*으로만 표현
- **금지 1**: *앤트로퍼 특전* 용어는 1-5 카엘 본인 자각 시점까지 본문 등장 안 함. 세파의 *왕가 비전 진단*과 함께 본격 명명
- **금지 2**: 본 Part 의수 1·2단계 + 강철 대포 변종까지. *강화강철 의수 등장 금지* (Ch2 P3 그라스 노 장인 시술로 이관)
- **금지 3**: *그라스 노 장인 등장 금지* (Ch2 P3 이관). 본 Part 의수 제작자 = *변경 마을 의료시설 의료인 (나무) + 시르베인 마법 장인 (강철) + 비공식 대장간 장인 (강철 대포)*
- **회상 대응**: Ch1 P6 5-5 *공중 발길질 직후 암전* 직후 의식 페이드 인. *입으로 공주 옷자락 무는 자세*는 1-1 본문 시작 시 *순간 회상으로 회수*
- **그린드래곤 카자르드 영역의 *미세 독성* 회수**: 1-2 둥지 외곽 도주 중 *식물·이끼·균사의 미세 마비* + *세파 신성 해독*. *카엘 두 팔 절단 단면 회복 지연 매개의 물리적 근거*
- **균형 캡**: 카엘의 *마력→내공*은 *근육 강화 + 신경 가속 부분 활성화 + 물리 위력 1.2배까지*만. *공허검 호흡과의 융합·감각 예민화 등은 Ch2 P3 이후*

## narrative-devices

- **잔향 약화 곡선** [§1.7 v2]: Ch1 100% → 1-1 빈사 0% → 1-2 회복 시작 → 1-3 변경 마을 30% → 1-4 시르베인 40% → 1-5 정착 50%. *세파의 불쾌감 점진 회복 관찰 → 카엘이 의도한 게 아님 확신*
- **붉은 보석 팔찌** [§2.5 신규]: 1-5 §5-C 세파 양도. *외부 객관 신호로 카엘 자기 의심 측정 도구*. 1-6 비밀 대장간 결단 직전 *팔찌 붉은 빛 발현* + 1-7 자기 측정 첫 실전 검증
- **드래곤 발톱 비늘 1조각** [§2.6 시드]: Ch1 P6 5-5 카엘 절단된 팔 옆 잔존 → 세파 무의식적 챙김 → 본 Part 1-7 *세파가 조용히 카엘에게 보임*. Ch3 역금제 의식 매개 시드
- **의수 시스템 4단계**: 본 Part = 1단계 (나무) → 2단계 (강철, 왼손 = 비밀 대포 변종). 3단계 (강화강철) = Ch2 P3 이관
- **마력→내공 운용**: 1-5 *우연한 발견*. *근육·신경·호흡* 3중 회로 첫 활성화. *감각 예민화*는 Ch2 P3 공허검 사사 후
- **부두인형 재발견**: 1-4 의수 시술 도중 *세파가 카엘 가방 재정리 중 부두인형 발견*. 카엘이 *짧게 응시 후 다시 가방에 넣음*. *카엘 본인 무의식 표현*
- **알레고리**: 카엘의 *두 팔 상실 + 마법 봉인 = 자기 정체성의 외부 박탈* — 동시대 *능력 박탈로 다시 시작하는 자*의 환유. 명시적 설명 금지

### 1-1 — 멀리도 날아왔네

- **공간**: 드래곤 둥지 영역 외곽 절벽 아래 깊은 골짜기 (Ch1 P6 5-5 절벽 위에서 *공중 부유로 멀리 날아간* 직후 낙하 지점)
- **시점**: 카엘 1인칭 (의식 페이드 인) + 세파 3인칭 짧은 교차 (카엘 의식 부재 시점)
- **사건**:
  - *카자르드 발차기 직격 직후 — 공중 부유 0.5초 짧은 명료한 시야* ★ (사용자 추가 보강) — *카엘의 *잘려나간 두 팔이 *자기 몸과 *다른 궤도로 *멀리 날아가는 것을 봄*. *오른팔은 절벽 안쪽 바위 더미 방향 + 왼팔은 더 멀리 골짜기 아래쪽으로 회전하며 날아감*. **왼팔의 *마나응축 보석 장갑 중심 보석이 햇빛에 한 번 반짝임***. 카엘의 *짧은 자각 — 저거… 내 팔이군. 저 보석 장갑… 1년 걸려 만든…*. 의식 페이드 아웃 직전 *마지막 시야 단편*
  - *공중 회상 1초* — *입으로 공주 옷자락 무는 자세 + 자기 몸으로 아래쪽 처박힘*
  - *바닥 직격* — 척추·갈비뼈 다수 골절 + 두부 충격 + 두 팔 출혈 가속 + 의식 거의 소실
  - *세파 3인칭 교차* — *카엘의 가슴 위에 충격 완화 + 다리 한쪽 뒤틀림 + 즉시 신성마법 응급 처치*
  - **낙하 충격 생환 3중 메커니즘** (엔트로퍼 이득 + 보호구 3종 + 세파 신성마법)
  - 공주의 분투 — *자기 다리 부상 무시 + 임시 부목 + 카엘 회복 우선 결단*
  - 카엘 의식 회복 직후 — *미안하오. 당신의 다리가 그 정도로 다쳤는데… 내 회복부터*
  - 세파 — *살아남으시오. 다리는 내가 알아서 합니다*
  - 카엘 — *멀리도 날아왔네. 아직도 뱃속이 뒤집히는 거 같아*
  - **품 안쪽 재산 무사 확인** ★ (사용자 추가 보강) — 카엘이 *짧은 자각 후 가슴 옷솔기를 의수 없는 단면으로 어색하게 더듬음 — 세파의 도움으로 옷자락 안쪽 봉합 부분 확인 → 금괴·금화 갈무리 무사*. *Ch1 P5 §10-E 환전·갈무리의 직접 회수*. 카엘의 짧은 내적 한 줄 — *…남았군. 이게 우리 노자돈이다*
- **단서**: 가슴 보호구 균열·변형, 신발 부스터 부츠 충전 중, 블링크 반지 영구 소실. **왼팔에 결합돼 있던 *마나응축 보석 장갑은 절단된 왼팔과 함께 골짜기 아래쪽 어딘가에 잔존*** (1-2 회수 시도 매개). *품 안쪽 금 재산 무사*. 카엘 *마법 시전 시도 무자각 안 함* (아직 의식 명료 부족)

### 1-2 — 둥지의 끝자락

- **공간**: 드래곤 둥지 영역 외곽 절벽 아래 골짜기 (1-1 낙하 지점 주변) → 둥지 영역 외곽 산악 → 변경 마을 의료시설

### 1-3 — 망명지의 소식

### 1-4 — 시르베인의 강철

  - 세파의 짧은 위로 — *살아 있으면 됩니다*
  - 변경 도시 대장간에서 *마검사 검 부품으로 임시 단검 1자루 완성*
- **[부두인형 재발견]** 의수 시술 도중 *세파가 가방을 재정리하다 부두인형 발견*. 카엘이 *짧게 응시 후 다시 가방에 넣음*. *Ch1 P6 5-5 세파 첫 발견 → 본 Part 1-4 재발견 → Ch2 P3 그라스 노 장인 회수 연결*
- **단서**: 카엘의 *체외 마법 봉인의 3중 추정* (의수 마나 회로 미세 + 두 팔 절단의 신체 회로 + 어떤 다른 정통 가능성). 본 회차에서 *3번째 가능성은 모름*

### 1-5 — 의수의 검 ★

- **공간**: 시르베인 외곽 훈련장 (임시 — 빈 들판 또는 작은 사원 뒷마당)
- **시점**: 카엘 1인칭 (훈련 + 발견 + 진단 + 양도)
- **사건**:

#### 검술 훈련 시작

- 카엘이 *건강 회복 안 됨에도 강철 의수 검술 훈련 강행* — 세파의 반복 만류
- *마력 운용 강철 의수에 활용 시도 + 반복 실패* — *마나가 검에 안 흘러감*

#### 마력 → 내공 운용 *첫 발견* (앤트로퍼 특전) ★

- *우연한 발견* — 체내 마나가 *손가락 끝에 머무르는 감각* 자각
- *그 상태에서 검을 휘둘렀더니 물리 위력 1.2배 강해짐*
- *반복 시도 → 근육 강화 회로 1번 활성화 + 신경 가속 회로 일부 활성화*
- 카엘 본인도 *왜 이게 되는지 모름 — 몸이 이렇게 하라고 부른다*
- 세파의 짧은 한 마디 — *당신은 마법사가 아닌 다른 무엇이 되고 있는 것 같아요*

#### 세파의 *엔트로퍼 진단*

- 진단 절차 — *카르샤 신성제국 의료·진단 기술 + 핵심 질문 시리즈*:
  - *어릴 때 마을에서 누군가가 원인 모를 불쾌감을 호소했나요?*
  - *마법 시전 시 마나가 외부에서 흘러 들어오는 감각이 있나요?*
  - *살의의 시작 시점이 자기 의지 이전인지 자기 의지 이후인지 구별 가능한가요?*
  - *주변 마나가 자기 회로로 빨려 들어가는 감각이 있나요?*
- 카엘 처음에는 *방어적 침묵 — 비뚤린 시야가 위협으로 인지*. *세파의 살리는 자 동기를 50% 회복된 본성으로 인정 → 점차 답함*
- **카엘 본인이 처음으로 자기 특성 정확히 인지**:
  - *나는 엔트로퍼다*
  - *역마나 흡수 + 살상 충동의 금제 마기 + 비뚤린 시야의 정통이 내 안에 공식적으로 존재한다*
  - *Ch1 카르엔 학파 잠적 분파 비전 + 친조부 알리스의 책 여백 + 카르엔 봉인 문서 단편*의 모든 단편이 *하나로 묶임*

#### 세파의 *금제 균열 첫 인식*

- *세파의 신성마법 + 진단 기술로는 엔트로퍼 금제를 깨는 것은 절대 불가능*
- 그러나 *카엘의 금제에 이미 균열이 가 있음 감지*. *균열의 폭 = Ch1 누적 사건의 흔적*
- **세파의 깨달음 — *이 균열은 내 아버지 (알레이언 6세)의 뜻에 의한 것*** — Ch1 4-4 알레이언 한마디 + 5합 결투의 자기 희생 + 평화 협정 의미가 *카엘에게 누적된 균열의 정통 근원*
- *세파의 복수의 그림자가 처음으로 명확히 옅어지는 순간* — *아버지의 뜻이 카엘의 균열을 의도했다면 나의 복수도 그 뜻에 어긋날 수 있다*

#### 붉은 보석 팔찌 양도

- 세파의 *카르샤 왕가 비전 호신 아티팩트 중 하나* — *붉은 보석 박힌 작은 팔찌*. 일상에서는 평범한 장신구처럼 보임
- 기능 — *주변 인물의 악의 기운·악한 생각·살의 의도를 감지해 붉은 빛으로 알려줌*. *카르샤 신성제국 신성마법의 진단 보조 매개*
- 세파가 *어릴 때부터 차고 있던 개인 보호용*. 왕족 호신 표준 장비 중 하나

**팔찌 양도 장면**:

> 세파가 *팔찌를 천천히 손목에서 풀어 카엘에게 건넨다*. *카엘의 강철 의수 손가락 위에 조심스럽게 놓는다*.
>
> 세파: *"이 팔찌는 저보다 카엘이 더 필요할 것 같군요."*
>
> 카엘이 *팔찌를 한 번 본다*. *낮은 음성으로* — *"이게 뭐지?"*
>
> 세파: *"악의 기운·생각 등의 의도를 붉은빛으로 알려주는 팔찌에요. 당신과 있으면 이게 항상 빛이 났지요. 아마 그 금제의 영향 같네요. 전 당신의 습성이 완전한 악이라고 생각했어요. 지금은 가끔씩 그러지만."*
>
> 짧은 침묵.
>
> 세파: *"당신의 금제가 당신에게 영향을 미칠 때는 그 팔찌가 알려줄 거에요."*

#### 카엘의 *이중 생각 시드 정착*

- 본 §5 이후 카엘이 *모든 결단 순간에 내적 이중 음성 명확히 자각*:
  - *"죽여!"* — 비뚤린 시야의 살상 충동·금제 마기의 자동 응답
  - *"멈춰!"* — 50% 회복된 정의로운 본성·금제 균열 너머 잠재 의식
- *카엘의 자기 질문 — …나의 진짜 생각은 어느 거지?*
- *팔찌가 붉은 빛으로 답해줌* — *지금 너의 결단은 금제 영향 우세다 → 멈춰. 또는 금제 영향 없음 → 본성의 결단이다*
- **이중 생각의 정통 자각이 Ch2~Ch3 카엘 인격 곡선 본격 시드**

#### 훈련 강도 + 건강 부진

- *훈련이 건강 회복을 지연시킴 — 척추·갈비뼈 통증 회복 1~2주 지연*
- 카엘의 결단 — *나는 지금부터 내 몸을 새로 짠다*
- 세파의 *조용한 응급 처치 + 약초 갈음 + 신성 회복으로 간신히 균형 유지*

### 1-6 — 비밀 대장간 ★

- **공간**: 시르베인 비공식 무기 대장간
- **시점**: 카엘 1인칭 (결단 + 비밀 + 자기 의심)
- **사건**:
  - 카엘이 *강철 의수로는 자기·세파 보호 불충분 판단* — *세파가 위험 시 물리적 거리에서 치명적 한방이 필요*
  - *세파가 신성 의식에 잠시 자리 비운 동안* — 카엘 비공식 무기 대장간 단독 방문
  - *왼손 강철 의수의 추가 개조 의뢰* — *대포 1회 한정 장착 + 입 트리거 + 손부분 뚜껑 개폐*
  - 의뢰비 = *바이서스 시기 챙긴 가슴 보호구 남은 균열 부품 + 마검사 검 1자루 남은 부품 = 부분 양도 + 세파 호신 아티팩트 추가 1점*
- **대포 의수 사양**:
  - *왼손 (오른손잡이) 강철 의수에 대포 1회 한정 장착*
  - *대포알 작음에도 매우 무거움* + *짐·의수 무게 부담*
  - *입으로 왼팔뚝 트리거 물어 당김 → 손부분 뚜껑 열리며 발포*
  - **1회 한정** (충전·재장전 절차 복잡)
- 카엘의 *비뚤린 시야와 본성의 경합* — *팔찌가 대장간 의뢰 시점에 *붉은 빛 발현*
- 카엘이 *팔찌를 본 후 짧은 자기 질문* — *이게 나의 본성인가 금제 영향인가*
- 결단 — *둘 다일 수 있다. 그러나 세파를 지키지 못하면 둘 다 의미 없다*. 비밀 결단 유지
- 외형적으로 *기존 강철 의수와 거의 동일 — 세파 의심 안 함*
- **단서**: *카엘이 비밀 카드의 도덕적 부담을 자기 안에 조용히 짊어짐*. 향후 Ch2 P7 §20 자식 척살 결단의 *내적 시드*

### 1-7 — 망명지의 문

- **사건**:
  - 시르베인 출발 + 망명지 이동 *약 3~5일 산악 도보 + 호위 코드 매개*
  - *세파의 조용한 동행 + 짧은 대화 단편*
  - 카엘의 *자기 측정 첫 실전 검증* — 작은 위험 (변경 도적 1~2회 + 작은 마수 1~2회) 회피
    - *세파의 신성마법 보조 + 카엘의 마력→내공 + 강철 의수 검술 물리 위력 1.2배*
    - *카엘이 결단 직전마다 팔찌를 한 번 봄*
    - *팔찌의 신호가 카엘의 결단을 부분적으로 보정*
  - 세파의 짧은 한 마디 — *팔찌가 도움이 되는 것 같네요*
- **[드래곤 발톱 비늘 1조각]** 세파가 *가방에서 짧게 꺼내 카엘에게 보임*. *Ch1 P6 5-5 카엘 절단된 팔 옆 잔존 — 세파가 무의식적으로 챙김*. **Ch2~Ch3 역금제 의식 매개 시드**. 카엘이 *짧게 본 후 가방에 다시 넣음*
- 망명지 외곽 진입 직전 — 호위 안내인 (망명지의 작은 인물 단편) 멀리서 다가옴
- 카엘의 한 마디 — *…내가 살아남았다. 지금부터다*
- **단서**: *세파의 손목에 팔찌가 없음 → 카엘의 강철 의수 손가락에 조용히 걸려 있음* + 양인 *조용한 동맹의 비공식 정착*

>
> 카엘이 *낮은 음성으로 한 마디*. *"…내가 *살아남았다. *지금부터다."*
