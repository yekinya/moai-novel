## Summary

Chapter 1 종결 (공중 발길질 직후 암전) 직후 — *낙하 + 바닥 직격 → 카엘 의식 페이드 + 공주의 신성마법 분투 + 엔트로퍼 마나 흡수 비정상 효율로 간신히 생환*. 둥지 영역 외곽 도주 (몬스터 회피·생사의 길 반복·3~5일) → 카르샤 외부 망명지 (키얀 왕자 측) 합류 → *그라스 노 장인의 의수 1단계 시술* + *친조부의 책 출처 회수* + *사프란 비망록 마지막 페이지* + *붉은 머리끈 의수 봉인 회로* + *카엘의 회복 + 강해지는 과정 시작*. 본 Part 후반 = **카엘의 *바이서스 복수 결단* + 드래곤 복수 *분노 시드*** 형성. *바이서스 복수는 *장기적 정치·추가 전쟁 방식* 으로 *긴 호흡* 진행 결단 + 드래곤 복수는 *역량 부족이라 도주하며 분노 누적*. 본 Part 종결 후크 = *카엘의 *결전 의지 첫 외부 신호* (드래곤 추적 분기점). 본 Part 부터 *카엘이 두 팔 없는 몸 + 의수 1단계로 *재정체화* + 평등 살상 가치관 정착*.

## 핵심 비트

### 1. 낙하 + 생환 + 둥지 영역 외곽 탈출

(Chapter 1 Part 6 v3 §6~§10 콘텐츠 흡수)

**낙하 + 바닥 직격 (1-1)**:
- 절벽 위에서 *둘 다 공중으로 멀리 날아가는 동안* — 카엘이 *공중에서 의식 일부 유지*. *두 팔이 없어 공주를 끌어안을 수 없음* → *대안으로 입으로 공주의 옷자락을 물고 위로 올림*
- 자신의 *남은 몸 (어깨·가슴·다리)* 으로 *아래쪽으로 처박힘 자세*. *자기 희생 자세*
- *바닥 충돌 — 카엘 척추 + 갈비뼈 다수 골절 + 두부 충격 + 두 팔 출혈 가속 + 의식 거의 소실*
- 세파 = *카엘의 가슴 위에 충격 완화된 채 떨어짐*. *다리 한쪽 뒤틀림 (Chapter 1 발길질 시점 1차 손상 유지)*

**낙하 충격 생환 3중 메커니즘**:
1. *엔트로퍼 이득* — 정신 안 잃음 + 정신력으로 고통 상쇄 + 빠른 회복력
2. *바이서스 시기 챙긴 보호구 3종 작동*:
   - *가슴 보호구* — 척추 충격 30~40% 완화 (작동 후 *균열·변형·부분 사용 불능*)
   - *신발 부스터 부츠* — 5-5 기습 시 사용 → 24시간 충전 중 (낙하 직전 사용 0)
   - *블링크 반지* — 풍압 칼날에 *팔과 함께 영구 소실*
3. *공주의 신성마법 보조* — 즉시 응급 처치 + 엔트로퍼 마나 흡수 비정상 효율로 *치유 영약 10배 효율*

**공주의 분투 + 카엘 *미안하다*** (1-1):
- 공주가 *자기 다리 부상 무시 + 임시 부목 + 카엘 회복 우선 결단*
- 카엘 의식 회복 직후 *미안하오. 당신의 다리가 그 정도로 다쳤는데 …내 회복부터*
- 공주: *살아남으시오. 다리는 내가 알아서 합니다*
- 카엘의 한 마디: *멀리도 날아왔네. 아직도 뱃속이 뒤집히는 거 같아*

**둥지 영역 외곽 탈출 (1-2)**:
- *발길질로 수십 미터 멀리 날아왔으나 외곽 가까운 상태*. *오우거·트롤·와이번 활동권은 여전히 영역 외곽까지 미침*
- *카엘 = 두 팔 없음 + 척추 손상 = 전투 불가*. *공주 = 다리 뒤틀림 + 신성마법 약화 = 보조만 가능*
- 공주가 *카엘을 부축·견인* + 카엘이 *희미한 의식으로 마법 한 줌씩 보조* (결계 분쇄·빙판 보법·광명 눈동자 직격 등) + *생사의 길 반복 3~5일*
- *공주가 *식수·식량을 카르샤 왕가 비전 기본 채집 기술로 확보* (야생 약초·강물 정수)
- *공주의 *카엘 = 엔트로퍼* 의 희미한 첫 의심 시드* — *왜 자기 마나가 카엘에게 이렇게 흡수되는지 의문*

### 2. 카르샤 외부 망명지 합류 + 키얀 왕자 첫 정면

**망명지 도착 (1-3)**:
- *카르샤 외부 망명 라인* = *Chapter 1 캐논 — 카르샤 왕비 엘라리아 + 셀라엔 대주교 + 카일렌 베르하르트의 처남 1인* 이 *키얀 왕자 (9세)* 와 함께 외부로 도주한 망명지. 위치 = *바이서스 영토 밖 — 헤븐즈 8왕국 다른 영역 (정확한 위치는 Chapter 2 후속에서 결정)*
- 카엘 + 공주가 *둥지 영역 외곽 탈출 후* 망명 라인의 *비공식 정보망* 을 통해 망명지 도착
- *망명지 = 작은 산골 수도원 + 카르샤 외부 인척 가옥들*. *경비 카르샤 잔존 기사 약 10~15명*

**카르샤 외부 망명자들의 *카엘 직면***:
- **카르샤 왕비 엘라리아** — *공주의 친모 + 알레이언 왕의 미망인*. 카엘 = *남편을 죽인 자*. 첫 대면에서 *침묵의 분노 + 짧은 한 마디 — "당신이 데려왔군요. 우리 딸을."*
- **셀라엔 대주교** — *신성 신앙 최고 권위*. 카엘에 대한 *공식 종교적 단죄 입장*. *너는 이 신성한 땅에 발을 들였다*. 그러나 *공주의 단호한 보호 진술로 즉결 처분 보류*
- **카일렌 베르하르트의 처남 1인** — *카르샤 정찰병 출신*. *카엘에 대한 직접 적의 + 공주 호위 결단*

**공주의 *카엘 보호 진술***:
- 공주: *나는 알아요. 이 자가 *무엇을 했는지*. 그러나 이 자가 *나를 살렸고, 내 아버지의 진짜 마지막을 알아요*.*
- *공주가 처음으로 카엘에 대해 *외부 다수 앞에서 공식 보호 결단을 표명*. *복수의 그림자 아래에서도 *생명을 살린 자에 대한 의리* 의 우선 결단*

**키얀 왕자 (9세) 의 *카엘 첫 정면***:
- 키얀이 *카엘 앞에 정면으로 섬*. *작은 검 (Chapter 2 후속의 *작은 검 돌격* 시드) 을 손에 쥔 채*. *분노 + 호기심 모순 시드*
- 키얀의 *첫 한 마디 — 너는 내 아버지를 죽였어. 그런데 …나를 데려온 누이를 살렸어*. *어린아이의 *논리적 모순 직면*
- 카엘이 *말없이 무릎을 꿇음 (한쪽 무릎 + 검 내려놓음 자세 = Chapter 3 후속의 *공식 무릎* 의 *축약 첫 버전*). *어린 키얀의 *분노 표현을 받아들이는 자세*
- 키얀이 *작은 검을 카엘의 어깨에 *살짝 가져다 댐* + 거두기*. *아이의 첫 결단 — 지금은 죽이지 않는다. 그러나 잊지 않는다*
- *Chapter 2 후속의 *키얀-카엘 결투* 의 *진짜 정통 시드*. 본 Part 에서는 *3합 결투의 *유아 버전 (검을 어깨에 살짝 가져다 댐)***

### 3. 그라스 노 장인의 의수 1단계 시술

(Chapter 1 Part 7 6-2 콘텐츠 흡수)

**그라스 노 장인 첫 등장 (1-4)**:
- *그라스 = 데미안 마법 장인 도시 — 헤븐즈 8왕국 중 데미안 제국 영지의 마도공학 중심지*. *카르샤 외부 망명지에서 *비공식 경로로 카엘 + 공주를 *그라스로 안내*. *키얀 왕자 측의 *비공식 동맹 첫 협력*
- **그라스 노 장인** = 60대 후반의 노 마법 장인. *카엘의 친조부 알리스와 같은 가문 (카르엔 학파 잠적 분파의 외부 인척)*. *카엘의 책의 진짜 출처를 아는 인물*

**친조부 책 출처 회수**:
- 카엘이 *그라스 노 장인에게 부친 책을 보여줌*. 노 장인이 *책의 여백 3중 필체를 즉시 인식*. *알리스 — 알린 — 카엘* 의 *3대 양도 정통 폭로*
- 노 장인의 한 마디 — *너의 친조부 알리스를 알았다. 너의 친조모 이렐라도. *그녀가 어떤 존재였는지도 안다.* 너는 *그 핏줄의 끝이다.*
- **Chapter 1 Part 1 시드의 진짜 회수** — *친조모 이렐라 = 강 엔트로퍼 미기록 발현* + *친조부 알리스 = 카르엔 학파 잠적 분파 학자* + *알린 = 약 캐리어* 의 *3대 비극* 이 *카엘 앞에 공개*

**의수 1단계 시술**:
- *재료 = 강철 + 룬 회로 + 카엘의 *마검사 검 1자루의 부서진 부품* + 그라스 노 장인의 *카르엔 학파 비전 마도공학 기법*
- *기능 = 기본 손가락 5개 동작 + 마나 회로 미세 연결 + 마법 시전 보조 통로 + 무게 = 일반 인간 팔의 1.2배*
- *시술 시간 = 약 3일*. *카엘이 시술 도중 의식 일부 유지*. *극심한 통증을 정신력으로 견딤*
- *의수 1단계의 한계* — *세밀한 시전은 어려움 + 일부 마법 시전 효율 30% 저하 + 무게 부담 + 24시간 충전형 마나 보조 시스템*

**사프란 비망록 마지막 페이지** (1-4):
- 노 장인이 *친조부 알리스의 *비밀 보관물 — 사프란 비망록 마지막 페이지* 1장 공개*. *사프란 학파의 *최종 카드 자결* 의 의미 첫 노출*
- *비망록의 핵심 — 엔트로퍼는 *자기 본성을 거부하는 자결로만 *진정한 평화에 도달*. 그러나 *그 결단은 자기 의지로만 가능*
