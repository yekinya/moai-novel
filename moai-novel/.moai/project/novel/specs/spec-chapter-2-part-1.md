## Summary

Chapter 1 종결 (공중 발길질 직후 암전) 직후 — *절벽 아래 골짜기로 낙하 + 카엘 두 팔 절단 + 척추·두부 손상 + 의식 거의 소실. 공주 다리 한쪽 뒤틀림*. *세파의 신성마법 응급 처치 + 엔트로퍼 마나 흡수 비정상 효율로 *겨우 숨만 쉬는 회복***. 둥지 외곽 도주 3~5일 → 변경 마을 의료시설에서 **나무 의수 1조 임시 장착** + *기초 행랑만 가능*. *세파의 돌봄 + 신성마법 회복 + 망명 친지들에게 *길드 경유 소식 전함* + 2주 대기 → *망명지 위치 파악 → 이동 결단*. 이동 중 *중간 대도시 경유 → 강철 의수 제작*. *카엘 본인 *마법 시전 완전 봉인 자각 + 강철 의수 검술 훈련 + **마력을 내공으로 활용하는 앤트로퍼 특전 발견***. *강철 의수로는 자기·세파 보호 불충분 판단 → 세파 몰래 대장간 의뢰 → **왼손 강철 대포 의수 비밀 제작** (1회 한정·입 트리거·뚜껑 열림)*. 본 Part 종결 시점 = *망명지 외곽 진입 직전*. **카엘이 *두 팔 없는 몸으로 의수 2단계 + 마력→내공 + 비밀 대포 = 첫 자기 보호 무력 회복***. 본 Part 의 *내적 정통 = *비뚤린 시야 vs 본성의 50% 회복 정착 + 세파의 *카엘 = 엔트로퍼 자각 + 붉은 보석 팔찌 양도 + 카엘의 *이중 생각 자기 의심 시드*.

## 핵심 비트

### 1. 낙하 + 생환 + 둥지 영역 외곽 탈출 (1-1·1-2)

(v3 §1 유지 + 그린드래곤 종족 특화 *미세 독성* 반영)

**낙하 + 바닥 직격 (1-1)**:
- 절벽 위에서 *둘 다 공중으로 멀리 날아가는 동안* — 카엘이 *공중에서 의식 일부 유지*. *두 팔이 없어 공주를 끌어안을 수 없음* → *대안으로 입으로 공주의 옷자락을 물고 위로 올림*
- 자신의 *남은 몸 (어깨·가슴·다리)* 으로 *아래쪽으로 처박힘 자세*. *자기 희생 자세*
- *바닥 충돌 — 카엘 척추 + 갈비뼈 다수 골절 + 두부 충격 + 두 팔 출혈 가속 + 의식 거의 소실*
- 세파 = *카엘의 가슴 위에 충격 완화된 채 떨어짐*. *다리 한쪽 뒤틀림* (Ch1 발길질 시점 1차 손상 유지)

**낙하 충격 생환 3중 메커니즘**:
1. *엔트로퍼 이득* — 정신 안 잃음 + 정신력으로 고통 상쇄 + 빠른 회복력
2. *바이서스 시기 챙긴 보호구 3종 작동*:
   - *가슴 보호구* — 척추 충격 30~40% 완화 (작동 후 *균열·변형·부분 사용 불능*)
   - *신발 부스터 부츠* — Ch1 P6 5-5 기습 시 사용 → 24시간 충전 중 (낙하 직전 사용 0)
   - *블링크 반지* — 풍압 칼날에 *팔과 함께 영구 소실*
3. *세파의 신성마법 보조* — 즉시 응급 처치 + 엔트로퍼 마나 흡수 비정상 효율로 *치유 영약 10배 효율*

**공주의 분투 + 카엘 *미안하다*** (1-1):
- 공주가 *자기 다리 부상 무시 + 임시 부목 + 카엘 회복 우선 결단*
- 카엘 의식 회복 직후 *미안하오. 당신의 다리가 그 정도로 다쳤는데 …내 회복부터*
- 공주: *살아남으시오. 다리는 내가 알아서 합니다*
- 카엘의 한 마디: *멀리도 날아왔네. 아직도 뱃속이 뒤집히는 거 같아*

**둥지 영역 외곽 탈출 (1-2)** — v4 보강:
- *발길질로 수십 미터 멀리 날아왔으나 *외곽 가까운 상태*. *몬스터 자연 회피 구배의 *외곽 가장자리에 가까움 → 몬스터 일부 서식권 진입 가능성*
- **그린드래곤 영역 특화 — *미세 독성 식물·이끼·균사***: 카자르드의 영역 정통 일부. *카엘의 절단된 두 팔 단면에 *미세 마비 잔여 (회복 지연 매개)*. *세파의 카르샤 왕가 비전 신성 해독 = 결정적 매개*. *식수·식량 채집 시 세파 신성 해독 필수*
- *카엘 = 두 팔 없음 + 척추 손상 = 전투 불가*. *공주 = 다리 뒤틀림 + 신성마법 약화 = 보조만 가능*
- 공주가 *카엘을 부축·견인* + 카엘이 *희미한 의식만 유지*. *마법 시전 자체는 *체외 출수 통로 0 = 완전 봉인*. (카엘 본인 *왜 마법이 안 되는지 아직 모름 — 단지 마나가 몸 안에 머무를 뿐*)
- *생사의 길 반복 3~5일 → 변경 마을 의료시설 도착*

**변경 마을 의료시설 — 나무 의수 장착 (1-2 종결)**:
- 세파가 *길드 경유로 변경 마을 의료시설 정보 입수* + *임시 의료비 = 세파의 *왕가 비전 호신 아티팩트 1점 양도* (호신 부적·왕가 표지 안 보이는 단편)*
- **나무 의수 1조 장착** — 단단한 목재 + 가죽 끈 + 청동 못. *자세 잡기 + 음식 그릇 받침 + 짐 끈 걸치기 정도만 가능*
- 정통 사양: [`../../world/prosthetics.md`](../../world/prosthetics.md) 1단계
- 카엘 본인 *마법 시전 완전 봉인 자각의 *첫 단편* (의수 자체는 마나 회로 0 — *왜 시전이 안 되는지 더 모름*)

### 2. 세파의 돌봄 + 망명지 소식 + 이동 결단 (1-3)

**변경 마을 체류 (약 2주)**:
- 세파의 *전적 돌봄* — *카엘의 식사·약초 갈음·신성마법 회복 보조*
- *세파 본인 다리 뒤틀림은 부목으로 임시 고정 — *완치 안 됨. 절뚝거리며 활동*
- *카엘은 거의 누워 있는 상태 → 약 1주 차부터 짧은 일어섬 가능*

**길드 경유 소식 전송**:
- 세파의 *망명 중인 친지들 소문 입수* — *바이서스 영토 밖 헤븐즈 8왕국 다른 영역 어느 산골 수도원 + 카르샤 외부 인척 가옥들*
- *변경 마을 길드 사무소를 통해 *비공식 정보 채널로 *살아 있음 알림 + 망명지 좌표 요청*
- *2주 대기 후 답신 도착* — 정확한 망명지 위치 + 안전 동선 + 호위 코드 명시

**세파의 *엔트로퍼 자각 시드*** (v3 §5-A 잔향 약화 정통 유지):
- *세파가 1-1·1-2 시점에 *카엘 옆에서 불쾌감 전무*. 이를 *극심한 부상으로 인한 *마기 무력화* 로 일단 해석*
- *1-3 변경 마을 체류 회복기 — 카엘의 회복과 함께 *불쾌감이 *조금씩 살아남*. 그러나 *Ch1 시기의 강도와 명확히 다름*
- *불쾌감의 강도가 *카엘 본인의 의식적 통제가 아닌 *생리적 자동 작동임을 세파가 자각*. *Ch1 시기 *살의 표현의 *위장* 일 가능성을 의심했던 세파가 *이것은 카엘이 의도한 게 아님* 을 확신
- *세파의 *왕가 비전 옛 자료 회상 + 본인 직접 관찰 정황 통합* → *카엘 = 엔트로퍼* 결론
- **본격 진단 + 직접 대화 + 붉은 보석 팔찌 양도는 *1-5 검술 훈련 시기 회차에서 진행* (§5 참조)**

**이동 결단**:
- *세파 + 카엘이 *망명지 이동 결단* — *바이서스 일반 추격대는 둥지 영역 외곽에서 단절. 변경 마을은 안전이나 카르샤 외부 망명지 합류가 *장기 안전 + 신성제국 잔존 호위 + 지원 매개*
- *이동 동선 = *변경 마을 → 중간 대도시 (강철 의수 제작 매개) → 망명지 외곽***
- 정통 시기 = 1-3 종결

### 3. 중간 대도시 + 강철 의수 + 마법 시전 봉인 자각 (1-4)

**대도시 진입**:

### 5. 세파의 *엔트로퍼 진단 + 붉은 보석 팔찌 양도 + 카엘 이중 생각 시드* ★ (v3 §5 유지)
