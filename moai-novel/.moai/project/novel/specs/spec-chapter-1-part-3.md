## Summary

셀라리아 외곽 *카르엔 외곽 연구소* (왕실 소속이나 정치색 없는 *한직 도서관*) 에 비정규 계약직 연구원으로 입소한 카엘. 5년간의 *외로운 학습* 끝에 5써클 도달 (앤트로퍼 이득 고려해도 사기 수준). 카르엔 사이엔의 *왕실 권위 회복 야망*의 도구로 활용되다가, 카르엔의 *우발적 죽음* (정황상 카엘 책임처럼 보이는 *마나 폭발 사고*) 으로 외곽 연구소 떠나며 라이세린에 *흡수*됨. 본 Part 종결 시 카엘 직급 = **외교실 자문 + 5써클 공식 인증** (~~왕실 마법 고문~~ 폐기).

## 핵심 비트

1. **외곽 연구소 = 왕실 소속 한직 도서관** — 정치색 거의 없음. 카엘 직급 = ~~왕실 마법 고문~~ **외곽 연구소 *비정규 계약직 연구원***. 공식 직위 거의 무명. 5년 동안 *외부 노출 거의 없음*
2. **카엘의 3중 니즈** (입소 동기) — *돈* (안정된 거처·식사) / *기술* (1.5부 마린·델란 자각 후 정통 학습 필요) / *힘* (추방 트라우마의 보상 — *강해지면 다시는 추방당하지 않는다*. 앤트로퍼 *살인 갈구의 학문적 갈구로 가장된 형태*)
3. **5년 → 5써클 도달의 4가지 개연성 풀이**:
   - 카르엔 = 옛 7써클 마법사 (시전 능력 유지, 다리 절음). *7써클까지의 정통 로드맵 + 시연 직접 견식* 전수
   - 외곽 연구소 *풍족한 도서 + 자료 정리* (정통 5단계 + 데미안 마도공학 기초 + 마녀령 흑마법 + 일레인 응용 + 사프란 일부)
   - 카르엔의 *수단 방법 안 가리는 빠른 성장* (금기 *엔트로피 역연산* 권장)
   - 앤트로퍼 이득 + 자기 직관 1단계 융합 = *5써클 등록증 + 9써클급 위력*
4. **카르엔 사이엔 4중 정체** — 옛 7써클 영광 / 마법 사고로 한직 좌천 (다리 절음·시전 능력 유지) / **카르엔 학파 잠적 분파 8대 계승자** / *왕실 권위 회복 야망*. 카엘 = 맞춤형 도구 = *1회성 카드*
5. **카르엔의 *깊은 한 줄*** (2-2) — *"세상에는 두 종류의 차가 있네. 마시는 차와… 마시지 않는 차."* (표면 = 신뢰의 비유 / 진짜 = *선택권은 나에게*)
6. **카엘의 외로움 5년 정착** — 식당 안 감 → 방에서 식사 → 카르엔만 *예외적 정기 사사*. 동료 잔향 작동
7. **2-5 미라 베스턴 사건** — 미라가 *카엘의 외로움 동정 + 호감*으로 어깨에 손 → 카엘이 *정보 절취*로 해석 → 손목 골절. *0.5초 망설임이 잔향 매개*. 미라의 사직서 *두려움 때문*. **외곽 연구소 내부 *카엘 위험 인물 분류*의 분기점**
8. **2-8 카르엔 *우발적 죽음*** — 카엘이 *직접 살의로 죽인 것 아님*. *환상의 붕괴 → 마나 회로 폭주 → 연구실 마나 도구·책장 동시 폭발 → 카르엔 직격*. 카엘이 *불 끄려 마나로 잡아 보려* 시도하나 *비뚤린 시야에서 자기 마나 통제 불능*
9. **카엘 → 카르엔 감정 변화 3단** — *1~2년차 도구로 인식* → *2~5년차 옅어지는 도구 인식 (가까움이 단의 기억 위에 겹쳐짐)* → *5년차 환상의 정점 (카르엔이 나를 사랑한다고 믿음)* → *환상의 붕괴 = 2-8 분노 폭발의 정통*
10. **자연스러운 정치 인과 체인** — 2-1~2-5 카엘 외부 노출 0 → 2-6 카르엔 부풀림으로 *공식 마법사 명부 등록* → 라이세린 우연 발견 → 2-6 외곽 연구소 직접 면담 (*왕립 연구원 시험 아님*) → 2-7 미라 사건 인지 (*통제 가능 + 약점*) → 2-8 카르엔 사망 후 *수습 명목 외교실 자문 흡수*

## narrative-devices

- **잔향** [§1.7]: 학문 공동체의 *까닭 모를 어색함* — 카르엔 (잠적 분파 직계) = *완전한 인지*, 시험관·동료들 (5~6써클) = *명확한 불쾌, 정체 모름*, 라이세린 (고위 정치인) = *잔향을 유용한 도구로 알아챔*. **잔향 단어 본문 미등장 유지** (5-2부터 등장)
- **카엘 외모** [§9.5]: *준수한 외양*이 학문 공동체 무대에서 *더 두드러짐*. 귀족 영애·여성 학자들의 *짧은 시선* 1~2회. 카엘은 *호의를 시기로 해석*
- **짠내 잔존**: 1.5부 갈비뼈 부상이 2-1까지 *통증으로 남음*. *카엘이 한밤중 침대에서 옆구리를 만지는 1줄* — 짠내의 *지속성*
- **부두인형·엘 단검의 잔존**: 2-2 짐 정리 중 *카엘이 가방 깊은 곳에서 두 물건을 짧게 봄*. 즉시 다시 넣음. *기억의 무덤* 1줄
- **재미** [§2.1]: 카엘의 *비정형 응용*이 학문 공동체에서 *천재 또는 위협*으로 받아들여짐. 2-5 *엔트로피 역연산*의 *정통 절차 무시 응용* 가시화
- **알레고리** [§8]: 라이세린 = *국익을 외치는 사익*의 정치인 알레고리. *시종의 시선 떨군 1줄 = 내부고발의 침묵된 양심 환유*

## 회차 비트

### 2-1 — 회색 도시

- **공간**: 셀라리아 외성문 → 외곽 연구소 등록처. 마법 안개로 늘 흐린 도시
- **사건**: 신분증 없는 자를 받는 *외곽 연구소 비정규 계약직 등록* (델란의 추천장 매개). 시험관이 카엘 마나 회로 효율에 놀라되 *눈을 보고 한순간 망설임*
- **[단서]** 시험관이 등록증에 *작은 표식* 추가로 찍음 (관찰 대상 표시 — 본인은 모름)
- **[잔향]** 시험관이 카엘 *마나 회로 효율*에 놀라 *손을 잠시 카엘 어깨에 댔다가 즉시 거둠*. *왜 손을 거뒀는지 본인도 모름*. 후일 시험관은 *카엘이 등록된 날*을 *왠지 모르게 잠 못 든 밤*으로 기억함
- **[외모 1컷]** 등록 대기실의 *귀족 영애 둘*이 카엘을 *조용히 본다*. *변경에서 온 평민 청년이 왜 저렇게 잘생겼느냐* 짧은 속삭임. *옆에 다가가지 못함* — 잔향이 시선을 *멀리서만* 유지. 카엘은 *비웃음*으로 해석
- **[짠내 잔존]** 도시 야경을 보던 카엘이 *옆구리 깊은 곳의 통증*으로 잠시 멈춤. 1.5-3 부상이 *두 달 지나도 완전히 낫지 않음*
- **마무리**: 처음 보는 큰 도시의 야경. 카엘 시야에서 도시는 *무수한 거짓의 등불*

### 2-2 — 스승의 그림자

- **공간**: 외곽 연구소 카르엔의 서재
- **사건**: 노마법사 카르엔이 *역마나 감각*을 알아보고 사사 제안. 첫 수업, 차 한 잔. 카르엔의 *왼쪽 다리 약간 절음*이 첫 정면 묘사
- **[카르엔 사고 정황 명시]** 카르엔 본인이 *짧게 말함* — *"전 왕대 군 자문 시절 시연 사고. 마법 회로 일부 손상. 시전은 여전히 되네."* 단편 단 1줄. 다리 절음과 함께 *7써클 마법사의 한직 좌천*의 정통 시드
- **[단서 — 회상 컷 #2 대응]** 카르엔이 *손수* 차를 따름. 카엘은 *독을 타려는 손길*로 봄. 객관 사실: *카르엔이 자기 잔에도 같은 분량을 따라 마신 뒤 위장에 좋다고 권함*. 카엘 차 *마시지 않고 비운 척*. 카르엔은 알면서도 웃음
- **[카르엔의 깊은 한 줄]** *"세상에는 두 종류의 차가 있네. 마시는 차와… 마시지 않는 차."* (표면 = 신뢰의 비유. 진짜 = *내가 너에게 무엇을 마시게 할 것인가의 선택권은 나에게*)
- **[잔향 — 카르엔 완전 인지]** 카르엔 (잠적 분파 직계) 이 *카엘이 들어오는 순간 눈썹을 약간 움직임*. *사프란 비망록의 그 묘사가 맞군*의 짧은 표정. 카엘은 *늙은이의 무관심*으로 해석. **카르엔만이 잔향의 정체를 정확히 안다**
- **[부두인형·단검 시드]** 첫 수업 후 카엘이 *기숙사 짐 정리* 중 가방 깊은 곳에서 *부두인형과 엘 단검을 짧게 봄*. 다시 덮음. 본문 1줄 — *그는 그것들이 거기 있는지 잠시 확인하고 다시 덮는다. 왜 확인하는지 묻지 않는다*

### 2-3 — 1년차의 학습

- **공간**: 외곽 연구소 도서관·연구실·복도
- **사건**: 첫 정통 1~2등급 학습 시작. 자기 직관 시전법과의 차이 첫 자각. 외로움 정착 — 식당 안 감 → 방에서 식사 → 카르엔이 *방으로 식사 운반 조용히 허락*
- **[자동 필기 깃펜]** 카르엔 서재의 보조 도구 — 카엘이 *반복 정독에 사용*
- **[마나 측정기]** 카르엔 책상 위 — 5-2에서 *세파 매개로 다시 등장*
- **[정통 학습 충돌]** 카엘의 *1단계 직관 시전*이 정통 1~2등급 학습과 충돌. *자기 회로가 정통 절차를 거부*

### 2-4 — 외전격 단편: 2년차의 갈증

- **공간**: 외곽 연구소 연구실 침대 — *어느 늦은 밤*
- **시점**: *본 Part 유일한 1인칭 단편* (3인칭 근접 → 1인칭 전환). 약 800~1,200자
- **내용**: 카엘의 첫 자기 의문 — *내가 왜 여기 있는가. 강해지고 싶다. 그런데 왜 강해지고 싶은가. 답이 추방의 보복이라면 — 나는 마을 사람들 따위에게 내 인생을 종속시킨 자가 된다. 답이 세상의 잘못된 것을 바로잡는 것이라면 — 누가 나에게 그 권한을 줬는가. 답이 없다. 그래서 나는 답을 미룬다. 강해지고 나서 답하기로 한다. 그것이 가장 쉬운 길이기 때문이다.*
- **외전의 의의**: *비뚤린 시야 너머 잠재한 정의로운 본성의 첫 흔적*. 4-5 자각의 *심리적 시드*. 다음 날 *카르엔의 도구화 권유로 다시 비뚤린 시야 회복 → 카엘이 잊음*
- **3단 의문 정통 시드**: 2-4 외전 (2년차) → 5년차 카르엔 사망 → Part 4 3-3 직전 외전. *카엘이 자기를 의심한 흔적 누적이 4-5 자각의 정통성*

### 2-5 — 3~4년차의 가속 + 미라 사건

- **공간**: 외곽 연구소 지하 금서고 + 공동 실험실 야간
- **사건**:
  - 카르엔의 *빠른 성장 권장* — 금기 분야 *엔트로피 역연산* 첫 시도. 죽은 곤충을 *형태만* 되돌림 (생명은 돌아오지 않음). 부작용으로 카엘의 시각이 *짧게 색을 잃음*
  - **색 사라진 1분 동안** — 카엘이 처음으로 *카르엔의 표정에서 적의가 보이지 않음*. 그 사실 자체에 *기만술의 일종*이라 결론. 색 돌아오자 다시 *적의가 보임* — *시야가 곧 인식*임을 독자에게만 누설
  - 부작용을 카르엔에게 보고하지 않음
- **[미라 베스턴 사건]** 동료 연구원 미라가 카엘의 *어깨에 손* + *실험 일지가 너무 무리야, 좀 쉬어*
  - 카엘이 *정보 절취 시도*로 해석 → 손목을 잡아 비틀어 *손목 골절*. 미라는 비명을 지르지 않고 *조용히 사과* — *미안해, 놀라게 했지*. 카엘은 *증거 인멸 시도*로 해석
- **[잔향 — 미라의 본능적 망설임]** 미라가 어깨에 손을 *얹기 직전 0.5초 망설임* 묘사. *본인도 왜 망설였는지 모름*. 카엘은 *공격 자세 준비*로 해석. **잔향이 친절을 공격으로 보이게 만든 핵심 비트**
- **[미라의 진짜 마음]** *카엘에게 호감이 있었음* — *준수한 외양 + 천재 마법사*의 매혹. 그러나 *까닭 모를 거리감*이 멈칫. 4-5 자각 시 카엘이 *그녀의 진심을 처음 이해*. 미라가 *사직 전 마지막 밤 일기에 한 줄 — 그를 좋아했다고 생각했는데, 옆에 있으면 왜 그렇게 무서웠을까. 답은 모르겠다*
- **[카엘 본인 야간 동요]** 그날 밤 *카엘이 처음으로 잠들지 못함*. *과로 탓*으로 해석. 본문 1줄 — *그는 옆구리를 만진다. 통증은 더 이상 없는데도*. *작은 인간성 잔재의 가시화*
- **마무리**: 다음 날 미라 사직. 사직서에 *두려움 때문*만. 카엘은 *위장 후퇴*로 봄

### 2-6 — 5년차의 부풀림 + 라이세린 우연 발견

- **공간**: 외곽 연구소 5써클 인증 의식장 (외부 학자 일부 초청)
- **사건**:
  - 카르엔의 *루키 포지셔닝* 책동 — *5써클 인증 의식*. 실제 카엘 숙련은 3등급 + 비상식 응용. 그러나 *외부 학자에게 5써클로 부풀려 공인*
  - **카엘이 *공식 마법사 명부* 등록**
  - **라이세린 우연 발견** — 공식 마법사 명부 정기 검토 중 *변경 출신 + 5년 만에 5써클*의 비정상적 기록 주목 → **외곽 연구소 직접 방문 시도**
  - 라이세린이 외곽 연구소로 직접 옴. 카르엔 + 카엘과 함께 차 자리. *자네 같은 사람을 기다렸다* 한마디
  - **이 시점 라이세린은 *카엘 흡수 계획만 있고 카르엔 사망 정황 모름***. 본심 = *변경 정세 격화 직전 강 발현 엔트로퍼를 전쟁 도구로 활용*
- **[단서]** 라이세린의 *자네 같은 사람을 기다렸다* 발언 시 *옆 시종이 잠시 시선을 떨굼* (시종의 단 한 줄 묘사 = 알레고리 §8)
- **[카르엔의 추천장]** 라이세린에게 천거용 추천장 — *어조가 1.5-7 델란 추천장과 일치* (정통: 카르엔이 *델란 추천장 어조를 모방*해 자기 천거 권위를 *낯설지 않게* 만듦)

### 2-7 — 시기의 거울

- **공간**: 외곽 연구소 내부 회의실 + 라이세린 추가 면담
- **사건**:

  - 라이세린이 *2-5 미라 사건도 알게 됨* — *카엘의 통제 가능성 + 약점* 동시 인지. *전쟁 도구 활용 계획*이 *확신*으로 굳어짐
  - 카엘의 외로움 신체화 정점 — *입 다물고 코로 호흡 (1.5-1 글라렌의 잔향)* + 방 안 식사 + 밤 도서관 혼자 정독
  - 외곽 연구소 동료들의 *카엘 옆자리 비움*의 잔향 작동 다수 묘사

### 2-8 — 마나의 형해 (카르엔 사망)

- **공간**: 외곽 연구소 카르엔 연구실. 야간
- **사건**:
  - 카르엔의 *왕실 시연회 강요* — *내일의 시연은 *나의 영광이지 자네의 영광이 아니야*. 자네는 *내가 만든 마법사*일세*
  - **카엘의 환상 붕괴** — *카르엔이 나를 *사랑한다고 믿었던 5년의 환상* 단숨에 깨짐*. *5년 동안 가족이라 믿었던 자가 물건으로 봤다*
  - **분노의 마나 회로 폭주** — 연구실의 *마나 도구 + 책장* 동시 폭발. 카엘 본인도 *어떻게 했는지 모름*
  - 카르엔이 *폭발 직격* — 연기 흡입 + 광역 화상. 카엘이 *불 끄려 마나로 잡아 보려* 시도하나 *비뚤린 시야에서 자기 마나 통제 불능*
  - 카르엔 사망. 외곽 연구소 *부분 화재 진압 후*
- **[객관 사실 vs 외부 인식]**:
  - 객관: *카엘이 직접 살의로 죽이려 한 것 아님*. *카엘의 마나 폭발이 직접 사인*. *정상 시야였다면 대화로 해결 가능했을 일*
  - 외부 인식: *고의적 살해처럼 보이는 정황* + 마나 도구 폭발 = *고의적 시전 결과처럼 외부에 보임* + 동료들의 *격렬한 언쟁 목격*
  - 라이세린이 *이 정황을 빠르게 파악 → 카엘의 약점으로 무기화*
- **[정통적 진실]** 사용자 HARD — *카엘이 그 정도로 비뚤린 인간은 아님*. **카엘이 *완전한 살인자가 아닌* 증거**. 엔트로퍼 마기로 인한 분노 폭발 결과. *카엘의 정의로운 성격은 마기 너머에 살아있음* — 4-5 자각·6부 회복의 정통 출처
- **[카르엔 미완성 경고 서한]** *"네가 보는 세계가 세계의 전부가 아니다…"*. 사고 직전 작성. 카엘이 *비겁한 모욕*으로 오독 → 소각. 4-5 자각 시 *기억 표면 회수*
- **[카엘의 내적 갈등]** *내가 그를 죽였나 / 아닌가*의 확신 못함. *영구 모호*. *카르엔이 나를 사랑했다는 환상이 깨진 상실*의 더 큰 무게
- **[라이세린 즉시 흡수]** 다음 날 새벽 *라이세린 공작의 마차가 외곽 연구소 앞에*. *수습 명목 — 위험 인물 안전 관리*. *실질 — 전쟁 도구화*. **카엘 직급 = 외교실 자문 + 5써클 공식 인증** (왕실 마법 고문 폐기)
- **종결 한 줄**: 라이세린 — *"노이엔 군. 자네 같은 사람을 *기다렸다*. 이제 *진짜 시작*일세."*

## 카르엔 외곽 연구소 — 2대 연구 과제 (학문 정통)

### 9-A. 신체 마나 응축 향상 *매개체* 연구

- **가설**: 인간 신체 마나 저장 한계 = *써클 등급의 천장*. 외부 매개체 정합·삽입 시 저장 용량 폭증 가능
- **매개체 후보**:
  - *결정화 심장 계열* — 드래곤·마왕급·정령왕류 핵·고위 몬스터 심장 (드래곤·마왕은 사례 0건, 고위 몬스터만 현실 채취 가능, 효과 매우 낮음)
  - *순도 99.9% 미스릴 압축공* — 본 세계관에서 *아다만티움 다음으로 강하고 다루기 어려운 금속*. 정제 한계 99.9%. *드워프 왕족 비전 제련술로만 99.99% 도달* 소문. 왕실 직속 대장간급 자원 요구
- **현실**: 매개체 모두 *외곽 연구소 한직 예산 압도*. 5년 연구 산물 = *이론서 + 매개체 정합 공식 + 실패 실험 기록 (고위 몬스터 한정)*. 실증 단계 미도달
- **카엘 흡수**: 5년 학습 중 이 이론서 통독. 후일 *Part 5+ 결정화 심장급 매개체 우연 입수의 정통 시드*
- **함의**: 카엘의 *9써클급 위력 발휘*가 *매개체 없이 가능한 살아있는 반증*. 카르엔이 *3~5년차 인지 → 루키 카드화 동기에 학문적 광기 합산*

### 9-B. 드래곤 *용언마법* 연구

- **분석 대상**: 드래곤 브레스 + 용언. *강대한 드래곤 내부 마나가 몸속 자체 변환 완료 후 외부 토출*. 인간 마법사의 *주문 영창 = 외부 변환 명령*과 *역순*. 시전 시간 0 + 영창 없음 + 위력 압도적
- **핵심 메커니즘**: *마나의 쿼크 단위 물질 변환*. 드래곤 = 쿼크 단위 분해·재조합으로 *질량과 부피 직접 통제*. 인간 = *상위 분자·원소 단위에서 변환 명령으로 우회* (영창 시간 + 효율 손실의 원인)
- **이론적 함의**: 인간이 *자체 물질 변환 능력 + 강대한 마력 보유량* 두 조건 동시 충족 시 *드래곤과 동일한 시전 시간 0 대형 마법* 이론 가능
- **카르엔의 좌절**: *인간의 뇌로는 쿼크 단위 통제 불가* — 이것이 카르엔이 *7써클에서 멈춘 진짜 이유* (역마나 폭주 사고는 *표면 변명*). 수학적 모델링 도달, 신체 실행 불가
- **카엘과의 연결**: 카엘의 *1단계 직관 + 앤트로퍼 마기 보너스 + 5써클 정통 융합 = 주문 영창 우회 비율이 비정상적으로 낮음*. 카르엔 4~5년차 인지. **카엘 후일 광역 학살 마법을 시전 시간 거의 0으로 발현하는 정통 근원**
- **함의**: 카엘 = *카르엔의 두 연구 가설을 동시에 부분 실증하는 살아있는 유일 표본*. 그래서 카르엔이 *수단 방법 안 가리고 가르친 진짜 이유* = *자기 평생 연구의 마지막 실험체*

### 9-C. 카엘의 *기형성* 교차점

| 카르엔의 이론 | 인간 한계 | 카엘의 예외 |
|---------------|-----------|-------------|
| 9-A 매개체 응축 | 외부 매개체 없이 7써클 천장 | 매개체 없이 9써클급 위력 발휘 |
| 9-B 쿼크 단위 변환 | 인간 뇌로 쿼크 통제 불가 | 직관 시전 + 마기로 *부분 우회* |

→ 2-8 사망의 *정통 비극* = 카르엔이 *이 학문적 보석을 외부 정치 도구로 부풀려 사용*하려 한 순간, *보석 자신이 부풀림에 반발해 폭발*. **학자의 야망이 자기 학문을 죽인 구조**

## 정치 인과 자연성 점검

- **카엘 외교실 진입 공식 명분 충분?** — 카르엔 사망 정황 수습 + 위험 인물 안전 관리 = 명목 충분
- **라이세린의 흡수 권한?** — 외교실 차관 권한 + 라카르 9세의 *외교실 자문 임명* 위임 권한
- **외곽 연구소 반발 부재?** — 한직 도서관 + 카르엔 사망 후 지도자 부재. 라이세린에 반발할 권력 없음
- **카엘의 반발 부재?** — 카르엔 사망 죄책감 + *내가 무엇인지 모르는 공허* + 라이세린의 *내가 너의 약점을 안다*의 무언 압박
