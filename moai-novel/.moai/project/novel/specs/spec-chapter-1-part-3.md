## Summary

셀라리아 외곽 *카르엔 외곽 연구소* (왕실 소속이나 정치색 없는 *한직 도서관*) 에 비정규 계약직 연구원으로 입소한 카엘. 5년간의 *외로운 학습* 끝에 5써클 도달 (앤트로퍼 이득 고려해도 사기 수준). 카르엔 사이엔의 *왕실 권위 회복 야망* 의 도구로 활용되다가, 카르엔의 *우발적 죽음* (정황상 카엘 책임처럼 보이는 마나 폭발) 으로 외곽 연구소 떠나며 라이세린에 *흡수* 됨.

## 핵심 비트 (사용자 보강 5항 통합)

### 1. 외곽 연구소 = *왕실 소속 한직 도서관*

- **카르엔 외곽 연구소**: 왕실 직속 명목상 소속이나 *정치색 거의 없는 *도서관 같은 한직***. 영향력 매우 낮음 → 외부 정치 견제 부재 → *비정규 계약직 등록 정치 문제 낮음*
- **카엘 직급**: ~~왕실 마법 고문~~ **외곽 연구소 *비정규 계약직 연구원*** — *공식 직위는 거의 무명*. 5년 동안 *내부 도서관 출입 + 보조 연구 + 본인 연구 시간 자유*
- **5년 동안의 외부 노출**: 거의 없음. 카엘은 *외곽 연구소 안에 처박힘*. 셀라리아 도시 전체에서 *모르는 존재*

### 2. 카엘이 연구소 들어가고 싶은 *니즈* (개연성 확보)

본 Part 진입 시 카엘의 동기 — *돈·기술·힘*:

- **돈**: 1.5부 떠돌이 짠내 + 부상 누적 → *안정된 거처 + 식사 보장* 절실. 외곽 연구소 *비정규 계약직 = 평민 5인 가족 식비 + 연구실 침대 + 도서관 접근*
- **기술**: 1.5-4 마린의 *3단계 정통 견식* + 1.5-7 델란의 *너는 *다른 무엇* 이다* 한마디로 카엘이 *정통 마법학 본격 학습* 필요성 자각
- **힘**: *추방의 트라우마 + 떠돌이의 무력함* 의 보상 — *강해지면 다시는 추방당하지 않는다* 의 무의식. **앤트로퍼의 *살인 갈구* 가 *학문적 갈구* 로 가장된 형태**

→ 본 Part 카엘 본인 의식은 *학문적 호기심 + 안정 거처* 만. 진짜 동기 *힘에의 갈구* 는 *후일 회수*

### 3. 5년 → 5써클 도달의 *개연성* (사기 수준 인정 + 풀이)

#### 본작 캐논 — 사기 수준 인정

- 정통 마법사 5년에 5써클 = *극히 드문 천재*
- 앤트로퍼 이득 (마나 효율 ↑, 기술 탐구 가속) 고려해도 *여전히 사기 수준*
- 본 Part는 *이 사기성을 풀어내야 함*

#### 4가지 개연성 풀이

1. **카르엔 사이엔 = 옛 7써클 마법사**:
   - 30대에 7써클 도달. 헤븐즈 영웅 시대 후기 최정상급
   - 50대 중반 *시전 사고* (라카르 9세 전 왕대 군 자문 시 신성마법 결박 시연 중 *역마나 폭주*) — **마법 기술은 모두 유지** (7써클 시전 가능). 다만 *왼쪽 다리 영구 손상으로 약간 절음*. 사건 책임 떠안고 외곽 연구소로 좌천. 시전 능력은 그대로지만 *외부 정치 무대에서 물러남*
   - **카르엔 = 본작 *최고급 마법학 지도자***. 카엘에게 *7써클까지의 정통 학습 로드맵* + *시연 가능한 7써클 마법 직접 견식* 전수
2. **외곽 연구소의 *풍족한 마법학 도서 + 자료 정리***:
   - 도서관 한직이라 *외부 출판 → 도서관 자료 정리 흐름* 이 지속됨
   - 카엘이 *5년간 도서관 자료 거의 전부 정독* — 정통 5단계 마법학 + 데미안 마도공학 기초 + 마녀령 흑마법 일부 + 일레인 응용 마법 + 사프란 학파 일부 (카르엔이 *조각으로 보여줌*)
3. **카르엔의 *수단 방법 가리지 않는 빠른 성장 연구***:
   - 카르엔은 *옛 야망 — 왕실 권위 회복* 을 위해 *카엘의 *루키 사례* 가 필요*. 자기 마법체계의 우수성 증명 도구
   - *가장 빠른 성장 경로 + 위험한 학습 방법* 도 카엘에게 *기꺼이 시킴* (예: 금기 분야 *엔트로피 역연산* 의 *부작용 알면서 시도 권장*)
   - 카엘은 *왜 카르엔이 자기에게 *이렇게까지* 가르치는가* 의문. *비뚤린 시야로 자기 천재성의 인정* 으로 오독
4. **앤트로퍼 이득 + 자기 직관 1단계 시전 정통 융합**:
   - 1.5부에서 형성한 *1단계 직관 시전* 이 정통 학습 후에도 *깊이 살아남음*
   - 5써클 *공식 등급* + *자기 직관 융합 → 실제 위력 9써클급* 발휘 가능 (앤트로퍼 보너스)
   - 즉 *써클 등급은 인간 편의의 척도* (magic-system §6) 대로 — 카엘 = *5써클 등록증 + 9써클급 위력* 의 *기형적 천재*

#### 본 Part 종결 시 카엘 상태

### 4. 카르엔 사이엔 — *옛 야망의 한직 학자*

#### 카르엔의 4중 정체

1. **옛 7써클 영광**: 30~50대 헤븐즈 영웅 시대 후기 최정상. 칠흑성 전투 시기 (CC 870) 라카르 9세의 *전 왕대 군 자문 마법사* 역임
2. **마법 사고 + 한직 좌천**: 50대 중반 *시전 사고* — 라카르 9세 전 왕대 군 자문 시 *신성마법 결박 시연 중 역마나 폭주*. **마법 기술은 모두 유지** (7써클 시전 모두 가능). 다만 *왼쪽 다리에 영구 손상이 남아 약간 절음* — *특유의 걸음걸이* 가 카르엔의 캐릭터 표식. 사건 책임 떠안고 외곽 연구소 *한직* 으로 좌천. 본인은 *명목상 *학문 연구 시간*** 으로 받아들이나 *실은 자존심 깊이 상처*
3. **카르엔 학파 잠적 분파의 8대 계승자**: 학파 비전 보유. 본 Part 이전 캐논 유지. 카엘의 *역마나 감각* 을 알아본 진짜 이유
4. **야망 — 왕실 권위 회복**:
   - 카르엔의 진짜 목표 = *외곽 연구소를 왕실 직속 권력축으로 복권*
   - 그러려면 *세계가 인정할 *루키 마법사*** 가 필요
   - 카엘 = *맞춤형 도구*. 강 발현 엔트로퍼의 *비정상적 성장 속도* + *카르엔 자기 마법체계의 산물* 로 외부 포장 가능
   - *1회성 카드* — 카엘이 *전쟁 + 권위 회복* 의 도구 역할 후 *어떻게 되든 관계 없음*

#### 카르엔의 카엘 간파

- 카르엔은 *카엘의 비틀린 시야* 를 *3년차쯤 간파*
- 그러나 *기본 전제 — 자신이 믿는 *정의의 신념* 으로 움직임* 도 간파
- **결론 — 카엘은 *정의로운 도구로 활용 가능*** = 카르엔 계획의 핵심
- 카엘의 *기본 정의감* 을 *카르엔 자신이 원하는 방향* 으로 유도하면 *완벽한 1회성 무기*

#### 카르엔의 *깊은 한 줄*

> *"세상에는 두 종류의 차가 있네. 마시는 차와… 마시지 않는 차."* (2-2 첫 사사 시. 표면 의미 — 신뢰의 비유. 진짜 의미 — *내가 *너에게 무엇을 마시게 할 것인가* 의 *선택권은 나에게* 있다*)

### 5. 카엘의 *외로움* (잔향 → 사회적 고립)

- 외곽 연구소 5년 거의 *혼자만의 시간*
- *식당 가지 않음*. **방에서 식사**. 방값과 함께 *연구실 침대로 식사 운반* 옵션을 카르엔이 *조용히 허락*
- 동료 연구원과의 교류 *거의 없음*. *복도에서 마주칠 때만 짧은 인사*. 카르엔만 *예외적으로 *정기 사사*** 받는 유일한 사람
- 외곽 연구소 내부에서도 *카엘에게 잔향 작동* — 동료들이 *왜인지 모르게 카엘 옆 자리 비움*
- **2-5 미라 베스턴 사건**의 진정한 정통:
  - 미라가 *카엘의 외로움을 동정* 해 *어깨에 손* 을 얹음 (잔향 거리감 *극복한 *드문 호의***)
  - 카엘이 *정보 절취 시도* 로 해석 → 손목 비틀어 골절
  - 미라의 *침묵 사과 후 사직* — 정통은 *두려움 + 카엘에 대한 *마지막 호의의 흔적***
  - 이 사건이 *외곽 연구소 내부에서도 카엘이 *공식 위험 인물*** 로 분류되는 분기점
- 본 Part 카엘의 *외로움 신체화* — *입을 다물고 코로 호흡* (1.5-1 글라렌의 충고 잔향) + *방 안에서 혼자 식사하는 모습* + *밤에 도서관 혼자 정독*

### 6. 카르엔의 *우발적 죽음* — 의도적 오해 스토리 ★

#### 사건 정황 (2-6)

본 Part 클라이맥스. 카엘이 *직접 죽이려 한 것은 아니나 *책임이 있는** 마나 폭발 사고.

**사건 흐름**:

1. 카르엔의 *왕실 권위 회복 계획* 절정 직전. 카엘에게 *왕실 시연회 출석 + 자기 마법체계 공개 시연* 강요
2. 카르엔이 *마지막 점검 — 카엘의 *충성도 검증*** 위해 *너무 명백한 도구화 발언* — *"자네는 *내가 만든* 마법사일세. 내일의 시연은 *나의 영광이지 자네의 영광이 아니야*."*
3. 카엘이 *비뚤린 시야에도 *불구하고*** 처음으로 *카르엔에 대한 분노 폭발* — *자기를 평생 사랑한다고 느꼈던 자가 *실은 자기를 *물건 으로 봤다***
4. 분노가 *마나 회로 폭주* 로 외화. 연구실의 *마나 도구 + 책장* 이 *동시 폭발* — 카엘 본인도 *어떻게 했는지 모름*
5. 카르엔이 *폭발 직격*. *연기 흡입 + 광역 화상*. 카엘이 *불을 끄려 마나로 잡아 보려* 시도하나 *비뚤린 시야에서 자기 마나 통제 불능*
6. 카르엔 사망. 외곽 연구소 *부분 화재 진압 후*

**객관 사실**:
- 카엘이 *직접 살의로 죽이려 한 것 아님*
- 그러나 *카엘의 마나 폭발이 직접 사인*
- 카엘 본인 *책임이 있음* — 정상적 시야였다면 *대화로 해결* 가능했을 일

**외부 인식**:

- 마나 도구 폭발 = *고의적 시전 결과처럼* 외부에 보임

#### 정통적 진실 — *카엘이 그 정도로 비틀린 인간은 아님* (사용자 HARD)

→ 사후 *카르엔이 나를 *사랑했다는 환상* 이 깨진 상실* 의 *영원한 죄책감 시드*. 4-5 자각·6부 회복 시 *단·카르엔·세파의 3중 가족 이미지 회수*

## 9. 카르엔 외곽 연구소 — *2대 연구 과제* ★ (v4 신설)

외곽 연구소가 *한직 도서관* 이라는 외피 아래에서 카르엔이 *진짜로* 몰두한 연구 과제는 두 가지. 둘 다 *왕실 권위 회복 야망* 의 기술적 발판이며, 카엘이 *5년간 도서관 자료를 거의 전부 정독* 하는 동안 *조각조각 흡수* 한 핵심 학문 토대.

### 9-A. 신체 마나 응축 향상을 위한 *매개체* 연구

#### 연구 가설

- 인간의 *신체 마나 저장 한계* 는 *써클 등급의 천장* 을 결정짓는 본질적 제약
- *외부 매개체* 를 신체 마나 회로에 정합·삽입하면 *저장 용량 자체를 폭증* 시킬 수 있다는 가설
- 카르엔의 *옛 7써클 천장 경험* 에서 출발 — *왜 8써클 이상은 신체 한계인가* 의 물음

#### 매개체 후보 — *결정화 심장* 계열

- **드래곤 심장** (결정화) — 이론상 최상위. *내부 마나 자체 변환 메커니즘* 의 잔향이 결정 안에 남아 있다는 추정. *세계 단위로 손에 넣은 사례 0건*
- **마왕급 존재의 심장** (결정화) — 마계 깊은 곳 외에는 채취 불가. *전설로만 존재*
- **정령왕류 핵** — 정령계 협상 사례가 *마법학사 기록상 헤븐즈 영웅 시대 1건 뿐*. 사실상 신화
- **고위 몬스터 심장** — 와이번·오우거·키메라 등. *드래곤·마왕 대비 효과 매우 낮음*. 그러나 *유일하게 현실적으로 채취 가능* 한 등급. 카르엔의 *실험 가능 범위는 여기까지*

#### 매개체 후보 — *순도 99.9% 미스릴 압축공*

- **미스릴** = 본 세계관에서 *아다만티움 다음으로 강하고 다루기 어려우며 구하기 힘든 금속*
- 정제 한계가 *순도 99.9%* — 그 이상은 *현 기술로 불가*. *과거 드워프 왕족의 비전 제련술로만 99.99% 도달* 했다는 소문
- 99.9% 미스릴을 *극한 압축* 하여 *마나 저장 매개체* 로 사용하면 결정화 심장에 준하는 효과 *이론 추정*
- 미스릴 정제 + 압축 공정 자체가 *왕실 직속 대장간급 자원* 을 요구 — 외곽 연구소 *현실적 자체 제작 불가*

#### 연구의 현실 — *이론으로만 정립*

- 매개체 후보 *모두 채취·제작 난이도가 외곽 연구소 한직 예산을 압도*
- 카르엔의 5년 연구 산물 = *이론서 + 매개체 정합 공식 + 실패 실험 기록 (고위 몬스터 심장 한정)*
- *실증 단계 미도달* — 다만 *공식 자체는 완성도 높음*
- **카엘의 흡수**: 5년 학습 중 *이 이론서를 통독*. 본인은 *흥미로 읽었으나* — 후일 *Part 5 이후 카엘 자신이 *결정화 심장급 매개체를 우연히 손에 넣게 되는 정통 시드***

#### 정통적 함의 (시드)

- 카엘의 *9써클급 위력 발휘* (§3 4번 풀이) 가 *순수 신체 마나만으로* 가능했다는 사실 자체가 — *카르엔 이론의 가설을 깨는 *예외 사례***

### 9-B. *주문에 종속되지 않은* 드래곤의 *용언마법* 연구

#### 분석 대상 — 드래곤 브레스와 용언

- **드래곤 브레스 분석** — *강대한 드래곤 내부 마나* 가 *몸속에서 자체 변환까지 완료된 후 숨과 함께 외부로 토출* 되는 메커니즘
- 인간 마법사는 *주문 영창* 으로 외부 마나에 *변환 명령* 을 부여하지만, 드래곤은 *내부에서 변환 완료* → *외부 구현* 의 *역순*
- 결과로 *시전 시간 0 + 영창 없음 + 위력 압도적* 의 *용언마법* 성립

#### 핵심 메커니즘 — *마나의 쿼크 단위 물질 변환*

- 드래곤은 *마나를 쿼크 단위까지 분해·재조합* 하여 *질량과 부피까지 직접 통제*
- 인간 주문 마법은 *쿼크 단위 직접 통제 불가* → *상위 분자·원소 단위에서 변환 명령* 으로 *우회*. 이 우회 비용이 *영창 시간 + 마나 효율 손실* 의 정통 원인
- **이론적 함의**: 인간이 *자체 물질 변환 능력* 과 *강대한 마력 보유량* 의 *두 조건* 을 동시에 충족하면 — *드래곤과 동일한 메커니즘으로 시전 시간 0 의 대형 마법 구현 이론적 가능*

#### 카르엔의 좌절

- *인간의 뇌로는 쿼크 단위 마나 변환을 *이해도 통제도 불가*** — 이것이 카르엔이 *7써클에서 멈춘* 진짜 이유 (역마나 폭주 사고는 *표면 변명*)
- 카르엔은 *드래곤 용언 메커니즘의 *수학적 모델링* 까지는 도달*. 그러나 *모델을 신체로 실행할 *뇌 구조 부재***
- 5년 연구의 *결론적 좌절* — *인간 한계 안에서는 불가*

#### 카엘과의 연결 — *반증 가능성*

- 카엘의 *1단계 직관 시전* + *앤트로퍼 마기 보너스* + *5써클 정통 융합* = *주문 영창 우회 비율이 인간 평균 대비 비정상적으로 낮음*
- 카르엔이 *4~5년차에 인지* — *카엘이 *우회 비용을 일부 부담하지 않는 듯한* 시전 패턴* 보임
- **시드**: 카엘이 후일 (Part 4 이후) *광역 학살 마법을 *시전 시간 거의 0* 으로 발현하게 되는 정통 근원* — *카르엔 이론의 *살아있는 부분 실증***. 카엘 본인은 *왜 자기가 그렇게 시전되는지 모름* — 카르엔 사후 *해설자 부재* 의 영구 모호

#### 정통적 함의 (시드)

- 카엘의 *광역 학살 가능성* (§3 종결 항목) 의 학문적 정통 = *이 연구의 *예외 사례화***
- 카르엔의 *왕실 권위 회복 야망* 의 진짜 도구 = *카엘 = 용언마법에 가장 근접한 인간 반증*. *시연회* 에서 카르엔이 보여주려 한 것 = *주문 영창 거의 없는 카엘의 시전* 그 자체
- **2-8 사망의 정통 비극** = 카르엔이 *이 학문적 보석을 *외부 정치 도구로 부풀려 사용* 하려 한 순간*, *보석 자신이 *부풀림에 반발해 폭발*. *학자의 야망이 자기 학문을 죽인 구조*

### 9-C. 두 연구의 *교차점* — 카엘의 *기형성*

| 카르엔의 이론 | 인간 한계 | 카엘의 예외 |
|---------------|-----------|-------------|
| 9-A 매개체 응축 | 외부 매개체 없이는 7써클 천장 | 매개체 없이 9써클급 위력 발휘 |
| 9-B 쿼크 단위 변환 | 인간 뇌로는 쿼크 통제 불가 | 직관 시전 + 마기로 *부분 우회* |

- **결론** — 카엘은 카르엔의 *두 연구 가설을 동시에 *부분 실증* 하는 살아있는 유일 표본*. 그래서 카르엔이 *카엘에게 *수단 방법 가리지 않고* 가르친* 진짜 이유 = *자기 평생 연구의 *마지막 실험체***
- *3년차 간파* (§4 카르엔의 카엘 간파) 의 의미가 *학문적 광기* 차원에서 *심화* — *카엘의 비뚤린 시야가 학문 표본으로서의 가치를 *훼손하지 않는다* 는 카르엔의 *학자적 무자비함***
