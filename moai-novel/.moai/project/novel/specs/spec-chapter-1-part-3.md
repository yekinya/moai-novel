## 핵심 비트 (사용자 보강 5항 통합)

### 1. 외곽 연구소 = *왕실 소속 한직 도서관*

- **카르엔 외곽 연구소**: 왕실 직속 명목상 소속이나 *정치색 거의 없는 *도서관 같은 한직***. 영향력 매우 낮음 → 외부 정치 견제 부재 → *비정규 계약직 등록 정치 문제 낮음*
- **카엘 직급**: ~~왕실 마법 고문~~ **외곽 연구소 *비정규 계약직 연구원*** — *공식 직위는 거의 무명*. 5년 동안 *내부 도서관 출입 + 보조 연구 + 본인 연구 시간 자유*
- **5년 동안의 외부 노출**: 거의 없음. 카엘은 *외곽 연구소 안에 처박힘*. 셀라리아 도시 전체에서 *모르는 존재*

### 2. 카엘이 연구소 들어가고 싶은 *니즈* (개연성 확보)

본 Part 진입 시 카엘의 동기 — *돈·기술·힘*:

- **돈**: 1.5부 떠돌이 짠내 + 부상 누적 → *안정된 거처 + 식사 보장* 절실. 외곽 연구소 *비정규 계약직 = 평민 5인 가족 식비 + 연구실 침대 + 도서관 접근*
- **기술**: 1.5-4 마린의 *3단계 정통 견식* + 1.5-7 델란의 *너는 *다른 무엇* 이다* 한마디로 카엘이 *정통 마법학 본격 학습* 필요성 자각
- **힘**: *추방의 트라우마 + 떠돌이의 무력함* 의 보상 — *강해지면 다시는 추방당하지 않는다* 의 무의식. **앤트로퍼의 *살인 갈구* 가 *학문적 갈구* 로 가장된 형태**

→ 본 Part 카엘 본인 의식은 *학문적 호기심 + 안정 거처* 만. 진짜 동기 *힘에의 갈구* 는 *후일 회수*

### 3. 5년 → 5써클 도달의 *개연성* (사기 수준 인정 + 풀이)

#### 본작 캐논 — 사기 수준 인정

- 정통 마법사 5년에 5써클 = *극히 드문 천재*
- 앤트로퍼 이득 (마나 효율 ↑, 기술 탐구 가속) 고려해도 *여전히 사기 수준*
- 본 Part는 *이 사기성을 풀어내야 함*

#### 4가지 개연성 풀이

1. **카르엔 사이엔 = 옛 7써클 마법사**:
   - 30대에 7써클 도달. 헤븐즈 영웅 시대 후기 최정상급
   - 50대 중반 *마법 사고* 로 *현역 시전 능력 대부분 상실* (시전 가능한 위력 4써클 미만). 그러나 *마법학 이론 조예 + 학습 지도 능력은 그대로*
   - **카르엔 = 본작 *최고급 마법학 지도자***. 카엘에게 *7써클까지의 정통 학습 로드맵* 직접 전수
2. **외곽 연구소의 *풍족한 마법학 도서 + 자료 정리***:
   - 도서관 한직이라 *외부 출판 → 도서관 자료 정리 흐름* 이 지속됨
   - 카엘이 *5년간 도서관 자료 거의 전부 정독* — 정통 5단계 마법학 + 데미안 마도공학 기초 + 마녀령 흑마법 일부 + 일레인 응용 마법 + 사프란 학파 일부 (카르엔이 *조각으로 보여줌*)
3. **카르엔의 *수단 방법 가리지 않는 빠른 성장 연구***:
   - 카르엔은 *옛 야망 — 왕실 권위 회복* 을 위해 *카엘의 *루키 사례* 가 필요*. 자기 마법체계의 우수성 증명 도구
   - *가장 빠른 성장 경로 + 위험한 학습 방법* 도 카엘에게 *기꺼이 시킴* (예: 금기 분야 *엔트로피 역연산* 의 *부작용 알면서 시도 권장*)
   - 카엘은 *왜 카르엔이 자기에게 *이렇게까지* 가르치는가* 의문. *비뚤린 시야로 자기 천재성의 인정* 으로 오독
4. **앤트로퍼 이득 + 자기 직관 1단계 시전 정통 융합**:
   - 1.5부에서 형성한 *1단계 직관 시전* 이 정통 학습 후에도 *깊이 살아남음*
   - 5써클 *공식 등급* + *자기 직관 융합 → 실제 위력 9써클급* 발휘 가능 (앤트로퍼 보너스)
   - 즉 *써클 등급은 인간 편의의 척도* (magic-system §6) 대로 — 카엘 = *5써클 등록증 + 9써클급 위력* 의 *기형적 천재*

#### 본 Part 종결 시 카엘 상태

- *공식 등급 5써클* (외곽 연구소 인증)
- *실제 시전 위력 9써클급* (앤트로퍼 + 직관 융합 — 본인은 *위력 측정 안 해봐서 모름*)
- *광역 학살 마법 가능 — 단 본인 미체험* (5년간 *실전 0회*. 연구소 안 *시연 + 도서관 학습* 만)
- **후일 Part 4 첫 광역 학살 (4-1 세번 강 1,400명) 시 카엘 본인도 *내가 이정도였나* 충격 받음**

### 4. 카르엔 사이엔 — *옛 야망의 한직 학자*

#### 카르엔의 4중 정체

1. **옛 7써클 영광**: 30~50대 헤븐즈 영웅 시대 후기 최정상. 칠흑성 전투 시기 (CC 870) 라카르 9세의 *전 왕대 군 자문 마법사* 역임
2. **마법 사고 + 한직 좌천**: 50대 중반 *시전 사고* (정확한 정황 미상) 로 시전 능력 대부분 상실. 외곽 연구소 *한직* 으로 좌천. 본인은 *명목상 *학문 연구 시간*** 으로 받아들이나 *실은 자존심 깊이 상처*
3. **카르엔 학파 잠적 분파의 8대 계승자**: 학파 비전 보유. 본 Part 이전 캐논 유지. 카엘의 *역마나 감각* 을 알아본 진짜 이유
4. **야망 — 왕실 권위 회복**:
   - 카르엔의 진짜 목표 = *외곽 연구소를 왕실 직속 권력축으로 복권*
   - 그러려면 *세계가 인정할 *루키 마법사*** 가 필요
   - 카엘 = *맞춤형 도구*. 강 발현 엔트로퍼의 *비정상적 성장 속도* + *카르엔 자기 마법체계의 산물* 로 외부 포장 가능
   - *1회성 카드* — 카엘이 *전쟁 + 권위 회복* 의 도구 역할 후 *어떻게 되든 관계 없음*

#### 카르엔의 카엘 간파

- 카르엔은 *카엘의 비틀린 시야* 를 *3년차쯤 간파*
- 그러나 *기본 전제 — 자신이 믿는 *정의의 신념* 으로 움직임* 도 간파
- **결론 — 카엘은 *정의로운 도구로 활용 가능*** = 카르엔 계획의 핵심
- 카엘의 *기본 정의감* 을 *카르엔 자신이 원하는 방향* 으로 유도하면 *완벽한 1회성 무기*

#### 카르엔의 *깊은 한 줄*

> *"세상에는 두 종류의 차가 있네. 마시는 차와… 마시지 않는 차."* (2-2 첫 사사 시. 표면 의미 — 신뢰의 비유. 진짜 의미 — *내가 *너에게 무엇을 마시게 할 것인가* 의 *선택권은 나에게* 있다*)

### 5. 카엘의 *외로움* (잔향 → 사회적 고립)

- 외곽 연구소 5년 거의 *혼자만의 시간*
- *식당 가지 않음*. **방에서 식사**. 방값과 함께 *연구실 침대로 식사 운반* 옵션을 카르엔이 *조용히 허락*
- 동료 연구원과의 교류 *거의 없음*. *복도에서 마주칠 때만 짧은 인사*. 카르엔만 *예외적으로 *정기 사사*** 받는 유일한 사람
- 외곽 연구소 내부에서도 *카엘에게 잔향 작동* — 동료들이 *왜인지 모르게 카엘 옆 자리 비움*
- **2-5 미라 베스턴 사건**의 진정한 정통:
  - 미라가 *카엘의 외로움을 동정* 해 *어깨에 손* 을 얹음 (잔향 거리감 *극복한 *드문 호의***)
  - 카엘이 *정보 절취 시도* 로 해석 → 손목 비틀어 골절
  - 미라의 *침묵 사과 후 사직* — 정통은 *두려움 + 카엘에 대한 *마지막 호의의 흔적***
  - 이 사건이 *외곽 연구소 내부에서도 카엘이 *공식 위험 인물*** 로 분류되는 분기점
- 본 Part 카엘의 *외로움 신체화* — *입을 다물고 코로 호흡* (1.5-1 글라렌의 충고 잔향) + *방 안에서 혼자 식사하는 모습* + *밤에 도서관 혼자 정독*

### 6. 카르엔의 *우발적 죽음* — 의도적 오해 스토리 ★

#### 사건 정황 (2-6)

본 Part 클라이맥스. 카엘이 *직접 죽이려 한 것은 아니나 *책임이 있는** 마나 폭발 사고.

**사건 흐름**:

1. 카르엔의 *왕실 권위 회복 계획* 절정 직전. 카엘에게 *왕실 시연회 출석 + 자기 마법체계 공개 시연* 강요
2. 카르엔이 *마지막 점검 — 카엘의 *충성도 검증*** 위해 *너무 명백한 도구화 발언* — *"자네는 *내가 만든* 마법사일세. 내일의 시연은 *나의 영광이지 자네의 영광이 아니야*."*
3. 카엘이 *비뚤린 시야에도 *불구하고*** 처음으로 *카르엔에 대한 분노 폭발* — *자기를 평생 사랑한다고 느꼈던 자가 *실은 자기를 *물건 으로 봤다***
4. 분노가 *마나 회로 폭주* 로 외화. 연구실의 *마나 도구 + 책장* 이 *동시 폭발* — 카엘 본인도 *어떻게 했는지 모름*
5. 카르엔이 *폭발 직격*. *연기 흡입 + 광역 화상*. 카엘이 *불을 끄려 마나로 잡아 보려* 시도하나 *비뚤린 시야에서 자기 마나 통제 불능*
6. 카르엔 사망. 외곽 연구소 *부분 화재 진압 후*

**객관 사실**:
- 카엘이 *직접 살의로 죽이려 한 것 아님*
- 그러나 *카엘의 마나 폭발이 직접 사인*
- 카엘 본인 *책임이 있음* — 정상적 시야였다면 *대화로 해결* 가능했을 일

**외부 인식**:
- *카엘이 카르엔을 의도적으로 살해한 것처럼 보이는 정황*
- 마나 도구 폭발 = *고의적 시전 결과처럼* 외부에 보임
- 외곽 연구소 동료들이 *카엘이 사고 직전 카르엔과 격렬한 언쟁* 을 목격
- 라이세린이 *이 정황을 빠르게 파악* — 카엘의 *약점* 으로 *무기화*

#### 정통적 진실 — *카엘이 그 정도로 비틀린 인간은 아님* (사용자 HARD)

- 본 사건은 **카엘이 *완전한 살인자가 아닌* 증거**
- *엔트로퍼 마기로 인한 분노 폭발* 의 결과이지 *카엘의 본성이 아님*
- *정상적인 시야였다면* 카엘은 *카르엔의 도구화 발언을 듣고 *대화·이별·도주* 의 길* 을 택했을 것
- 즉 *카엘의 정의로운 성격은 마기 너머에 살아있음* — 4-5 자각·6부 회복의 정통 출처

#### 미완성 경고 서한의 의미

- 카르엔의 미완성 서한 — *"네가 보는 세계가 세계의 전부가 아니다…"*

- 카엘이 사고 직후 발견 → *비겁한 모욕* 으로 오독 → 소각
- 4-5 자각 시 *기억 표면 회수*

#### 카엘의 *내적 갈등* (본 Part 종결)

- *내가 그를 죽였나 / 아닌가* 의 확신 못함. *영구 모호*
- *카르엔이 나를 *사랑했다는 환상* 이 깨진 상실* 의 더 큰 무게
- 외곽 연구소 떠나며 *방향 없음*. 라이세린이 *그 공허를 채우러* 다가옴 (2-4 만남의 정치적 의미가 본 Part 종결에서 *새 의미로 활성화*)

## 7. 정치 개입 인과 자연성 (검토 3 답변)

본 Part 의 정치 흐름이 *부자연스럽지 않게* 진행되도록 인과 명시:

### 자연스러운 정치 인과 체인

1. **2-1~2-5 — 카엘 외부 노출 0**: 외곽 연구소 비정규 계약직 + 카엘 내부 처박힘. 외부 정치 그물 안 진입 안 함
2. **2-6 — 카르엔의 부풀림 시작**: 카르엔이 *5써클 인증 의식* 을 *외부 학자 일부 초청* 으로 거행. 카엘이 *5써클 인증* 으로 *공식 마법사 명부* 에 등록됨
3. **2-6 — 라이세린 우연 발견**: 라이세린이 *공식 마법사 명부 정기 검토* 중 *카엘의 5써클 인증* 발견. *변경 출신 + 5년 만에 5써클* 의 *비정상적 기록* 에 주목. 외곽 연구소 방문 시도
4. **2-6 — 라이세린 첫 면담**: 외곽 연구소로 라이세린이 직접 옴. 카르엔 + 카엘과 함께 차 자리. *자네 같은 사람을 기다렸다* 한마디. **이 시점 라이세린은 *카엘 흡수 계획만 있고 사망 정황 모름***
5. **2-7 — 미라 사건**: 외곽 연구소 *내부 위험 인물 분류*. 라이세린이 *이 정황도 알게 됨* — *카엘의 통제 가능성 + 약점* 동시 인지
6. **2-8 — 카르엔 사망 + 라이세린 즉시 흡수**: 카르엔의 *왕실 시연회 직전 사망*. 외곽 연구소 *권력 진공 + 카엘 책임 의혹*. 라이세린이 *수습 명목으로 카엘을 외교실로 데려옴* — *왕실 명목상 *위험 인물 격리* + 라이세린의 *실질 활용***. **자연스러운 정치 흡수**

### 정치 인과의 자연스러움 점검

- **카엘이 외교실에 들어가는 *공식 명분* 이 있는가?** — *카르엔 사망 정황 수습 + 위험 인물의 안전 관리* = 명목 충분
- **라이세린이 카엘을 *공작 권한으로* 흡수 가능한가?** — 외교실 차관 권한 + 라카르 9세의 *외교실 자문 임명* 위임 권한
- **외곽 연구소가 *반발하지 않는가?*** — 한직 도서관. 카르엔 사망 후 *지도자 부재*. 라이세린의 흡수에 반발할 *권력 없음*
- **카엘 자신이 *반발하지 않는가?*** — 카르엔 사망 죄책감 + *내가 무엇인지 모르는 공허* + 라이세린의 *내가 너의 약점을 안다* 의 무언 압박

→ **모든 인과가 *자연스럽게 연결*** — 카르엔 야망의 부풀림이 *우연히* 라이세린의 손에 들어감
