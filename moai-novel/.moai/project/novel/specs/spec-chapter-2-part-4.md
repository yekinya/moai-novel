## Summary

Ch2 P3 종결 (강철제국 본가 도착 직전) 직후 — *카엘이 본가 진입 + 본격 작업 시작*. **강화 대포 강철 의수 제작** (왼손의 *비밀 대포 + 강화강철 격상 + 룬 결각 작업*) + **룬 결각 + 아티팩트화** (마녀령 비전 지식 활용) + **팔 전체에 검기를 두르는 방식 개발**. 카엘의 *마력→내공 + 공허검 호흡 + 의수 마나 감응의 *3중 융합* 으로 *팔 전체 검기 방식의 *기본 작동 확인*. 단 *효과 약함 — 검기 두께가 *일반 검의 70%* 만 형성*. *원인 추적 → 산속 도장에서 *세이른 스틸만 직접 직면* + **마법력 부족 진단**. *카엘의 마법 자체는 봉인되지 않았으나 *체외 출수 통로가 0 + 강화강철의 마나 전도율 한계 + 카엘의 마법력 총량 부족이 합산되어 *효과 약함 결과*. *세이른의 결단 — 너의 길은 *마법력의 *총량을 높이거나 *전도율을 *극단까지 끌어올리는 것이다*. 그를 위해 *앨더스톰의 룬 마법사 + 드워프 지하세계의 아다만티움·미스릴 의수가 필요. 본 Part 종결 시점 = *카엘의 앨더스톰 이동 결단 + 세이른의 짧은 작별*.

## 핵심 비트

### 1. 강화 대포 강철 의수 제작 (4-1·4-2)

**왼손 의수 격상**:
- *Ch2 P3 3-3 강화강철 의수 격상 이후 *왼손 = 강화강철 본체 + 강철 대포 부분*. 본 Part 4-1 에서 *대포 부분도 강화강철로 재제작 + 마력 회로 미세 결각 추가*
- *대포 무게 감소 + 발포 정확도 추가 향상 + 마력 운용 1회 추가 가능 (충전 시간 단축 — 24시간 → 12시간)*

**룬 결각 1차 작업 (4-2)**:
- *마녀령 비전 룬 지식 활용* + 강철제국 본가 룬 장인 1인 (작명 후속 — 강철제국 + 마법 거부 정통 내에서도 *룬 결각 자체는 마법 거부 아닌 검의 정통* 으로 인정) 의 보조 매개
- *룬 = 의수 표면 미세 새김 + 마나 전도 향상 + 일부 봉인 회로*
- *1차 작업 룬 = 3~5종 부분 결각*. *완전 결각은 *아다만티움/미스릴 단계에서 본격*

### 2. 팔 전체 검기 두르는 방식 개발 (4-3·4-4)

**3중 융합 시도**:
- *마력→내공 (체내 마나 변환 + 4중 회로)* + *공허검 호흡 (3박자)* + *의수 마나 감응 (강화강철 + 룬 결각)*
- *팔 전체 = 의수 + 어깨·가슴·등의 *남은 체내 마나 회로 + 공허 호흡으로 *팔 전체에 검기 막을 두르는 방식*

**반복 실패 곡선**:
- *처음에는 *기본 작동도 안 됨* — *마나가 *손가락 끝에 머무를 뿐 *팔 전체로 흐르지 않음*
- *카엘의 자기 분석 — *공허 호흡과 내공 호흡의 *일치 조건 + 의수 룬 결각의 *마나 흐름 매핑*
- *수십 회 반복 후 *기본 작동 확인* (4-4)

**70% 한계 자각**:
- *팔 전체 검기는 *기본 작동* 하나 *두께가 *일반 검의 70%* — *검기 면적은 *팔 전체이나 *밀도는 약함*
- 의수 위력 = *물리 위력 + 약한 검기 = 일반 검사의 1.2배 정도*. *소드 마스터·강력한 마검사 상대로 *결정타 불가*
- *카엘의 *깊은 답답함* — *길은 보이는데 *효율이 *근본적으로 부족하다*

### 3. 세이른 스틸만 직접 직면 (4-5)

(Ch1 시드의 *Ch2 본격 회수*)

**산속 도장 진입**:
- 강철제국 본가 → 산속 도장 *도보 3일* (지역 거주 정보는 본가의 *극히 일부에게만 공개*)
- 도장 = *작은 산속 가옥 + 공허검 5인 마스터 중 가장 젊은 세이른의 거처*

**세이른의 첫 인지**:
- *카엘의 정체 + 옛 동문 추천장 + 마녀령 사사 정통 인지*
- 세이른의 한 마디 — *"마녀령에서 *공허를 *3분의 2 봤군. 나머지는 *너의 시야가 *허락하지 않은 것이지."*
- 카엘 *짧은 묵례 — 그렇습니다*

### 4. 마법력 부족 진단 (4-6) ★

**세이른의 진단**:
- *카엘의 *팔 전체 검기 시연 직접 관찰* — *짧은 침묵 후 결단*
- **원인 3중 진단**:
  1. *체외 출수 통로 0* (두 팔 상실의 결과 — 회복 불가)
  2. *강화강철의 마나 전도율 한계* (의수 단계의 결과 — 재료 진화로 해결 가능)
  3. *카엘의 *마법력 총량 자체가 부족* — *체내 마나량은 5써클 + 4중 효율 1.5배 (Ch1 정통)이지만, *내공 변환 + 공허 호흡 + 의수 감응의 *3중 손실로 *실효 마법력이 *3써클 수준에 머무름*

**해결 두 길**:
- *길 1 — 마법력 총량 향상*: *체내 마나량 자체를 *2배 이상 늘림. 그러나 카엘의 정통상 어려움 — *체외 마법 시전이 봉인된 상태에서 마나량 증대는 *반복 명상·내공 훈련의 *10년 단위 작업*
- *길 2 — 전도율 극단 향상*: *의수 재료를 *아다만티움 + 미스릴로 격상*. *마나 전도율 일반 강철의 3~5배*. *카엘의 마법력 총량은 그대로지만 *효율이 2~3배 증가하여 *팔 전체 검기 두께가 *일반 검의 100~120%* 까지 가능

**세이른의 결단**:
- *너의 길은 *길 2 다. 길 1은 *시간이 *너에게 *허락하지 않을 것이다*
- *아다만티움·미스릴은 *돈으로 살 수 없다. *드워프 지하세계가 *유일한 출처*
- *앨더스톰의 룬 마법사 1인이 *드워프 부족의 *비공식 거래 매개로 *유일하게 연결된다*

### 5. 작별의 모루 (4-7)

**세이른의 짧은 작별**:
- *추천장 1장 양도* — *앨더스톰의 룬 마법사 *마르덴 라온* (작명 후속)에게 *카엘 진입 매개*
- 한 마디 — *"공허는 *너의 길의 *전반부다. 나머지 *반은 *너의 *팔의 무게에 달렸다. *살아 돌아오면 *다시 본다."*

**카엘의 결단**:
- *앨더스톰 이동 결단* — *마법력 진단 + 아다만티움/미스릴 의수 필요성 확인 후 *드워프 지하세계 진입의 매개*
- *왼손 강화 대포 강철 의수 + 1차 룬 결각 + 팔 전체 검기 기본 작동 + 공허검 2원리 + 마력→내공 4중 회로 일부 = 본 Part 종결 시점 자산*

**종결**:
- 산속 도장 → 강철제국 본가 복귀 → 앨더스톰 출발 (도보·말 약 *3~4주*)

## 회차 비트

### 4-1 — 본가의 모루

- **공간**: 강철제국 본가 (오르난) 도장 + 룬 작업장
- **시점**: 카엘 1인칭 (조용한 노동)
- **비트**:
  - 본가 도착 + 본격 작업 시작
  - **왼손 강화 대포 강철 의수 작업** — 대포 부분 강화강철 격상 + 마력 회로 미세 결각 추가
  - 무게 감소 + 발포 정확도 향상 + 충전 시간 24시간 → 12시간 단축
  - 본가의 *나이 많은 검사 + 그라엔 호란 (룬 장인) 직접 보조*

### 4-2 — 룬의 첫 결각

- **공간**: 그라엔 호란의 작업장
- **시점**: 카엘 1인칭
- **비트**:
  - **마녀령 비전 룬 지식 활용 + 의수 표면 룬 결각 1차 작업**
  - *3~5종 부분 결각 + 마나 전도 향상*
  - 그라엔 호란의 한 마디 — *너의 룬은 *마녀령 정통과 강철 정통이 *섞였다. 깨끗하지 않다. 그러나 *그게 너의 길이다*

### 4-3 — 팔의 길

- **공간**: 본가 외곽 훈련장
- **시점**: 카엘 1인칭 (반복 실패)
- **비트**:
  - *팔 전체에 검기를 두르는 방식 개발 시도*
  - *마력→내공 + 공허검 호흡 + 의수 마나 감응 3중 융합 시도*
  - 반복 실패 + 카엘의 자기 분석 — *공허 호흡과 내공 호흡의 일치 조건 + 의수 룬 결각의 마나 흐름 매핑*

### 4-4 — 두께의 70%

- **공간**: 본가 외곽 훈련장
- **시점**: 카엘 1인칭 (짧은 명료성 + 깊은 답답함)
- **비트**:
  - 수십 회 반복 후 *팔 전체 검기 *기본 작동 확인*
  - *검기 두께가 일반 검의 70%* 한계
  - *의수 위력 = 물리 위력 + 약한 검기 = 일반 검사의 1.2배*. *소드 마스터·강력한 마검사 상대로 결정타 불가*
  - 카엘의 깊은 답답함 — *길은 보이는데 효율이 근본적으로 부족하다*

### 4-5 — 산속 도장

- **공간**: 오르난 → 세이른 스틸만의 산속 도장 (도보 3일)
- **시점**: 카엘 1인칭 (긴 침묵 + 직면)
- **비트**:
  - 산속 도장 진입 — 작은 산속 가옥 + 공허검 5인 마스터 중 가장 젊은 세이른의 거처
  - **세이른 스틸만 직접 직면** (Ch1 시드의 본격 회수)
  - 세이른의 한 마디 — *"마녀령에서 *공허를 *3분의 2 봤군. 나머지는 *너의 시야가 *허락하지 않은 것이지."*
  - 카엘 *짧은 묵례 — 그렇습니다*

### 4-6 — 마법력의 진단

- **공간**: 세이른의 도장 — 시연장
- **시점**: 카엘 1인칭 (진단의 무게 + 결단)
- **비트**:
  - 세이른이 카엘의 *팔 전체 검기 시연 직접 관찰* + 짧은 침묵 후 결단
  - **3중 진단**:
    1. *체외 출수 통로 0* (두 팔 상실 결과 — 회복 불가)
    2. *강화강철 마나 전도율 한계* (의수 단계 결과 — 재료 진화로 해결 가능)
    3. *카엘의 마법력 총량 부족* (체내 마나 5써클 + 4중 효율 1.5배지만 *내공 변환 + 공허 호흡 + 의수 감응의 3중 손실로 실효 마법력 3써클 수준*)
  - 세이른의 결단:
    - *너의 길은 *길 2다. 길 1은 *시간이 *너에게 *허락하지 않을 것이다*
    - *아다만티움·미스릴은 *돈으로 살 수 없다. 드워프 지하세계가 유일한 출처*
    - *앨더스톰의 룬 마법사 1인이 드워프 부족의 비공식 거래 매개로 유일하게 연결*

### 4-7 — 작별의 모루

- **공간**: 세이른 도장 → 강철제국 본가 복귀 직전
- **시점**: 카엘 1인칭 (조용한 결단)
- **비트**:
  - **세이른의 짧은 작별 + 앨더스톰 추천장 1장 양도** — 마르덴 라온 (앨더스톰 룬 마법사 대원로) 매개
  - 한 마디 — *"공허는 *너의 길의 *전반부다. 나머지 *반은 *너의 *팔의 무게에 달렸다. *살아 돌아오면 *다시 본다."*
  - 카엘의 *앨더스톰 이동 결단*. 본가 복귀 후 *앨더스톰 출발 직전 종결*
- **단서**: *팔찌 조용 — 본성의 결단 안정*

## 종결 후크

> 강철제국 본가의 *낯익은 굴뚝이 *카엘의 등 뒤로 천천히 멀어진다*. *팔 전체 검기를 한 번 짧게 두른다 — *얇은 막이 *팔을 감싸지만 *밀도는 분명 부족하다*.
>
> 카엘의 내적 한 줄 — *…전도율을 *극단까지 끌어올리려면 *돈으로 살 수 없는 금속이 필요하다. *앨더스톰으로 간다*.
>
> 멀리 — *동쪽 산맥 너머의 *앨더스톰이 *룬으로 빛나는 흰 첨탑을 *늘어놓고 있다*.
