# Chapter 2 Part 4 Spec: 그림자의 후계

> Chapter 2 overview: [`./spec-chapter-2.md`](./spec-chapter-2.md)

## Summary

라이세린 잠적 후 권력 진공을 채우려는 *3왕자 하르엔 + 청혈 가문 후원* 의 음모 폭로. 헤븐즈 군부 *하르덴 에프란 백작* 의 비밀 사절 + 라카르 9세 비공개 답신. 데미안 카루린 황후의 *베르카르 황자에게 밀약 폭로*.

## 회차 구성 (9회차 — 제안)

| 회차 | 제목 (가제) | 비트 |
|------|-----------|------|
| 2-4-1 | 비밀 사절 | 하르덴 에프란 백작이 *반라이세린 라인* 으로 비밀 사절을 카엘에게 보냄 |
| 2-4-2 | 청혈의 그림자 | 하르엔 왕자 + 청혈 가문 *비밀 후원* 정황 추적 |
| 2-4-3 | 왕녀의 검 | 카리엔 왕녀의 *반(反)하르엔 라인* + 카엘에게 *간접 협력 의향* |
| 2-4-4 | 셀라리아 잠입 | 카엘이 *간접 사신* 으로 셀라리아 진입 (변장). *5년 만의 귀환* |
| 2-4-5 | 왕의 침묵 | 라카르 9세 *비공개 친서*. *짐도 자네를 더 이상 도구로 보지 않네* |
| 2-4-6 | 외척의 균열 | 데미안 카루린 황후의 *베르카르 황자 폭로* — 라이세린 밀약 잔재 |
| 2-4-7 | 백색 사냥단의 추격 | 카엘의 *그라스 의수 시술자* 가 백색 사냥단 표적화 되었다는 풍문 |
| 2-4-8 | 라이세린의 잠적지 | 카엘이 라이세린 위치 확인 — *데미안 황실 변경 별궁*. 직접 대결 결정 *보류* |
| 2-4-9 | 음모의 골격 | 본 음모의 *큰 그림 정리* — 데미안 + 하르엔 + 청혈 라인 vs 카리엔 + 라카르 + 하르덴 라인 |

## 핵심 비트

1. **라이세린 잠적 후 권력 진공** — 본작 *정치 게임* 본격 재개
2. **하르엔 왕자의 *마녀혈통 의심*** — 청혈 가문 사생녀 측실 셀라엔 소생. 본 Part에서 *진실 확정*
3. **카리엔 왕녀의 등장** — 1왕녀 31세, 8써클 마법사. 차기 왕위 1순위 + 하르엔 정적
4. **라카르 9세의 *왕의 두려움 한 줄*** — *짐의 적이 되지 않기만 비네*. 본작 *왕도 인간이다* 의 첫 직접 노출
5. **데미안 카루린의 *밀약 폭로*** — 본작 *최강 권력자 그룹의 첫 균열*
6. **백색 사냥단의 *카엘의 *지인* 표적화*** — 카엘 본인이 아닌 *그의 그라스 시술자* 표적. 책임의 무게

- 유머 *낮음* — 단 세파의 *변장 카엘에 대한 비웃음* 1~2회

## ★ Review points

1. **2-4-4 *셀라리아 잠입 변장*** — 변장 정도? 어떤 외양으로? (현 안: 검은 후드 망토 + 의수 가림). 카엘 *외모 풍문* 이 *5년 후에도 살아있음* 디테일?
2. **2-4-5 라카르 9세 친서의 *진심도*** — *왕의 두려움* (현 안) vs *왕의 공식 해방* (외교적 거리만) — 어느 톤?
3. **2-4-6 카루린의 *폭로 동기*** — Chapter 2 spec 검토 5에서 이미 제기됨. *베르스 8세와의 정치적 불화* / *외척 자기 안전 도모* / *카엘 동정* — 어느 것?
4. **하르엔의 *직접 등장* vs *간접 그림자*** — 본 Part에서 *직접 대면* (1회) vs *간접만* (다음 Chapter 본격 등장)?
5. **9회차 분량** — 본 Part가 *Chapter 2 중 가장 김*. 7회차로 압축 가능?
6. **미라 베스턴 *동부국 정보원* 풍문 단편** — 본 Part에 1줄 등장 시드 vs Chapter 3로 미룸?
