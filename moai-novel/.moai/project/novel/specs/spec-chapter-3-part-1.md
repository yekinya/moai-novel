# Chapter 3 Part 1 Spec: 강철의 손

> Chapter 3 overview: [`./spec-chapter-3.md`](./spec-chapter-3.md)

## Summary

의수 3단계 (왼팔 강철대포 확장) + 마법검사 진화 본격. 자체 아티팩트 1호 — *몸의 마나 측정 결정* 설계. 키메라 정수 벨트 도입.

## 회차 구성 (7회차 — 제안)

| 회차 | 제목 (가제) | 비트 |
|------|-----------|------|
| 3-1-1 | 그라스의 재방문 | 노 장인 + 데미안 마도공학 장인 2차 합작. 강철대포 의수 *왼팔 확장* 시작 |
| 3-1-2 | 대포의 시연 | 강철대포 첫 시연. 5~30m 사격. 마나 응축 탄환. 1분당 3발 |
| 3-1-3 | 자체 아티팩트 1호 | 카엘이 *몸의 마나 측정 결정* 설계 — 셀라엔과의 의식 매개 보조 |
| 3-1-4 | 키메라 정수 벨트 | 마계 진주 1개 가공 (베르난 호산 협력). 마나 소모 보조 도구 |
| 3-1-5 | 마법검사 진화 | 카엘이 *팔에서 검까지 검기 흐르는 정통* 첫 형성. 검과 의수가 *한 몸* |
| 3-1-6 | 키얀 일행의 도착 | 키얀 추적 일행이 그라스 외곽 도달. *3일 거리 → 도착 임박* |
| 3-1-7 | 첫 동맹 회의 | 카엘 + 세파 + 카이엔 + (간접) 셀라엔 + 모르간 솔테의 *원격 회의*. 키메라 위협 + 마왕 카운트다운 |

## 핵심 비트

1. **의수 3단계 — 강철대포** — 왼팔 *전완에 마나 응축 포구 내장*. 데미안 화기보다 우수
2. **양팔 비대칭 운용** — 왼팔 공격 / 오른팔 방어·검 운용
3. **자체 아티팩트 1호** — 카엘이 *제작자* 가 된 첫 시점. 본작 후반부 *자체 아티팩트 12점* 라인의 시작
4. **키메라 정수 벨트** — 마계 진주의 *합법 활용*. 금혈 협력의 가시화
5. **마법검사 정통 형성** — *팔이 곧 검* 의 첫 시연
6. **키얀 도착 임박** — Part 3에서 결투

## 인물

- 카엘 / 세파 / 카이엔 / 그라스 노 장인 / 데미안 마도공학 장인 / 베르난 호산 (금혈, 간접 협력) / 키얀 일행 (4인, 도착 임박)

- *성장의 카타르시스* — 본작 *재기 의 정점* 의 첫 신호
- 짠내 *낮음* — 의수 시술 고통 묘사 1~2회
- 유머 *증가* — 그라스 노 장인의 *깐깐한 농담* + 데미안 장인의 *상인 농담*

## ★ Review points

1. **3-1-2 강철대포 시연** — *어떤 표적* 에 시연? *허수아비* (안전) vs *변경 도적 1인* (실전) vs *카이엔* (서로 시연)?
2. **3-1-3 자체 아티팩트 1호 *외형*** — 결정체 + 끈 (몸에 부착)? 가슴 펜던트? 의수 통합?
