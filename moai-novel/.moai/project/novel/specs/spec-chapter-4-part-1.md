## Summary

드워프 살라멘티 종파 산악 본거 방문 — 사피엔 사회 *천 년 만의 외교 사절*. 살라멘티의 축복 의식 통과 + 아다만티움 의수 (4단계) 완성.

## 회차 구성 (7회차)

| 회차 | 비트 |
|------|------|
| 4-1-1 | 강철제국 북부 너머 *빙벽 협곡 너머* 드워프 산악 도착. 입구의 *침묵의 경계* |
| 4-1-2 | *모루의 왕 그라하르* 와의 면담. 카엘의 *붉은 사슬을 본태 시야로 인지* — 첫 드워프 *공식 식별* |
| 4-1-3 | 드워프의 *조건* — *살라멘티의 축복 의식* 통과 시 의수 제작. 미통과 시 *영구 거절* |
| 4-1-4 | 축복 의식 *준비* — 잉그리드 사원 몽크의 *돌화 비전* 차용. 카엘 의수의 *부분 돌화 시험* |
| 4-1-5 | 축복 의식 — 오른팔 의수 일부 *임시 돌화* 통과 |
| 4-1-6 | 의수 4단계 *아다만티움 제작*. 1주일 시술 |
| 4-1-7 | 의수 4단계 완성 + 시연. 드워프와의 *비공식 동맹* + 모루의 산 *영구 방문 권한* |

## 핵심 비트

1. **드워프 외교 사절** — 본작 *사피엔 천 년 만의 첫 공식 접촉*
2. **살라멘티의 축복** — 잉그리드 몽크 비전 + 드워프 신앙 결합
3. **의수 4단계 완성** — 아다만티움 + 양팔 비대칭 + 마운트 인터페이스 + 비상 상비 주사 + 소형 방어 키트
4. **드워프와의 비공식 동맹** — Chapter 4 후반 *기계화 사단 격돌* 시 *간접 협력* 가능성
