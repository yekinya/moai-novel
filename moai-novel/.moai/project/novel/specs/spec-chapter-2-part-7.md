## Summary

Ch2 P6 종결 직후 — *카엘 단독 출발 + 앨더스톰 룬 마법사 1~2명 비공식 동행 + 카르샤 망명지 협공 시드 활성화 (세파 + 키얀 + 잔존 호위 + 신성제국 형제·소드 마스터들의 *바이서스 외곽 동시 압박)*. 카엘이 *변경 침투 → 외곽 → 셀라리아 진입* — *마르덴이 양도한 5왕국 비공식 인맥 라인 + 바이서스 비공식 외교 기록 + 카엘 본인 셀라리아 5년 거주 정보의 *3중 매개*로 *왕궁 대전당 진입 가능*. **결전 — 라카르 9세 + 자식들 척살**. *왕궁 호위 + 라이세린 잔존 측 + 호스반의 *대응 (8써클 전쟁 참모) 과의 *광범위 결전*. *카엘의 *아다만티움 마력 대포 의수 + 공허검 2원리 + 마력→내공 + 룬 결각 + 마력포 무한 = *Ch1 5써클 시기 광역 학살의 *2~3배 위력*. *라카르 9세 척살 + 자식들 척살 + 라이세린·호스반 처분*. **그 자식들 중 일부 = 행커만 후예 (반인반룡 혈통)** — *카엘이 *Ch2 P6 마르덴 정보로 *알고 있으나 결단 변하지 않음*. 결전 종결 시점 = *카엘이 *왕좌의 방 한가운데에 *조용히 서 있다*. *드래곤 종족 적대 발화의 *시드 점화*. 본 Part 종결 = *카엘의 셀라리아 출발 직전 + Ch2 P8 카자르드 인간계 유랑 시드 진입 직전*.

## 핵심 비트

### 1. 결전 진입 (7-1·7-2·7-3·7-4)

**카르샤 망명지 협공 시드 활성화 (7-1)**:
- *세파 + 키얀 + 카르샤 잔존 호위 + 신성제국 형제·소드 마스터들이 *바이서스 외곽 동쪽 동시 압박*. *직접 침투 아닌 *외곽 정찰병 분산 매개*
- *카엘이 *셀라리아 직접 진입을 위한 *시선 분산 매개*

**외곽 침투 (7-2)**:
- *카엘 단독 + 앨더스톰 룬 마법사 1~2명 비공식 동행* (작명 후속)
- *변경 정찰병 5~10인 *소리 없이 격퇴* — *카엘의 *공허검 2원리 + 의수 검기 정통의 *완벽한 침투력*

**셀라리아 진입 + 외곽 연구소 잔존 협력 (7-3)**:
- *카엘이 *Ch1 P3 5년 학습한 *외곽 연구소 잔존 인물 1~2명 (카르엔 사이엔 사후 잠적 분파의 *조용한 잔존자)*과 *비공식 협력*
- *왕궁 대전당 진입 매개 — 외곽 연구소의 *마법학자 자격으로 입성 위장 + 마르덴 비공식 외교 기록 활용*

**왕궁 진입 + 팔 전체 검기 첫 본격 발현 (7-4)**:
- *왕궁 호위 일부 무력화 — 마르덴 양도 비공식 외교 기록 매개*
- **카엘의 *팔 전체 검기 첫 본격 발현*** — *아다만티움 의수 + 공허검 호흡 + 마력→내공 4중 회로 + 룬 결각 전체 = 100~120% 두께*. *왕궁 호위 마법사 + 기사 다수 *동시 무력화*

### 2. 자식 척살 시작 (7-5)

- *카엘이 *팔찌를 한 번 봄 — 붉은 빛 없음*. *결단 변하지 않음*

### 3. 호스반의 응답 (7-6) ★

**호스반 직면**:
- *Ch1 P4-P5 의 8써클 전쟁 참모. 카엘의 1회성 조커 카드 운용자*
- *호스반이 *카엘의 *변화한 정통 직접 확인 — 짧은 평가 한 줄 — "공허는 *어디서 배웠나."*

**호각 결투**:
- *호스반의 *광역 8써클 마법 vs 카엘의 *마검사 변형 + 의수 마력포 + 공허검 호흡 + 마력→내공*
- *호스반 = 광역 위력 압도적*. *카엘 = 호스반의 *광역 마법을 *공허검 1원리 (불교 — 받지 않음) 로 비키게 함*
- *호스반의 *결정적 시전 직전 — 카엘의 *바람의 길 응용 + 마력포 = 호스반 옆구리 직격*
- *호스반 척살*

### 4. 라이세린의 마지막 (7-7)

**라이세린 직면**:
- *라이세린 = 잠적 후 비공식 권력 라인 운영*. *바이서스 결전 시점에 *왕궁 비공개 거처에 *잠시 복귀해 있음* (마르덴 정보)
- *카엘의 *조용한 단문* — *"너의 *위조 첩보는 *우리 어머니의 *침묵보다 무거웠지."*

**라이세린의 짧은 응답**:
- *너는 *결국 *돌아왔구나*. *내가 *너를 키운 것이 *내 잘못이었다*
- *라이세린 척살 — 공허검 한 점 침*

**비공식 권력 라인 부분 해체**:
- *라이세린 잠적 라인의 *측근 다수 *그의 죽음과 동시에 잠적 또는 항복*
- *카엘의 *비공식 정보 라인 흡수 매개 활성화* (Ch3+ 매개 시드)

### 5. 옥좌의 피 (7-8) ★

**라카르 9세 5합 결투**:
- *Ch1 P5 4-4 알레이언 5합 결투의 *역방향 거울 정통*
- *알레이언 = 자기 희생으로 카엘 사슬에 균열을 냄*
- *라카르 9세 = 자기 보존 + 비명 + 도망치려 함*. *카엘의 *조용한 결단 — 도망쳐도 잡힌다*

**5합 흐름**:
1. 라카르 9세의 *근위 마법사 1인 보조 시전* — 카엘 공허검 1원리로 비킴
2. 라카르 9세의 *광역 결계 시전* — 카엘 의수 마력포로 결계 분쇄
3. 라카르 9세의 *도주 시도* — 카엘 바람의 길 응용 (Ch1 카자르드 용언 응용 시드)으로 추격
4. 라카르 9세의 *광역 1회성 조커 카드 시전 시도* — 카엘 공허검 2원리 (공출) 로 비움 + 빈 자리 침
5. 라카르 9세의 *마지막 한 마디 — 너는 *나의 *도구가 아니었다* + 카엘의 척살

### 6. 자식들의 끝 — 행커만 후예 척살 ★ (7-9)

- *3왕자 하르엔 척살 — 공허검 한 점 침*. *4공주 리에네 척살 — 의수 마력포 직격*. *2왕자 데이렌 척살 — 마검사 변형*

**행커만 후예 *처음 죽음* — 드래곤 종족 적대 발화 시드 점화**:
