## Summary

Ch2 P6 종결 직후 — *카엘 단독 출발 + 앨더스톰 룬 마법사 2인 (세렌 마르덴·베라트 알린) 비공식 동행 + 카르샤 망명지 협공 시드 활성화 (세파 + 키얀 + 잔존 호위 + 신성제국 형제·소드 마스터들의 바이서스 외곽 동시 압박)*. 카엘이 *변경 침투 직후 첫 자각 — 정면 돌파 불가*. 바이서스 = *9써클 1명·소드 엠퍼러급 1명·왕실 직속 소드 마스터 5인·정규 소드 마스터 100명+·12만 정규 검사·10만+ 마법사·길드 용병 5만 동원 가능* — *카엘이 아무리 강해졌어도 정면 직격 시 즉시 압도당함*. **결단 — 마르덴이 양도한 5왕국 비공식 인맥 라인 + 바이서스 비공식 외교 기록 + 카엘 본인 셀라리아 5년 거주 정보의 3중 매개로 *치밀하게 사전 조사한 정적·고수들의 위치 다수를 게릴라 척살하는 정통*. 셀라리아 외곽 비밀 거점 구축 + 분산 잠입 + *4~6주에 걸친 게릴라전*. 정적·고수 척살 = *바이서스 8써클 2~3명 중 호스반 + 정규 소드 마스터 약 8~12명 + 라이세린의 사적 권력 사슬 핵심 인물 다수 + 마법 학회 7써클 대마법사 일부*. **호스반의 죽음 + 라이세린의 죽음 + 라이세린 사적 이권 사슬의 즉시 부분 해체 + 바이서스 측 *공식 적 = 카엘 단독 자각 + 군사 비상령 검토 진입***. 본 Part 종결 시점 = *카엘이 셀라리아 외곽 비밀 거점에서 다음 단계 결단 + 바이서스 측 *군사 비상령 점화 직전*.

## 핵심 비트

1. **카르샤 망명지 협공 시드 활성화** (7-1) — *세파 + 키얀 + 카르샤 잔존 호위 + 신성제국 형제·소드 마스터들이 바이서스 외곽 동쪽 동시 압박*. *직접 침투 아닌 외곽 정찰병 분산 매개*. *카엘이 셀라리아 직접 진입을 위한 시선 분산 매개*. **새 정통**: *카르샤 측에는 *식솔 피난 임무로 잔존한 소드 엠퍼러급 1명·소드 마스터 2인이 *시선 분산 + 외곽 압박의 핵심 무력*. 직접 셀라리아 진입은 안 함

2. **카엘의 정면 돌파 불가 자각** (7-1) ★ — *변경 진입 직후 첫 정찰 결과 + 마르덴 양도 바이서스 군사력 기록 재확인*. 바이서스 = *9써클 1명·엠퍼러급 1명·왕실 마스터 5인·정규 마스터 100명+ + 12만 검사 + 10만+ 마법사 + 길드 5만 동원 가능*. *카엘이 아무리 강해졌어도 *왕궁 정면 진격 시 *왕실 호위 5인 마스터·엠퍼러급·9써클의 동시 직격에 즉시 압도*. **결단 — 정면 돌파는 자살. 게릴라·치밀 정찰·정적/고수 척살 중심으로 *바이서스의 무력 균형을 *내부에서 무너뜨림*

3. **치밀 사전 정찰 + 셀라리아 외곽 비밀 거점** (7-2) — *마르덴 양도 5왕국 비공식 인맥 라인 활용 + 카엘 본인 셀라리아 5년 거주 정보 + 외곽 연구소 잔존 인물 1~2명 (카르엔 사이엔 사후 잠적 분파의 조용한 잔존자)의 비공식 협력*. *외곽 산악 깊숙이 비밀 거점 구축 (안전 가옥 신부 라인의 *Ch1 P5 §10 안전 가옥과 다른 별개 거점*)*. *주요 정적·고수 위치 약 30~40명 분량 사전 조사 완료*. 세렌 마르덴·베라트 알린의 *룬 마법 원거리 시야 차단 + 위치 미세 추적*

4. **분산 잠입 + 외곽 연구소 잔존 협력** (7-3) — *카엘이 단독 잠입 + 룬 마법사 2인 외부 매개 지원*. *분산 잠입 = 1개 거점에 머무르지 않고 *야간 이동 + 변장 + 위장 출입증으로 *주요 거점·왕궁 외곽·정적 거처 다수 정찰*

5. **게릴라 — 첫 척살들** (7-4) ★ — 약 *4~6주에 걸친 다수 척살*:
   - **정규 소드 마스터 약 3~5명** = *야간 거처 잠입 + 잠자던 중 공허검 한 점 침으로 즉살*. *마스터가 정통 검술 1합 못 받아침* 캡 유지 — *카엘이 *비전·비대칭 매개로만 가능*
   - **7써클 대마법사 2명** = *시전 직전 마력포 직격 + 룬 마법사 2인의 *원거리 시야 차단 매개*
   - **라이세린 사적 사슬 핵심 인물 4~6명** = 외교실 비공개 거상 + 라이세린 후원 라인의 핵심 자금원·정보원. *그들의 죽음으로 라이세린의 사적 이권 사슬이 *부분 마비 시작*
   - **마법 학회 일부 7써클** = *카엘의 비전·합성 매개로 *제거 가능한 표적만 우선*. 9써클·왕실 직속은 *본 Part 절대 잠금*

6. **호스반 호각 결투 + 척살** (7-5) ★ — *Ch1 P4-P5의 8써클 전쟁 참모. 카엘의 1회성 조커 카드 운용자*. *바이서스 외부 마법사 1인 (학회 6써클·정적 척살 정보 보고자)이 카엘 변화한 정통 짧게 분석 → 호스반 직접 추격*. *호스반의 짧은 평가 한 줄 — "공허는 어디서 배웠나"*. **호각 결투** — *호스반의 광역 8써클 마법 vs 카엘의 마검사 변형 + 의수 마력포 + 공허검 호흡 + 마력→내공*. *호스반 = 광역 위력 압도적. 카엘 = 호스반의 광역 마법을 공허검 1원리 (불교 — 받지 않음) 로 비키게 함*. *호스반의 결정적 시전 직전 — 카엘의 바람의 길 응용 + 마력포 = 호스반 옆구리 직격 척살*

7. **라이세린의 마지막** (7-6) — *라이세린 = 잠적 후 비공식 권력 라인 운영. 결전 시점에 왕궁 비공개 거처 잠시 복귀*. *카엘의 조용한 단문 — "너의 위조 첩보는 우리 어머니의 침묵보다 무거웠지."* 라이세린의 응답 — *너는 결국 돌아왔구나. 내가 너를 키운 것이 내 잘못이었다*. *공허검 한 점 침으로 척살*. **라이세린 사적 이권 사슬 부분 해체** — *라카르 9세 형제·자식들의 사적 후원 라인 즉시 마비 + 마도공학 사적 거래 라인 마비 + 외교 비밀 정보 라인 마비*. 보석/이권 정통은 [`../../world/commerce-and-interests.md`](../../world/commerce-and-interests.md)

8. **바이서스 측 카엘 단독 자각** (7-7) — 호스반·라이세린·다수 정적·소드 마스터 다수 사망 후 *바이서스 외부 분석 마법사·왕실 정찰관·외교실 잔존 라인의 분산 정보가 약 4~5주 만에 *왕실 본가에 집중 보고됨*. *라카르 9세 + 왕실 호위 + 정규 사령관들이 *카엘 단독 가설로 결단*. *Ch1 P5 광역 학살 매개와 본 결전 정통이 부분 일치*

9. **군사 비상령 점화 시드** (7-7) — *라이세린 사망 직후 = 사적 권력 통제의 즉시 마비*. *라카르 9세 본인이 비상령 결단 = 사적 이권 사슬이 마비되어 *공식 군 무력 외에 다른 의존 라인이 없음*. **결단 — *왕실 직속 호위 5인 + 정규 소드 마스터 100명+ 즉시 비상 소집 + 길드 용병 5만 추가 동원 발령***. *Ch2 P8 본격 발화*

10. **카엘의 4~6주 누적 피로** (7-8) — *게릴라전 + 다수 척살 + 분산 잠입의 *체력·정신력 누적 소모*. *팔찌 조용 — 본성의 결단 안정*. 카엘의 짧은 내적 한 줄 — *…절반은 끝났다. 그러나 절반은 아직이다. 비상령이 떨어지면 — 진짜가 시작된다*

## 회차 비트

### 7-1 — 변경의 새벽

- **공간**: 앨더스톰 → 바이서스 변경 진입
- **시점**: 카엘 1인칭 (조용한 결단)
- **비트**:
  - 앨더스톰 → 바이서스 변경 진입 + 세렌 마르덴·베라트 알린 비공식 동행
  - 카르샤 망명지 협공 시드 활성화 — *세파 + 키얀 + 신성제국 형제·식솔 피난 잔존 카르샤 엠퍼러급·마스터 2인의 바이서스 외곽 동쪽 동시 압박*
  - **카엘의 정면 돌파 불가 자각** — 바이서스 군사력 정통 재확인 + 결단 *게릴라·치밀 정찰·정적 척살*

### 7-2 — 외곽의 정찰·치밀 사전 조사

- **공간**: 바이서스 외곽 산악 → 셀라리아 외곽 비밀 거점
- **시점**: 카엘 1인칭
- **비트**:
  - 마르덴 양도 5왕국 비공식 인맥 라인 활용
  - 변경 정찰병 5~10인 소리 없이 격퇴 — 공허검 2원리 + 의수 검기 완벽 침투력
  - **셀라리아 외곽 비밀 거점 구축** (안전 가옥 신부 라인 별개 거점)
  - **정적·고수 위치 약 30~40명 분량 사전 조사 완료** (세렌·베라트 룬 마법 원거리 시야 차단·위치 미세 추적 매개)

### 7-3 — 분산 잠입

- **공간**: 셀라리아 시내 다수 거점
- **시점**: 카엘 1인칭 (깊은 회상 + 야간 침묵)
- **비트**:
  - 셀라리아 진입 + Ch1 P3 5년 학습 도시의 깊은 회상 단편
  - **외곽 연구소 잔존 인물 1~2명 (카르엔 사이엔 사후 잠적 분파 조용한 잔존자) 비공식 협력**
  - 야간 이동 + 변장 + 위장 출입증 + 거처 잠입의 분산 매개

### 7-4 — 게릴라, 첫 척살들 ★

- **공간**: 셀라리아 시내·외곽 거점 다수
- **시점**: 카엘 1인칭 (압축된 단문)
- **비트**:
  - **4~6주에 걸친 다수 척살**:
    - *정규 소드 마스터 3~5명 — 야간 거처 잠입 + 공허검 한 점 침*
    - *7써클 대마법사 2명 — 시전 직전 마력포 직격 + 룬 마법 시야 차단 매개*
    - *라이세린 사적 사슬 핵심 4~6명 — 외교실 비공개 거상·정보원·자금원*
  - *팔찌 조용 — 결단의 본성 안정. 비뚤린 시야 우세 없음*

### 7-5 — 호스반의 응답

- **공간**: 셀라리아 외곽 → 마법 학회 본관
- **시점**: 카엘 1인칭 (호각 결투의 압축된 단문)
- **비트**:
  - 바이서스 외부 분석 마법사 1인이 카엘 변화한 정통 짧게 분석 → 호스반 직접 추격
  - **호스반 셰이드런 직면**
  - 호스반의 짧은 평가 — *"공허는 어디서 배웠나."*
  - **호각 결투** — 호스반의 광역 8써클 마법 vs 카엘의 마검사 변형 + 의수 마력포 + 공허검 호흡 + 마력→내공
  - 카엘 = *공허검 1원리 (불교)로 광역 마법 비키게 함*
  - *결정타 — 카엘의 바람의 길 응용 + 마력포 = 호스반 옆구리 직격 척살*

### 7-6 — 라이세린의 마지막

- **공간**: 왕궁 비공개 거처
- **시점**: 카엘 1인칭 (짧은 응답 + 분노 응축)
- **비트**:
  - **라이세린 데오릭 직면**
  - 카엘의 조용한 단문 — *"너의 위조 첩보는 우리 어머니의 침묵보다 무거웠지."*
  - 라이세린의 응답 — *"너는 결국 돌아왔구나. 내가 너를 키운 것이 내 잘못이었다."*
  - *공허검 한 점 침으로 척살*
  - **라이세린 사적 이권 사슬 즉시 부분 해체** — 외교 비밀 정보 라인 마비 + 마도공학 사적 거래 라인 마비 + 왕가 자식 후원 라인 마비
  - 팔찌 *부분 붉은 빛 → 안정 (분노 응축 표지)*

### 7-7 — 비상령의 그림자

- **공간**: 왕실 본가 (외부 단편 + 카엘 직접 등장 없음)
- **시점**: 외부 단편 (간접 묘사)
- **비트**:
  - 바이서스 측 *4~5주 누적 정보 집중 보고 → 카엘 단독 결전 가설 확정*
  - *라카르 9세의 자기 보존 결단 + 라이세린 사망으로 사적 권력 통제 즉시 약화 → 공식 군 무력 의존만 남음*
  - **결단 — 군사 비상령 본격 검토 + 왕실 직속 호위 5인 + 정규 소드 마스터 100명+ 즉시 비상 소집 시드 점화 + 길드 용병 5만 추가 동원 발령 검토** (P8 본격 발화)

### 7-8 — 거점의 침묵

- **공간**: 셀라리아 외곽 비밀 거점
- **시점**: 카엘 1인칭 (침묵 + 다음 단계 결단)
- **비트**:
  - 4~6주 게릴라 누적 피로
  - 세렌·베라트의 짧은 보고 — *비상령이 떨어진다. 며칠 안에. 결단해야 합니다*
  - 카엘의 짧은 내적 한 줄 — *…절반은 끝났다. 그러나 절반은 아직이다. 비상령이 떨어지면 — 진짜가 시작된다*
  - 팔찌 조용 — 본성의 결단

## 종결 후크

> 셀라리아 외곽 비밀 거점의 *작은 창문 너머로 셀라리아 종소리가 멀리 들린다*. *호스반·라이세린·다수 마스터·대마법사들의 죽음이 *낮은 종소리 사이에 묻혀 있다*. 그러나 카엘은 안다 — *그 종소리 뒤에서 *수많은 발걸음이 모이고 있다는 것을*.
>
> 카엘의 내적 한 줄 — *…수십 명이 동시에 온다. 그것을 받아낸다*.
>
> 멀리 — *셀라리아 왕궁 첨탑에서 *비상령의 깃발이 *천천히 오른다*. *왕실 직속 5인 마스터·엠퍼러급·9써클이 *움직이기 시작한다*.
