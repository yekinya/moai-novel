# Chapter 1 Part 4 Spec: 정의의 가면 — *30% 분기점*

> 회차 시나리오: [`../chapters/part-1-4.md`](../chapters/part-1-4.md)
> Chapter 1 overview: [`./spec-chapter-1.md`](./spec-chapter-1.md)

## Summary

카르샤 평화 사절단 도래 → 라이세린 위조 첩보 → 국경 사주 학살 → 카르샤 악마화 강연 → 선전포고. 본작 *30% 분기점*. 3-5는 *마하 시점 교차* — 본작 첫 시점 전환.

## 회차 구성 (6회차)

| 회차 | 제목 | 비트 |
|------|------|------|
| 3-1 | 동쪽의 사절 | 카일렌 베르하르트 경 사절단 + 손 모음 인사 *조롱 오인* |
| 3-2 | 위조된 서한 | 라이세린 위조 첩보 + *바이서스산 종이* 단서 흘림 + 카엘 *진위 못 잡음* |
| 3-3 | 국경의 불 | 변경 사주 학살 + **에리언 사망** (갈색 머리 농부, 두 손 들고) |
| 3-4 | 악마화 | 카엘의 카르샤 악마화 강연 + 청중 *증거 부족* 외침 |
| 3-5 | **[교차] 카르샤의 어머니** | **마하의 시점** — *우리 아이가 미워한 사람이 없게 해주세요* |
| 3-6 | 선전포고 | 왕 라카르 9세 망설임 + 카엘 전권 사령관 임명 + 출정 |

## 핵심 비트

1. **30% 단서 완전 노출 분기점** — 모든 핵심 단서가 본 Part 종료까지 노출됨
2. **3-5 시점 교차** — *마하* 의 시점으로 카엘의 학살을 *바깥에서* 본다. *우리 아이가 미워한 사람이 없게 해주세요* = 본작 *진짜 신앙*
3. **라이세린-카루린 밀약** 그림자 가시화 — 3-2 시종 + 3-4 라이세린 *이제 시작이군*
4. **왕 라카르 9세의 망설임** — *손을 모으려다 멈춤* (적국과 같음을 의식). 카엘은 *노쇠한 결단력* 으로 봄
5. **에리언 = 무명의 인격** — 3-3 학살 직격 → 3-5 가족·이름·기도로 *인격 회수*

## 시점·문체

- **3인칭 근접** (카엘) — 3-1~3-4·3-6
- **3-5 단편 시점 교차** — *마하 시점*. 1,500자 한정. *완전 신뢰* 톤
- *왜곡된 시야 정점* — 카엘이 가장 *정의로워 보이는* 시기

## 인물

- 카엘 (왕실 마법 고문) / 라이세린 데오릭 공작 / **카일렌 베르하르트 경** (사절단장) + 부사 베르난 류엔 자작 + 호위 5명 + 신관 1명 + 기록관 1명 / 라카르 9세 (3-6) / 카멜라 4세 후예 왕녀들 (간접) / **에리언** (3-3 학살) / **마하** (3-5 어머니) / **셋째 동생** (3-5) / 청중 1인 (3-4 *반대 청중*) / 라이세린 측근들 (백색 사냥단)

## 아티팩트·단서

- **라이세린의 위조 첩보 문서** (바이서스산 종이)
- **카르샤 사절단의 손 모음 인사** (왕가 문장과 동일)
- **에리언의 빈 그릇** (3-5 마하가 차림)
- **카르엔 학파 로브 청중** (3-4 강당)
- **카엘 강연집 출판물** (라이세린 자금)

## 지명

- 셀라리아 영빈관 / 외교실 비밀 분석실 / **세번 강** 변경 (양국 경계) / 셀라리아 대학 강당 / **카르샤 변경 농가** (3-5, 세이렌에서 사흘 거리) / 셀라리아 왕좌의 방

## Tone

- 정치 드라마. 음모와 광신
- 카엘이 *가장 정의로워 보임* — 독자 입장에서 *불편한 흡인력*
- 3-5만 *완전히 다른 톤* — 따뜻함, 일상, 평범한 사랑
- 짠내·유머 *0*

## ★ Review points

1. **3-5 시점 교차 1,500자**의 적정성 — 회차 분량의 25%. 더 짧게 (1,000자) / 길게 (2,500자)?
2. **라이세린 *이제 시작이군* 대사** — 3-4에 등장. *너무 명백한 음모 표현* 아닌지? 더 미묘하게?
3. **왕 라카르 9세의 망설임 명시 정도** — 3-6 현재 안: *손을 모으려다 멈춤*. 더 명시 (대사) vs 더 모호 (행동만)?
4. **에리언의 *두 손 들기*** — 3-3 현재 안: 평화 의례 (왕가 문장과 동일 자세). 더 강하게? 단순 *항복* 으로?
