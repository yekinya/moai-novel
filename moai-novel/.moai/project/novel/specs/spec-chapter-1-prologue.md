## Summary

본작 *in medias res* 개막. 카엘이 카르샤 왕 알레이언 6세를 호각의 5합 끝에 살해하는 순간, 왕이 *균열의 빛 즉발판* 으로 카엘의 사슬에 *균열* 을 낸다.

## 회차 구성

| 회차 | 비트 |
|------|------|
| 프롤로그 (단일 회차, 6,000자) | 1막 옥좌까지 + 2막 호각 5합 + 3막 균열의 0.5초 자각 |

## 핵심 비트

1. **백색 회랑** 통과 — 호위 시신 + 머리카락 목걸이 단서
2. **5합 호각 결투** — 알레이언 신성안으로 *붉은 사슬 7마디* 확정 (3합 후)
3. **균열의 빛 즉발판** — 알레이언이 검을 내려놓고 자기 죽음으로 시전
4. **카엘의 0.5초 자각의 창** — *얇은 금속음* + *가장자리 이중상* 잔류
5. **마지막 한마디** — *너는… 무엇을 보고 있는가*
6. **회상 4컷** + *고 나는 믿었다* 패턴

## 시점·문체

- **1인칭** (카엘) — 본작 *지배 시점 첫 등장*
- 짧은 호흡. 행동·대사 중심. 후각·청각 강조 (피·향·*얇은 금속음*)
- *완벽한 묘사·정보 덤프 금지* — 독자가 *무엇이 일어났는지 정확히 모른 채* 끌려가야 함

## 인물

- 카엘 (1인칭, *그·그 자* 로 알레이언 호칭) / 카르샤 알레이언 6세 (정식명 미공개)
- 카일렌 베르하르트 경 (회랑 시신만, 4-3에서 인격 회수)

## 아티팩트·단서

- 머리카락 목걸이 / *도망치소서* 비명 / 손 모음 문장 (왕가 벽화) / 평화 협정 양피지 / 옅어진 일렁임 (거울) / *얇은 금속음* / *가장자리 이중상* / *7대를 기다린 자*

## 지명

- 카르샤 수도 세이렌 — 왕궁 백색 회랑 + 옥좌의 방

## Tone

- 무겁고 격렬. 유머 *0*
- 짠내 *최고급* — 5합 모두 카엘에게 *치명상* (한쪽 폐 손상, 자상 다수, 마나 회로 자해)
- *극적 비트 모든 5종 동시 발현* — 결정적 한마디·물리 충격·재해석·시점 충돌·선언

## ★ Review points

1. *너는 무엇을 보고 있는가* — 본작 *시그니처 대사* 로 확정 OK?
2. *호각의 5합* — 결투 묘사 분량 (현재: 3,000자 / 전체 6,000자 중 50%). 적정한지?
3. *균열의 빛 즉발판 = 시전자 사망* 의 비가역성 명시 시점 — 프롤로그(이번)? 4-4 (회수)? 둘 다?
