## Summary

Ch2 P2 종결 직후 — 카엘 단독 강철제국 변경 진입. *세이른 스틸만의 옛 인연 (강철제국 본가 방계) 매개로 본가 진입 + 강철제국 검술 정통 일부 학습 + 그라스 노 장인 의수 시술 (강화강철 1단계)*. *친조부 책 출처 회수 + 사프란 비망록 마지막 페이지 + 붉은 머리끈 의수 봉인 회로*의 **Ch1 시드 정통 본격 회수**. 강화강철 의수 적응 + 마나 운용 미세 자연 훈련 + *검기 운용 약함 한계 자각*. 카엘이 *검기 없이 검사를 제압하는 공허검 정통*의 존재를 *카르샤 잔존 소드 마스터 정통 전수 + 세이른 옛 동문 정통에서 교차 인지* → **동부마녀연맹 진입 결단**. *모르간 솔테 (흑혈 종주) 의 비공식 사사 허락* + *9혈 원탁 외부 손님 정통의 8번째 등록 시드* + *모르간 린이엔 (차기 종주, 25세) 과의 첫 정면* + 공허검 *불교·공출* 2원리 부분 습득 (3원리 허는 *비뚤린 시야 50%로 불완전*). *마녀령 비전 룬 결각 + 아티팩트화 지식 부분 양도*. 사사 약 3~4개월 후 *강철제국 복귀 결단*. 본 Part 종결 = *강화강철 의수 + 공허검 2원리 + 룬 지식을 가지고 강철제국 본가 도착 직전*.

## 핵심 비트

1. **강철제국 진입 + 세이른 옛 인연** (3-1) — 카엘이 *세이른 스틸만의 옛 동문 추천장 1장 매개*로 오르난 본가 진입. 추천장 출처 = *Ch1 P3 알린 마스반 경 → 마스반 옛 동문 → 세이른 옛 동문*의 *3중 인연 라인*. *Ch2 P2 카르샤 잔존 소드 마스터 (라이아 모르난) 가 추천장 작성 매개*
2. **본가 = 마법 거부 검술 정통의 공식 도장** — 카엘의 *마법 봉인 + 두 팔 상실 + 의수* 상태가 *마법 거부 검술 정통의 유일한 정당성 매개*. *마법 없이 검만으로 강해지려는 자만 강철제국이 받아들임*
3. **강철제국 검술 정통 1·2단계 학습** (3-1) — *기초 검술 (자세·기본 호흡·1합 받아치기·짧은 거리 진입) + 강철 검의 물리 위력 정통*. 카엘이 이미 *Ch1 P3 알린 마스반 경 사사로 기초(상) 단계*. *본가에서 2단계 (정통 검사 기초 정점) 까지 학습*
4. **그라스 노 장인 첫 등장 + 3대 양도 정통 폭로** (3-2) ★ — 60대 후반, *카엘 친조부 알리스와 같은 가문* (카르엔 학파 잠적 분파 외부 인척). 친조부의 책 *여백 3중 필체 즉시 인식* — *알리스 → 알린 → 카엘* 3대 양도. 한 마디 — *"너의 친조부 알리스를 알았다. 너의 친조모 이렐라도. 그녀가 어떤 존재였는지도 안다. 너는 그 핏줄의 끝이다."*
5. **3대 비극 본격 공개** (3-2) — *친조모 이렐라 = 강 엔트로퍼 미기록 발현 + 친조부 알리스 = 카르엔 학파 잠적 분파 학자 + 알린 = 약 캐리어*. **Ch1 P1 시드의 진짜 회수**
6. **사프란 비망록 마지막 페이지 공개** (3-2) — *엔트로퍼는 자기 본성을 거부하는 자결로만 진정한 평화에 도달*. *그러나 그 결단은 자기 의지로만 가능*. **Chapter 3·4 자결 결단 시드**
7. **붉은 머리끈 의수 봉인 회로** (3-2) — *Ch1 5-2 충동 구매 의미 회수*. 노 장인이 *붉은 머리끈을 의수의 마나 봉인 회로 매개로 사용*. *봉인 회로 = 마나 폭주 방지 + 살상 충동 일부 봉인 + 평등 살상 가치관 외부 발현 억제*
8. **강화강철 의수 시술** (3-3) — *재료 = 강철 + 정교한 경강철화 관절 + 룬 회로 + 마검사 검 부서진 부품 + 카르엔 학파 비전 마도공학 기법*. *손가락 3관절 모두 굽힘·회전 + 손목 회전 360° + 마나 회로 정밀 결각*. 시술 약 7~10일·극심한 통증·정신력 견딤. **왼손 강화강철 대포 의수 함께 격상** (입 트리거 방식 유지 + 무게 감소 + 발포 정확도 향상)
9. **각성 신체화 두 번째 단계** ★ (3-3) — *그라스 노 장인의 식전 손 모음 인사* (Ch1 시드의 오릴 노 장인 정통과 유사). *카엘이 위협으로 보지 않음 → 본작 각성 신체화 두 번째 단계*. **Ch1 P1 1-4 *손 모음 = 위협* 인식의 진짜 회수**
10. **검기 한계 자각** (3-5) — 강화강철 의수 검술 훈련 + 검기 두르기 시도 → *효율 일반 사람 검의 70% 한계*. 원인 = *강철의 마나 전도율 한계 + 카엘 마법력 부족 (마법 봉인 부산물)*. 카엘 *조용한 답답함*
11. **공허검 정통 3중 교차 인지** (3-6) — *그라스 노 장인 짧은 언급 + 강철제국 본가 검사의 세이른 스틸만 5인 마스터 명시 + 카르샤 잔존 소드 마스터 추가 전수*. *공허검 = 마법 봉인 + 의수 검기 한계의 유일한 우회 정통* 자각 → 동부마녀연맹 진입 결단
12. **린엔 모르간 비공식 안내** (3-7) — *모르간 가문 외곽 분파 출신 30대 후반 여성*. *그라스 노 장인 → 마녀령 진입 매개*. 모르간 솔테 직면 매개
13. **모르간 솔테 비공식 사사 허락 + 9혈 외부 손님 8번째 등록 시드** (3-7) — *9혈 원탁 외부 손님 정통의 8번째 등록 시드*. Chapter 3 본격 공식화
14. **모르간 린이엔 첫 정면** (3-8) — *25세 차기 흑혈 종주 + 흑혈/백혈 마검술 + 황실 검술 융합 3중 정통 보유*. **v3 정정** (카이엔 = 이실 폴리모프 캐논 폐기 결과) — *린이엔이 카엘을 외부 마검술 변종의 4번째 정통 후보로 조용히 관찰*. *호감 + 거리 두기 + 호기심* 의 복합 감정 시드. *당신의 정통이 무엇인지 지켜보겠습니다*
15. **공허검 2원리 부분 습득** (3-9) — *기본 호흡 3박자 + 1원리 불교 (받지 않음) + 2원리 공출 (비움에서 출수)*. *3원리 허 (빈 곳을 침) 는 비뚤린 시야 잔존 50%로 완전 습득 불가*. Ch3+ 본격 정점 시드
16. **마력→내공 + 공허 호흡 완벽 융합** ★ (3-9) — *공허검 3박자 호흡 + 마력→내공의 호흡 안정 회로 완벽 융합*. *지속성 = 5~10분 운용 후 *30~40분 회복으로 단축* (Ch2 P1 1-5의 1~2시간 회복에서 단축). **카엘의 *재정체화 정통의 생리적 매개 정착***
17. **카이렌 스틸반의 짧은 평가** (3-9 종결) — 세이른 옛 동문, 마녀령 잔존 공허검 마스터 1인. *너의 공허는 온전하지 않다. 그러나 너에게는 그게 더 맞을지도. 남은 길은 너의 손에 달렸다*. Ch2 P4 세이른 직접 진단의 시드
18. **마녀령 비전 룬 결각 + 아티팩트화 지식 부분 양도** (3-10) — 9혈 외부 손님 정통의 부수 특전. *완전 정통은 앨더스톰 + 드워프 지하세계에서 본격 보강*. Ch2 P4 강화 대포 강철 의수 + 룬 결각 작업의 핵심 매개
19. **모르간 솔테의 짧은 한 마디** — *너의 공허가 우리의 정통과 다른 정통이 되겠지. 그러나 그것도 좋다*

## 회차 비트

### 3-1 — 강철의 도시

- **공간**: 강철제국 변경 → 본가 (오르난) 진입
- **시점**: 카엘 1인칭
- **비트**: 변경 진입 + 라이아 모르난의 *세이른 스틸만 옛 동문 추천장 1장 매개* + 본가 진입 + *마법 거부 검술 정통 1·2단계 학습 (정통 검사 기초 정점)* + 본가의 나이 많은 검사가 카엘을 *조용히 받아들임*

### 3-2 — 그라스의 노 장인

- **공간**: 오르난 → 그라스 (도보·말 2주)
- **시점**: 카엘 1인칭 (정통 회수의 무거움)
- **비트**:
  - 그라스 노 장인 첫 등장 (60대 후반, 카엘 친조부 알리스와 같은 가문)
  - 카엘이 친조부의 책을 보여줌. **3중 필체 즉시 인식** — *알리스 → 알린 → 카엘*의 3대 양도 정통
  - 노 장인의 한 마디 — *"너의 친조부 알리스를 알았다. 너의 친조모 이렐라도. 그녀가 어떤 존재였는지도 안다. 너는 그 핏줄의 끝이다."*
  - **3대 비극 공개** — *친조모 이렐라 = 강 엔트로퍼 미기록 발현 + 친조부 알리스 = 카르엔 학파 잠적 분파 학자 + 알린 = 약 캐리어*
  - **사프란 비망록 마지막 페이지 공개** — *엔트로퍼는 자기 본성을 거부하는 자결로만 진정한 평화에 도달*. *카엘 짧게 본다. 말이 없다*
  - **붉은 머리끈 의수 봉인 회로** — Ch1 5-2 충동 구매 의미 회수
- **단서**: 카엘의 *첫 공식적 자기 핏줄 인식*. *팔찌가 짧게 부분 붉은 빛 → 조용해짐 — 본성과 비뚤린 시야의 공평한 응시*

### 3-3 — 강화강철의 손가락

- **공간**: 그라스 노 장인의 작업장
- **시점**: 카엘 1인칭 (극심한 통증 + 짧은 명료성)
- **비트**:
  - **강화강철 의수 시술** — 강철 본체 + 정교한 경강철화 관절 + 룬 회로 + 마검사 검 부서진 부품 + 카르엔 학파 비전 마도공학 기법
  - *시술 시간 약 7~10일*. 극심한 통증 + 정신력 견딤
  - *손가락 3관절 모두 굽힘·회전 + 손목 회전 360° + 마나 회로 정밀 결각*
  - **왼손 강화강철 대포 의수 함께 격상** (입 트리거 유지 + 무게 감소 + 발포 정확도 향상)
  - **그라스 노 장인의 식전 손 모음 인사** — *카엘이 위협으로 보지 않음 → 본작 각성 신체화 두 번째 단계*
  - Ch1 P1 1-4 *손 모음 = 위협* 인식의 *진짜 회수*

### 3-4 — 의수의 마나

- **공간**: 그라스 작업장 + 외곽 훈련장
- **시점**: 카엘 1인칭
- **비트**: 강화강철 적응 + 마나 운용 미세 자연 훈련 + 일상 거의 정상 회복 (컵·바늘질·글씨·악기 시도) + *세밀 작업 0.5초 지연 한계 자각*

### 3-5 — 검기의 한계

- **공간**: 그라스 외곽 훈련장
- **시점**: 카엘 1인칭 (답답함)
- **비트**: 강화강철 의수 검술 훈련 → *검기 두르기 효율 일반 사람 검의 70% 한계 자각* + 카엘의 *조용한 답답함*. *마력→내공 + 공허 호흡 + 의수 감응의 3중 융합 시도 → 첫 기본 작동 실패*

### 3-6 — 공허의 소문

- **공간**: 그라스 외곽 → 오르난 본가 + 라이아 모르난 추가 정보
- **시점**: 카엘 1인칭 (정통 인지 + 결단)
- **비트**: 공허검 정통 = *그라스 노 장인 + 강철제국 본가 + 카르샤 잔존 소드 마스터 정보 3중 교차 확인*. *원류 = 동부마녀연맹 자각 + 마녀령 진입 결단*

### 3-7 — 마녀령의 문

- **공간**: 그라스 → 동부마녀연맹 진입
- **시점**: 카엘 1인칭 (배타적 거리감)
- **비트**: 마녀령 진입 + 배타적 성격 직면 + **린엔 모르간 매개** + **모르간 솔테 (흑혈 종주) 직면 + 비공식 사사 허락** + *9혈 원탁 외부 손님 8번째 등록 시드*

### 3-8 — 흑혈의 손녀 *(린이엔 시점 짧은 교차)*

- **공간**: 동부마녀연맹 본가 — 흑혈 도장
- **시점**: 카엘 1인칭 + **린이엔 3인칭 짧은 교차** (린이엔이 카엘을 조용히 관찰하는 시점)
- **비트**:
  - **모르간 린이엔 (25세, 차기 흑혈 종주) 과의 첫 정면**
  - v3 정정 — 린이엔이 카엘 본인을 *외부 마검술 변종의 4번째 정통 후보로 조용히 관찰*. *호감 + 거리 두기 + 호기심*의 복합 감정 시드
  - 린이엔의 짧은 한 마디 — *당신의 정통이 무엇인지 지켜보겠습니다*
  - 카엘 *짧은 묵례 + 답 거부* — *나는 아직 내 정통을 모릅니다*
  - 공허검 사사 정식 시작 (모르간 솔테의 결단 + 린이엔이 옆에서 짧게 관찰만)

### 3-9 — 공허의 호흡

- **공간**: 흑혈 도장 사사장 + 외곽 훈련장
- **시점**: 카엘 1인칭 (짠내 노동 + 짧은 깨달음)
- **비트**:
  - *공허검 기본 호흡 3박자* + *불교 (받지 않음) + 공출 (비움에서 출수) 2원리 부분 습득*
  - *3원리 허는 비뚤린 시야 잔존 50%로 완전 습득 불가*
  - **마력→내공 + 공허 호흡 완벽 융합** — *5~10분 운용 → 30~40분 회복으로 단축*
  - **카이렌 스틸반의 짧은 평가** — *너의 공허는 온전하지 않다. 그러나 너에게는 그게 더 맞을지도. 남은 길은 너의 손에 달렸다*

### 3-10 — 룬의 작은 정통

- **공간**: 흑혈 도장 룬 결각 작업장 + 모르간 솔테의 거처
- **시점**: 카엘 1인칭 (조용한 정통)
- **비트**:
  - *마녀령 비전 룬 결각 + 아티팩트화 지식 부분 양도* — 9혈 외부 손님 정통의 부수 특전
  - *완전 정통은 앨더스톰 + 드워프 지하세계에서 본격 보강*
  - 모르간 솔테의 짧은 한 마디 — *너의 공허가 우리의 정통과 다른 정통이 되겠지. 그러나 그것도 좋다*
  - **카엘의 강철제국 복귀 결단** — Ch2 P4 강화 대포 강철 의수 + 룬 결각 + 팔 검기 개발
- **단서**: 본 Part 종결 = 오르난 본가 도착 직전. *팔찌 조용 — 본성의 결단 안정*
