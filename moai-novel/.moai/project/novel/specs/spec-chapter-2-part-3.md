## Summary

Ch2 P2 종결 직후 — *카엘 단독 강철제국 변경 진입*. *세이른 스틸만의 옛 인연 (강철제국 본가 방계) 매개로 본가 진입 + 강철제국 검술 정통 일부 학습 + 그라스 노 장인 (Ch2 P1 v3에서 이관) 의수 시술 (강화강철 1단계) 받음*. *친조부 책 출처 회수 + 사프란 비망록 마지막 페이지 + 붉은 머리끈 의수 봉인 회로* 의 *Ch1 시드 정통 본격 회수*. 강화강철 의수 적응 + 마나 운용 미세 자연 훈련 + *검기 운용 약함 한계 자각*. 카엘이 *검기 없이 검사를 제압하는 공허검 정통의 존재를 *카르샤 잔존 소드 마스터의 정통 전수와 *세이른 스틸만의 옛 동문 정통에서 *교차 인지* → **동부마녀연맹 진입 결단**. *모르간 솔테 (흑혈 종주) 의 비공식 사사 허락* + *9혈 원탁 외부 손님 정통의 8번째 등록 시드* + *모르간 린이엔 (차기 종주, 25세) 과의 첫 정면* + 공허검 *불교·공출* 2원리 부분 습득 (3원리 허는 불완전). *마녀령 비전 룬 결각 + 아티팩트화 지식 부분 양도*. 사사 약 3~4개월 후 *카엘이 강철제국 복귀 결단*. 본 Part 종결 시점 = *카엘이 강화강철 의수 + 공허검 2원리 + 룬 지식을 가지고 *강철제국 본가로 돌아옴*.

### 1. 강철제국 진입 + 세이른 옛 인연 (3-1)

**진입 매개**:
- *카엘이 강철제국 변경 도시에서 *세이른 스틸만의 *옛 동문 추천장* 1장 매개로 본가 진입*
- 추천장의 출처 = *Ch1 P3 알린 마스반 경 (카엘 검술 사부) → 알린 마스반 경의 옛 동문 → 세이른 스틸만 옛 동문* 의 *3중 인연 라인*. *Ch2 P2 카르샤 잔존 소드 마스터 1~2명이 *추천장 작성 매개*

**강철제국 본가 진입**:
- 본가 = *마법 거부 검술 정통의 *공식 도장 + 강철 제련 + 의수 제작 시설*. 본가의 *나이 많은 검사 (이름 미정 — 후속) 가 *카엘을 *조용히 받아들임*
- *조건 — 카엘의 *마법 봉인 + 두 팔 상실 + 의수 *상태가 *마법 거부 검술 정통의 *유일한 정당성 매개*. *마법 없이 검만으로 강해지려는 자만 *강철제국이 받아들임*

**강철제국 검술 정통 일부 학습**:
- *기초 검술 (자세·기본 호흡·1합 받아치기·짧은 거리 진입) + 강철 검의 *물리 위력 정통 + 마법 없이 검만으로 *마검사·마법사 상대하는 기본기*

  - v3 정정 — *린이엔이 *카엘 본인을 *외부 마검술 변종의 4번째 정통 후보로 *조용히 관찰*. *호감 + 거리 두기 + 호기심* 의 복합 감정 시드
  - 린이엔의 짧은 한 마디 — *당신의 정통이 *무엇인지 *지켜보겠습니다*
  - 카엘 *짧은 묵례 + 답 거부* — *나는 *아직 *내 정통을 *모릅니다*
  - 공허검 사사 정식 시작 (모르간 솔테의 결단 + 린이엔이 *옆에서 짧게 관찰만*)

### 3-9 — 공허의 호흡

- **공간**: 흑혈 도장 사사장 + 외곽 훈련장
- **시점**: 카엘 1인칭 (짠내 노동 + 짧은 깨달음)
- **비트**:
  - *공허검 기본 호흡 3박자* + *불교 (받지 않음) + 공출 (비움에서 출수) 2원리 부분 습득*
  - *3원리 허(虛) 는 *비뚤린 시야 잔존 50%로 *완전 습득 불가*
  - **마력→내공 + 공허 호흡 융합** — *5~10분 운용 → 30~40분 회복으로 단축*
  - **카이렌 스틸반 (세이른 스틸만 옛 동문, 마녀령 잔존 공허검 마스터 1인) 의 짧은 평가** — *너의 공허는 *온전하지 않다. 그러나 *너에게는 *그게 더 맞을지도. 남은 길은 *너의 손에 달렸다*

### 3-10 — 룬의 작은 정통

- **공간**: 흑혈 도장 룬 결각 작업장 + 모르간 솔테의 거처
- **시점**: 카엘 1인칭 (조용한 정통)
- **비트**:
  - *마녀령 비전 룬 결각 + 아티팩트화 지식 부분 양도* — 9혈 외부 손님 정통의 부수 특전
  - *완전 정통은 *앨더스톰 + 드워프 지하세계에서 본격 보강*
  - 모르간 솔테의 짧은 한 마디 — *너의 공허가 *우리의 정통과 *다른 정통이 되겠지. 그러나 그것도 *좋다*
  - **카엘의 강철제국 복귀 결단** — *Ch2 P4 강화 대포 강철 의수 + 룬 결각 + 팔 검기 개발*
- **단서**: 본 Part 종결 = 강철제국 본가 도착 직전. *팔찌 조용 — 본성의 결단 안정*
