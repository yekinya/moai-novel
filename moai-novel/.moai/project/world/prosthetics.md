# 의수 시스템 — 4단계 진화 정통

> 본 문서는 *카엘의 양 팔 상실 (Ch1 P6 5-5) 이후 의수 4단계 진화 곡선* 의 도메인 사양서. Chapter 2~4 의수 진화 라이프사이클의 정통 근거.
>
> 본 시스템은 *세계관 전체에 적용되는 의수 표준* — 카엘 외 다른 의수 사용자에도 동일하게 적용. 단 *드워프 제작 의수의 희소성* 으로 *고급 단계는 대부분 카엘 등 *예외적 인물에게만 도달*.

| 단계 | 명칭 | 재료 | 제작자 | 마나 운용 | 일상 | 검술 | 가격 | 카엘 도달 |
|------|------|------|--------|---------|------|------|------|---------|
| **1** | 나무 의수 | 단단한 목재 + 가죽 끈 | 의료시설·일반 장인 | 0 (불가) | 자세 잡기·기초 활동 보조 | 불가 | 저렴 | **Ch2 P1 초반** (둥지 탈출 직후) |
| **2** | 강철 의수 | 강철 (전체 또는 부분) | 일반 대장간·마법 장인 | *어느 정도 가능* (숙련 편차 大) | 약함 — 무게 부담 | 약함 — 단단함 위주 | 비쌈 | **Ch2 P1 후반** (대도시 경유) |

## 1. 1단계 — 나무 의수 (Wooden Prosthetic)

### 재료·구조
- 단단한 목재 (참나무·박달나무·물푸레나무 등)
- 가죽 끈 + 청동 못 + 베어링 없음
- 손가락 = 고정 또는 *판쪽 관절 1축* — 회전 불가
- 무게 = 일반 인간 팔의 1.0~1.2배

### 기능
- *자세 잡기* — 의식적 균형·체위 유지
- *기초 활동 보조* — 음식 그릇 받침·문 손잡이 누르기·짐 끈 걸치기
- *세밀 작업 불가* — 단추 채우기·글씨 쓰기·검 잡기 모두 불가
- **마나 운용 0** — 마법 시전 보조 매개체 기능 없음

### 카엘 적용 시점 (Ch2 P1)
- 절벽 낙하 + 둥지 탈출 직후 *간신히 숨 쉬는 상태* 에서 *세파의 응급 처치 매개로 마련*
- 세파가 길드 경유로 *임시 의료 키트* 확보 → *마을 의술원의 임시 나무 의수 2조* 장착
- 카엘 본인은 *마법 시전 완전 봉인* + *기초 행랑만 가능*
- 정통 시기 = Ch2 P1 1-1 ~ 1-4 (회복 + 망명지 소식 대기 + 이동 결단 단계)

### 사회적 시각
- 평민·전상 군인·노동 사고자의 *일반적 의수*. *특별한 시선 없음*. 신분 위장에 적합

## 2. 2단계 — 강철 의수 (Steel Prosthetic)

### 재료·구조
- *강철 전체* 시 = *너무 무겁고 지탱 안 됨* → 실용성 0. 따라서 **부분만 얇은 강판** 적용
  - 손바닥 + 검지 끝 + 회전 관절 3축에만 얇은 강철판
  - 나머지는 단단한 목재 + 가죽 결박
- 무게 = 일반 인간 팔의 1.5배 (강철 부분 위주 무게 분포 어려움)
- 내구도 *약함* — 강한 충격 시 *목재 부분 파손 빈발*
- 가격 *비쌈* — 대장간 의뢰 + 마법 회로 미세 결각 비용

### 기능
- *어느 정도의 움직임 마나 제어* — 강철 부분에 *마나 흐름 미세 결각*. 사용자의 마나 운용에 따라 *손가락 일부 굽힘·펴기·약한 그립* 가능
- **숙련 편차 매우 큼** — *마나 운용 자체가 미숙하면 거의 작동 안 함*. 카엘 같은 *마나 자체 운용 천재* 는 *상대적으로 빠른 적응*
- *일상* — *컵 잡기 어색·문 손잡이 회전 가능·짧은 글씨 가능* 정도. *세밀 작업 불가* (단추·바늘질·정밀 도구 사용 불가)
- *검술* — *검 자루 잡기 가능*. 그러나 *강한 힘을 내는 동작은 오히려 더 단단함* — 강철의 *경직성* 이 *유연한 검기 운용 방해*. **마나 검기 운용 불가**

### 카엘 적용 시점 (Ch2 P1 후반)
- 망명지 이동 도중 *중간 대도시 경유 시 제작*. 정통 시기 = Ch2 P1 1-5 ~ 1-6 (대도시 경유 → 강철의수 제작 → 망명지 도착 직전)
- 카엘이 *기본적인 일상 회복 + 검술 훈련 시작*
- *마법 시전 완전 봉인 상태* → *마력을 내공으로 활용하는 특이 운용 발견* (앤트로퍼 특전 — 별도 시스템 문서 [`../systems/anti-entroper-internal-energy.md`](../systems/anti-entroper-internal-energy.md))

### 강철 대포 의수 변종 (왼손 — 카엘 비밀 제작)
- *왼손 (검을 오른손에 듬) 강철 의수* 에 *대포 기능 1회 장착*
- *대포알이 작음에도 매우 무거움* — *짐 무게 + 의수 자체 무게 부담*
- *입으로 왼팔뚝쪽 트리거 물어 당김 → 손부분 뚜껑 열리며 발포*
- **1회 한정 장착** (충전·재장전 절차 복잡 → 본 작품 중 추가 사용 보류)
- 카엘이 *세파 몰래 대장간 의뢰* — 자기·세파 보호 결단의 *비밀 카드*
- 정통 시기 = Ch2 P1 1-7 종결 직전

## 3. 3단계 — 강화강철 의수 (Reinforced Steel Prosthetic)

### 재료·구조
- *상당부분 강철* + *세부 관절까지 정교한 경강철화*
- 손가락 = *3관절 모두 굽힘·회전 가능*. *손목 회전 360°·팔꿈치 굽힘 정상 범위*
- 강철의 *비밀 — 경강철화 처리* — 강철을 *마나 감응 가능한 상태로 *결정 구조 미세 재배열*. *마법 장인 도시 (그라스·강철제국·앨더스톰) 의 비전 기술*
- 무게 = 일반 인간 팔의 1.8배 (강철 비중 高 + 정밀 관절로 일부 상쇄)
- 가격 *매우 비쌈* — 평민 평생 임금 + α. *왕족·고위 귀족·소드 마스터급* 만 부담 가능

### 기능
- *마나 운용에 따라 컵 잡기·세밀 동작 가능* — *바늘질·글씨·정밀 도구 사용 모두 가능*
- *일상 거의 정상* — 단 *세밀한 작업의 *0.5초 지연*. *손가락 끝 감각 부재* 의 한계 (의수의 *촉각 감응*은 아다만티움/미스릴 단계에서 첫 가능)
- *검술* — *검 자루 잡기 + 휘두름 가능*. 그러나 *검기 운용은 여전히 약함* — *강철의 *마나 전도율 한계* 로 *검기를 두르는 효율이 일반 사람 검에 비해 70%*
- *강한 힘을 내는 동작은 오히려 더 단단함* — 강화강철의 *경직성* 이 *3단계에서도 일부 잔존*. *유연한 검기는 어려움*

### 카엘 적용 시점 (Ch2 P3~4)
- 강철제국 1차 방문에서 제작. *세이른 스틸만의 옛 인연 + 카르샤 망명지 형제들의 강철제국 추천*
- 정통 시기 = Ch2 P3 (강화강철의수 제작) ~ P4 (공허검 원류 찾아 동부마녀연맹 이동)
- 카엘이 *제한적 마나 검기 응용 확인* — 일상 일부 정상 회복 + *마나 운용을 훨씬 민감하게 자연 훈련*
- **세밀 작업은 여전히 불가** — *왼손 강철 대포 의수의 트리거 입 당김* 은 *강화강철 단계에서도 입 당김 정통 유지*

## 4. 4단계 — 아다만티움·미스릴 의수 (Adamantium / Mithril Prosthetic)

### 재료·구조
- *아다만티움* — 검은 회색의 *극도로 단단한 금속*. *마나 감응 특화 + 인간 장인 일부 제련 가능 (조잡한 수준)*. *세밀 가공은 인간 불가*
- *미스릴* — 푸른 은빛 광택 금속. *마나 감응 *최고 효율*. *인간 제련 완전 불가* — *드워프 지하세계 비전 기술 전용*
- 본작 *카엘 의수는 *드워프 지하세계 장인 전용 제작***. 인간 장인의 제련 불가 한계 정통
- 두 재질 *모두 마나 감응에 매우 특화* — 검기 운용 효율 *일반 강철의 3~5배*. *마검사 풀 운용 가능*
- 무게 = 일반 인간 팔의 1.3배 (밀도는 높으나 *합금 구조 + 마나 부유 결각* 으로 체감 무게 상쇄)
- **돈으로 살 수 없는 재료** — *드워프 부족의 *비공식 거래·교환·은혜 양도* 만 가능*

### 기능
- *마검사 풀 운용 + 룬 결각 + 아티팩트화 가능*
- *손가락 촉각 감응* 첫 부분 가능 — *마나를 의수 표면 미세 감응으로 변환*
- *팔 전체에 검기를 두르는 방식* (Ch2 P6 카엘 본격 개발) 가능
- *대포 장착 + 마력포로 변환 가능* — 마나가 있으면 *계속 발사 가능* (강철 대포의 *1회 한정* 폐기)
- *세밀 작업 정상* — 도자기 잡기·바늘질·악기 연주까지 가능
- *내구도 극강* — 본작 정통 *카엘 의수가 부서지는 사건은 첫 마왕 강림 (Chapter 3 후반) 까지 한 번* 만 발생

### 카엘 적용 시점 (Ch2 P7~8)
- *앨더스톰 → 드워프 지하세계* 이동 + 드워프 부족의 *공식 의뢰 + 카엘의 *대가 (마나석·고대 정통 정보·미래의 약속) 양도*
- 정통 시기 = Ch2 P7 (드워프 지하세계 진입) ~ P8 (아다만티움 대포 의수 완성)
- *바이서스 결전 (Ch2 P8 후반 ~ Ch3 초반) 직전 핵심 무기*

### 사회적 시각
- *전 대륙에서 *50명 이하의 보유자*. *왕·소드 엠퍼러·드래곤 폴리모프* 정도가 알려진 사례
- 카엘이 *공개 보유* 시 *드래곤 로드 분파의 즉시 인지* 매개 — 후속 *추적 가속의 핵심 단서*

## 5. 룬 결각·아티팩트화 (3~4단계 적용)

의수 단계별 *룬 결각 + 아티팩트화* 가능성:

| 단계 | 룬 결각 | 아티팩트화 | 비고 |
|------|--------|-----------|------|
| 1 나무 | 불가 | 불가 | 마나 회로 0 |
| 2 강철 | *극히 약한 결각 1~2종* | 불가 | 충전형 부스터 부츠 수준 |
| 3 강화강철 | *3~5종 룬 결각* | *낮은 등급 1~2 아티팩트화* | 마나 봉인 회로·일부 회복 결각 |
| 4 아다만티움/미스릴 | *전체 표면 결각 가능* | *고등급 다중 아티팩트화* | 룬·아티팩트·마법 회로·대포 통합 |

룬 결각 제작자 — *그라스 노 장인 (강화강철 단계 까지)* / *앨더스톰 룬 마법사 (4단계)* / *드워프 지하세계 룬 장인 (4단계 최고)*. 자세한 인물·정통 = [`./legendary-figures.md`](./legendary-figures.md) + [`../novel/characters.md`](../novel/characters.md)

## 6. 카엘 의수 진화 라이프사이클 — Ch2~4 전체

```
[Ch1 P6 5-5]   두 팔 절단 + 절벽 낙하
       ↓
[Ch2 P1 초반]  나무 의수 1조 (세파 응급 매개·임시)
       ↓        + 마법 시전 완전 봉인 자각
       ↓        + 앤트로퍼 특전 — 마력 → 내공 운용 첫 발견
       ↓
[Ch2 P1 후반]  중간 대도시 강철 의수 제작
       ↓        + 왼손 강철 대포 의수 비밀 제작 (1회 한정)
       ↓        + 검술 훈련 시작
       ↓
[Ch2 P3~4]    강철제국 강화강철 의수 제작
       ↓        + 일상 회복 + 마나 검기 운용 검증 (약함)
       ↓        + 동부마녀연맹 공허검 원류 추적 결단
       ↓
[Ch2 P5]      공허검 습득 + 룬 결각·아티팩트화 지식 획득
       ↓
[Ch2 P6]      강철제국 복귀 + 강화 대포 강철 의수 + 룬 + 팔 전체 검기 개발
       ↓        + 마법력 부족으로 효과 약함 진단 (세이른 스틸만)
       ↓
[Ch2 P7]      앨더스톰 — 전도율 진단 (아다만티움·미스릴 필요)
       ↓        + 드워프 지하세계 이동 결단
       ↓
[Ch2 P8]      드워프 지하세계 — 아다만티움 + 마력 대포 의수 완성
       ↓        + 행커만 과거·바이서스 악행 공유 (앨더스톰 회수)
       ↓
[Ch3+ 본격]   바이서스 결전 → 행커만 자식 척살 → 드래곤 적대 → 카자르드 추격
```
