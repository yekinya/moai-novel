# Part A — 상업·이권 6대


본작 리투니아 대륙의 *상업·무역·기술·생산·광산·마법 부여 정통과 그 이권 (Vested Interest) 분배 시스템*. **각 강대국이 누리는 이권의 본질 + 약소국이 빼앗긴 이권의 구조 + 강대국 멸망·약화 시 이권 재분배의 본격 정통**.

본 문서의 핵심 정통: **바이서스 헤븐즈 = 헤븐즈 킹덤의 종주국 = 대륙 이권 강제 편취 체계의 정점**. *바이서스 약화 시 = 주변국이 빠르게 이권 침탈·약소국 주권 복원*.

## 1. 6대 이권 (Vested Interests)

각 국가의 경제력·정치력·군사력의 *진정한 토대*. 외관상 군사력은 도구. **이권 = 본질**.

### (1) 상업권 (Commerce)

- *물건 중개 수수료·관세·특정 상인 집단의 세금*
- 도시 내·국경 내 거래의 *세금 정통*
- 길드 인가권 (상인 길드·운반 길드 등)
- **대표 사례 — 바이서스의 셀라리아 상업권**: 대륙 최대 도시. 카르샤·데미안·강철제국·마녀령에서 수입되는 모든 사치품·마법 도구·아티팩트가 *셀라리아 중개상을 거침*. 중개 수수료 5~15%

### (2) 무역권 (Trade)

- *특정 지리 통행료·관세*
- *특정 물건의 독점 + 시세 차익 운용*
- *강대국의 약소국 수수료 강제 징취*
- **대표 사례 — 헤븐즈 킹덤 8왕국 무역 동맹**: 바이서스 종주 + 데미안·강철제국 등 7왕국이 *명목상 평등 가맹*. 실제는 *바이서스가 모든 외부 무역 통과 시 5~10% 부가 수수료 징취*. 데미안의 마도공학 수출, 강철제국의 강철 수출, 마녀령의 마법 약초 수출 모두 *바이서스 통과·인가가 필요*

### (3) 기술 독점권 (Technology Patent)

- *마법공구·마법 물건의 특허 보호*
- 특허료·로열티
- *특허 유효 기간 = 발명자 사후 50~100년 (왕실 인가 시 연장 가능)*
- **대표 사례 — 데미안의 마도공학 특허**: 데미안의 *마법 회로 결각 정통 + 룬 결각 정통*은 대륙 1위. 그러나 *바이서스가 헤븐즈 종주국 매개로 데미안 특허의 30~40%를 강제 공유 또는 저가 매입*. *블링크 반지·부스터 부츠 등 데미안 시제품을 바이서스가 사적 라인 통해 입수했던 것도 이 매개*
- **앨더스톰의 룬 마법 특허**: 외관상 비공식·실제는 대륙 최고 정통. 바이서스의 직접 강탈 시도는 *드워프 비공식 동맹망 + 마녀령 외곽 보호망*으로 *부분 회피*

### (4) 생산 (Production)

- *강철 무기·방어구 생산*
- *마법 도구·아티팩트 생산*
- *잡화·일상품 생산*
- **대표 사례 — 강철제국의 강철 생산권**: *대륙 최고 강철 정통 + 의수 제작 정통 + 마법 거부 검술 정통*. 강철 무기·방어구의 대륙 1위 생산국. 그러나 *수출 시 헤븐즈 무역 동맹 수수료를 바이서스에 5~8% 납부*

### (5) 광산권 (Mining)

- *강철·각종 보석 원석·원자재 채굴·채집권*
- 광산 1곳의 *연 산출·세금·노동력 매개*
- **대표 사례 — 헤븐즈 킹덤의 광산 강제 편취 정통**:
  - **카르샤 산악 광산** — 원래 카르샤 왕가 직속. *바이서스 헤븐즈 종주 매개로 *연 산출의 20~30%를 형제 동맹세*로 강제 납부*
  - **데미안 룬 광산** — 카보나이트·일부 마나석 산출. *바이서스가 *특허 매개로 산출의 일부 우선 매입*
  - **앨더스톰 산령** — 태양석 광산 일부. *드워프 부족의 비공식 거래로 보호되어 바이서스 직접 편취 불가*
  - **마녀령 외곽** — 월광석 일부. *모르간 가문 외곽 분파가 비공식 운영. 바이서스 매개 부분 편취*
- *드워프 지하세계 = 광산권의 대륙 최정점*. *외부 인간 국가의 직접 편취 절대 불가*. *비공식 거래·미래 약속·은혜 양도로만 산출 이동*

### (6) 마법 부여·신성 부여 (Enchantment / Blessing)

- *2차 가공 무역*. *일반 무기·방어구에 마법력·신성력을 부여하여 가치 향상*
- 사례:
  - **블링크 반지** — 짧은 거리 공간 이동 부여
  - **충격 완화 갑옷 마법 부여** — 가슴 보호구·전투 갑옷
  - **힘 강화 부여** — 일반 검에 검기 운용 보조
  - **신성 결계 봉지** — 카르샤 왕가 비전 (비매매 영역)
  - **신성 가호 부여** — 카르샤 신성 사제단의 *대규모 부여 정통*
  - **룬 결각 부여** — 앨더스톰·드워프 정통
- **수익 정통**: 일반 검 1자루 ≈ 5~10 금환 → 마법 부여 검 1자루 ≈ 50~500 금환. *부여 = 가치 10~100배 증가*
- **이권의 정점**: 일반 인구는 부여 무기·방어구를 못 만들 → *왕실·고급 학회·신성 사제단이 독점*

## 2. 헤븐즈 킹덤 — 대륙 이권 강제 편취 체계의 정점

### 헤븐즈 킹덤 본질

- *명목상 = 헤븐즈 8왕국 평등 동맹*
- *실제 = 바이서스 종주국 + 7왕국 종속 + 약소국 1차 편취 + 약소국 2차 편취*
- **본작 시점 (CC 1000~1001)**: 라카르 9세 시기 = *바이서스 권력 정점 + 헤븐즈 동맹 통제 본격*

### 6대 이권의 바이서스 편취 비율

| 이권 | 바이서스 편취 정통 | 대상 |
|------|---------------------|------|
| 상업권 | 셀라리아 중개 5~15% | 모든 대륙 무역 |
| 무역권 | 헤븐즈 동맹 부가 수수료 5~10% | 데미안·강철제국·카르샤·앨더스톰 등 |
| 기술 독점권 | 데미안 특허 30~40% 강제 공유/저가 매입 | 데미안 |
| 생산 | 강철 수출 수수료 5~8% | 강철제국 |
| 광산권 | 카르샤 광산 형제 동맹세 20~30% | 카르샤·데미안 일부 |
| 부여 | 자체 마법 학회 + 데미안 부여 기술 흡수 | 자체 + 데미안 |

### 약소국의 짠내

- **카르샤**:
  - 신성 사제단·신성 정통·산악 광산 = 약점 적음
  - 그러나 *광산 형제 동맹세 20~30% 강제 납부* + *해외 무역 시 바이서스 통과 수수료 5~10%* = 국가 GDP의 *15~25% 강제 편취*
  - 본작 시점 카르샤 결전 결단의 *경제적 배경의 진짜 매개*
- **데미안**:
  - 마도공학 정통 대륙 1위
  - 그러나 *특허 30~40% 강제 공유* + *바이서스 본국·왕실 우선 공급* + *외부 수출은 바이서스 인가 필요*
  - *비공식 망명·정보 양도 라인만 비공식 운영*
- **강철제국**:
  - 마법 거부 정통 + 강철 정통
  - *수출 수수료 5~8% 매개 + 외교적 중립 유지*
  - *비공식 동맹은 마녀령·드워프 라인으로 분산*
- **앨더스톰**:
  - *외부에 룬 정통을 노출하지 않음 + 드워프 비공식 동맹망 보호*
  - 바이서스의 *직접 편취 가능성 매우 낮음* — 그러나 *공식 군사 동맹은 거부 (제제 두려움)*
- **마녀령 동부국**:
  - 9혈 마법 정통 + 외곽 분파의 비공식 산출
  - *모르간 가문 외곽 분파 일부 편취 + 본가는 자기 보호*

### 바이서스 약화 시 — 이권 재분배의 본격 정통 ★

본작 캐논 (Ch2 P8 결전 종결 후 적용):

1. **상업권** = 셀라리아 중개 = *바이서스 왕권 다툼 + 쿠데타 + 셀라리아 시정 마비*로 *수수료 징취 능력 즉시 마비*. *대륙 상인 길드들이 *다른 중심 도시 (카르샤 세이렌·데미안 본수도·앨더스톰)로 중개 라인 분산*
2. **무역권** = 헤븐즈 동맹 부가 수수료 = *주변국이 일제히 거부 + 자체 무역 정통 회복*
3. **기술 독점권** = 데미안 특허 강제 공유 = *데미안이 즉시 회수 + 자기 시장 회복*
4. **생산 수수료** = 강철제국 = 즉시 회수
5. **광산권** = ★ **카르샤 광산 형제 동맹세 즉시 회수 + 카르샤 주권 복원의 *경제적 정통 매개***. 카르샤가 결전 패배에도 *주권 회복 가능한 배경*
6. **부여 시장** = 바이서스 자체 학회 마비 → 데미안·앨더스톰으로 시장 이동

**결과 = 바이서스 헤븐즈 종주국 체계의 본질적 해체**. *멸망은 아니나, 헤븐즈 8왕국 정통은 사실상 와해*. *주변국의 빠른 이권 침탈 + 카르샤·데미안의 자기 산업 복원이 동시 진행*

## 3. 라이세린의 사적 이권 시스템

### 라이세린 데오릭 공작의 비공식 권력 라인

- **외형**: 외교실 차관 + 카르샤 사절 외교 담당
- **실제**: *바이서스 내 사적 이권 운용의 *대륙 최대 운용자 중 1인*

### 라이세린의 사적 이권 4중 라인

1. **외교 비밀 정보 라인** — 카르샤 외교·신성 사제단 내부 정보 흡수 → 셀라리아 정치판 활용 + 자기 자금 운용 매개
2. **라카르 9세 형제·자식들의 *비공식 후원자 라인*** — *왕가 핏줄의 분파별 후원으로 *각 자식이 미래 왕위 다툼 시 자기에게 빚지게 만듦*. *Ch2 P8 결전 후 왕권 다툼의 *분파 라인 다수가 라이세린 후원 사슬에서 출발한 자들*
3. **카엘 양육·후원 라인** — *카엘의 외교실 자문·연구소 자금·강연 출판의 *후원자 위치*. *카엘 = 라이세린의 사적 자산*. *Ch1 P4 강연 자금 + Ch1 P5 1회성 조커 카드 운용도 이 라인의 직접 표현*
4. **마도공학·아티팩트 사적 거래** — *블링크 반지 같은 *비싸고 구하기 힘든 고대 마도공학 유물*의 *사적 라인 거래*. *카엘 본인이 사재 1/4 들여 구입한 블링크 반지도 *실은 라이세린 사적 라인 거상을 통한 거래*. *수수료 일부가 라이세린에게 환류*

### 라이세린의 권력 본질

- *왕실 마법 고문·전권 사령관 같은 *공식 직위는 회피*
- *외형은 외교실 차관 + 셀라리아 평민 친화 + 종교적 광신*
- *실제는 *왕가 분파의 후원자 사슬 + 사적 이권 운용의 정점*
- **본작 시점**: *라카르 9세의 *진정한 약점 = 라이세린의 사적 이권 사슬에 *왕가 자식들이 묶여 있음*. *라카르 9세 본인이 *힘이 없는 왕*인 진짜 사유

### 라이세린의 죽음 (Ch2 P8 결전 종결 시점)

- 라이세린 척살 시 *그의 사적 이권 사슬이 즉시 해체*
- *후원받던 왕가 자식들 = 후원자 사망으로 즉시 자기 자금원 상실 + 다른 분파에 의존 또는 자기 권력 다툼 본격 진입*
- **이는 Ch2 P8 결전 후 *왕권 다툼·국력 쇄락의 직접 매개***

## 4. 각 국가별 상업·이권 시스템 매핑

### 바이서스 헤븐즈

- **주력**: 상업권 (셀라리아 중개) + 무역권 (헤븐즈 동맹 수수료) + 기술 흡수 (데미안 특허) + 자체 마법 학회 부여
- **약점**: *모든 이권이 *바이서스 중심 통제 매개에 의존*. *통제 마비 시 즉시 와해*
- **본 전쟁 후 운명**: 결전 → 왕 척살 → 쿠데타 → 통제 마비 → 이권 와해 → 헤븐즈 동맹 사실상 해체 → *바이서스 = 헤븐즈 6왕국 중 한 평범한 왕국으로 축소*

### 카르샤 신성제국

- **주력**: 신성 사제단 (대규모 부여·축복) + 산악 광산 (마나석·은·금 일부) + 신성 결계 봉지 등 비매매 비전 아티팩트
- **약점**: 군사력 + 동맹망 의존
- **본 전쟁 후 운명**: 결전 패배에도 *바이서스 헤븐즈 동맹세 즉시 회수 → 주권 복원 + 광산 회복 + 신성 정통 재정비 → 약 1~2년 내 평시 수준 회복*

### 데미안

- **주력**: 마도공학 특허 + 기술 정통
- **본 전쟁 후 운명**: *특허 회수 + 자기 시장 즉시 회복 + 앨더스톰·드워프 연계 강화*

### 강철제국

- **본 전쟁 후 운명**: *수수료 즉시 회수 + 외교 중립 유지*

# Part B — 화폐·보석


본작 리투니아 대륙의 *화폐 체계 + 금환 정통 + 보석류 가치 체계 + 마나·기계·검 부속용 보석의 특성과 거래 정통*.

대규모 거래·국가 자산·왕실 보고는 *금화/금환*보다 *고가치 보석류*가 압도적으로 선호됨. 사유 = *무게·휴대성·은닉성·가치 밀도*의 4중 우위.

---

## 1. 화폐 체계

### 동전 단위 (일반 통화)

| 단위 | 재료 | 일반 가치 환산 | 사용 |
|------|------|---------------|------|
| 동화 (Copper) | 구리 합금 | 1 동화 | 일상 식료·소액 거래 |
| 은화 (Silver) | 은 + 미세 합금 | 100 동화 | 정규 상점·임금·여관비 |
| 금화 (Gold Coin) | 금 + 합금 | 100 은화 = 10,000 동화 | 대형 거래·세금·정규 군 임금 |
| **금환 (Gold Ingot)** ★ | 순금 (둥글고 넓적한 큰 덩어리) | 50~100 금화 = 50~100만 동화 | 왕실 자산·대형 무역·전쟁 자금·사적 비공개 보관 |

### 금환 (Gold Ingot)의 정통

- *금을 둥글고 넓적하게 단조·압축한 큰 덩어리*. 일반 금화보다 *순도가 매우 높음 + 무게당 가치가 안정적*
- 1 금환 ≈ 50~100 금화 가치 (왕실 발행 기준). 표면에 *발행 왕가의 인장 압인*
- 사적 비공개 보관·대형 자산 이전·전쟁 자금 운용의 *주력 매개*
- **그러나 단점**: *무거움 + 부피 + 인장의 출처 추적 가능성*. *대규모 자산을 *원거리 휴대·은닉·국경 통과해야 하는 경우*에는 *보석류로 환전*하는 것이 정통

### 환전 정통

- 정규 환전상 = 왕실 인가·세금 추적 대상
- **비공식 환전상** = *왕가 비공식 라인·길드 비공식 매개·외곽 거상의 사적 라인*. *세금·인장 추적 안 됨*. *수수료 10~30%*. *카엘의 Ch1 P5 §10-E 환전이 이 경로*

---

## 2. 보석류의 가치 정통

대규모 자산의 *최종 저장 매개*는 *보석류*. 4중 우위:

1. **휴대성 우위** — 금환 1개 무게 = 보석 50~100개 휴대 가능
2. **가치 밀도 우위** — 같은 부피에 *100~1,000배 가치*
3. **은닉성 우위** — *옷솔기·반지·장신구·의수 회로*에 자연 위장 가능
4. **출처 추적 회피** — 보석은 *왕가 인장이 없음*. *연마·재가공으로 원천 위장 가능*

### 보석 가치 결정 4축

- **희귀성** — 채굴·수입 가능 빈도
- **특성 (Property)** — 마나 응축 / 마나 전도 / 단단함 / 가벼움 / 신성 친화 / 정신 보호 등의 *기능적 가치*
- **순도·결정 품질** — 결각·재가공 가능 여부
- **수요** — 왕실·마법사·마도공학·신성 사제단·기계 정통의 *직접 사용처*

본작 캐논 = **특성**이 가치의 *주력 결정 요소*. *희귀성보다 어디에 쓸 수 있느냐*가 정통.

## 3. 보석 카탈로그

### 마나 응축 가능 보석 (대륙 정통 — 단 2종만)

> **본작 정통**: 대륙에서 *마나를 응축·저장할 수 있는 보석은 단 2종*. 다른 보석은 *전도·증폭·반사·차폐는 가능하나 응축 자체는 불가*.

#### 1) 태양석 (Sunstone) ★

- **특성**: *마나 응축 최대 정점*. 1개당 *5~9써클 마법 1발 분량의 마나를 저장 가능*
- **희귀성**: 대륙 최희귀. *드워프 지하세계 + 앨더스톰 비공식 거래 매개로만 입수 가능*
- **주요 사용처**:
  - **후반 타이탄 (기계로봇) 주축전지** ★ — *본작 후반 등장 인간형 거대 기계 무력의 *유일한 마나 동력원*. *타이탄 1체당 태양석 3~7개 소비*. 대륙에 *수십 체 미만의 타이탄이 존재하는 결정적 사유 = 태양석 부족*
  - 9써클 대마도사의 *대규모 광역 시전 보조 매개*
  - 왕실 비밀 보고의 *전략 자산*
- **가치**: *돈으로 환산 불가 — 보통 외교적 거래·부족 의뢰·왕가 비전 교환으로만 이동*. 굳이 금환 환산 시 *1개 = 1만~5만 금환 (수도 1년 운영비 수준)*
- **물리 특성**: 햇빛 받으면 미세 황금 빛 발산. 결각 시 *위험 — 응축 마나 폭발 가능성*
- **본작 등장**: Ch2 P5 드워프 부족 거래·Ch2 P6 마르덴 양도 미래 시드·Ch3+ 타이탄 본격

#### 2) 월광석 (Moonstone)

- **특성**: *마나 응축 중급*. 1개당 *3~5써클 마법 1발 분량의 마나를 저장 + 천천히 회복 (체내 마나·주변 마나 흡수)*
- **희귀성**: 중상급. *데미안·마녀령 일부 광산에서 채굴*. *연 채굴량 한정 — 왕실·고급 마법사·아티팩트 장인 우선 분배*
- **주요 사용처**:
  - **개인 아티팩트 매개** — *마나 회복 반지·마나응축 장갑·비상 회복 캡슐 등의 주재료*
  - **카엘의 *마나응축 보석 장갑 (Ch1 P4 1년 자기 제작) = 큰 월광석 1개 + 보조 회로 결각*** ★
  - 마법사·마검사의 *전투 중 마나 비상 회복 매개*
- **가치**: 큰 결정 1개 = *50~200 금환*. 카엘의 보석 장갑에 사용된 등급 = *상급 결정 (약 100~150 금환 가치 + 1년 자기 제작 인건비)*
- **물리 특성**: 달빛에 미세 푸른 빛 발산. 결각 안정성 양호 — 카엘 같은 5써클 마법사가 자기 제작 가능
- **본작 등장**: Ch1 P4 카엘 자기 제작 + Ch2 P1 절단 왼팔 회수·판매 (보석 미세 균열·판매 가치 유지)

### 단단함 우위 보석 (검·기계 부속용)

#### 3) 다이아몬드 (Diamond)

- **특성**: *대륙 최강 경도*. *왠만한 강철보다 단단함*. *마나 응축·전도 모두 약함 — 순수 물리 정통*
- **희귀성**: 상급. *드워프 지하세계·일부 외곽 광산 채굴*
- **주요 사용처**:
  - **검의 주요 부속** ★ — 검날 가장자리 박힘·검자루 핵심 결정·검 끝 결각으로 *경도 극단 향상*. *소드 마스터급 검 다수 = 다이아몬드 결각 매개*
  - 강철·아다만티움 의수의 *경도 보강 부속*
  - 마도공학 무기의 *압력 견딤 핵심 부품*
- **가치**: 결정 크기에 따라 *5~100 금환*. *검 부속용 다이아몬드 등급 = 30~80 금환 (1~2캐럿 기준)*
- **본작 등장**: Ch1 P5 카엘 마검사 검 2자루 부품 (다이아몬드 결각) + Ch2 P5 드워프 아다만티움 의수 보강 부속

#### 4) 카보나이트 (Carbonite) ★ — *카엘 갈무리 보석*

- **특성**: ★ 본작 정통 핵심 보석
  - **매우 가벼움** — 같은 부피 다이아몬드의 *1/5 무게*. *대규모 가치 휴대의 최적 매개*
  - **매우 단단함** — *다이아몬드보다는 약하나, 일반 강철의 2~3배 단단*. *충격·압력에 안정*
  - **마나 저장 불가** — *응축은 절대 안 됨*
  - **마나 전도 매우 우수** — *대륙 최고 수준 마나 전도율*. *룬 결각 매개·마법 회로 전도 매개·아다만티움/미스릴 보강의 핵심 부속*
- **희귀성**: 중상급. *드워프 지하세계·일부 외곽 광산*. 다이아몬드보다 흔하나 *마나 전도 특성으로 수요 매우 높음*
- **주요 사용처**:
  - **대규모 자산 휴대·은닉의 *최적 매개*** ★ — *왕실·귀족·거상의 사적 비공개 자산 보관 표준*. *옷솔기·반지·장신구·의수 회로에 위장 가능*. *국경 통과·망명·도주 시 핵심*
  - **마법공학 무기·아티팩트의 마나 전도 부속**
  - **룬 결각 매개** (앨더스톰·드워프 지하세계 정통)
  - **카엘의 아다만티움+미스릴 의수의 *룬 회로 전도 부속 일부*** (Ch2 P5 시드)
- **가치**: *원석 1개 = 10~50 금환*. *연마·결각 후 = 50~200 금환*. **가치 밀도 우위**: *카보나이트 결정 100개 ≈ 금환 5,000~10,000개 = 수도 평민 1만 가구의 1년 생활비 수준*
- **물리 특성**: 무색 또는 옅은 청회색. 빛 통과 시 *희미한 푸른 잔상*. *결각 안정 매우 우수*
- **본작 등장**: ★ **카엘의 Ch1 P5 §10-E 환전·갈무리 보석 = 카보나이트 (작은 결정 다수, 옷솔기 봉합 보존)**. Ch2 P1 품 무사 확인·Ch2 P2 노자돈 점검의 핵심 자산. Ch2 P5 의수 룬 회로 일부 매개로도 후속 사용

### 친화·신성·정신 보호 보석

#### 5) 루비 (Ruby)

- **특성**: *화염 친화 + 정신 응시 + 악의·살의 감지*
- **희귀성**: 중급
- **주요 사용처**:
  - **호신 아티팩트** — 악의·살의 감지·붉은 빛으로 경고
  - **카르샤 왕가 비전 *붉은 보석 팔찌* = 루비 핵심 + 신성 결각의 융합** ★ (Ch2 P1 시드 회수 — 세파 → 카엘 양도)
  - 화염 마법사의 시전 보조
- **가치**: 결정 크기·순도에 따라 *5~50 금환*. *왕가 비전급 = 100+ 금환 + 신성 결각 추가 가치*

#### 6) 에메랄드 (Emerald)

- **특성**: *신성 친화 + 치유 보조 + 자연 마나 친화*
- **희귀성**: 중급
- **주요 사용처**:
  - 신성 사제단의 *치유 보조 매개* (대규모 힐 효율 향상)
  - 신성 결계 봉지의 *결정 핵심 일부*
  - 자연 마법사·드루이드 정통 시전 보조
- **가치**: 결정 크기에 따라 *5~40 금환*. 카르샤 신성 사제단의 *대규모 수요*

#### 7) 사파이어 (Sapphire)

- **특성**: *정신 보호 + 사고 안정 + 잔향·금제 부분 차폐*
- **희귀성**: 중상급
- **주요 사용처**:
  - 정신 마법 시전자의 *시전 안정 매개*
  - 의수 회로의 *정신 안정 부속* (의수 마나 운용 시 시전자 정신 흔들림 차단)
  - *앤트로퍼 잔향·금제 부분 차폐 매개* — *완전 차폐는 불가능. 부분 약화*. 카엘의 Ch3+ 자기 정통 시드
- **가치**: 결정 크기에 따라 *10~60 금환*

#### 8) 검은 옵시디언 (Black Obsidian)

- **특성**: *어둠·엔트로피 친화 + 마기 흡수 부분 + 살상 충동 매개*. **금기 보석** — 대륙 종교·왕실 다수에서 금기 분류
- **희귀성**: 중상급. *외곽 비공식 채굴 또는 드워프 지하 일부*
- **주요 사용처**:
  - 흑마법사·마계 정통의 *시전 매개* (금기)
  - 일부 마기 봉인·차폐 결각 (역설적 활용)
- **가치**: 거래 자체가 *대륙 다수 국가에서 불법*. *비공식 가격 = 50~200 금환* (위험 수당 포함)
- **본작 등장**: Ch3+ 마계 영지 군주 출현 + 첫 마계 영지 정통의 시드

#### 9) 소울 가넷 (Soul Garnet)

- **특성**: *영혼 친화 + 신성 사제 매개 + 죽음 직전 영혼 정착 보조*
- **희귀성**: 중급
- **주요 사용처**:
  - 신성 사제단의 *임종 의례 + 영혼 안정 매개*
  - 망자 추모 아티팩트
- **가치**: 결정 크기에 따라 *5~30 금환*. 카르샤 신성제국 대량 수요

## 4. 본작 카엘 자산 정통 (보석 체계 적용)

### Ch1 P4 (5년 셀라리아 연구소 시기) 아티팩트 정통

- **블링크 반지** = *비싸고 구하기 힘든 고대 마도공학 유물*. 사재 1/4 매개 사적 구입. *구입가 = 약 5,000~8,000 금환 수준 (대마법사 1인의 5년 봉급에 해당)*
- **마나응축 보석 장갑 (왼손)** = *상급 월광석 1개 + 카엘 자기 회로 결각 1년 + 부속 보조 (작은 사파이어 1~2개 정신 안정용)*. *재료비 + 1년 인건비 + 회로 비전 = 약 200~300 금환 가치*
- *카엘 5년 누적 재산 = 약 5만~8만 금환 추정* (외교실 자문 봉급 + 연구소 봉급 + 라이세린 후원 + 강연 수익). *블링크 반지 + 보석 장갑 + 자기 제작 아티팩트 3종에 약 1/3 소비*

### Ch1 P5 §10-E 환전·갈무리 = *카보나이트 결정 다수* ★

- **갈무리 보석 = 카보나이트** (사용자 정통 결정)
- *비공식 환전상 매개*로 *비공개 보관 재산 약 2만~3만 금환 → 카보나이트 작은 결정 약 200~400개로 환전*
- *옷솔기 안쪽 봉합 보존* — *부피 작음·무게 작음·왕가 인장 없음으로 추적 회피*
- *카보나이트 = 옷자락 작은 봉합 주머니 다수에 분산*. *바이서스 압수 대상에서 누락*
- 카엘 짧은 한 줄 — *…내가 정의로 부려졌을 때 받은 것들을, 이제 다른 형태로 가져간다*

### Ch2 P1 회수·확인

- *품 안쪽 카보나이트 결정 무사 확인* — 200~400개 분산 봉합이 *낙하 충격에도 흩어지지 않음 (카보나이트 자체 단단함 + 옷솔기 분산 봉합)*
- *왼팔 마나응축 보석 장갑 = 월광석 핵심 + 작은 사파이어 부속 → 비공식 중개상에 판매*. 판매 대금 ≈ *100~150 금환 (보석 미세 균열·재가공 필요 차감)*

### Ch2 P2 노자돈 점검

- *카보나이트 보유 잔량 + 보석 장갑 판매 대금 = 강철제국 단독 이동·체류·검술 학습 1차 비용 + Ch2 P3 그라스 노 장인 시술 보증금 + 사적 거래 매개의 본격 자금원*
- 카보나이트 일부를 *세파에게 비공식 양도* — 키얀 호위·망명지 운영비 보조

### Ch2 P5 의수 매개

- 드워프 부족 거래 시 *카보나이트 결정 다수가 마나석 양도의 일부로 함께 양도될 수 있음* (선택지). 또는 *카엘 자기 보유 유지 + 의수 룬 회로 일부에 직접 결각 사용*
