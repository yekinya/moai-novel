## 1. 본작 핵심 아티팩트 — *7비전의 도구*

사프란이 7제자에게 분할 봉인한 비전이 각각 *물건* 형태로 보존됨. 본작 진행 중 7개 중 4개가 카엘 주변에 모인다.

### 1.1 루멘브링거 (Lumenbringer) — 카르샤 왕가 비전 검

- **유형**: 신성검. 양손 바스타드 소드. 길이 1.4m
- **재질**: 은백색 강철 + 황금 룬. 코르넬리스 1세가 *알레이엔 1세의 검 잔재* 로 제작
- **결박된 마법**: *균열의 빛 축소판* — 왕대 1회만 발동 가능 (검의 결박을 푸는 행위)
- **시전 시간**: 일반 균열의 빛 = 3시간 / 루멘브링거 시전 = *3분*
- **단점**: 1회 발동 후 *검 자체가 평범한 강철로 변함*. 다음 왕대까지 *수십 년 봉인 의식* 필요
- **본작 등장**: 프롤로그 2합에서 *마나검 vs 루멘브링거* 5합 교차. *균열의 빛 즉발판* 은 검 결박과 *별도 시전* (알레이언이 *생명 매개*)
- **현재 위치 (CC 1000)**: 알레이언 사망 후 옥좌의 방. 카엘이 *건드리지 않음*. **6부 이후 키얀 왕자가 회수 가능성**

### 1.2 부친의 책 — 『고대마법총론』 발췌본 (변형)

- **유형**: 양피지 묶음 책 1권. 약 200페이지
- **출처**: 카엘의 친조부 (그라스 카르엔 학파 잠적 분파) 의 *수기 사본*
- **내용**: 『고대마법총론』 1·2·3·7권의 *부분 발췌* + 친조부의 *수기 주석*
- **표지**: 지워진 마탑 문장
- **여백의 비밀**: 같은 문장 *세 번* 다른 필체로 옮겨 적힌 흔적 — *자기 자신을 설득하려 한* 친조부의 손
- **본작 등장**:
  - 1-4 알린이 카엘에게 전수
  - 1-5 폐허 마탑에서 첫 마나 점화 — 첫 페이지 *"세상을 미워하는 자는 마나를 다루지 마라"* 를 *비겁한 변명* 으로 오독
  - 6-2 그라스에서 *책의 진짜 출처* 회수
  - 후반부 *친조부의 주석을 다시 읽음* — 자기 핏줄에 대한 새 이해

### 1.3 카르엔의 봉인 문서 — *학파 비전*

- **유형**: 7장의 양피지 묶음. 카르엔 사이엔 자필
- **내용**:
  - 1장: 사프란의 본명·생애·죽음
  - 2장: 앤트로퍼 메커니즘 (붉은 사슬 7~9마디)
  - 3장: 이득과 충동 (마나 효율·살인 갈망)
  - 4장: 표적 우선순위 (드래곤·마왕·정령왕 → 인간 왕족 → 일반인)
  - 5장: 변질의 역사 + 16대 숙주 명단
  - 6장: 7비전 7조각의 위치
  - 7장: *마지막 카드 — 자결의 의미* (사프란 비망록 마지막 페이지 인용)
- **봉인**: 르세프 결계 봉인. 카르엔 사이엔의 사망 시 자동 해제 — *후계자가 펴기 전까지 무용지물*
- **본작 등장**: 5-2 카르엔 외곽 연구소에서 세파가 카엘에게 전달

### 1.4 카르엔의 미완성 경고 서한

- **유형**: 미완성 편지 1장. 잉크 다 마르지 않음
- **내용**: *"네가 보는 세계가 세계의 전부가 아니다. 너의 눈은 정직하지 않다. 너의 정직함은 더더욱 정직하지 않다. 부디 — 부디 한 번이라도 — 너의 분노가 *너의 분노가 아닐 수 있다* 는 가능성을 의심해다오."*
- **본작 등장**: 2-6 카르엔 사망 현장. 카엘이 *읽고 불태움*. 4-5에서 *기억 표면에 떠오름*

### 1.5 사프란 비망록 (마지막 페이지)

- **유형**: 작은 양피지 1장. 800년 보관으로 변색
- **내용**: *"만약 내 모든 설계가 변질된다면. 그때는 단 하나의 답만 남는다. 앤트로퍼 숙주 자신의 *자결*. 그 한 명의 죽음이 내가 만든 모든 시스템보다 더 큰 해소가 된다."*
- **보존자**: 카르엔 학파 잠적 분파 (그라스 가문)
- **본작 등장**: 6-2 그라스 의수 시술자가 카엘에게 마지막 페이지만 보여줌. *전체 비망록 7권* 은 별도 위치

### 1.6 9혈 흑혈 가문 회수 회로 (Recovery Circuit)

- **유형**: 7개의 룬 결정체 + 원형 의식 도해
- **보존**: 마녀령 흑혈 가문. 모르간 가문이 1세대로부터 7대 보존
- **상태**: *불완전* — 9혈 중 4혈 멸문으로 *결정체 3개 분실*
- **기능**: 앤트로퍼 사슬 *완전 해체* 의 *나머지 절반*
- **본작 등장**: 6부 후반 ~ 중반부 (마녀령 진입 시)

### 1.7 그라가르드 룬 도식 — 의수 비전

- **유형**: 마도공학 *룬 회로* 설계도. 데미안 마도공학원 + 그라스 비전 보존
- **본작 카엘 적용**: 6-2 의수에 *룬 회로 재설계*
- **함의**: 사프란의 6제자 그르긴의 비전이 *카엘의 신체에 결합* — 7비전 4번째가 *카엘 자신에게 들어옴*

## 2. 점멸·결계 매개 — 휴대 마법 도구

### 2.1 점멸 양피지 (Blink Scroll)

- **유형**: 룬어 새겨진 양피지 1매
- **크기**: 손바닥만 함
- **사용**: *찢는 순간* 시전. 5~10m 점멸
- **가격**: 1매 = 평민 한 달 수입
- **고급형**: 20m 점멸 — 평민 1년 수입
- **재사용 불가**

### 2.2 점멸 반지 (Blink Ring) — *비밀 카드*

- **유형**: 룬어 결박 반지 1개
- **사용**: 1일 *정해진 횟수* 만 발동. 보통 1~3회
- **가격**: 평민 평생 수입 +
- **특이성**: *공개 표시 없이* 발동 — 적이 모르는 카드
- **본작 의심자**: 라이세린 데오릭 공작 (보유 가설)

### 2.3 결계 부적 (Ward Charm)

- **유형**: 천 또는 종이에 룬 그림 + 마나 결박
- **종류**:
  - 광역 결계 — 5m 반경
  - 인체 결계 — 자기 자신만
  - 마계 결계 — 마계 창조물 전용 (신성마법)
- **가격**: 평민 한 달 수입
- **생산자**: 카르샤 신관단·셀라리아 르세프 본원교
- **재사용 가능**: 결계 발동 후 마나 회복까지 1시간

### 2.4 마나 코어 (Mana Core)

- **유형**: 작은 결정체. 마나 보관 매개
- **크기·가격**:
  - 작은 코어: 평민 1년 수입 / 마나 1회 보조 분
  - 큰 코어: 9써클 마법사 1년 수입 / 마나 5~10회 분
- **사용처**: 기사 갑주·검·도시 인프라
- **본작 등장**: 6-2 카엘 의수의 핵심 동력원

### 2.5 마나 측정기 (Mana Meter)

- **유형**: 작은 황동 원반. 마나 농도 표시
- **사용**: 슐로브스 마레 발명. 학자·셉터·자혈 마녀 표준 도구
- **본작 등장**: 5-2 카르엔의 책상 위에 *낡은 측정기 1개*

## 3. 무기 — 특수 검·갑주

### 3.1 카엘의 마도 의수 (Mage's Prosthetic)

- **유형**: 양 팔의 의수. 그라스 제작
- **재질**: 아다만티움 골조 + 강화 가죽 + 룬 회로
- **시술자**: 그라스의 노 장인 (이름 미정, 60대). 카엘 친조부와 같은 가문
- **마나 회로**: *새로 설계됨* — 기존 회로 7개 중 5개 단절 후, 2개 잔존 회로에 *우회* 연결
- **특이성**:
  - *공격* 보다 *방어·감지* 특화
  - *감응 정밀도 ↑* — 마나 잔향 직접 감지
  - *마법 공격 자동 결계* — 적의 마법이 닿기 직전 *0.5초간* 결계 발동
  - 단점: *원래 9써클 마법 시전 불가*. *3~5써클* 범위만
- **본작 등장**: 6-2 시술 → 6-7 결전 직전 *마지막 점검*
- **내포된 룬**: 그라가르드 기관왕의 갑주 룬 변형 — 7비전 4번째의 결합체

### 3.2 카이엔의 작별의 검

- **유형**: 동부국 마검술 양식. 한손검. 길이 1.1m
- **재질**: 흑색 강철 + 자수정 자루
- **새긴 문구**: *"인간으로 살게 해줘서 고맙다"* (6-7에서 카이엔이 황녀에게 남김)
- **특이성**: 카이엔이 *드래곤 본태 마나로 결박* 한 검. 인간형의 *마지막 잔재*
- **본작 등장**: 6-7 황녀에게 남김 → 후반부 *황녀가 주전파에게 들이댐*

### 3.3 강철제국의 다섯 검 — *식칼들*

5명의 마스터급 강철검사가 보유한 *황실 직속* 검.

| 보유자 | 검 별명 | 특이성 |
|--------|--------|--------|
| 칼 호르난 | *큰 식칼* | 가장 무거움. 황실 대장간 표준 |
| 가르 베르덴 | *베는 식칼* | 절단력 1위 |
| 베른헤르 라움 | *얼음 식칼* | 검기 빙결 응용 |
| 모르 셀란 | *그림자 식칼* | 검기 *환각 결합* — 검사형 셉터에 가까움 |
| 셀 라우엔 | *어린 식칼* | 가장 새 검. 1황자 카르덴 사부 검 |

### 3.4 베른하르트 강철왕의 *최후의 검*

- 역사적 보물. 강철제국 황실 보검. 일반 전투 불가 (의례용)
- 베른하르트가 신좌침묵 이후 50년간 사용. 의식적 *결정화* 된 부분 있음

### 3.5 일레인 맥필렌의 *룬어 팔찌*

- 9써클 마스터의 정통. *점멸 + 즉발 시전 결합*
- 일레인 사후 엘더스톰 황실 보관. 외부 사용 불가
- 일레인 카타르 백작 (방계 후예) 가 *모조 팔찌* 보유 (성능 70%)

### 3.6 무명의 *공허검 1세대 마스터의 검*

- 검은평원 전투에서 *적장 리치 바하르의 척추를 벤* 검. 린넬 허프 백작 회고록 등장
- 현재 위치 미상. *후반부 카드*

## 5. 마법공학 무기 — 사피엔 특이 응용

### 5.1 연사식 화염 발사기

- 데미안 마도공학 표준 보병 무기
- *아다만티움 + 화약 + 부싯돌* 결합
- 마법 재능 없는 평민도 운용 가능
- 1분당 *3발 발사*. 사거리 50m

### 5.2 용접기형 해머

- 데미안 마도공학 근접 무기
- 망치 형태이나 *지속 발열* — 적 갑주를 *녹여 관통*
- 마검사보다 효율적인 *근접 갑주 파괴* 도구

### 5.3 마나 자동 사격탑

- 미라돈 외성 방어 무기
- 적 진입 시 *자동 마나 측정 + 화기 발사*
- 도시 방어 표준

### 5.4 마나 펌프

- 도시 인프라. 우물·하수도·항만
- 미라돈 항만이 *세계 최대 마나 펌프 시스템* 보유

## 6. 일상 마법 도구

### 6.1 자동 필기 깃펜

- 시전자의 사고를 글로 옮김
- 학자·셉터 표준
- 카르엔 사이엔 서재에 *다수 보유* (대부분 5-2 직후 카엘이 가져감)

### 6.2 마나 잉크

- *거짓을 적으면 글자 사라짐*
- 카르샤 신성 재판 + 데미안 상인 계약 표준

### 6.3 녹음 결정 (Voice Crystal)

- 음성을 *수 시간 보관*. 데미안 상인 계약 표준
- 본작 후반부 *비밀 협약의 증거* 회수 가능성

### 6.4 마나 가로등

- 광명 마법 결박. 도시 야간 조명
- 셀라리아·미라돈·세이렌 표준
- 변경 도시는 *희소* — 가격 부담

### 6.5 결계 항아리

- 음식 *수개월 보관*. 미라돈 무역품

### 6.6 마나 차

- 약한 마나 결박된 찻잎
- *정신 집중 효과* — 셉터 표준
- 카르엔 사이엔이 카엘에게 처음 권한 차 = *마나 차*. *독으로 오인된 그것*

## 7. 신화·역사 보물 — 본작 미등장 또는 후반부

### 7.1 모티어스 매리앤의 *거인의 지팡이*

- 사피엔 최초 9써클 마스터의 상징
- 현재 국립상아탑 보관. 외부 출입 금지
- *후반부 카드*

### 7.2 행커만의 *원본 고대마법이론 원고*

- 행커만 자필 원본 1권
- 국제마법과학원 보관. 사본 392쇄
- 일부 학자: *행커만 본인이 *드래곤의 가르침* 을 *암호로 숨겼다* 는 가설*

### 7.3 알레이엔 1세의 *균열의 빛 원형 의식 도해*

- 루멘 신성제국 멸망 시 코르넬리스 1세가 *절반만 가지고 망명*. 나머지 절반은 마녀령 흑혈 가문
- 본작 후반부 *7비전 결합* 시 두 절반이 다시 만남

### 7.4 사프란의 비망록 (전체 7권)

- 그라스 카르엔 학파 잠적 분파 보존
- 6-2에서 *마지막 페이지만* 공개
- 후반부 *전체 7권 공개* 가능성

### 7.5 세계의 알 (World Egg)

- 리펠리아 잔재에 남은 거인족 마지막 유물
- 영구 봉인 상태
- *부화 시 새 거인족 출현* — 다음 리셋 가속의 *최악의 카드*
- 본작 종결부 *최종 변수*

## 9. 작품 활용 — 5원칙

1. **물건은 *인물 정체성의 연장*** — 카엘의 의수 / 알레이언의 루멘브링거 / 카이엔의 검
2. **물건의 *역사* 가 *현재* 를 만든다** — 부친의 책이 *6-2에서 출처 회수* 되는 구조
3. **물건의 *공간 이동* 이 *플롯의 이동*** — 카르엔의 서한이 *기억으로 남고*, 봉인 문서가 *진실로 옮겨감*
4. **물건의 *파괴 또는 소실* 은 무거운 비트** — 4-4 *루멘브링거의 결박이 풀린 검은 평범한 강철* / 2-6 *서한 소각*
5. **물건의 *재해석* — 같은 물건이 다른 의미로** — 1-5에서 *비겁한 변명* 으로 읽은 책 첫 문장이 4-5에서 *나에게 주는 경고* 로 재해석
