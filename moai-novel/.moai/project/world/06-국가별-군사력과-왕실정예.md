# Part A — 국가별 군사력


본작 리투니아 대륙 8왕국의 *국가별 군사력 정통*. 등급 체계 본격은 [`../systems/06-군사-등급과-교리.md`](../systems/06-군사-등급과-교리.md) 참조.

본 문서는 *전쟁 시점 (CC 1000~1001) 의 평시 군사력 기준*. 실제 출정 동원은 *왕가 직접 결단 + 외교 관계 + 동맹·후방 사정*에 따라 변동.

## 1. 바이서스 헤븐즈 (대륙 1위 군사 강국)

대륙 *최대 인구·생산력·마법공학 정통의 융합*. 라카르 9세 시대 = 군사력 정점기.

### 왕실 직속 호위대 (최정점 무력)

- **9써클 대마도사 1명 = *칼란 베르냐스 (Kalan Vernyas)*** — 95세·*위력 + 기동 융합 정통의 대륙 유일*. *공중부양 (5분 부유·30초 낙하) + 블링크 반지 2종 1분 반복 시전 + 바람 방어막 지속 + 온몸 마나 감응 (손·스펠 없이 시전) + 마법 양피지 15~25개 휴대 + 얼음 송곳·대형 얼음창·대지 마법 4중 패턴*. *전장 전면 출전 거의 없음 — 왕가 안전 결계 보호 최우선*. *왕실의 결단보다 자기 정통 우선 = 카엘과 직접 직면 거부 정통의 직접 표현*. 상세 = [`./bisus-royal-elite.md`](./bisus-royal-elite.md)
- **소드 엠퍼러급 1명 = *카르난 발도르 (Karnan Valdor)*** — 58세·바이서스 변경 발도르 가문·*마법 자질 없음 → 30년 검 수련 끝 엠퍼러급*. *온몸 검기화 폭발적 힘·스피드 + 한 검에 적중 시 몸 두동강 + 마법 시전 1초 이내만 회피 가능 (그 이상 = 접근·목 날림) + 검은 검 (검기 응축 정점 일격, 일반 강철 동강·아다만티움 조각) + 카보나이트 합금 갑옷 (경량·마나 주입 가능·반사) + 검기 단검 던지기 (1.5초 유지·자동 추적 보조)*. *라카르 9세의 어린 시절 호위·35년 평생 충성*. 상세 = [`./bisus-royal-elite.md`](./bisus-royal-elite.md)
- **소드 마스터 5인** — 왕실 직속 호위 + 왕가 핵심 인물 호위 분배 + 카르난 보조 매개
- **8써클 마도 거장 2~3명** — *호스반 셰이드런 (전쟁 참모) 포함*. 라이세린 데오릭은 *마법 직접 시전보다 정치·자금·정보 매개*. *호스반이 바이서스 8써클의 실질 전쟁 운용자*

### 기타 정규 군

- **소드 마스터 100명 이상** — *각 군단·지역 사령부 지휘관*. *대륙 마스터 보유 1위*
- **익스퍼트급 정규 기사 약 5,000명** — *각 부대 장교·정규 호위*
- **일반 검사 약 12만 명** — 정규 보병
- **일반 마법사 약 10만 명** — *대부분 3~4써클*. *5써클 이상은 약 500명 수준 (왕실 인가 매개)*
- **마검사 약 50~80명** — *5써클 + 익스퍼트 융합 정통*. 변칙 부대 운용
- **6~7써클 전쟁 마법사·대마법사 약 30~50명** — *각 전선 마법 총책*

### 마법공학·아티팩트

- **대규모 공성 무기 다수** — 마력 공성포·연발 발사기·이동식 결계 발생기·마법 차단 결계 등
- **아티팩트 보유량 대륙 1위** — *결계용·증폭용·전략 차단용 다수*
- **마도공학 시제품** — 데미안 + 자체 개발 + 비공식 룬 결각 등 다양

### 신성 사제단 (취약점)

- *바이서스 = 세속 왕권 중심 + 신성 권위 약화 정통* (자세한 사유는 [`../systems/06-군사-등급과-교리.md`](../systems/06-군사-등급과-교리.md) §4 + [`./eight-kingdoms-modern.md`](./eight-kingdoms-modern.md) 종교 항목 참조)
- *신성 마법 위계는 4단계 신부급이 정점*. *대규모 힐·축복 사실상 불가능*
- *전장 회복은 마법공학 회복 매개 + 약품 + 익숙한 야전 의술로 보완*. **결과 = 군의 지속 가능성·사기 약점**

### 동맹·후방

- **헤븐즈 8왕국의 비공식 종주국 정통**. *공식 동맹은 약하나 다른 왕국이 바이서스 적대 결단을 두려워함*
- **길드 용병 약 5만 명 추가 동원 가능** — 비용 매개. *전쟁 장기화 시 핵심 추가 자원*

### 약점 종합

- 신성 사제단 약화 → 지속 가능성·사기
- 9써클 1명에 과도 의존 → 잃으면 마법 정통 즉시 약화
- 정치적 분열 잠재 — 라이세린·호스반·왕가의 *세 축이 표면 결속 + 본질 분열*. *Ch1~Ch2 본격 노출*

## 2. 카르샤 신성제국 (정통 2위·실전 군사력 4~5위)

*신성 권위의 대륙 정점*. 그러나 *군사력 자체는 바이서스 비교 약함*. 동맹망·신성 가호·왕가 결단으로 균형.

### 왕실 직속 호위대

- **소드 엠퍼러급 1명** — *카르샤 왕실 친위대장. 본작 시점 (CC 1000~1001) 알레이언 6세 직속*. *바이서스 엠퍼러급과 동급 추정*
- **소드 마스터 3인** — *왕가 핵심 인물 호위 분배*
- **9써클 마법사 없음** — *카르샤는 마법 정통이 신성 사제 정통으로 흡수됨*. *대주교의 신성 위력은 8써클 광역 마법 수준의 전장 영향력*
- **8써클 마도 거장 약 1~2명** — *대부분 신성 사제 정통과 융합된 변종*

### 기타 정규 군

- **신성 기사단 약 300명** ★ — *순수 검술 단계는 익스퍼트급이나 신성 가호·축복 무장 매개로 *전장 영향력은 마스터급에 근접*. *대규모 진형에서 마스터 1인 분량의 영향*. *카르샤 군의 결정적 자산*
- **일반 검사 약 4만 명** — 정규 보병
- **일반 마법사 약 3,000명** — *카르샤 마법사 수 자체가 적음 — 마법 정통이 신성 정통으로 흡수됨*
- **신성 사제단 약 1,000명** ★ — *대주교 1인 (셀라엔) + 주교 다수 + 신부·수도사 다수*. *대규모 힐·축복·결계 방어의 전장 결정적 자산*

### 마법공학·아티팩트

- *마법공학 정통은 약함 — 신성 정통 중심*
- *신성 결계 봉지·축복 의식구·신성 무장 다수* (마법공학과는 별도 정통)
- *왕가 비전 호신 아티팩트* (붉은 보석 팔찌·신성 결계 봉지·해독 약초 정통)

### 동맹·후방 — *본 전쟁의 핵심 문제*

- **외부 동맹국들의 군사 보조 동맹에 의존하는 군 구조** ★
- 평시 = 동맹망이 카르샤 군의 약 30~40%를 보충
- **본 전쟁 (CC 1000~1001 바이서스 침공)의 결정적 약점**:
  - 바이서스가 *위조 첩보 + 카르샤 악마화 강연 + 광범위 정치 압박*으로 *카르샤 동맹망을 사전 마비*
  - **주변국 = 바이서스의 제제·보복이 두려워 *공식적 동맹 참전 거부***. *비공식 망명 수용 정도만 허용 (셀라엔 망명지 라인)*
  - 결과 = 카르샤 본국 = *동맹 보조 30~40% 부재 + 자국 군사력만으로 결전*

### 약점 종합

- 일반 마법사·일반 검사 수 절대적으로 부족
- 동맹망 의존도 높음 → 동맹 마비 시 *실전 군사력이 평시의 60~70%로 급감*
- 마법공학·공성 무기 정통 약함 → 공방전에서 불리

## 3. 본작 CC 1000~1001 바이서스 vs 카르샤 전쟁 정통

### 객관적 군사력 격차

| 항목 | 바이서스 | 카르샤 |
|------|---------|--------|
| 9써클 | 1명 | 0명 |
| 소드 엠퍼러급 | 1명 (근사치) | 1명 |
| 소드 마스터 | 5인 직속 + 100명+ 정규 | 3인 직속 (신성 기사단 300명 보조) |
| 8써클 | 2~3명 | 1~2명 (신성 융합 변종) |
| 일반 검사 | 12만 명 | 4만 명 |
| 일반 마법사 | 10만+ 명 | 3천 명 |
| 마검사 | 50~80명 | 거의 없음 |
| 마법공학·공성 | 대륙 1위 | 약함 |
| 신성 사제단 | 거의 없음 | 1,000명 + 대주교 |
| 길드 용병 동원 | 5만 명 가능 | 어려움 |

### 카르샤가 그래도 결전 결단한 이유

1. **신성 사제단의 결정적 자산** — 대규모 회복·축복으로 *전력 격차 일부 상쇄*
2. **신성 기사단 300명** — *마스터급 영향력*
3. **알레이언 6세 본인의 결단** — *물러서면 카르샤 신성 정통 자체가 무너짐. 결전이 그나마 협상 매개*
4. **외부 동맹 부분 회복 기대** — *전쟁이 길어지면 헤븐즈 다른 왕국이 균형을 위해 일부 개입할 가능성* (결과적으로 실패)
5. **알레이언 6세의 자기 희생 계산** — *자기 한 명의 결단으로 카엘의 비뚤린 시야에 균열을 만든다면, 다음 세대가 진실을 회수할 매개를 남길 수 있다*. **5합 결투·해독 거부의 정통 근원**

### 카르샤 측 *왕의 지시 — 식솔 피난 우선*

★ 본작 캐논:

- **알레이언 6세가 결전 직전 *왕실 호위 소드 엠퍼러급 1명 + 소드 마스터 3인에게 식솔 피난 명령***
- 명령 내용 — *너희는 *전장이 아니라 *왕가의 다음 세대를 *살리는 데 *써야 한다. *세파와 키얀, 왕비를 *국경 너머로 *피난시켜라*
- 결과 = *카르샤의 핵심 무력 4인이 *결전 자체에 직접 투입되지 않음 → 카르샤 본국 군사력 추가 약화*
- 대신 *세파의 망명·키얀의 호위·왕비의 셀라엔 망명지 합류*가 가능해짐 (Ch2 P2 시드의 정통 근원)
- 알레이언 6세 본인은 *식솔 피난 결단 후 자기 자리에 남아 5합 결투 직면*

### 카르샤 소드 마스터 1인 호스반 기습 사망

★ 본작 캐논 (Ch1 P5 §10 보강):

- 카르샤 왕실 직속 소드 마스터 3인 중 *1인이 식솔 피난 임무 수행 중 호스반의 기습 마법 직격으로 사망*
- 위치 = *카르샤 변경 외곽 — 세파·키얀 피난 호위 동선 중*
- 호스반의 *광역 8써클 마법이 아닌 *고도 정밀 7써클 즉결 마법으로 *마스터의 호흡 단 한 점에 직격*
- 마스터의 *마지막 결단 — 자기 몸으로 세파·키얀을 가림 + 신성 결계 봉지 1매를 세파에게 양도*. **이 봉지가 Ch1 P5 §10-E 탈출 시 호스반의 8써클 압살을 분쇄한 *바로 그 봉지***
- *세파의 호위 잔존 2인 = 외부 망명지로 도주 매개*. *나머지 1인 = 왕비·셀라엔 호위로 분담*
- **남은 카르샤 소드 마스터 = 2인 (전쟁 시점)**

### 카엘의 최초 광역 폭발 — *세이렌 외성벽 한쪽 박살 + 500명 타격* (사용자 정통 변경)

★ 본작 정통 변경 (이전 *세번강 1,400명 광역 학살*에서 *세이렌 외성벽 한쪽 박살 + 500명 타격*으로):

- **4-1 → 4-2 본격 무대 전환**: 카엘의 *최초 대량 폭발*은 *세번강 전장이 아닌 세이렌 외성벽 진입 시점*
- 효과 = **세이렌 외성벽의 한쪽이 박살남 + 성벽 위·아래·근접 진형에 몰려 있던 약 500명이 직접 타격 (사망·중상 합산)**
- 정통 = *카엘의 *동시성 제어 아티팩트 + 5써클 + 4가지 마법 압축 폭발 + 호스반의 후방 6써클 합작 + 마력 공성포 1발 동시 작동의 *5중 합성*. 일반 마법사 1인이 단독으로 만들 수 없는 효과 — 그러나 *공식 발표는 카엘 명의로*
- 카엘의 *미세한 위화감* — *내가 한 게 맞나. 4가지 마법 + 아티팩트로 *성벽 한쪽 + 500명이 가능한가*. *호스반의 후방 판 깔기 + 바이서스 군 환호가 *의심을 덮음*
- **세번강 전장 광역 학살 1,400명은 *없던 일로 변경*** — *세번강 전장 = 카엘 마검사 백병전 + 부분 진형 격퇴 (수십~수백 명 단위)로 격하*. *대규모 학살은 세이렌 외성벽에서만 단 1회*
- *공식 발표 + 라이세린 자금 매개 출판 + 호스반 시각 효과 조작 결과 = *카엘 = 학살자 호칭의 핵심 시드*. 이후 *바이서스 군의 카르샤 광범위 독살이 합산되어 *카르샤 시민·왕실 광범위 중독 + 카엘 명의의 추가 학살자 호칭 확대*

### 주변국이 나서지 않은 이유 (사용자 정통 보강)

★ 본작 캐논:

- **주변국 = 바이서스의 제제·보복이 두려워 *공식 참전 거부***

- *적의 마법 시전 차단·근접 회피·기동 매개* — *1초 이내 마법 시전 차단 정통과 결합되어 *적의 거리 매개 자체를 차단*
- **이 정통의 본격 적용**: *카엘 같은 *광역 마법 + 기동 마검사도 *카르난의 단검 던지기 + 검은 검 직격 + 카보나이트 갑옷의 3중 매개에 *결정타 매개가 매우 어려움*

### 사적 정통 — *왕가에 대한 평생 충성*

- 카르난의 직무 = *라카르 9세의 어린 시절 호위 → 친위대장 25년 = 평생 충성*
- *Ch2 P8 결전 = 카엘이 라카르 9세에게 직격하려면 *반드시 카르난을 먼저 처분해야 함*
- *카르난의 결전 = 자기 정통적 결단 + 왕가 충성 + 평생 직무의 통합*
- 카르난의 마지막 한 마디 (Ch2 P8 8-6 종결) — *"…너는… 처음 보는 길이군."* = *50대 후반 평생 검에 매진한 자의 *마지막 인정 + 자기 정통의 한계 자각*

### 카엘과의 직격 결과 — Ch2 P8 정통

- *카엘의 *마력 대포 총동원 + 공허검 1·2원리 합성 + 룬 결각 광역 함정 + 신성 결계 봉지 1매 + 호위 마스터 5인 순차 처리의 *5중 매개*로 *가까스로 척살*
- *카엘의 *자기 무기 전부 소진 + 의수 부분 균열 + 룬 회로 추가 손상 + 마력 회로 80% 소진의 *결정적 대가*
- **이는 카엘 4단계 의수의 본격 한계 자각 매개**. *카르난과 같은 엠퍼러급이 *2명이라면 카엘 결단 불가능 = 본작 정통에서 *바이서스 엠퍼러급이 1명인 이유 = 그 1명조차 가까스로 처리될 수준의 정점*

## 3. 바이서스 함락이 쉽지 않은 서사 정통

본 인물 2인의 정통 = **바이서스 함락이 *카엘의 강해진 무력에도 결코 쉽지 않은 서사의 핵심 매개***.

### 정면 돌파 불가의 5중 사유

1. **9써클 칼란 베르냐스** — *대륙 1~3명 중 1인 + 위력 + 기동 융합 정통*. *직접 직면 시 카엘 30~50% 승률*
2. **소드 엠퍼러급 카르난 발도르** — *검은 검 + 1초 시전 차단 + 카보나이트 갑옷 + 단검 던지기*. *직접 직면 시 카엘 자기 무기 전부 소진 대가로 가까스로 척살*
3. **왕실 직속 소드 마스터 5인** — *각자 정점 마스터. 카르난 보조 + 카엘 회피 공간 제거 매개*
4. **정규 소드 마스터 100명+** — *수십 명 협공 = 카엘 압도당함 (Ch2 P8 1차 협공 정통)*
5. **8써클 호스반 + 정규 7써클 대마법사·5써클 마법사 다수** — *마법 후방 매개 + 정찰 + 합성 시전*

### 카엘 정면 돌파 불가 자각의 정통 (Ch2 P7 1-1)

> 카엘의 4단계 의수 + 공허검 + 마력→내공 + 룬 결각 + 마력포 무한 정통으로도 *바이서스 왕실 직속의 *2 정점 (칼란·카르난) + 5인 마스터 + 100명+ 정규 마스터 + 호스반 + 학회 대마법사들의 *동시 정통을 *동시 직격하는 것은 불가능*. 만약 정면 돌파했다면:
> - *카엘이 셀라리아 외곽 진입 시점에 칼란의 광역 9써클 1발에 즉시 사망*
> - *또는 카르난의 검은 검 + 단검 던지기 + 카보나이트 갑옷의 3중 매개에 *가까스로 견뎌도 *나머지 마스터·대마법사들의 협공에 즉시 압도*

### 결단 — *칼란 직접 직면을 회피하는 매개로만 결전 가능*

- *Ch2 P7 게릴라전 = *칼란을 *왕성 깊은 결계 보호 직무에 매어두는 매개*
- *호스반·라이세린 척살 + 사적 이권 사슬 해체 → 라카르 9세 비상령 발화 → 9써클은 왕가 안전 결계 보호 직무로 자동 후퇴 (자기 정통적 결단)*
- *Ch2 P8 결전 = 카엘 vs 카르난 + 5인 마스터 + 잔존 대마법사들. *9써클 칼란은 끝까지 결계 안*

### 결전 후 정통

- 카르난 = *척살*. *바이서스 친위대장 정점 + 평생 충성의 *마지막 직무 완수 + 자기 정통의 한계 자각*
- 칼란 = *결계 해제 + 자기 거처로 잠적*. *왕실 와해 후 본래 학자형 정통으로 회귀*. **Ch3+ 미래 재출현 시드** — *마계 영지 군주 출현 시 *자기 정통적 결단으로 다시 나설 가능성*
