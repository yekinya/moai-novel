## 1. 결혼 풍습

### 헤븐즈 킹덤 — *르세프 본원교식*

- **장소**: 셀라리아 르세프 본원교 대성당 또는 마을 사당
- **의례**: 신랑·신부가 *손을 모아 신관에게 내밀고*, 신관이 *광명의 빛* 으로 손등에 작은 표식 (3개월 후 사라짐)
- **선물 의례**: 신부 가족에게 *마나 코어 1개* 또는 동등 가치
- **하객**: 평민 50명 수준 / 귀족 200명+
- **신혼**: *침묵절(12월)* 회피 (불운). *광명절(4월)* 최선

### 엘더스톰 — *전쟁 동맹식*

- **장소**: 그라니움 일레인 광장. 일레인 묘 앞에서 의례
- **특이성**: *결혼 = 가문 간 군사 동맹*. 양 가문의 셉터들이 *공동 영창* 으로 *축복 결계* 시전
- **신랑신부 행위**: 검 한 자루를 *둘이 함께* 들고 *부수는* 의례 — *과거의 무력을 함께 버린다* 는 상징
- 평민 결혼도 *형식상 검* 사용 (싸구려)

### 강철제국 — *모루 위 의례*

- **장소**: 카르나 모루의 광장 또는 마을 대장간
- **특이성**: *대장간 화로 옆에서* 의례. 신랑이 *작은 못* 을 망치로 박는 것으로 *서약 확정*
- **의례 길이**: 약 5분. 짧고 단호
- 신혼 = *함께 검 1자루 직접 제작* — 가문 보관용

### 카르샤 — *루멘 통합 4축식 (가장 정교)*

- **장소**: 세이렌 신성 대성당 또는 작은 신성 수도원
- **의례 길이**: 약 3시간. 4축 신 (이그너스 + 저그넛 + 로렌 + 잉그리드) 각각 별도 의례
- **핵심 자세**: 신랑신부가 *서로 손을 모아* 마주 댐 — 손 모음 인사의 변형
- **로렌의 *맹세 결박***: 신성마법 *맹세 결박* 으로 *영혼 단위 서약*. 위반 시 *영구 저주*. 매우 무거운 결혼
- **이혼**: *불가능* (영혼 결박이라). 사별 후 재혼만 가능
- **본작 함의**: 카르샤 키얀 왕자 = *로렌 맹세 결박* 으로 출생. 그의 *왕가 정통성* 의 종교적 기반

### 마녀령 — *9혈별 다름*

- 흑혈 = *밤의 호수에서 둘이 손을 *피로 적심*
- 적혈 = *서로의 손목을 단검으로 그어 *피를 섞음*
- 백혈 = *식물 화환을 서로 머리에 씌움* (가장 평화로움)
- 회혈·녹혈·자혈·금혈·무혈 = 각 종파별 비전 의례
- **공통**: *서로 다른 종파 간 결혼은 *원탁회의 사전 합의 필요*** (정치 동맹 결정)

### 제국 데미안 — *상인 계약식*

- **장소**: 미라돈 재믹 거래소 또는 길드 회당
- **특이성**: *결혼 = 양 가문 자산 결합 계약*. 마나 잉크로 *결혼 계약서* 작성
- **의례**: 양가 가장이 *계약서에 서명* + *상인 길드 의장* 의 *공증*
- **종교적 요소 약함** — 신관이 *축복 1줄* 만 첨가

### 마녀령·다크엘프 통혼 (드문 사례)

- 녹혈 가문 + 칠흑숲 변경 엘프 부족 간 *천 년에 30회* 정도. 매번 *9혈 회의 논쟁*

## 2. 장례 풍습

### 보편 원칙

- **장소**: 룔(환생) 사원 또는 자연 묘지
- **신앙 등급별**:
  - 평민: 매장 + 묘비 (작음)
  - 귀족: 매장 + 가문 묘지 + 룔 의식
  - 왕족: 화장 + 유골 봉인. *마나 잔향 영구 보존* 도구 사용

### 8왕국별 특이성

- **헤븐즈**: 르세프 사제 주관. *광명의 빛으로 영혼 인도* 의례
- **엘더스톰**: 일레인 묘 옆 *기념 비석* 1개 추가. 가족 단위 *집단 묘*
- **강철제국**: *검을 함께 묻음* — 평생 사용한 검을 *유해와 함께*
- **카르샤**: *손 모음 자세* 로 안치. 룔 + 이그너스 이중 의식
- **마녀령**: 흑혈 가문 주관. *영혼 결박 해체 의식* — *시신은 자연 분해* 시킴 (매장 없음)
- **데미안**: *상인 길드 추모식* 후 매장. 부유층은 *마나 코어 묘지 결계*
- **엘프**: *나무에 영혼 결박*. 시신은 *대지에 그대로*. 그 자리에 *영혼 나무* 가 자란다는 전승

### 본작 핵심 장례

- **카르엔 사이엔 (2-6 사망)**: 카르엔 학파 비공식 장례. 외곽 연구소에서 *7명만 참석*. 카엘은 *참석 의무 없다 판단해 가지 않음* — 후일 회한
- **카르샤 알레이언 6세 (프롤로그 사망)**: 카엘이 *시신을 옥좌에 그대로 둠*. 카르샤 측의 *공식 장례 부재*. 키얀 왕자가 *언젠가 돌아와 의식 거행* 시드
- **에리언 (3-3 사망)**: 마하가 *시신 못 찾음*. 빈 그릇만 차림 (3-5 묘사)

## 3. 축제·절기

[`calendar-and-time.md`](./calendar-and-time.md) §절기 참조 — 5대 절기 (잉태절·광명절·응결절·해체절·침묵절)

### 8왕국별 추가 축제

#### 헤븐즈

- **광명절 (4월) 협정 기념 대축제** — 7일간. 셀라리아 전체 점등. 마나 가로등 *2배 밝기*
- **수확절 (10월)** — 마을 단위 평민 축제

#### 엘더스톰

- **일레인 기념일 (CC 925, 일레인 사망일 추정)** — 그라니움 군대 행진. 도시 봉인 1일
- **칠흑숲 모험 시즌** (가을 1개월) — 관광 성수기

#### 강철제국

- **모루의 날** (월 1회) — 황실 대장간 *공개 시연일*. 평민 입장 가능
- **다섯 검 시연** (연 1회) — 다섯 검 호위가 *공개 검술 시연*

#### 카르샤

- **광명절** — 손 모음 의례 *전 시민 동시 시전*. 가장 경건한 절기
- **알레이엔 1세 기념일 (CC 0 종전일)** — 7대 왕가 추모. *루멘브링거 공개*
- **본작 시점**: *광명절 직후 알레이언 6세 사망* 의 시기적 충격

#### 마녀령

- **9혈 합의 기념일** (연 1회) — 원탁회의 *공개 의식*. 보통 외부 출입 불가
- **해체절 (10월)** — 마녀의 절기. 변경에서 *작은 학살* 의식 (정확한 표적, 작은 규모)

#### 데미안

- **노바 항만 축제 (3개월 1회)** — 출항 시즌마다 폭풍 신 노바에게 *공물 의식*
- **재믹 거래소 도박 페스티벌** (연 4회) — 분기별 대규모 도박. 막대한 자본 회전

## 4. 식문화

### 평민 식단

- 빵 (보리·호밀) + 양고기 또는 돼지고기 + 야채 스튜
- 변경: *마법 보존 항아리* 로 *겨울 식량* 비축

### 귀족 식단

- 양고기·소고기 + 해산물 (미라돈) + 마법 화로 요리
- *마나 차* — 약한 마나 결박된 차. 정신 집중 효과 (셉터 표준)

### 지역 특산

- **헤븐즈**: 평원 곡물·소고기. *셀라 호수 잉어*
- **엘더스톰**: 산악 양고기. *오우거 환각 향신료* (강력)
- **강철제국**: 한랭 보존 식품. *얼린 늑대 고기*
- **카르샤**: 동변 강 물고기. *손 모음 빵* (특수 모양)
- **마녀령**: 늪지 식물·버섯. *9혈별 비전 약초 차*
- **데미안**: 해산물 풍부. *향료 무역* 으로 *대륙 최고 향신료*
- **엘프**: 채소·과일 위주. 육식 거부

### 술

- 평민: 맥주·과실주
- 귀족: 와인·증류주
- 카이엔이 6-1에서 *처음 술* 을 경험 — *어지러움의 즐거움*

## 5. 의복

### 평민

- 거친 천 (린넨·양모). 색 = 갈색·회색·청색 (염료 비쌈)
- 변경: 가죽 + 모피

### 귀족

- 비단·벨벳. 화려한 자수
- 가문 문장 부착

### 직업별

- **셉터**: *로브* — 종파색. 후드 달림
- **기사**: 갑주 + 서코트. 가문 문장
- **신관**: 종파색 사제복 + 머리쓰개 (수녀)
- **마녀**: 무복 + 머리 풀어 늘어뜨림 + 깃털 장식
- **상인**: 화려하나 *기능적*. 미라돈 길드별 색

### 본작 인물 의복

- 카엘 1~3부: *바이서스 왕실 마법 고문 로브* (검은색 + 은색 자수)
- 카엘 4부 옥좌의 방: *피와 재로 검게 절은 망토*
- 카엘 5~6부: *변경 평민 옷*. 의수 위에 장갑
- 알레이언 6세: 카르샤 *백색 신성 의복 + 황금 자수 + 두 손 모음 의례용 띠*
- 카이엔 6부: *동부국 검술대회 우승 후 검은 마검사 의복*

## 6. 일상 풍습

### 인사

- **헤븐즈**: *손을 가슴에 한 번*. 정중함 표시
- **엘더스톰**: *오른팔을 옆으로 펼침* — 검을 보이는 자세 (적의 없음)
- **강철제국**: *주먹을 가슴에 한 번*. 군대식
- **카르샤**: *양손 모음* — 손 모음 인사. 의례 매개
- **마녀령**: 종파별 다름. 흑혈 = 손바닥을 바닥 향함 / 백혈 = 손바닥을 위로 향함
- **데미안**: *오른손 악수*. 상인식

### 호칭

- 평민: 이름만
- 귀족: 성 + 작위
- 왕족: *폐하* / *전하*
- 셉터: *학장* / *교수* / 시니어 셉터는 *경*
- 기사: *경*
- 신관: *대주교* / *주교* / *신부* / *수녀*
- 마녀: *종주* / *대모* / 외부 호칭 *황녀* 등

### 명함

- 도시 귀족 + 상인 = *마나 잉크 명함*. *접근 시 부드러운 음악 발생* 같은 기능
- 평민 = *문서 자체 없음*. 마을 단위 신원

## 7. 본작 활용 — 일상 묘사 5원칙

1. **회차마다 1~2개 풍습 디테일 끼우기** — *손 모음·검을 함께 묻기·양손 모음 인사* 등 자연스럽게
2. **인사·호칭의 일관성** — 카엘이 카일렌 경을 *경* 으로 부른다 / 카르엔을 *스승* 으로 부른다 / 라이세린을 *각하* 로 부르나 5-1 이후 *공작* 으로 호칭 변경
3. **식·의복은 *인물 정체* 의 텍스처** — 카이엔의 *어지러움의 즐거움* / 카엘 5부의 *변경 평민 옷* 의 무게
4. **축제·절기는 *시점 표지로* 활용** — *광명절 직후* = 카르샤 충격의 시기적 무게
5. **결혼·장례는 *큰 무게로* 다룸** — 카르엔 장례에 카엘이 가지 않은 회한, 알레이언 장례 부재의 의미
