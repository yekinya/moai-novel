## 1. 본질 정의

### 마계 창조물 (Demon Creature) 이란

- **표면 인식**: 마계의 악마. 인간을 학살하러 온 적
- **실제 본질**: *마계 균열을 통해 분출되는 *해체 마나 응집체***. 의지 없거나 일부만 있음
- **출현 조건**: 엔트로픽 농도가 임계에 도달해 *마나 역류* 가 발생할 때
- **목적**: *과밀한 생명 마나 구조* 를 해체. 그 결과로 인간을 *공격하는 것처럼 보임*
- **종족 아님**: 자기 영구 종족이 아닌 *균열에 한정된 일시 존재*. 균열이 닫히면 *자연 소멸*

> *"마계 창조물은 *생명을 증오하지 않는다*. 그들은 *과밀한 마나를 풀어주는* 자연 현상이다. 우리가 그들에게 *증오의 얼굴* 을 씌운 것뿐이다."*
> — 사프란, 『앤트로퍼 비망록』 CC -10

## 2. 6대 유형 — 등급별 분류

### 2.1 잔재 (Remnant) — 1등급

- **외형**: 형체가 *불완전*. 그림자 + 빛의 잔해처럼 일그러진 형태. 1~3미터
- **개체 마나량**: 일반 사피엔 1~5명 분
- **자아**: *없음*. 본능적 *마나 응집체로의 이끌림* 만
- **약점**: 일반 검·화살로도 처치 가능. 다만 자기가 다친 자리에서 *마나 결박을 풀고* 다시 응집
- **출현**: 마계 균열 *주변 100미터* 내. 균열이 작아도 다수 출현
- **본작 등장**: 5-3 변경 엘프 마을에서 *3주 전 잔재 2체 격퇴* 흔적 (배경)

### 2.2 사역마 (Familiar) — 2등급

- **외형**: 작은 4족 짐승 형태. 검은 비늘 + 붉은 눈. 키 50cm~1m
- **개체 마나량**: 9써클 마법사의 1/100
- **자아**: *기본 사역 지능*. 누군가가 *소환* 했거나 *영지 군주의 종속체*
- **약점**: 마법 결계로 *경계 정지*. 직접 공격은 무리. 떼로 공격
- **출현**: 균열 + *소환사* 가 매개. 마녀령 금혈 가문이 *유일하게 합법 소환*
- **본작 등장**: 후반부 마왕 강림 전조에 *무리* 출현 가능

### 2.3 마계 영지 군주 (Demon Lord) — 3등급

- **외형**: 인간형 + 변형. 키 2~3미터. 뿔·이빨·날개·꼬리 중 1~2가지 보유. 검은 갑주 같은 외피
- **개체 마나량**: 일반 사피엔 *2,000~5,000명 분* (왕급 인간 응축)
- **자아**: *완전 지능*. 언어 사용 (다양한 마계 방언). 협상·기만 가능
- **약점**: 신성마법에 약함. 단, 일반 9써클 마법은 *통하나 위력 절반*
- **출현**: 큰 마계 균열 + 농도 임계. 자연 발생 또는 *금혈 마녀가 의도적 소환*
- **사망 시 해소량**: 약 20,000~50,000 단위 (`high-density-targets.md`)
- **역사 사례**: 사프란이 직접 처치 1체 (CC 4) / 1대 모나르가 처치 3체 (CC 4~50). 그 외 약 15체 자연 발생 → 인간 사회가 공동 격퇴

### 2.4 마왕 (Demon King) — 4등급

- **외형**: 거대 인간형 + 강력 변형. 키 5~8미터. 검은 광휘 + 8개 손 / 다중 머리 / 박쥐 날개 같은 *상징적 압도 외형*
- **개체 마나량**: 드래곤 성체급. 일반 사피엔 *100,000~150,000명 분*
- **자아**: 완전 + 카리스마. *통솔력* 으로 마계 영지 군주들을 *연합 지휘*
- **약점**: *없음* — 단 *균열의 빛* 같은 *결박 해체 마법* 에 약함
- **출현 조건**: 마계 균열 *임계 임박*. 농도 시한 *6개월 이내*. 강림 = *리셋이 거의 확정* 된 신호
- **사망 시 해소량**: 약 150,000 단위 — 드래곤보다 *약간 더 큼*
- **부작용**: 마왕 살해 시 *마나 역류 일시 가속* — 한 번의 큰 균열이 자동 발생. 즉 마왕은 *죽여도 작은 대가*
- **역사 사례**: 신좌침묵 시대 *1체* 만 알려짐. 사프란의 *균열의 빛 원형 의식* 으로 처치 (CC 0)
- **본작 함의**: **중반부 첫 마왕 강림** — 카엘의 *진짜 속죄전 본격 무대*

### 2.5 마계 균열 핵 (Rift Core) — 5등급

- **외형**: 균열 그 자체. *공간의 일그러진 구멍*. 인간이 *물리적으로 볼 수 있는 형태가 아님*
- **마나량**: 측정 불가 — *무한에 가까움*
- **자아**: *없음*. 다만 *균열을 유지하려는 본능적 기제*
- **약점**: 일반 마법 무력. *마녀령 회수 회로 + 카르샤 균열의 빛* 의 *결합 의식* 만 해체 가능
- **출현**: 임계 시 *대륙에 1~3개* 동시 형성
- **해체 시 해소량**: 50,000~100,000 단위 + *균열 영구 봉인 불능 부작용*
- **본작 함의**: 8대 앤트로퍼 셀라가 *유일하게 1체 해체* (CC 580). 그녀는 직후 *자기 분해* 사망

### 2.6 마계 진주 (Demon Pearl) — 6등급 / 특수

- **외형**: 작은 검은 구슬. 지름 5cm. *마계 균열의 결정화 잔재*
- **마나량**: 큰 마계 영지 군주 분
- **자아**: 없음. 단 *주변에 영향*
- **위험**: 인간이 가까이 가면 *영혼 결박이 풀려나가는 부작용*. 만지면 즉사
- **활용**: 마녀령 금혈 가문이 *극도의 위험* 으로 *마계 계약 매개* 로 사용
- **출현**: 마왕이 사망한 자리에 *3~5개* 형성. 균열 해체 자리에도 가끔

## 3. 출현 패턴 — 시기·지역·전조

### 시기

| 농도 수준 | 출현 양상 |
|----------|----------|
| 안정기 (~CC 0 평형) | 0체 |
| 농밀기 (CC 0~500) | 잔재 산발 출현 |
| 정체기 (CC 500~800) | 사역마·마계 영지 군주 *연 1~2체* |
| 역류 전조 (CC 800~1000 현재) | 영지 군주 *연 3~5체* + 균열 핵 *5년에 1개* |
| 임계 임박 (다음 50년 예상) | **마왕 강림 + 균열 핵 동시 다발** |

> **현 본작 시점 (CC 1000) = *역류 전조의 정점***. 마왕 강림이 *10~30년 내* 예상.

### 지역

- **균열 다발 지역**:
  - 칠흑숲 — 변경 엘프 마을이 *3~5년에 1회 잔재 격퇴*
  - 강철제국 북부 빙벽 협곡 — 추정 (이실 빙벽 협곡 일대)
  - 미드그레이드 외곽 — 카르마서스 사건 (CC ~870) 이후 *균열 다발 의심*
  - **남부 사막** — 침묵의 사원 인근. *고요한 손* 사제들이 *균열 동향 관측*
- **균열 없음**:
  - 8왕국 수도 (강력 결계로 차단)
  - 미라돈 (마도공학 결계 차단)
  - 마녀령 오르하임 (9혈 공동 결계)

### 전조

- **마나 농도 급변** — 측정 가능 (상아탑·자혈 마녀·카르엔 학파 모니터링)
- **르세프 신성마법 일시 작동 불능** — 신좌침묵 직전과 동일
- **드래곤 갑작스러운 깸** — 카르마서스 사건이 *균열 직전 신호* 였다는 가설
- **앤트로퍼 활성도 급증** — 본작 카엘의 비정상적 회로 효율 = *씨앗 마지막 활성기 + 균열 임박* 의 동시 신호 가설

## 4. 처치 방법 — 무기·전술

### 1등급 잔재

- 일반 검 / 화살로 가능
- 단 *마법 결계로 응집 방지* 후 처치 권장 (재응집 차단)

### 2등급 사역마

- 마법 결계로 *경계 정지* → 화살 일제 사격
- 9써클 광역 마법 1발로 *떼 전체 소각*

### 3등급 마계 영지 군주

- **신성마법 5등급 + 마법사 9써클 + 마스터급 검사** 의 3인 조합이 정통
- 영지 군주 1체 격퇴 = *대도시 한 구역의 동원력* 필요
- *적은 자원* 으로 처치 시도 = *시도자가 모두 사망*

### 4등급 마왕

- 일반 군대 무력
- **균열의 빛 + 마녀령 회수 회로 결합 의식** 만 처치 가능
- 사프란 + 8왕의 *연합 의식* 이 유일한 역사 사례
- *부작용 = 마나 역류 일시 가속* — 균열 추가 발생

### 5등급 균열 핵

- **앤트로퍼 숙주 + 마녀령 회수 회로 + 카르샤 균열의 빛** 의 *3중 결합 의식* 만 가능
- 셀라(8대 앤트로퍼) 처치 = 흑혈 가문 회수 회로 + 본인 *자기 분해* 의 2축
- *균열 영구 봉인 불능 부작용* — 그 자리에는 *다시는 결계 불가*

### 6등급 마계 진주

- 처치 대신 *밀봉 + 영구 격리*
- 금혈 마녀가 *극도 위험* 으로 *마계 계약 매개* 사용

## 5. 본작 후반부 적용 — 카엘의 속죄전 무대

### 진행 순서 (예상)

1. **중반부 진입**: 농도 측정 + 균열 핵 1체 발견 → 본격 위기 도래
2. **마왕 강림** (중반부 중간): 첫 마왕 등장. 카엘이 *진짜 적* 을 본다
3. **마왕 처치 시도** (중반부 종결): *균열의 빛 결합 의식 시도* — 카르샤 잔재 수녀단 (셀라엔 + 엘라리아) + 마녀령 흑혈 (모르간 솔테) + 카엘 (앤트로퍼). **8왕 연합 의식의 *작은 재현***
4. **고룡 + 정령왕 사냥** (후반부 전반): 카엘이 *드래곤 로드의 추종파* 와 *대등한 무력* 으로 도전
5. **균열 핵 동시 해체** (후반부 후반): 마지막 표적군
6. **카엘의 최종 카드** (종결부): 본인의 *자결* 로 마지막 농도 해소

### 핵심 인물 동선 매트릭스

| 표적 | 처치 매개 | 핵심 인물 |
|------|---------|----------|
| 잔재·사역마 | 일반 마법·검 | 카엘 + 세파 + 세이른 일행 |
| 마계 영지 군주 | 3인 정통 조합 | 카엘 (9써클) + 세이른 (검) + 셀라엔 대주교 (신성마법) |
| **마왕** | 균열의 빛 + 회수 회로 결합 | 카엘 + 엘라리아·셀라엔 (카르샤 잔재) + 모르간 솔테 (흑혈) + 베르난 호산 (금혈 협상) |
| 균열 핵 | 3중 결합 의식 | 카엘 본인 매개 (자기 분해 위험) |
| 고룡 | *밀착 거리 매복* | 카엘 + 카이엔 (만약 화해) + 세이른 (공허검) |
| 정령왕 | 부작용 감수 + 광역 의식 | 카엘 + 자혈 점성 예언 |
| 마계 진주 | 밀봉 | 금혈 가문 |
| **앤트로퍼 숙주 자신** | 자결 | 카엘 본인 |

## 6. 묘사 가이드 — 본문 등장 시

### 시각 묘사 원칙

- 잔재 = *형체 없는 음영*. *그림자가 굳어진 듯한*
- 사역마 = *작은 짐승 그림자가 비늘로 굳어진*
- 영지 군주 = *키 큰 인간 그림자가 뿔과 이빨을 얻은*
- 마왕 = *압도적 크기. 본문이 *전체 묘사가 불가능* 함을 시인*
- 균열 핵 = *공간의 일그러짐. 묘사 불가*
- 마계 진주 = *작은 검은 구슬. 그러나 *보는 것만으로 메스꺼움**

### 후각·청각 묘사 원칙

- 마계 창조물 = *비린내 + 탄 냄새 + 금속 냄새*
- 영지 군주 + 마왕 = *낮은 진동 같은 울림*. 직접 *말* 하지 않을 때도 *공기가 진동*
- 균열 핵 = *모든 소리가 *눌리는 듯한* 정적*

### 본작 본문 활용

- 5-3 변경 엘프 마을 = *잔재 격퇴 흔적* 1줄
- 6부 후반 변경 도시 = *영지 군주 1체 격퇴 사건* 짧게 언급
- 중반부 = *마왕 강림 묘사 1회 본격*. 본문 5,000자 이상 할애
- 후반부 = *균열 핵 해체* + *고룡 사냥* 의 클라이맥스 묘사
