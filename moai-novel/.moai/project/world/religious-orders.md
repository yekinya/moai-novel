## 1. 5대 직분

신을 섬기는 방식에 따라 분화된 5개 직분. 같은 신을 모시더라도 직분에 따라 *수련법·복장·도구·사회 위치* 가 다르다.

### 1.1 몽크 (Monk, 수행자)

- **방식**: 신을 *체화* 하려는 자. 신의 성격을 자기 신체에 새기는 *무술·금욕·내적 수련*
- **도구**: *없음*. 신체 자체가 매개
- **복장**: 무복 (회색·갈색 무지). 머리 면도 또는 짧게
- **거처**: 산속 사원·동굴·변경. 도시 회피
- **신성마법**: 거의 사용하지 않음. *마나가 아닌 *기(氣)*** 라 부르며 신체 강화에 사용
- **사회 위치**: 권력 밖. 존경받되 의사결정 권한 없음
- **주류 신**: 잉그리드 (대지) / 룔 (환생) / 아샤스 (영광) 일부
- **본작 관련**: 직접 등장 없음. 5부 빙벽 협곡 가는 길에 *변경 잉그리드 사원* 의 늙은 몽크 1인 단편 등장 가능

### 1.2 성기사 (Paladin)

- **방식**: 신의 의지를 *검* 으로 실현. 기사도 + 신성마법 결합
- **도구**: 장검 (보통 양손 바스타드 또는 한손 롱소드) + 강철 갑주 + 신성 결계
- **복장**: 갑주 + 신의 문장이 새겨진 *서코트*
- **거처**: 각국 왕성·신전 + 변경 순례
- **신성마법**: 4~5등급. *검에 신성력 결박* 기술 필수
- **사회 위치**: 귀족 동급. 왕실 호위·종교 재판·이단 처벌
- **주류 신**: 아샤스 (영광) / 저그넛 (정의) / 이그너스 (순결·이단 정화 — 카르샤)
- **본작 관련**: 카르샤 신성검단 = *이그너스 + 저그넛 + 잉그리드 4축* 의 성기사 결사 (`holy-magic.md` 참조). 카일렌 베르하르트 경 + 카르샤 도르난 단장이 정점

### 1.3 수녀 (Nun)

- **방식**: 신의 의지를 *결계·치유·정화* 로 실현. 성기사와 *동격* 의 직분이되 무력보다 *수호·돌봄·의례* 가 중심
- **도구**: 의식 봉 (석장) + 결계 부적 + 약초 도구
- **복장**: 백색 또는 종파색 수녀복 + 머리쓰개
- **거처**: 수도원 (도시 외곽 또는 변경 산속) + 야전 병원
- **신성마법**: 4~6등급. *치유·결계·해독* 전문
- **사회 위치**: 종교적 권위 + 의료 권위. 일반인의 마지막 의지처
- **주류 신**: 에필링 (치유) / 이그너스 (순결·잉태) / 엘라 (조화·엘프계)
- **본작 관련**: 카르샤 셀라엔 대주교 휘하 수녀단이 4-4 함락 시 *왕비 엘라리아와 함께 변경 수도원 망명*. 5~6부 *7비전 보존 잔재* 라인의 핵

### 1.4 무녀 (Shamaness / Oracle)

- **방식**: 신과 *직접 교감*. 무아경·꿈·점·접신. 영적 매개
- **도구**: 방울 + 점복 도구 (뼈·돌·잎) + 무복
- **복장**: 종파색 무복 + 머리 장식 (깃털·뼈). 머리 풀어 늘어뜨림
- **거처**: 변경 사당 + 마녀령 종파 본거 + 엘프 부족
- **신성마법**: 3~7등급 (편차 큼). *예지·환각·접신* 전문. 다른 직분이 못 쓰는 *시간 감응* 가능
- **사회 위치**: 변경·소수민족·이단 경계. 도시 권력층은 *경계* 하나 *완전히 무시 못함*
- **주류 신**: 학카르 (어둠·영혼) / 룔 (환생) / 엘라 (조화) / 자혈 마녀 종파의 자혈신 (점성)
- **본작 관련**: 마녀령 9혈 종주들이 *무녀 정점* — 모르간 솔테 / 베리아 솔 (130세 자혈 원로) 등이 본격 무녀 등급. 5-3 변경 엘프 마을의 *침묵의 환대* 도 엘라 무녀가 주관

### 1.5 성직자 (Clergy, 모닝스타 상징)

- **방식**: 신의 율법을 *세속 사회에서 집행*. 결혼·장례·축복·재판 의례 주관
- **도구**: **모닝스타** (Morning Star) — 1.2~1.8미터 봉 끝에 가시 박힌 철구. 무기이자 *신의 무게* 를 상징
- **복장**: 종파색 사제복 + 어깨에 두른 서약 띠 (시통대)
- **거처**: 도시 신전 + 왕실 예배당
- **신성마법**: 3~5등급. *축복·결박·맹세 봉인* 전문. *전투용 신성마법* 도 일부 (모닝스타에 신성력 부여)
- **사회 위치**: 도시 행정 + 종교 권위의 결합. 왕실·귀족과 동행
- **주류 신**: 로렌 (장미·혼인) / 저그넛 (정의·재판) / 르세프 본원교 (헤븐즈 통합) / 노바 (해상·미라돈)
- **본작 관련**: 헤븐즈 셀라리아의 *르세프 본원교 대성당* + 데미안 미라돈의 *노바 항만 신전* 이 거대. 본작 본편에서는 직접 등장 없으나 *카엘이 통과하는 도시* 의 *배경* 으로 항상 존재

## 3. 나라별 주요 종파

### 3.1 헤븐즈 킹덤 — *르세프 본원교*

- **수도 셀라리아 대성당**: *8 하위신 통합 신앙*. 정통 르세프 종합
- **수석 사제**: 르세프 본원교 대주교 **에피르엔 대주교** (Epirien, 65세)
- **본작 카엘 관련**: 1-1 출생 시 마을 *르세프 본원교 산파* 가 카엘 눈을 보고 도주. 1-4 마을 회의 시 *르세프 사제* 가 *추방 결정에 동의*. 카엘에게 신앙적 트라우마
- **종교 정치**: 라이세린 공작은 본원교 대주교와 *형식적 협력*. 실제로는 데미안 노바 종파에 *외척 신앙* 으로 더 가까움

### 3.2 엘더스톰 — *아샤스 + 저그넛 + 일레인 추모*

- **수도 그라니움 일레인 광장**: 일레인 맥필렌 묘가 사실상 신전화. *영웅 숭배 + 아샤스 정통*
- **수석 사제**: 일레인 추모 대제관 **카타르 대제관** (Katar) — 군 마법 총사령과 동일 인물 (일레인 카타르 백작)
- **특이성**: 사피엔이 *살아 있던 영웅* 을 *반신반인* 으로 모시는 유일 사례. 정통 종교계의 *이단 비판* 받음
- **직분 분포**: 성기사 중심 (아샤스). 일레인 묘 호위단 100명 = *살아 있는 모범 성기사*

### 3.3 강철제국 — *아샤스 + 레코르 + 살라멘티*

- **수도 카르나 모루의 광장**: 신전이 아니라 *대장간 자체가 의례 공간*. 황실 검 가공 시 *살라멘티 (드워프 신)* 에게 감사
- **이중 신앙 — 검의 두 얼굴**:
  - **아샤스 (영광)** — 검을 *명예와 영광* 으로 모심. 성기사 중심
  - **레코르 (검과 살육)** — 검을 *살육의 도구* 로 모심. 전사형 사제 중심
  - 한 검사가 두 신을 동시에 모심. 모순이 강철제국 정체성
- **레코르 교단의 특이성**: 사피엔 사회에서 *유일하게 공식적인 살육 신앙*. *적정량의 살육이 사회를 구한다* 는 본원교 비판 + 마녀령과 *닮은 사상* 보유
- **세이른 스틸만의 *공허검* = 두 신앙 모두 거부**. 강철제국에서 그가 추방된 *진짜 종교적 이유*

### 3.4 카르샤 — *루멘 통합신앙* (이그너스 + 저그넛 + 로렌 + 잉그리드 4축)

- **수도 세이렌 신성 대성당**: 카르샤 왕궁과 동일 건물군. 알레이엔 1세가 정립
- **루멘 통합신앙 4축**:
  - **이그너스 (순결·잉태)** — 신성 정화. 신성검 정통
  - **저그넛 (정의)** — 진실 식별 + 재판
  - **로렌 (장미와 피·혼인)** — 왕가 혼인 의례 + *균열의 빛 봉인 의식의 핏빛 부분*
  - **잉그리드 (대지)** — 봉인 의식의 *대지 결박 부분*
- **손 모음 인사** = 4축 통합의 *대중화된 의례 자세*. 신성마법 *귀의 단계* 자세 그대로
- **카르샤 대주교 셀라엔** = 4축 통합 5등급 직전 신성마법사. 알레이언 6세의 *종교적 동료*. 4-4 직후 왕비 엘라리아와 함께 변경 수도원 망명
- **본작 함의**: 카르샤가 *작고 가난한 신정국가* 임에도 *천 년을 견딘 이유* = 4축 통합신앙의 *내부 결속력*

### 3.5 마녀령 동부국 — *학카르 본원 + 9혈별 보조신*

- **수도 오르하임 — 신전 없음**. 9혈 종파 본거가 각각 *작은 사당*
- **공통 본원신**: **학카르 (어둠·영혼)** — 모든 마녀의 공통 시조 신
- **9혈별 보조신** (각 종파가 학카르 외에 *주로 모시는 신*):
  - 흑혈 — 학카르 단독
  - 적혈 — 학카르 + **레코르** (검·살육)
  - 청혈 — 학카르 + 환각계 (신 미정)
  - 백혈 — 학카르 + **에필링** (치유) — *르세프 신을 모시는 마녀* 라는 이단성의 원천
  - 회혈 — 학카르 + **거스터** (질병)
  - 녹혈 — 학카르 + **잉그리드** (대지) — 자연 친화. 엘프 신앙과 일부 융합
  - 자혈 — 학카르 + 자혈 고유 신 *베라엔* (점성. 9혈 외부에 알려지지 않은 자혈 비전 신)
  - 금혈 — 학카르 + 마계 계약신 (이름 비전 — 마계 측 호명)
  - 무혈 — 학카르 + **무**(無) 그 자체. 신 부재. *무신앙적 신앙*
- **직분 분포**: 마녀는 *무녀 직분이 거의 전부*. 5직분 중 *몽크 일부*, *수녀 (백혈만)* 있음. 성기사·성직자 없음
- **모닝스타의 부재**: 마녀령에는 모닝스타가 *없다*. 마녀의 무기는 *지팡이 + 단검*

### 3.6 제국 데미안 — *살라멘티 + 노바 + 재믹*

- **수도 미라돈 신전 구역**: 3대 신전 병립
  - **살라멘티 대신전** — 마도공학자들의 신앙. 드워프 신을 *사피엔이 모시는 드문 사례*
  - **노바 항만 신전** — 항해사·상인의 신. *폭풍 신을 두려움으로 모심*. 출항 전 의례 필수
  - **재믹 거래소** — 신전 형태가 *증권·도박장과 결합*. 상인 길드의 *신앙적 정당화* 도구
- **수석 사제**: 살라멘티 대신전 *기관 사제* **그르긴 대사제** (Grgin, 70대) — 데미안 1대 황제 그르긴 1세의 직계 후예 가문
- **본작 함의**:
  - 라이세린 공작이 *외척 신앙* 으로 노바를 모심 (황후 카루린의 친정 신앙)
  - *백색 사냥단의 신앙적 정당화* = *노바의 분노를 가라앉히려면 *부정한 마법사를 베어야 한다** 는 비공식 교리
  - 6-2 그라스 의수 시술도 *살라멘티 사제의 축복* 후 진행 (작중 묘사)

### 3.7 국립상아탑 — *무신론 / 마나 평형 본질주의*

- **공식 입장**: *신은 인격적 존재가 아니다. 마나의 방향성을 의인화한 것이다*
- 학장 세이언 라모르 = 무신론자. *마나 평형 본질주의* 자
- 그러나 학자 개인은 *사적 신앙 자유* — 일부는 르세프 본원교 / 일부는 카르엔 학파의 *사프란 추모* / 일부는 무신앙
- **본작 함의**: 카르엔 사이엔이 *사적으로 사프란을 신처럼 추모* 한 흔적 — 2-2 서재에 *작은 무명 사당* (사프란 추정)

### 3.8 칠흑숲 변경 엘프 — *엘라 + 룔 + 자연 신앙*

- **공식 신앙**: 엘라 (조화) 본원 + 룔 (환생) 보조
- **다크 엘프 분파**: 학카르 (어둠) 으로 *전향*. 마녀령 일부 종파와 *교차 신앙*
- **무녀 중심**. 성기사·성직자 없음

## 4. 모닝스타 — 성직자의 상징 무기

성직자 직분의 *공식 상징* 이자 실전 무기.

### 형태

- 1.2~1.8미터 봉 (재질: 떡갈나무·철·드워프 합금 명품)
- 끝에 가시 박힌 철구 (지름 15~25cm)
- 봉에 *종파 룬 + 신앙 서약문* 새김
- 평신도 모닝스타 = 단순한 무기. 성직자 모닝스타 = *신성마법 결박 매개*

### 종파별 변형

| 종파 | 모닝스타 변형 | 특징 |
|------|--------------|------|
| 르세프 본원교 (헤븐즈) | *광명의 모닝스타* | 가시에 광명 마법 결박. 어둠 정화 |
| 저그넛 (정의) | *진실의 모닝스타* | 휘두를 때 *거짓 발화자* 의 혀가 묶임 |
| 로렌 (혼인) | *맹세의 모닝스타* | 결혼·서약 의례에서만 사용. 보통 장식 |
| 노바 (해상) | *폭풍의 모닝스타* | 가시가 13개 (폭풍의 13파). 항해사 축복 |
| 살라멘티 (드워프) | *모루의 모닝스타* | 드워프 합금. 무게 일반의 2배. 휘두름 자체가 *대장간 의례* |
| 재믹 (도박) | *주사위의 모닝스타* | 가시 끝에 작은 주사위 박힘. *결과를 무작위로* 만드는 농담조 무기 |

### 비종파 사용

- 변경 용병이 *신앙 없이* 모닝스타를 휘두를 수 있다. 그러나 *축복 없는 모닝스타* 는 신성마법 결박 불가 — *그냥 무거운 둔기*

### 본작 모닝스타 등장 가능

- 4-3 옥좌의 방 앞 회랑에서 카일렌 경 사망 직후, *카르샤 신성검단의 후위 성직자* 가 *저그넛 모닝스타* 로 카엘을 막아서다 사망 가능 (회차 비트 추가 옵션)
- 6-2 그라스 시술 시 *살라멘티 모닝스타* 가 시술실 벽에 걸려있음 (배경)
- 후반부 *마왕 강림* 시 *르세프 본원교 광명 모닝스타* 가 첫 격퇴 도구로 등장 가능

## 5. 종파 간 갈등·동맹 관계도

```
                  [르세프 본원교 (헤븐즈)]
                          ↑
                    "정통 신앙" 자처
                          │
          ┌───────────────┼───────────────┐
          │               │               │
   [루멘 통합 (카르샤)]  [아샤스 (강철)]  [살라멘티 (데미안)]
          │               │               │
       이단 비판        검의 두 얼굴     공업·무역
                          │
          ┌───────────────┼───────────────┐
          │               │               │
     [레코르 교단]    [노바 (미라돈)]   [엘라 (엘프)]
       살육 정당화      폭풍 두려움      자연 조화
          │               │
          └───────────────┘
                  ↓
            [학카르 본원 — 마녀령 9혈]
                  ↓
            [무혈 — 무신앙적 신앙]
                  ↓
              [무신론 — 상아탑]
```

### 핵심 갈등 라인

1. **르세프 본원교 ↔ 루멘 통합**: 헤븐즈가 *카르샤 신앙을 이단으로 분류* 한 적이 3차례. 마지막은 CC 900. 그 적대의 잔재가 *손 모음 인사 금지령* (3-4 카엘의 강연 직후)
2. **아샤스 ↔ 레코르**: 강철제국 내부의 *영원한 자기 모순*. 같은 검사가 두 신을 모시는 *심리적 분열*
3. **르세프 본원교 ↔ 학카르**: *천 년의 종교 전쟁*. 백색 사냥단의 *신앙적 근거*
4. **살라멘티 (데미안) ↔ 잉그리드 (드워프 원본)**: 사피엔이 드워프 신을 *횡령했다* 는 드워프 측 비판. 외교적 불씨

### 비공식 동맹 라인

- **카르엔 학파 + 마녀령 9혈** — 학카르 본원과 *루멘 통합신앙* 의 *진실 기반* 비공식 채널 (`witch-eastern-council.md` 레우라엔)
- **백혈 마녀 + 카르샤 잔존 수녀단** — 둘 다 *르세프-크세프 경계* 에서 활동. 본작 후반부 동맹 카드
- **무혈 + 상아탑 무신론자** — *신을 모시지 않는다* 는 공통점. 종결부 *카엘 사슬 해체* 작전의 토대

| 1부 (유년~추방) | **르세프 본원교 표면 신앙** | 마을 사제가 그를 *저주받은 자* 로 분류 → *신앙적 트라우마* |

## 7. 작품 활용 5원칙

1. **종파는 *배경* 으로만 등장**. 본문이 직접 설명하지 않는다. 인물의 *행동·복장·도구* 로 종파를 드러낸다.
2. **카엘은 1~3부에 *어떤 신앙도 신뢰하지 않는다***. 4-5 이후 *신앙 자체를 의심* 한다. 본문 *신* 단어 사용은 카엘 시점이 아닐 때만.
3. **모닝스타는 *신성마법 결박 매개* 임을 한 번 묘사** (4-3 옵션 또는 6-2 배경). 이후는 *그 무게* 만으로 작동.
4. **종파 간 갈등은 *외교 묘사로* 등장**. 직접 종교 전쟁은 본작 무대 아님 (배경으로만).
5. **신성마법 시전 자세 = 종파 의례 자세**. 손 모음 (루멘 통합) / 무릎 꿇음 (저그넛) / 가슴에 한 손 (르세프 본원) 등 동작이 시전 매개.
