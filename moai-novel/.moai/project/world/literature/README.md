## 인용 형식 (작품 표준)

본문 회차에 단편을 삽입할 때는 다음 형식을 따른다.

```markdown
> *"본문 인용 본문 인용."*
> — 저자명 (『책 제목』 N장 / N페이지)
```

발언·구전 형식:

```markdown
> *"본문."*
> — 저자명, 발언 시기·맥락
```

---

## 작품 활용 원칙

1. **단편은 *서술자의 말이 아니라 인용* 이다**. 본문 톤을 깨지 않는다.
2. **인용 출처는 *반드시 명기*** — 출처 자체가 *세계관 텍스처* 가 된다.
3. **같은 단편이 두 번 인용될 수 있다** — 1차는 오독, 2차는 재해석. (예: 사프란 A-1 *세상을 미워하는 자* 는 1-5와 4-5에서 다른 의미로 재인용)
4. **단편의 의미는 *본문이 직접 설명하지 않는다*** — 독자가 본문과 단편 사이의 *공명* 을 스스로 잡는다.
5. **금기 단편 (G 그룹)** 은 *본작 후반부* 에만 등장. 초·중반 누설 금지.

## 주인공의 *동료 결핍 → 자각 후 동료 형성* 의 문헌적 인과

> 사용자 요청 핵심 — 카엘은 *친구·동료 없음*. 1부 자각 후 인성 변화로 동료가 생긴다.

| 시기 | 동료 상태 | 매개 문헌 | 의미 |
|------|----------|---------|------|
| 1부 (출생~추방) | **0명**. 단·리엔과의 관계는 *왜곡으로 파괴* | 친조부의 책 = 1세대 동료의 *간접 유증* | 카엘이 인식하지 못한 *최초의 호의* |
| 2부 (학도) | **0명**. 카르엔·미라의 호의를 모두 *적의로 번역* | 카르엔의 미완성 경고 서한 (D-1) | *동료가 되려 했던 자의 마지막 외침* |
| 3부 (사령관) | **0명**. 부관·동료 셉터 모두 *도구로 사용* | 라이세린의 편지 (D-2) | *후원자조차 동료가 아니었음* |
| 4부 (각성) | **0명 → 자기 회의** | 알레이언의 마지막 한마디 + 평화 협정 | *적이었던 자가 동료의 자리에 있었음* |
| 5부 (잠적) | **1명 — 세파 류르** (5-2 합류) | 카르엔 봉인 문서가 *세파를 데려옴* | 학파의 비전이 *동료를 만든다* |
| 5부 후반 | **1명 + 오릴 (일시)** | 사프란 격언 A-2 *사랑을 약점이라 하는 자들* | *받음* 의 가능성 |
| 6부 (재기) | **1명 → 2명** (세이른 추가) | 강철제국 무경 공허검 부록 + 린넬 허프 F-3 | *검의 옆자리* — 동료의 자리 |
| 6-7 (결전) | **세파 *떠나지 않음* 으로 동료 라인 확정** | A-2 사랑의 격언 재인용 가능 | 동료의 *선택* 으로 동료가 된다 |

> **모든 동료 형성의 매개는 *문헌* 이다**. 사프란이 8왕을 모았던 방식 — *지식을 통한 결합* — 의 작은 반복이 카엘의 동료 형성 과정에도 적용된다. 이것이 본작이 *지식·문헌의 무게* 를 강조하는 구조적 이유.

## 확장 권장 (향후 작업)

집필 진행 중 다음 단편이 필요해질 가능성:

- 행커만의 *드래곤 정체 폭로 직전* 마지막 강의록 한 페이지
- 모티어스 매리앤의 *3년 침묵 직전* 마지막 말
- 카르마서스의 *미드그레이드 공격 직전* 단 한 줄 외침 (구전)
- 데미안 그르긴 1세의 룬 도식 메모
- 헬리오스의 *상아탑 분열 직후* 첫 회보
- 8왕 회담의 *유일한 기록자* 의 짧은 메모
- 침묵절(12월) 마법사들의 *기도문* 한 줄
- 모리간 대모의 *9혈 회담 직전* 결단의 시
- 대수호자 아엘의 *마지막 노래* 전문
- 첫 마왕 강림 전조에 대한 변경 마을 일기 (중반부 진입 시)

각 항목은 회차 진행 중 필요 시 [`fragments.md`](./fragments.md) 에 추가한다.
