# 8왕국 — 신좌침묵의 동맹과 그 운명

신좌침묵 전쟁(CC -3 ~ CC 0) 당시 대륙을 분할 통치하던 8개의 강대국. 본세대 대현자 *사프란* 의 *연합 의식* 에 참가한 군주들이 이 8왕이다.

타임라인 맥락: [`timeline.md`](./timeline.md) IV ~ V
대현자: [`great-sage-and-entroper.md`](./great-sage-and-entroper.md)
달력: [`calendar-and-time.md`](./calendar-and-time.md)

## 8왕의 공통 성격

- **각 왕은 그 자체가 *마나 응축 개체* 다**. 한 국가의 정점에 평생 동안 마나가 모인 결과, 그들의 죽음은 *왕국 단위의 엔트로픽 농도 해소* 와 같은 효과를 낸다.
- 이 사실이 후일 *앤트로퍼의 진짜 표적 우선순위* 가 되는 이유다 (`high-density-targets.md` 참조).
- 8왕은 *연합 의식* 에서 자신의 마나 회로를 *결절점* 으로 봉헌. **5명이 의식 도중 사망, 3명이 생존하되 회로 손상**.

## 1. 알마리스 마법왕국 (Almaris) — 생존

- **위치**: 대륙 중앙 평원
- **마법 정통**: 원소 4계열(불·물·바람·대지) 정통. *고대마법이론* (행커만 저서)의 토대가 된 사고 방식.
- **왕**: **아르카논 4세 (Arcanon IV)** — 8왕 중 가장 젊었던 군주(당시 38세). 가장 *유연한* 마법사로 평가받음.
- **회로 봉헌**: *결절점 4개* 중 *균형 결절* 을 담당. 의식 후 마나 회로 80% 손상, 마법사로서는 사실상 은퇴.
- **의식 후 행적**: 협정 체결의 *조정자* 역할. CC 0~40 통치 후 평민으로 양위. 사망 시점 CC 47.
- **현재 후신**: **헤븐즈 킹덤**. 바이서스 왕가는 아르카논의 *외손계* 분파. 정통성 강함.
- **본작 연결**: 주인공이 진입한 *바이서스 왕립 연구원* 의 *역대 학장* 명단 1번이 아르카논. 학원 휘장은 아르카논의 *4원소 봉인진* 도식.

## 2. 벨라리온 마법왕국 (Belarion) — 사망

- **위치**: 알마리스와 인접한 서북 산악
- **마법 정통**: *중력·공간 응용*. 알마리스의 4원소 정통과 *대비되는* 학파. 두 왕국은 평시에는 *경쟁 관계*.
- **왕**: **일레안 7세 (Ilean VII)** — 9써클 직전 도달. 일레인 맥필렌의 먼 *방계 후예* 가 *벨라리온의 마지막 후예* 라는 가설 (`legendary-figures.md` 일레인 항목 참조).
- **회로 봉헌**: *공간 차단 결절* 담당. *역류의 분산 방향* 을 설계한 핵심 인물. 의식 도중 *마나 회로 완전 붕괴* — 사망.
- **왕국의 멸망**: 왕 사망 후 후계자 없음. 알마리스가 영토 흡수 시도 → 잔존 마법사들의 *반란*. 결국 *벨라리온 왕국 명목 소멸*, 잔존 학파가 동쪽으로 이주.
- **현재 후신**: **엘더스톰**. 일레안의 *중력·공간 응용 마법* 이 엘더스톰 군대의 정체성. *일레인 맥필렌* 의 *압축·중력 응용* 마법이 이 계보의 정점.
- **본작 연결**: 엘더스톰 vs 헤븐즈 킹덤의 *천 년 묵은 라이벌리* 의 본질 — *알마리스 vs 벨라리온* 의 학파 갈등 연장.

## 3. 카랑크 기사왕국 (Karangk) — 생존

- **위치**: 북부 한랭 고원
- **성격**: *마법 없는 검기* 의 정점. 왕족·기사 모두 *검에 마나를 흘려* 강화하는 *검기계* 사용.
- **왕**: **베른하르트 강철왕 (Bernhardt der Stahlkönig)** — 8왕 중 가장 *고령* (당시 67세). 평생을 전장에서 보낸 무인.
- **회로 봉헌**: *물리적 압력 결절* 담당 — 의식의 *충격을 받아내는* 역할. 신체 절반이 *결정화* 되었으나 *살아남음*.
- **의식 후 행적**: 잔존 신체로 50년 더 통치. CC 50 사망. 그의 *마지막 검* 이 강철제국 황실 보검.
- **현재 후신**: **강철제국**. 아라곤 스틸만 (`legendary-figures.md`)이 베른하르트의 *7대손* 이라는 공식 계보.
- **본작 연결**: 6부 세이른의 *마법 없는 검술* 도장은 *카랑크 비전 검술* 의 변경 분파. 세이른은 강철제국 본가의 *추방된* 무인.

## 4. 루멘 신성제국 (Lumen) — 사망

- **위치**: 서부 일조 강한 평원 + 해안
- **성격**: *신성마법* 정통. 르세프 계열 봉인·결계·치유. *균열의 빛* 봉인 의식의 *원형* 을 보유한 유일 왕국.
- **왕**: **알레이엔 1세 (Aleien I)** — 8왕 중 가장 *나이 어림* (즉위 25세, 신좌침묵 시 28세). 본인의 신성력이 *대륙 최강 등급*.
- **회로 봉헌**: *균열 봉합 결절* 담당 — 마계 균열을 *닫는* 핵심. 자신의 신성마법 전부를 의식에 쏟아 부음. *사망*.
- **왕국의 멸망**: 알레이엔 1세 외에는 *왕가 핵심 인물* 이 거의 없었음(젊은 왕). 잔존 신관단이 *왕가 비전 의식의 절반* 만 챙겨 동쪽으로 망명.
- **현재 후신**: **카르샤 왕국**. 알레이엔 1세의 *조카* 가 망명단을 이끌어 동쪽 변경에 *작은 신정 왕국* 건설. 1대 카르샤 왕 = 알레이엔의 조카 *코르넬리스*. 본작 알레이언 6세는 그의 *7대 직계*.
- **본작 연결**:
  - 왕의 마지막 대사 *"우리 왕가가 7대를 기다린 자"* = 코르넬리스부터 알레이언까지 정확히 7대.
  - *균열의 빛* 봉인 의식 = 알레이엔 1세의 *원형 의식* 의 *절반*. 카르샤 왕가는 *나머지 절반* 을 800년간 찾아왔다.
  - 손 모음 인사 = 알레이엔 1세의 *마지막 자세* 를 신앙으로 보존한 것.

## 5. 이실데 동부 마녀 연맹 (Isilde) — 생존

- **위치**: 동부 늪지·심림
- **성격**: 9개 혈맥의 *대모(大母)* 들이 합의로 운영하는 *연맹제*. 단일 왕 없음. 혈주술·저주·금기 마법.
- **대표 왕**: **모리간 대모 (Morrigan)** — 9혈 중 *흑혈 (Black Coven)* 의 수장. 8왕 회담에서 *연맹 전체* 를 대표.
- **회로 봉헌**: *해체 회수 결절* 담당 — 의식이 만든 *분산 에너지* 를 무스페름으로 되돌리는 역할. 모리간 본인은 *살아남았으나* 9혈 중 *4혈* 의 대모가 동시 사망.
- **의식 후 행적**: 모리간은 CC 30년경까지 생존. 그녀의 후계자들이 *현 마녀령* 의 9개 가문 시조.
- **현재 후신**: **마녀령 동부국** — *왕정 없음*. 9혈 중 멸문된 4혈 (회혈·녹혈·자혈·금혈·무혈)이 CC 800~1000 사이 *회복 1~2세대* 로 복원되었고, 현재 9혈 모두 *불완전 상태로 생존*. CC ~700 *모르간 가문의 왕정 시도* 는 9혈의 자율성을 침해해 *원탁회의에 의해 폐기*. 현재는 **마녀원탁회의 16인** (9혈 종주 9 + 중립 원로 7) 합의제. 자세한 구조는 [`witch-eastern-council.md`](./witch-eastern-council.md).
- **본작 연결**: 6부 *린이엔 황녀* 는 *흑혈 차기 종주*. *황녀* 호칭은 *외부 사피엔이 마녀령을 이해하지 못해 잘못 부르는 경칭* — 모르간 가문 옛 왕정 시도의 잔재. 내부 마녀들 사이에서는 *흑혈 차기 종주* 로만 호칭.

## 6. 타르모리스 마법공학 왕국 (Tarmoris) — 사망

- **위치**: 남서 광산 지대
- **성격**: *룬 각인 + 기계 결합* 의 시조. 신좌침묵 당시 *마도 기관* 의 원형(증기·마나 동력 혼합)을 가졌던 유일 왕국.
- **왕**: **그라가르드 기관왕 (Gragard der Maschinenkönig)** — 본인이 *마도공학자* 이자 *왕*. 의수·의안·기계 갑주를 평상시에도 착용.
- **회로 봉헌**: *동력 증폭 결절* 담당. 자신의 *기계 갑주를 의식 회로의 증폭기로 변환*. 갑주와 함께 *증발*.
- **왕국의 멸망**: 왕 사망 + 기술자 절반 사망. 잔존 기술자들이 *남부 망명* 후 *제국 데미안* 의 모태가 됨. *데미안* 이라는 국명은 *데미안 스펠른* (CC 900대 인물)에게서 따왔지만, 그 *기술 토대* 는 타르모리스의 잔재.
- **현재 후신**: **제국 데미안**. *스펠 마법 + 마도공학* 의 양대 정통성을 보유.
- **본작 연결**: 6-2의 *마도 의수* 시술 도시 *그라스* 는 타르모리스 잔존 기술자의 후예 집단. 주인공의 의수에 새겨진 룬어는 *그라가르드의 갑주 룬* 의 변형.

## 7. 상아탑 지식국 (Ivory Concord) — 사실상 사망

- **위치**: 대륙 중앙 호수의 인공 섬
- **성격**: *왕 없는 도시국*. 7명의 *현인(賢人)* 합의제. 모든 학문을 *수집·보관* 하는 학자 연합. 사프란 본인이 *여기 출신*.
- **회로 봉헌**: *지식 조율 결절* 담당 — 의식의 *변수 조정자*. 7현인 중 **5명 의식 중 사망**, 2명 생존.
- **왕국의 멸망**: 생존한 2현인이 *앤트로퍼 시스템의 도덕성* 을 놓고 분열. CC 4년 *상아탑 분열 사건* 으로 정치체 소멸. 한쪽은 헤븐즈 킹덤에 흡수되며 *국립상아탑* 의 모태가 되었고, 다른 쪽은 *변경으로 잠적* 하여 *사프란 학파 비전 보존단* (후일 카르엔 학파의 시조)이 됨.
- **현재 후신**:
  - **국립상아탑** (헤븐즈 킹덤 내) — 학술 잔재
  - **국제마법과학원** — 행커만 시대에 잔존 지식을 재집결시킨 후신
  - **카르엔 학파** — 잠적한 분파의 후예. 본작 *카르엔* + *세파* 가 직계.
- **본작 연결**: 사프란의 가장 중요한 *비전 지식* 은 *분열된 다른 쪽 (= 카르엔 학파)* 이 보존. 카르엔의 봉인 문서가 *왜 다른 학파에 없는가* 의 답.

## 8. 숲의 동맹 (Sylvan Pact) — 사망

- **위치**: 동대륙 심림 (현재 칠흑숲의 모태)
- **성격**: 엘프 + 드루이드 + 일부 정령 사역자의 *자연 동맹*. 왕정 아님. *대수호자* 가 명목 대표.
- **대표**: **대수호자 아엘 (Ael der Weiße)** — *백색 엘프* 라는 별명. 당시 *6백 살*. 9써클 도달.
- **회로 봉헌**: *자연 균형 결절* 담당. 자신의 *6백 년 마나* 전체를 의식에 부음. 의식 직후 *완전 분해* — 시신 없음. 엘프의 전승에서 *별이 되었다* 고 함.
- **왕국의 멸망**: 동맹 자체는 *정치체로서 소멸*. 잔존 엘프들이 *칠흑숲* 으로 후퇴.
- **현재 후신**: 정치체 없음. *다크 엘프 종족 형성* — 동맹 붕괴 후 어둠에 동화된 엘프들의 변형.
- **본작 연결**: 5부 빙벽 협곡으로 가는 길에 잠시 마주치는 *변경 엘프 마을* 이 숲의 동맹 후예. 그들은 *대현자를 원망하지 않지만* *인간을 신뢰하지 않는다*.

## 8왕국의 사후 정치 지형

```
        [3왕 생존]                       [5왕 사망 → 정치체 소멸/잔존]
   ┌─────────────────────┐         ┌──────────────────────────────────┐
   │ 알마리스(아르카논)  │         │ 벨라리온(일레안) — 후계 분파      │
   │   → 헤븐즈 킹덤     │ ↔ 경쟁   │   → 엘더스톰                      │
   ├─────────────────────┤         │ 루멘(알레이엔) — 조카 망명        │
   │ 카랑크(베른하르트)  │         │   → 카르샤 (소국)                 │
   │   → 강철제국        │         │ 타르모리스(그라가르드) — 기술자   │
   ├─────────────────────┤         │   → 제국 데미안                   │
   │ 이실데(모리간)      │         │ 상아탑(7현인) — 분열              │
   │   → 마녀령 동부국   │         │   → 국립상아탑 / 카르엔 학파       │
   └─────────────────────┘         │ 숲의 동맹(아엘) — 정치체 소멸     │
                                   │   → 칠흑숲 변경 엘프              │
                                   └──────────────────────────────────┘
```

## 정치적 함의 — 본작 갈등의 뿌리

- **3 생존왕국 vs 5 사망왕국 잔재**: 후자는 *우리가 더 많이 죽었다* 는 정서를 보존. 이 정서가 *대현자 비판* 과 *반앤트로퍼 음모* 의 동기.
- **알마리스 vs 벨라리온의 학파 갈등** → 헤븐즈 킹덤 vs 엘더스톰의 *마법 외교 라이벌리* 로 계승.
- **카르샤의 작음과 가난** → 7대 동안 *균열의 빛* 분할 의식의 *나머지 절반* 을 찾아 헤맨 *집착*. 알레이언 6세가 *주인공을 보자마자 사슬 7마디를 식별* 한 것은 7대의 비전 훈련의 결과.
- **이실데 5혈 회복** = 마녀령의 *현재 안정기*. 린이엔 황녀 라인의 *왕정 도입 시도* 가 9혈 가문들의 *전통 위협*. 6부 동부국 정치 갈등의 배경.
- **상아탑 분열의 두 후예** = *국가 권력* vs *비전 보존*. 카르엔 학파가 *왜 늘 변경에 숨어 있는가* 의 이유.
- **숲의 동맹 후예**의 *침묵* = 다음 위기에 *방관할지 동참할지* 의 정치 변수. 후반부 카드.

## 작품 활용

- 1~3부: 주인공의 행적은 모두 *알마리스-벨라리온 후예 (헤븐즈 킹덤-엘더스톰)* 권역에서 일어난다.
- 4부: *루멘 잔존 (카르샤)* 과의 충돌. 7대의 무게.
- 5부: *상아탑 분열 후예 (카르엔 학파)* 의 진실 폭로 + *숲의 동맹 후예* 변경의 침묵.
- 6부 이후: *이실데 (마녀령 동부국)*, *카랑크 (강철제국 검술 분파)*, *타르모리스 (마도 의수 기술자)* 가 모두 등장. **8왕국 후예 전부가 6부 시점에 다시 연결되며 협정의 의미를 재질문하게 된다**.
