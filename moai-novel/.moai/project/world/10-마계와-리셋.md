# Part A — 생명 리셋과 마계 전쟁


## 생명 초기화

마나 평형이 한계를 넘었을 때 발생하는 대륙적 해체 현상. 생명에 묶인 마나를 풀어 자연계로 되돌린다.

## 마계의 역할

- 마계는 단순한 악마 왕국이 아니다.
- 리투니아의 마나 흐름이 역류할 때 접속되는 해체 질서다.
- 마물은 생명을 증오해서가 아니라 과밀한 마나 구조를 해체하기 위해 움직인다.

## 신좌침묵

수백 년 전 발생한 대규모 생명 초기화 위기. 과거에는 여러 제국의 최고마법사, 기사, 성기사, 마녀들이 참전했다.

- 통칭 후보: `신좌침묵`.
- 민중 인식: 마계의 대침공.
- 실제 원인: 리투니아의 엔트로픽 농도 임계 초과.
- 신화적 의미: 신들이 침묵했고, 인간 영웅들이 스스로 세계를 지킨 전쟁.
- 은폐 이유: 신성 권위와 제국 정통성에 불리한 기록이기 때문.

## 대현자의 역할

- 마물을 전부 죽인 것이 아니라 역류의 흐름을 돌려 초기화를 중단시켰다.
- 전쟁의 본질이 침공이 아니라 평형 반작용임을 알아냈다.
- 전후에 `앤트로퍼` 계획을 구상했다.

## 전후 질서

- 기존 제국 다수가 붕괴했다.
- 살아남은 영웅들이 각지에 새 제국과 왕국을 세웠다.
- 새 질서는 인간의 번영을 이끌었지만, 인구 증가와 마법 문명 확장으로 다시 엔트로픽 농도를 높였다.
- 크고 작은 전쟁은 생명을 줄이면서도 원한과 잔향을 남겨 문제를 완전히 해결하지 못했다.

## 은폐된 진실

후대는 신좌침묵을 영웅들의 승리로 기억한다. 그러나 대현자의 후반기 연구와 앤트로퍼 계획은 금서, 비밀 조직, 일부 왕실 기록에만 남아 있다.

## 1. 본질 정의

### 마계 창조물 (Demon Creature) 이란

- **표면 인식**: 마계의 악마. 인간을 학살하러 온 적
- **실제 본질**: *마계 균열을 통해 분출되는 *해체 마나 응집체***. 의지 없거나 일부만 있음
- **출현 조건**: 엔트로픽 농도가 임계에 도달해 *마나 역류* 가 발생할 때
- **목적**: *과밀한 생명 마나 구조* 를 해체. 그 결과로 인간을 *공격하는 것처럼 보임*
- **종족 아님**: 자기 영구 종족이 아닌 *균열에 한정된 일시 존재*. 균열이 닫히면 *자연 소멸*

> *"마계 창조물은 *생명을 증오하지 않는다*. 그들은 *과밀한 마나를 풀어주는* 자연 현상이다. 우리가 그들에게 *증오의 얼굴* 을 씌운 것뿐이다."*
> — 사프란, 『앤트로퍼 비망록』 CC -10

## 2. 6대 유형 — 등급별 분류

### 2.1 잔재 (Remnant) — 1등급

- **외형**: 형체가 *불완전*. 그림자 + 빛의 잔해처럼 일그러진 형태. 1~3미터
- **개체 마나량**: 일반 사피엔 1~5명 분
- **자아**: *없음*. 본능적 *마나 응집체로의 이끌림* 만
- **약점**: 일반 검·화살로도 처치 가능. 다만 자기가 다친 자리에서 *마나 결박을 풀고* 다시 응집
- **출현**: 마계 균열 *주변 100미터* 내. 균열이 작아도 다수 출현
- **본작 등장**: 5-3 변경 엘프 마을에서 *3주 전 잔재 2체 격퇴* 흔적 (배경)

### 2.2 사역마 (Familiar) — 2등급

- **외형**: 작은 4족 짐승 형태. 검은 비늘 + 붉은 눈. 키 50cm~1m
- **개체 마나량**: 9써클 마법사의 1/100
- **자아**: *기본 사역 지능*. 누군가가 *소환* 했거나 *영지 군주의 종속체*
- **약점**: 마법 결계로 *경계 정지*. 직접 공격은 무리. 떼로 공격
- **출현**: 균열 + *소환사* 가 매개. 마녀령 금혈 가문이 *유일하게 합법 소환*
- **본작 등장**: 후반부 마왕 강림 전조에 *무리* 출현 가능

### 2.3 마계 영지 군주 (Demon Lord) — 3등급

- **외형**: 인간형 + 변형. 키 2~3미터. 뿔·이빨·날개·꼬리 중 1~2가지 보유. 검은 갑주 같은 외피
- **개체 마나량**: 일반 사피엔 *2,000~5,000명 분* (왕급 인간 응축)
- **자아**: *완전 지능*. 언어 사용 (다양한 마계 방언). 협상·기만 가능
- **약점**: 신성마법에 약함. 단, 일반 9써클 마법은 *통하나 위력 절반*
- **출현**: 큰 마계 균열 + 농도 임계. 자연 발생 또는 *금혈 마녀가 의도적 소환*
- **사망 시 해소량**: 약 20,000~50,000 단위 (`high-density-targets.md`)
- **역사 사례**: 사프란이 직접 처치 1체 (CC 4) / 1대 모나르가 처치 3체 (CC 4~50). 그 외 약 15체 자연 발생 → 인간 사회가 공동 격퇴

### 2.4 마왕 (Demon King) — 4등급

- **외형**: 거대 인간형 + 강력 변형. 키 5~8미터. 검은 광휘 + 8개 손 / 다중 머리 / 박쥐 날개 같은 *상징적 압도 외형*
- **개체 마나량**: 드래곤 성체급. 일반 사피엔 *100,000~150,000명 분*
- **자아**: 완전 + 카리스마. *통솔력* 으로 마계 영지 군주들을 *연합 지휘*
- **약점**: *없음* — 단 *균열의 빛* 같은 *결박 해체 마법* 에 약함
- **출현 조건**: 마계 균열 *임계 임박*. 농도 시한 *6개월 이내*. 강림 = *리셋이 거의 확정* 된 신호
- **사망 시 해소량**: 약 150,000 단위 — 드래곤보다 *약간 더 큼*
- **부작용**: 마왕 살해 시 *마나 역류 일시 가속* — 한 번의 큰 균열이 자동 발생. 즉 마왕은 *죽여도 작은 대가*
- **역사 사례**: 신좌침묵 시대 *1체* 만 알려짐. 사프란의 *균열의 빛 원형 의식* 으로 처치 (CC 0)
- **본작 함의**: **중반부 첫 마왕 강림** — 카엘의 *진짜 속죄전 본격 무대*

### 2.5 마계 균열 핵 (Rift Core) — 5등급

- **외형**: 균열 그 자체. *공간의 일그러진 구멍*. 인간이 *물리적으로 볼 수 있는 형태가 아님*
- **마나량**: 측정 불가 — *무한에 가까움*
- **자아**: *없음*. 다만 *균열을 유지하려는 본능적 기제*
- **약점**: 일반 마법 무력. *마녀령 회수 회로 + 카르샤 균열의 빛* 의 *결합 의식* 만 해체 가능
- **출현**: 임계 시 *대륙에 1~3개* 동시 형성
- **해체 시 해소량**: 50,000~100,000 단위 + *균열 영구 봉인 불능 부작용*
- **본작 함의**: 8대 앤트로퍼 셀라가 *유일하게 1체 해체* (CC 580). 그녀는 직후 *자기 분해* 사망

### 2.6 마계 진주 (Demon Pearl) — 6등급 / 특수

- **외형**: 작은 검은 구슬. 지름 5cm. *마계 균열의 결정화 잔재*
- **마나량**: 큰 마계 영지 군주 분
- **자아**: 없음. 단 *주변에 영향*
- **위험**: 인간이 가까이 가면 *영혼 결박이 풀려나가는 부작용*. 만지면 즉사
- **활용**: 마녀령 금혈 가문이 *극도의 위험* 으로 *마계 계약 매개* 로 사용
- **출현**: 마왕이 사망한 자리에 *3~5개* 형성. 균열 해체 자리에도 가끔

## 3. 출현 패턴 — 시기·지역·전조

### 시기

| 농도 수준 | 출현 양상 |
|----------|----------|
| 안정기 (~CC 0 평형) | 0체 |
| 농밀기 (CC 0~500) | 잔재 산발 출현 |
| 정체기 (CC 500~800) | 사역마·마계 영지 군주 *연 1~2체* |
| 역류 전조 (CC 800~1000 현재) | 영지 군주 *연 3~5체* + 균열 핵 *5년에 1개* |
| 임계 임박 (다음 50년 예상) | **마왕 강림 + 균열 핵 동시 다발** |

> **현 본작 시점 (CC 1000) = *역류 전조의 정점***. 마왕 강림이 *10~30년 내* 예상.

### 지역

- **균열 다발 지역**:
  - 칠흑숲 — 변경 엘프 마을이 *3~5년에 1회 잔재 격퇴*
  - 강철제국 북부 빙벽 협곡 — 추정 (이실 빙벽 협곡 일대)
  - 미드그레이드 외곽 — 카르마서스 사건 (CC ~870) 이후 *균열 다발 의심*
  - **남부 사막** — 침묵의 사원 인근. *고요한 손* 사제들이 *균열 동향 관측*
- **균열 없음**:
  - 8왕국 수도 (강력 결계로 차단)
  - 미라돈 (마도공학 결계 차단)
  - 마녀령 오르하임 (9혈 공동 결계)

### 전조

- **마나 농도 급변** — 측정 가능 (상아탑·자혈 마녀·카르엔 학파 모니터링)
- **르세프 신성마법 일시 작동 불능** — 신좌침묵 직전과 동일
- **드래곤 갑작스러운 깸** — 카르마서스 사건이 *균열 직전 신호* 였다는 가설
- **앤트로퍼 활성도 급증** — 본작 카엘의 비정상적 회로 효율 = *씨앗 마지막 활성기 + 균열 임박* 의 동시 신호 가설

## 4. 처치 방법 — 무기·전술

### 1등급 잔재

- 일반 검 / 화살로 가능
- 단 *마법 결계로 응집 방지* 후 처치 권장 (재응집 차단)

### 2등급 사역마

- 마법 결계로 *경계 정지* → 화살 일제 사격
- 9써클 광역 마법 1발로 *떼 전체 소각*

### 3등급 마계 영지 군주

- **신성마법 5등급 + 마법사 9써클 + 마스터급 검사** 의 3인 조합이 정통
- 영지 군주 1체 격퇴 = *대도시 한 구역의 동원력* 필요
- *적은 자원* 으로 처치 시도 = *시도자가 모두 사망*

### 4등급 마왕

- 일반 군대 무력
- **균열의 빛 + 마녀령 회수 회로 결합 의식** 만 처치 가능
- 사프란 + 8왕의 *연합 의식* 이 유일한 역사 사례
- *부작용 = 마나 역류 일시 가속* — 균열 추가 발생

### 5등급 균열 핵

- **앤트로퍼 숙주 + 마녀령 회수 회로 + 카르샤 균열의 빛** 의 *3중 결합 의식* 만 가능
- 셀라(8대 앤트로퍼) 처치 = 흑혈 가문 회수 회로 + 본인 *자기 분해* 의 2축
- *균열 영구 봉인 불능 부작용* — 그 자리에는 *다시는 결계 불가*

### 6등급 마계 진주

- 처치 대신 *밀봉 + 영구 격리*
- 금혈 마녀가 *극도 위험* 으로 *마계 계약 매개* 사용

## 5. 본작 후반부 적용 — 카엘의 속죄전 무대

### 진행 순서 (예상)

1. **중반부 진입**: 농도 측정 + 균열 핵 1체 발견 → 본격 위기 도래
2. **마왕 강림** (중반부 중간): 첫 마왕 등장. 카엘이 *진짜 적* 을 본다
3. **마왕 처치 시도** (중반부 종결): *균열의 빛 결합 의식 시도* — 카르샤 잔재 수녀단 (셀라엔 + 엘라리아) + 마녀령 흑혈 (모르간 솔테) + 카엘 (앤트로퍼). **8왕 연합 의식의 *작은 재현***
4. **고룡 + 정령왕 사냥** (후반부 전반): 카엘이 *드래곤 로드의 추종파* 와 *대등한 무력* 으로 도전
5. **균열 핵 동시 해체** (후반부 후반): 마지막 표적군
6. **카엘의 최종 카드** (종결부): 본인의 *자결* 로 마지막 농도 해소

### 핵심 인물 동선 매트릭스

| 표적 | 처치 매개 | 핵심 인물 |
|------|---------|----------|
| 잔재·사역마 | 일반 마법·검 | 카엘 + 세파 + 세이른 일행 |
| 마계 영지 군주 | 3인 정통 조합 | 카엘 (9써클) + 세이른 (검) + 셀라엔 대주교 (신성마법) |
| **마왕** | 균열의 빛 + 회수 회로 결합 | 카엘 + 엘라리아·셀라엔 (카르샤 잔재) + 모르간 솔테 (흑혈) + 베르난 호산 (금혈 협상) |
| 균열 핵 | 3중 결합 의식 | 카엘 본인 매개 (자기 분해 위험) |
| 고룡 | *밀착 거리 매복* | 카엘 + 카이엔 (만약 화해) + 세이른 (공허검) |
| 정령왕 | 부작용 감수 + 광역 의식 | 카엘 + 자혈 점성 예언 |
| 마계 진주 | 밀봉 | 금혈 가문 |
| **앤트로퍼 숙주 자신** | 자결 | 카엘 본인 |

## 6. 묘사 가이드 — 본문 등장 시

### 시각 묘사 원칙

- 잔재 = *형체 없는 음영*. *그림자가 굳어진 듯한*

- 균열 핵 = *공간의 일그러짐. 묘사 불가*
- 마계 진주 = *작은 검은 구슬. 그러나 *보는 것만으로 메스꺼움**

### 후각·청각 묘사 원칙

- 마계 창조물 = *비린내 + 탄 냄새 + 금속 냄새*

# Part C — 고밀도 표적


본 문서는 *사프란이 앤트로퍼를 설계한 진짜 의도* 와 그 *우선 표적* 을 정리한다. 본작의 후반부 *속죄전* 의 표적 카탈로그.

타임라인: [`timeline.md`](./timeline.md)
앤트로퍼 메커니즘: [`great-sage-and-entroper.md`](./great-sage-and-entroper.md)
대현자 계보: [`great-sages-lineage.md`](./great-sages-lineage.md)

## 핵심 원리

**같은 양의 마나가 해체될 때, 그 마나가 *얼마나 한 곳에 응축되어 있었는가* 가 엔트로픽 농도 해소 효율을 결정한다.**

- 일반 사피엔 1명 살해 = 농도 해소 단위 1 (기준값)
- 일반 사피엔 1만 명 학살 = 약 10,000 (단순 합)
- *고마나 응축 단일 개체* 1명 = **수만 ~ 수십만 단위** (응축 효율 보너스)

이유:
- 흩어진 마나는 *결박* 도 흩어져 있다 — 해체해도 *작은 결박 만 개* 가 풀릴 뿐.
- 응축된 마나는 *결박 자체가 한 덩어리* 다 — 핵심 매듭 하나가 풀리면 *주변 마나 전체* 가 연쇄 해방.
- 사프란의 비유: *"실타래 하나를 풀면 백만 가닥이 풀린다. 백만 가닥을 따로 푸는 것보다 훨씬 빠르다."*

## 표적 우선순위

### 1차 표적 — 고마나 응축 거대 개체

수가 적고, 한 명의 죽음이 *왕국 단위 농도 해소* 효과.

#### 1.1 드래곤

- **응축 효율**: 1마리 = 약 *왕국 1~2개* 의 일반 사피엔 인구 합과 동등.
- **이유**: 드래곤은 *그 자체로 거대한 세프 덩어리* (`races.md`). 폴리모프하지 않은 본태 드래곤 1마리의 마나량은 9써클 사피엔 마법사의 *수백 배*.
- **사냥 난이도**: *극악*. 사피엔 9써클 마스터 다수가 협공해도 1마리 처치가 불확실. *앤트로퍼 숙주가 *그래서* 필요* 했다.
- **본작 연결**: 5부 이실(폴리모프 드래곤). 6부 카이엔. 주인공이 의식적으로든 무의식적으로든 *드래곤을 추적한 것* 은 *앤트로퍼의 1차 표적 본능* 이 작동한 결과.

#### 1.2 마왕 (Demon King)

- **응축 효율**: 드래곤급. 1체 = 왕국 1~2개 분량.
- **본질**: *마계의 창조물* 중 *자아 + 통솔력* 을 가진 *해체 개체의 정점*. 마계 균열이 임계 임박 시 출현.
- **차이점**: 드래곤은 *생명* 이지만 마왕은 *해체 그 자체의 의인화*. 죽이면 해체 회수 효율이 *드래곤보다 약간 높다* — 그러나 죽이는 행위 자체가 *마나 역류* 를 일시 가속하는 부작용.
- **본작 시점**: 현재 (CC 1000) 마왕급 개체는 *아직 출현 전*. 본작 중반부에 *첫 마왕* 의 강림이 예고된다 — *진짜 적*.

#### 1.3 정령왕 (Spirit King)

- **응축 효율**: 드래곤급. 단, 죽이는 방식의 *부작용* 이 가장 큼.
- **본질**: 자연계의 *대정령* 의 정점. 숲의 왕(Sylvan King), 바다의 왕(Tidal King), 바람의 왕(Storm King), 대지의 왕(Mountain King) 등 *지역별로 1체* 씩 존재.
- **위험**: 정령왕은 *그 지역 자연 생태계의 결절점*. 죽이면 *해당 지역의 마나 균형이 붕괴* → 가뭄·홍수·산사태·전염병이 *수십 년간 지속*. 인간·동물·식물 모두 큰 피해.
- **사프란의 권고**: *최후의 수단으로만*. 정령왕 사냥은 *문명을 부분적으로 희생* 하는 선택.
- **본작 연결**: 후반부 *주인공이 마지막 수단* 으로 고려하는 표적군.

#### 1.4 기타 1차 표적 (사프란 분류)

- **고룡 (Ancient Dragon / 1000살 이상)**: 일반 드래곤보다 응축 *5배*. 1마리 = 왕국 5~10개 분량.
- **리치 로드 (Lich Lord)**: 자발적 언데드화한 *옛 9써클 마법사* 가 *천 년 이상* 영혼 결박을 누적한 경우. 드래곤보다 작지만 응축은 비슷. 예) *검은평원 전투* 의 적장 리치.
- **세계수 수호자 (World-Tree Guardian)**: *엘프 대수호자* 중에서도 *천 년* 을 넘긴 자. 숲의 동맹 1대 *아엘* 이 이 등급. 정령왕에 준하는 부작용 위험.
- **무스페름 균열 핵 (Muspherm Rift Core)**: 마계 균열이 *고정된* 지점. 균열을 *닫지 않고 핵을 직접 해체* 하면 큰 해소 효과. 단, 균열이 *영구 봉인 불능* 으로 변형되는 부작용.

### 2차 표적 — 인간 *왕·고위직* 중 고응축 개체

- **응축 효율**: 일반 사피엔의 *수십 ~ 수백 배*.
- **이유**: 평생 *최정상* 위치에서 마법·자원·인적 결박을 누적한 자들. *왕의 옥좌* 자체가 마나 결박의 결절점이며, 왕의 신체가 그 결절을 *흡수* 한다.
- **본작 핵심**: 알레이언 6세 = 카르샤 신성왕가 7대 응축 + 본인의 신성마법 정점 = *왕국 절반 분량 응축*. 그의 죽음 = 카르샤 왕국 인구 절반 학살과 동등한 해소 효과.
- **함의**: 라이세린 공작이 *주인공을 카르샤로 보낸* 진짜 동기 = *알레이언 한 명 죽이면 1만 명 죽이는 것보다 효율적* 이라는 정치적 계산. 8왕국 후예들이 *서로의 왕* 을 *앤트로퍼에게 죽이게* 하려고 음모를 짜는 진짜 이유.

### 3차 표적 — 일반 인구

- **응축 효율**: 기준값 1.
- **사프란의 원래 의도**: *1·2차로 부족할 때만*.
- **현재 변질**: 8왕국 분열 후 *1차 표적 (드래곤·마왕·정령왕) 사냥은 비용이 너무 큼*. 1차 표적 사냥은 *앤트로퍼 숙주를 잃을 위험* (이실의 분노가 그 증거). 2차 표적 사냥은 *정치 위기*. 따라서 후대 앤트로퍼들은 *3차 표적 (일반 학살) 으로 유도되어 왔다*.
- **본작 비극의 뿌리**: 주인공의 1~3부 학살은 *3차 표적 학살*. 1차 표적인 *이실* 과의 조우(5-4)는 의도가 아니라 *해주 시도* 였다. *진짜 사냥은 한 번도 한 적이 없었다*.

# Part D — 마나·엔트로피 밀도


## 마나

- 세계를 이루는 기본 구성요소.
- 물질, 생명, 정신, 마법은 모두 마나 배열의 차이로 구분된다.
- 마나는 자연 상태에서 순환하지만, 생명과 문명은 마나를 오래 붙잡는다.

## 생명이 문제를 만드는 이유

- 새 생명은 자연계의 자유 마나를 자신의 육체/영혼 구조에 묶는다.
- 인간은 기억, 감정, 언어, 사회성 때문에 특히 복잡한 마나 구조를 가진다.
- 문명은 도시, 마법, 왕권, 종교, 전쟁 기록을 통해 마나 결박을 강화한다.

## 엔트로픽 농도

`엔트로픽 농도`는 생명과 문명이 마나를 얼마나 복잡하게 묶어두었는지를 나타내는 압력이다.

- 단순한 마나량이 아니다.
- 순환하지 못하는 마나의 복잡도와 정체도를 뜻한다.
- 높아질수록 세계는 해체 방향의 반작용을 일으킨다.

## 단계 초안

1. 안정기: 순환이 정상. 마법과 생태계가 안정적.
2. 농밀기: 마법 효율 상승, 생태 변이, 이상 성장.
3. 정체기: 저주, 광증, 죽은 기억의 잔향 증가.
4. 역류 전조: 마계 균열, 마물 출현, 마나 흐름 반전.
5. 초기화: 생명 배열이 대륙 단위로 해체됨.

## 핵심 인과

생명 증가가 곧바로 악은 아니다. 문제는 생명과 문명이 마나를 과도하게 묶어 순환을 막는다는 점이다. 이 때문에 선한 번영도 장기적으로는 재앙의 원인이 된다.

## 엔트로픽 농축의 4대 경로

엔트로픽 마나는 다음 4가지 경로로 *고농축화* 된다. *가까이 있을수록 더 짙어짐*:

1. **생명의 잉태** — 새 생명이 자유 마나를 *육체·영혼 구조에 결박*
2. **마력의 축적** — 마법사·셉터·아티팩트가 마나를 *압축 보존*
3. **오래된 생명 활동** — 드래곤·언데드·정령 같은 *수백~수천 년 단일 결박체*
4. **집단 군체** — 대도시·종교 본산·전장 등 *생명 결박체 밀집 공간*

## 카오스 이론 역설

엔트로픽의 자연 원리는 *불확실성으로 흩어지려는 카오스* (평형 분산). 그러나 *고농축화* 는 이 원리에 *반대* 되는 현상. *임계점* 까지 농축되면 *큰 개연성으로 동작* — *해소(폭발·해체·붕괴)* 가 반드시 발생함. 이 누적이 곧 *마나 폭주 자연 리셋*.

## 다차원 평형 회귀

대륙 *한쪽* 에서 *대량 농축 엔트로픽 군체* 가 파괴되면(전쟁·역병·재해):

- 결박이 풀린 엔트로픽은 *증발하여 자연계로 회귀*
- 자연계 = 미드갈드(인간계) 단일 영역이 아니라 *다차원계 전체*. 영적·신적·정령적·마계적 영역까지 골고루 분산
- *중력* 도 본질은 *다차원 엔트로픽 평형의 가시화* — *질량은 다차원 마나 결박의 잔향*
- 이 다차원 분산은 *수일~수년* 에 걸쳐 천천히 진행

## 잉여 엔트로픽 → 마도공학의 명분

대륙 한쪽이 갑자기 비워지면 *다른 한쪽이 상대적 고농축 상태*. *잉여 농축 엔트로픽* 이 *흐를 개연성* 이 생기고, *흐름은 *이용 가능한 자원* 으로 보임*.

*마도공학 로봇·기계화 군단* 의 역사적 명분이 정확히 여기서 발생:
> *"비어 있는 쪽으로 흘러갈 마나라면, 차라리 우리 동력으로 쓰자."*

그러나 *이용은 자연 평형 회귀를 차단* 함 — *리셋 압력은 오히려 가속*. 본작 후반 *기계화 군단전* 의 세계관적 비극: *해결책으로 보이는 마도공학이 실제로는 리셋을 앞당기는 가속기*.
