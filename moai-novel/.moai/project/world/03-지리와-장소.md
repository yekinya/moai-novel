## 큰 단위

### 리투니아 (Lithunia)

작품의 무대가 되는 별/대륙의 이름. 마나가 충만하고 다종족이 거주한다.

### 리펠리아 (Lifelia)

태초의 대륙. 거인족이 유랑한 최초의 땅으로 전해진다. 현재는 흔적만 남았고 모험가들이 그 잔해에서 태초의 원리를 찾는다. **리투니아의 지질학적 모태**로 보는 것이 일반적.

### 무스페름 (Muspherm)

지하세계. 빛이 닿지 않는 곳에 크세프 계열 법칙과 종족들이 자라났다. 무차별한 지옥이 아니라 **해체와 회수의 영역**이다. 마계 균열은 본질적으로 무스페름의 마나가 표면으로 분출되는 현상.

## 인간 제국·왕국

신좌침묵 이후 생존 영웅들이 세운 국가들. 대륙의 정치 지형은 이들의 상호 견제로 유지된다.

### 헤븐즈 킹덤 (Heaven's Kingdom)

- 위치: 중앙.
- 성격: 마법 학문의 중심지. 마법과학원과 국립상아탑이 이곳에 있다.
- 왕조: 바이서스(Vyserus). 대표 인물 — 바이서스 카멜라 4세(최초의 여성 국왕).
- 역사적 위기:
  - 행커만이 사실은 변신 블랙 드래곤이었음이 칠흑성 전투로 밝혀지며 멸망 직전까지 갔다.
  - 일레인 맥필렌의 승리 덕분에 멸망을 면했다.
  - 이후 엘더스톰과 평화·화해 국면으로 전환.

### 엘더스톰 (Elderstorm)

- 위치: 헤븐즈 킹덤과 대치하는 또 다른 마법 강국.
- 상징 인물: 일레인 맥필렌.
- 군사 특성: 일레인의 마법 회복·사기 진작으로 강화된 군대. "마왕군"이라 불릴 정도의 위용.
- 훈련법: 칠흑숲의 오우거 환각을 이용한 악명 높은 훈련. 셉터·전사 모두에게 적용.
- 비밀 문서: 칠흑성 전투 보고서를 비롯한 군사·마법 기밀이 보존되어 있다.

### 강철제국 (Iron Empire)

- 위치: 북부.
- 성격: 군사 제국. 아라곤 스틸만의 후예 국가.
- 명물: 아라곤산 강철 소드(바스타드 소드). 무겁고 단단하며 명품 대장장이 가문에서 대대로 만든다.
- 상징: "강철제국의 다섯 검" — 황제 직속 호위 전사.
- 현재 위치: 전사형 셉터의 본산. 검기계 마법의 정통.

### 제국 데미안 (Empire Damian)

- 위치: 미정 / 데미안 스펠른의 출신지.
- 성격: 마법 명문 왕가. 스펠 마법의 발상지.
- 학문적 권위: 헤븐즈 킹덤의 학문과 양대 산맥을 이루기도 했다.

### 미드그레이드 (Midgrade)

- 위치: 변경 / 인구 밀집 지역.
- 역사적 사건: 과거 블랙 드래곤 카르마서스(Karmaceus)가 도시를 파괴하던 곳. 일레인이 이를 패퇴시켰다.
- 현재: 다민족·다종족이 혼재하는 변경 도시. 사피엔·엘프·하플링이 섞여 산다.

## 마법 기관

### 국립상아탑 (National Ivory Tower)

- 위치: 헤븐즈 킹덤 추정.
- 역할: 사피엔스 마법 지식의 총집결지. 모든 지식이 이곳에 쌓인다.
- 입학: 최연소 수석입학이 영광으로 여겨진다 (스펠른의 일화).

### 국제마법과학원 (International Arcane Academy)

- 역할: 대륙 차원의 마법 학술 기관.
- 소장: 행커만의 『고대마법이론』 원본.
- 권위: 마법 클래스(써클) 인증을 발급하며 셉터 등급의 기준점이 된다.

### 마법과학원 (헤븐즈 킹덤 소속)

- 슐로브스 마레가 명예교수로 활동했고 마법 클래스 개념을 정립한 곳.

## 지명 — 자연과 특수 지역

### 칠흑숲 (Black Forest / Chil-Heuk Forest)

- 어둠이 짙은 거대 숲. 동화되면 엘프조차 다크엘프가 된다.
- 오우거 환각이 유명하다 — 일레인의 훈련장으로 사용.
- 마계 균열이 자주 나타나는 지역 후보 중 하나.

### 칠흑성 (Black Castle)

- 칠흑숲 근처의 거대한 성. 한때 행커만이 머물던 곳.
- **칠흑성 전투**의 무대. 일레인 vs 행커만의 9써클 격돌이 벌어진 장소.

### 검은평원 (Black Plain)

- 대전장. 엘더리치 대전장이라고도 불린다.
- **린넬 허프 백작의 검은평원 전투**가 벌어진 곳. 적장 리치를 베고 전세를 사피엔스에게 기울게 한 결정적 전장.

### 칼날부리산맥 (Knife-Beak Range)

- 뾰족뾰족한 바위가 병풍처럼 둘러싼 산맥.
- 도적 소굴이 많고 변경 모험가들이 활동한다.
- 도시 사회에서 밀려난 자들이 모이는 지역.

### 붉은산맥 (Red Range)

- 독거미, 독사, 붉은늑대, 가고일, 와이번 등 다양한 위협이 서식하는 위험 지역.
- 살아 돌아온 자의 증언만으로 위험을 기록하므로, 실제 위험은 알려진 것보다 훨씬 클 가능성.

### 엘라의 평원 (Plain of Ela)

- 부드러운 바람과 맑은 하늘의 목가적 평원.
- 표면적 평화의 상징.
- 포링, 멧돼지 등 일견 귀여운 마물이 있으나 방심하면 포링의 위장 속에 흡수된다.

### 잉그리드 사원 (Temple of Ingrid)

- 드래곤을 모시는 사원.
- 99%의 사피엔스는 드래곤을 직접 본 적이 없으므로 드래곤을 오랜 현자·신으로 추앙한다.

## 정치 지형 요약

## 설계 기준

- 모든 도시·국가는 신좌침묵 이후의 산물이다 — 즉, 영웅 신화 위에 세워진 정통성을 가진다.
- 정치적 갈등은 표면이고, 본질은 엔트로픽 농도를 누가 더 빨리 쌓아 올리느냐의 경쟁이다.
- 변경·금기·잔존 마녀 권력은 대현자·앤트로퍼의 비밀이 숨어 있을 만한 공간이다.

## 3. 8왕국 내 부속 도시

### 헤븐즈 영토

- **셀라리아 호수 섬** — 국립상아탑 본탑. 사프란 시신 봉인 *전설*
- **노이엔** (Noien) — 변경 북서. 카엘 출생 마을. 평민 농촌. **본작 1부·6-7 결전지 (사흘 거리에 폐허 마탑)**
- **알마린 백작령** — 왕비 미리에 친정. 알마리스 직계 외척

### 엘더스톰 영토

- **칠흑숲 외곽 훈련장** — 오우거 환각 훈련 표준 코스
- **노 화산 분화구** — 그라니움 외곽. 마법 의식 거점

### 강철제국 영토

- **세이른의 산속 도장** — 강철제국 변경 산속. 6-4 카엘 입문지

### 카르샤 영토

- **변경 신성 수도원** (위치 미정) — 4-4 직후 왕비 엘라리아 + 셀라엔 대주교 망명지
- **에리언 농가 마을** (3-3 사주 학살지) — 세이렌에서 사흘 거리. 마하의 거주지
- **세번 강** — 양국 변경. **본작 4-1 회전 무대**

### 마녀령 영토

- **오르하임 9섬** — 각 섬이 9혈 한 종파 본거 (중앙=흑탑·동=철문의 회당·서=백의 신전 등)
- **원탁 회당** — 오르하임 중앙. 16인 회의장
- **바엘** (Bael) — 마녀령 변경 도시. **본작 6-1 카이엔 폴리모프 직후 첫 식사 도시**

### 제국 데미안 영토

- **미라돈 5대 길드 회당** — 마도공학·직물·향료·무기·금융
- **항만 세 등대** — 동방·서방·남방 항로
- **마도공학원** — 미라돈 중심. 카르덴 스펠른 원장
- **그라스** (Grass) — 미라돈 동북 내륙 마법 장인 도시. **본작 6-2 카엘 의수 시술지 / 6-6 시장 *5초 어긋남***
- **다섯 모루 결사 비밀 공방** — 미라돈 지하 + 그라스 인근 폐광

| 출발 | 도착 | 일수 |
|------|------|------|

| 노이엔 → 셀라리아 | | 14일 |
| 셀라리아 → 세이렌 | | 21일 (3주) |
| 세번 강 → 세이렌 | | 5일 |
| 세이렌 → 카르엔 외곽 연구소 | | 25일 |
| 카르엔 외곽 → 빙벽 협곡 | | 30일 (1개월) |
| 오릴 마을 → 그라스 | | 3일 |
| 그라스 → 세이른 산속 도장 | | 5일 |
| 그라스 → 노이엔 (6-7 결전지) | | 12일 |

### 말 (1일 60km — 도보의 2배)

위 표의 *절반*. 단 *말 + 보급* 필요.

### 마법 (점멸·순간이동)

- 점멸 양피지 1매 = 5~10미터 (전투용)
- 순간이동 스펠 (9써클) = 도시 간 가능. *시전자 + 동반자 1명* 한정. 막대한 마나 소모

### 마법 우편

- 미라돈 *순간이동 우체통* = 도시 내만
- 도시 간 우편 = *말 + 우편망* 으로 5~10일

## 9. 미정·확장 가능 지역

본작 후반부에 등장 가능성:

- **드래곤 로드 본거** (위치 *어디에도 없음* — 떠도는 잠) — 종결부
- **고룡 의회 산봉우리** (북부 빙벽 가장 깊은 곳) — 후반부
- **세계의 알 봉인지** (리펠리아 잔재) — 후반부 카드
- **카르샤 변경 신성 수도원** (왕비·대주교 망명지) — 5~6부 잠재
- **데미안 황실 비밀 회랑** (라이세린-카루린 외척 밀약 회의장) — 5~6부 잠재
- **다섯 모루 결사 본거** (미라돈 지하) — 중반부

각 지역은 본문 진행 시 *그때마다 위치 확정* 권장.
