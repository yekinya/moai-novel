## 1. 대륙 기본 지리

### 큰 단위

|------|------|
| **리투니아 (Lithunia)** | 별 + 본작 대륙 이름 |
| **리펠리아 (Lifelia)** | 태초의 모태 대륙. 거인족 멸종 시 침몰. *지질학적 모태* |
| **무스페름 (Muspherm)** | 지하세계. 크세프 계열 법칙·종족 거주 |

### 대륙 4방위 + 중심

```
                  [북부 — 강철제국 카르나]
                  [한랭 고원 + 빙벽 협곡]
                          │
[서북 — 엘더스톰 그라니움]┼[중앙 — 헤븐즈 셀라리아]
[서북 산악 칼데라]        │[중앙 평원 + 호수]
                          │
[남서 — 데미안 미라돈]    │[동변 — 카르샤 세이렌]
[해안 항만 + 그라스]      │[동변 강변 농지]
                          │
                  [동부 — 마녀령 오르하임]
                  [늪지 + 9섬 도시]
                          │
                  [남부 — 침묵의 사원]
                  [사막 + 변경]
                          │
                  [동대륙 심림 — 칠흑숲]
                  [엘프 부족 산재]
```

## 3. 8왕국 내 부속 도시

### 헤븐즈 영토

- **셀라리아 호수 섬** — 국립상아탑 본탑. 사프란 시신 봉인 *전설*
- **노이엔** (Noien) — 변경 북서. 카엘 출생 마을. 평민 농촌. **본작 1부·6-7 결전지 (사흘 거리에 폐허 마탑)**
- **알마린 백작령** — 왕비 미리에 친정. 알마리스 직계 외척

### 엘더스톰 영토

- **일레인 광장** — 그라니움 중심. 일레인 맥필렌 묘
- **칠흑숲 외곽 훈련장** — 오우거 환각 훈련 표준 코스
- **노 화산 분화구** — 그라니움 외곽. 마법 의식 거점

### 강철제국 영토

- **모루의 광장** — 카르나 황실 대장간 (드워프 명검 가공)
- **다섯 검 회당** — 카르나 황궁 내. 황제 직속 호위 회의장
- **세이른의 산속 도장** — 강철제국 변경 산속. 6-4 카엘 입문지

### 카르샤 영토

- **변경 신성 수도원** (위치 미정) — 4-4 직후 왕비 엘라리아 + 셀라엔 대주교 망명지
- **에리언 농가 마을** (3-3 사주 학살지) — 세이렌에서 사흘 거리. 마하의 거주지
- **세번 강** — 양국 변경. **본작 4-1 회전 무대**

### 마녀령 영토

- **오르하임 9섬** — 각 섬이 9혈 한 종파 본거 (중앙=흑탑·동=철문의 회당·서=백의 신전 등)
- **원탁 회당** — 오르하임 중앙. 16인 회의장
- **바엘** (Bael) — 마녀령 변경 도시. **본작 6-1 카이엔 폴리모프 직후 첫 식사 도시**

### 제국 데미안 영토

- **미라돈 5대 길드 회당** — 마도공학·직물·향료·무기·금융
- **항만 세 등대** — 동방·서방·남방 항로
- **마도공학원** — 미라돈 중심. 카르덴 스펠른 원장
- **그라스** (Grass) — 미라돈 동북 내륙 마법 장인 도시. **본작 6-2 카엘 의수 시술지 / 6-6 시장 *5초 어긋남***
- **다섯 모루 결사 비밀 공방** — 미라돈 지하 + 그라스 인근 폐광

## 7. 본작 핵심 장소 — *카엘의 여정 지도*

```
[CC 980] 노이엔 마을 (출생)
   ↓ (1부)
[CC 996] 폐허 마탑 (1-5 첫 마나 점화 = 6-7 결전지)
   ↓ (사흘 도보)
[CC 998] 셀라리아 (바이서스 진입)
   ↓ (2~3부)
[CC 999] 세번 강 (4-1 첫 회전)
   ↓
[CC 1000 봄] 세이렌 (4-2~4-5 함락 + 옥좌의 방)
   ↓ (5-1 잠적)
[CC 1000 봄] 카르엔 외곽 연구소 (5-2 봉인 문서)
   ↓ (1개월 여정)
[CC 1000 여름] 칠흑숲 → 변경 엘프 마을 → 잉그리드 사원
   ↓
[CC 1000 여름] 빙벽 협곡 (5-4 의식 실패 → 5-5 양팔 상실)
   ↓ (빙하 강 표류)
[CC 1000 여름] 오릴의 변경 마을 (5-6 회복)
   ↓
[CC 1000 가을] 그라스 (6-2 의수 시술)
   ↓
[CC 1000 가을~겨울] 세이른의 산속 도장 (6-4 공허검 입문)
   ↓
[CC 1001 봄] 그라스 시장 (6-6 카이엔과 5초 어긋남)
   ↓
[CC 1001 봄] 폐허 마탑 (6-7 결전 편지 발신)
```

### 동시 진행 — 카이엔의 여정

```
[CC 1000 여름] 빙벽 협곡 (5-4 깨움) → 비늘 회복 수개월
[CC 1000 가을] 바엘 (6-1 폴리모프 후 첫 식사)
[CC 1000 가을] 오르하임 (6-3 동부국 검술대회 우승 + 황녀 사사)
[CC 1001 봄] 그라스 시장 (6-6 어긋남)
[CC 1001 봄] 폐허 마탑 (6-7 출발)
```

## 8. 거리·이동 시간 표준

### 도보 (평민 1일 30km)

| 출발 | 도착 | 일수 |
|------|------|------|
| 노이엔 → 폐허 마탑 | | 3일 |
| 노이엔 → 셀라리아 | | 14일 |
| 셀라리아 → 세이렌 | | 21일 (3주) |
| 세번 강 → 세이렌 | | 5일 |
| 세이렌 → 카르엔 외곽 연구소 | | 25일 |
| 카르엔 외곽 → 빙벽 협곡 | | 30일 (1개월) |
| 오릴 마을 → 그라스 | | 3일 |
| 그라스 → 세이른 산속 도장 | | 5일 |
| 그라스 → 노이엔 (6-7 결전지) | | 12일 |

### 말 (1일 60km — 도보의 2배)

위 표의 *절반*. 단 *말 + 보급* 필요.

### 마법 (점멸·순간이동)

- 점멸 양피지 1매 = 5~10미터 (전투용)
- 순간이동 스펠 (9써클) = 도시 간 가능. *시전자 + 동반자 1명* 한정. 막대한 마나 소모

### 마법 우편

- 미라돈 *순간이동 우체통* = 도시 내만
- 도시 간 우편 = *말 + 우편망* 으로 5~10일

## 9. 미정·확장 가능 지역

본작 후반부에 등장 가능성:

- **드래곤 로드 본거** (위치 *어디에도 없음* — 떠도는 잠) — 종결부
- **고룡 의회 산봉우리** (북부 빙벽 가장 깊은 곳) — 후반부
- **세계의 알 봉인지** (리펠리아 잔재) — 후반부 카드
- **카르샤 변경 신성 수도원** (왕비·대주교 망명지) — 5~6부 잠재
- **데미안 황실 비밀 회랑** (라이세린-카루린 외척 밀약 회의장) — 5~6부 잠재
- **다섯 모루 결사 본거** (미라돈 지하) — 중반부

각 지역은 본문 진행 시 *그때마다 위치 확정* 권장.
